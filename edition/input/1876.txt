Osagai nagusiak hauek dira: <a href="/wiki/Nitrogeno" title="Nitrogeno">nitrogenoa</a> (%&#160;78), <a href="/wiki/Oxigeno" title="Oxigeno">oxigenoa</a> (%&#160;21,12), <a href="/wiki/Argon" title="Argon">argona</a> (%&#160;0,93), <a href="/wiki/Karbono_dioxido" title="Karbono dioxido">karbono dioxidoa</a> (%&#160;0,04) eta <a href="/wiki/Ur" title="Ur">ur</a> lurruna (%&#160;2 inguru). Gainerakoa beste gas batzuen aztarnak dira.</p><p>Atmosferak lur kanpoko <a href="/wiki/Erradiazio_elektromagnetiko" class="mw-redirect" title="Erradiazio elektromagnetiko">erradiazioetatik</a> babesten du, <a href="/wiki/Ultramore" title="Ultramore">izpi ultramoreak</a> xurgatuz (<a href="/wiki/Ozono_geruza" title="Ozono geruza">ozono geruzak</a> egiten du lan hori). Ez du muga zehatzik, gainazaletik aldendu ahala bere <a href="/wiki/Dentsitate_(fisika)" title="Dentsitate (fisika)">dentsitatea</a> jarraian jaisten baita. Askotan, atmosfera eta espazioaren arteko muga bezala Karman muga aipatzen da, 100 <a href="/wiki/Kilometro" title="Kilometro">km</a>-ko garaieran.</p><h2><span class="mw-headline" id="Presioa">Presioa</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Presioa»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera&amp;action=edit&amp;section=2" title="Aldatu atal hau: «Presioa»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><div style="font-style: italic; padding-left: 2em; margin-bottom: 0.5em;">Sakontzeko, irakurri: «<a href="/wiki/Presio_atmosferiko" title="Presio atmosferiko">Presio atmosferiko</a>»</div><p>Presio atmosferikoa lurzoruan atmosferak atmosfera barruan dauden gorputzen gainean egiten duen <a href="/wiki/Presio" title="Presio">presioa</a> da.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup></p><p>Lurreko atmosferak batez besteko 1013.25 <a href="/wiki/Pascal_(unitatea)" title="Pascal (unitatea)">hektopascal, hPa</a> (baztertua <a href="/wiki/Bar" title="Bar">milibar, mb</a>) presioa du <a href="/wiki/Itsas_maila" title="Itsas maila">itsas mailan</a>, 45º <a href="/wiki/Latitude" title="Latitude">latitudean</a>. Hala ere, <a href="/wiki/Nazioarteko_Unitate_Sistema" title="Nazioarteko Unitate Sistema">Nazioarteko Unitate Sistemaren</a> unitatea <a href="/wiki/Metro_koadro" title="Metro koadro">metro karratutako</a> <a href="/wiki/Newton_(unitatea)" title="Newton (unitatea)">newton</a> (N/m²) edo <a href="/wiki/Pascal_(unitatea)" title="Pascal (unitatea)">pascal</a> izanda, itsas mailan batez besteko presioa 101325 <a href="/wiki/N" title="N">N</a>/<a href="/wiki/Metro_koadro" title="Metro koadro">m²</a> edo <a href="/wiki/Urte#SI_aurrizkiak" title="Urte">Pa</a> da. Aire hotza bada, jaisten da, presioa igoz eta <a href="/wiki/Antizikloi" title="Antizikloi">antizikloi</a> termikoak eraginez. Aldiz, beroa bada, igotzen da eta <a href="/wiki/Zikloi" title="Zikloi">zikloi</a> edo <a href="/wiki/Behe_presioko_gune" title="Behe presioko gune">depresio</a> termikoak eragiten ditu. Era berean, <a href="/wiki/Aire" title="Aire">aire</a> hotza eta <a href="/wiki/Aire" title="Aire">aire</a> beroa nahasten ez direnez, bata besteari jaitsarazten du <a href="/wiki/Antizikloi" title="Antizikloi">antizikloi</a> eta <a href="/wiki/Behe_presioko_gune" title="Behe presioko gune">depresio</a> dinamikoak sortuz.</p><h2><span class="mw-headline"
