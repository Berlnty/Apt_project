style="margin-left: 60px;"><a href="https://io.wiktionary.org/wiki/Frontispico" class="extiw" title="wikt:Frontispico">Wikivortaro</a> explikas<br />ca rubriko<br />en altra lingui: <b><a href="https://io.wiktionary.org/wiki/Special:Search/Atmosfero" class="extiw" title="wikt:Special:Search/Atmosfero">Atmosfero</a></b></div></div><p><b>Atmosfero</b> esas la sferatra kuverto de <a href="/wiki/Gaso" title="Gaso">gaso</a>, <a href="/wiki/Vaporo" title="Vaporo">vaporo</a> e partikuli cirkondante <a href="/wiki/Tero" title="Tero">tero</a>, retenata da <a href="/wiki/Gravitado" title="Gravitado">gravitado</a> e formacante granda kompozanto di l'ambiento di la maxim multa terana vivanti, protektante oli de l'efekto di <a href="/wiki/Meteoro" title="Meteoro">meteori</a>, <a href="/wiki/Kosmala_radii" title="Kosmala radii">kosmala radii</a> e nociva <a href="/w/index.php?title=Sunala_radii&amp;action=edit&amp;redlink=1" class="new" title="Sunala radii (pagino ne existas)">sunala radii</a>.</p><p>La kompozeso e fizikala traiti di l'atmosfero varias segun <a href="/w/index.php?title=Alteso&amp;action=edit&amp;redlink=1" class="new" title="Alteso (pagino ne existas)">alteso</a>. Certena traiti uzesas por dividar olu aden plura zoni, di qui la basa ed alta limiti chanjas segun <a href="/wiki/Latitudo" title="Latitudo">latitudo</a>, di-kloko e yar-sezono. Cirkume 75% de la tota amaso e 90% de l'aquo-vaporo di atmosfero kontenesas en <a href="/wiki/Troposfero" title="Troposfero">troposfero</a>, la maxim basa zono. La kontenala aquo-vaporo fluktuas nam aquo vaporeskas de l'oceani, portesas en nubi e pluvas sur la kontinenti.</p><p>En troposfero eventas la vetero (videz <a href="/wiki/Meteorologio" title="Meteorologio">meteorologio</a>), anke maxim multa vivanti trovesas ibe. La <a href="/w/index.php?title=Stratosfero&amp;action=edit&amp;redlink=1" class="new" title="Stratosfero (pagino ne existas)">stratosfero</a> kontenas l' <a href="/w/index.php?title=Ozon-strato&amp;action=edit&amp;redlink=1" class="new" title="Ozon-strato (pagino ne existas)">ozon-strato</a> quo ekfiltras danjeroza <a href="/w/index.php?title=Ultreviolea_radii&amp;action=edit&amp;redlink=1" class="new" title="Ultreviolea radii (pagino ne existas)">ultreviolea radii</a> de <a href="/wiki/Suno" title="Suno">suno</a>.</p><p>Super stratosfero, <a href="/w/index.php?title=Mezosfero&amp;action=edit&amp;redlink=1" class="new" title="Mezosfero (pagino ne existas)">mezosfero</a> fuzesas aden <a href="/w/index.php?title=Ionosfero&amp;action=edit&amp;redlink=1" class="new" title="Ionosfero (pagino ne existas)">ionosfero</a>, strato qua kontenas diversa elektrizita partikuli importanta por la propago di <a href="/wiki/Radio" title="Radio">radio</a>-ondi. Mem plu alte, ionosfero fuzesas aden <a href="/w/index.php?title=Exosfero&amp;action=edit&amp;redlink=1" class="new" title="Exosfero (pagino ne existas)">exosfero</a>, regiono di <a href="/wiki/Helio" title="Helio">helio</a> e <a href="/wiki/Hidrogeno" class="mw-redirect" title="Hidrogeno">hidrogeno</a> gasi, quo fine fuzesas aden l'interplaneta medio.</p><p>L'altra <a href="/wiki/Planeto" title="Planeto">planeti</a> di <a href="/wiki/Sunala_sistemo" title="Sunala sistemo">sunala sistemo</a> (ecepte forsan <a href="/wiki/Plutono" title="Plutono">Plutono</a>), ma nur du ek lia <a href="/wiki/Satelito" title="Satelito">sateliti</a>, omna havas distinta atmosferi, ma nula kontenas tam multa viv-sustenanta <a href="/wiki/Oxo" title="Oxo">oxo</a> kam nia terala atmosfero.</p><!-- NewPP limit reportParsed by mw1258Cached time: 20180417024452Cache expiry: 1900800Dynamic content: falseCPU time usage: 0.012 secondsReal time usage: 0.013 secondsPreprocessor visited node count: 10/1000000Preprocessor generated node count: 0/1500000Post‐expand include size: 487/2097152 bytesTemplate argument size: 0/2097152 bytesHighest expansion depth: 2/40Expensive parser function count: 0/500Unstrip recursion depth: 0/20Unstrip post‐expand size: 0/5000000 bytes--><!--Transclusion expansion time report (%,ms,calls,template)100.00%    1.699      1 Shablono:Wikivortaro100.00%    1.699      1 -total--></div><!-- Saved in parser cache with key iowiki:pcache:idhash:3596-0!canonical and timestamp 20180417024452 and revision id 857190 --><noscript><img src="//io.wikipedia.org/wiki/Special:CentralAutoLogin/start?type=1x1" alt="" title="" width="1" height="1" style="border: none;
