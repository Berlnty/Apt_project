href="#p-search">барај</a>				</div>				<div id="mw-content-text" lang="mk" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><div class="dablink">За атмосферата на Земјата, видете <a href="/wiki/%D0%97%D0%B5%D0%BC%D1%98%D0%B8%D0%BD%D0%B0_%D0%B0%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%B0" title="Земјина атмосфера">Земјина атмосфера</a>. За други значења, видете <a href="/w/index.php?title=%D0%90%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%B0_(%D0%BF%D0%BE%D1%98%D0%B0%D1%81%D0%BD%D1%83%D0%B2%D0%B0%D1%9A%D0%B5)&amp;action=edit&amp;redlink=1" class="new" title="Атмосфера (појаснување) (страницата не постои)">Атмосфера (појаснување)</a>.</div><div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/%D0%9F%D0%BE%D0%B4%D0%B0%D1%82%D0%BE%D1%82%D0%B5%D0%BA%D0%B0:Portrait_of_Jupiter_from_Cassini.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/75/Portrait_of_Jupiter_from_Cassini.jpg/220px-Portrait_of_Jupiter_from_Cassini.jpg" width="220" height="312" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/75/Portrait_of_Jupiter_from_Cassini.jpg/330px-Portrait_of_Jupiter_from_Cassini.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/75/Portrait_of_Jupiter_from_Cassini.jpg/440px-Portrait_of_Jupiter_from_Cassini.jpg 2x" data-file-width="2260" data-file-height="3207" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/%D0%9F%D0%BE%D0%B4%D0%B0%D1%82%D0%BE%D1%82%D0%B5%D0%BA%D0%B0:Portrait_of_Jupiter_from_Cassini.jpg" class="internal" title="Зголеми"></a></div>Дел од активната атмосфера на <a href="/wiki/%D0%88%D1%83%D0%BF%D0%B8%D1%82%D0%B5%D1%80" title="Јупитер">Јупитер</a>, вклучувајќи ја и <a href="/w/index.php?title=%D0%93%D0%BE%D0%BB%D0%B5%D0%BC%D0%B0_%D1%86%D1%80%D0%B2%D0%B5%D0%BD%D0%B0_%D0%B4%D0%B0%D0%BC%D0%BA%D0%B0&amp;action=edit&amp;redlink=1" class="new" title="Голема црвена дамка (страницата не постои)">Големата црвена дамка</a>.</div></div></div><p><b>Атмосфера</b> (од <a href="/wiki/%D0%93%D1%80%D1%87%D0%BA%D0%B8_%D1%98%D0%B0%D0%B7%D0%B8%D0%BA" title="Грчки јазик">грчкото</a> <i>ατμός</i> - <i>атмос</i>, „<a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B5%D0%B0" title="Пареа">пареа</a>“ + <i>σφαίρα</i> - „<a href="/wiki/%D0%A1%D1%84%D0%B5%D1%80%D0%B0" title="Сфера">сфера</a>“) е слој од <a href="/wiki/%D0%93%D0%B0%D1%81" title="Гас">гасови</a> кои можат да опкружат едно материјално тело со одредена <a href="/wiki/%D0%9C%D0%B0%D1%81%D0%B0" title="Маса">маса</a>,<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> зависно од <a href="/wiki/%D0%93%D1%80%D0%B0%D0%B2%D0%B8%D1%82%D0%B0%D1%86%D0%B8%D1%98%D0%B0" title="Гравитација">гравитацијата</a> на телото, кои се задржуваат за подолго време ако гравитацијата е висока, а температурата на атмосферата е ниска. Некои <a href="/wiki/%D0%9F%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%B0" title="Планета">планети</a> се составени од различни видови гасови, но само нивната надворешна обвивка е атмосфера (видете <a href="/wiki/%D0%93%D0%B0%D1%81%D0%BE%D0%B2%D0%B8%D1%82_%D1%9F%D0%B8%D0%BD" title="Гасовит џин">гасовити џинови</a>).</p><p>Поимот <a href="/wiki/%D0%85%D0%B2%D0%B5%D0%B7%D0%B4%D0%B5%D0%BD%D0%B0_%D0%B0%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%B0" title="Ѕвездена атмосфера">ѕвездена атмосфера</a> го опишува надворешниот дел од ѕвездата. Ѕвездите со релативно ниска температура можат да формираат соединенија од молекули во нивната надворешна атмосфера. <a href="/wiki/%D0%97%D0%B5%D0%BC%D1%98%D0%B8%D0%BD%D0%B0%D1%82%D0%B0_%D0%B0%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%B0" class="mw-redirect" title="Земјината атмосфера">Земјината атмосфера</a>, која содржи <a href="/wiki/%D0%9A%D0%B8%D1%81%D0%BB%D0%BE%D1%80%D0%BE%D0%B4" title="Кислород">кислород</a> кој се користи од поголем дел <a href="/wiki/%D0%9E%D1%80%D0%B3%D0%B0%D0%BD%D0%B8%D0%B7%D0%B0%D0%BC" title="Организам">организми</a> за <a href="/wiki/%D0%94%D0%B8%D1%88%D0%B5%D1%9A%D0%B5" title="Дишење">дишење</a> и <a href="/wiki/%D0%88%D0%B0%D0%B3%D0%BB%D0%B5%D1%80%D0%BE%D0%B4_%D0%B4%D0%B8%D0%BE%D0%BA%D1%81%D0%B8%D0%B4" class="mw-redirect" title="Јаглерод диоксид">јаглерод диоксид</a> кој го користат <a href="/wiki/%D0%A0%D0%B0%D1%81%D1%82%D0%B5%D0%BD%D0%B8%D1%98%D0%B0" title="Растенија">растенијата</a>, <a href="/wiki/%D0%90%D0%BB%D0%B3%D0%B8" title="Алги">алгите</a> и <a href="/wiki/%D0%A6%D0%B8%D1%98%D0%B0%D0%BD%D0%BE%D0%B1%D0%B0%D0%BA%D1%82%D0%B5%D1%80%D0%B8%D1%98%D0%B0" class="mw-redirect" title="Цијанобактерија">цијанобактериите</a> за <a href="/wiki/%D0%A4%D0%BE%D1%82%D0%BE%D1%81%D0%B8%D0%BD%D1%82%D0%B5%D0%B7%D0%B0" title="Фотосинтеза">фотосинтеза</a>, исто така ги штити живите организми од штетното влијание на соларните <a href="/wiki/%D0%A3%D0%BB%D1%82%D1%80%D0%B0%D0%B2%D0%B8%D0%BE%D0%BB%D0%B5%D1%82%D0%BE%D0%B2%D0%B0_%D1%81%D0%B2%D0%B5%D1%82%D0%BB%D0%B8%D0%BD%D0%B0" class="mw-redirect" title="Ултравиолетова светлина">ултравиолетови зраци</a>. Нејзиниот моментален состав е продукт на билионите години од биохемичките модификации предизвикани од живите организми. Таа има четири слоја или сфери: Тропосфера: тоа е сфера во која има најмногу кислород и таму се случуваат сите појави и процеси и таму има услови за живот. Тоа е најнискиот слој кој се наоѓа до висина од 12км.
