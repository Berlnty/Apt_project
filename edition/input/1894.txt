Atmosfera lengviau išlieka, kai kūno <a href="/wiki/Gravitacija" title="Gravitacija">gravitacinis laukas</a> stipresnis, o <a href="/wiki/Temperat%C5%ABra" title="Temperatūra">temperatūra</a> žemesnė.</p><p>Kai kurios <a href="/wiki/Planeta" title="Planeta">planetos</a> yra sudarytos daugiausia iš dujų (<a href="/wiki/Planetos_mil%C5%BEin%C4%97s" title="Planetos milžinės">planetos milžinės</a>), tačiau tik išorinis dujų sluoksnis laikomas atmosfera. Žvaigždinė atmosfera&#160;– išorinė <a href="/wiki/%C5%BDvaig%C5%BEd%C4%97" title="Žvaigždė">žvaigždės</a> dalis, į kurią įprastai įeina dalis <a href="/wiki/Fotosfera" title="Fotosfera">fotosferos</a>.</p><p></p><div id="toc" class="toc"><div class="toctitle" lang="lt" dir="ltr" xml:lang="lt"><h2>Turinys</h2></div><ul><li class="toclevel-1 tocsection-1"><a href="#Slėgis"><span class="tocnumber">1</span> <span class="toctext">Slėgis</span></a></li><li class="toclevel-1 tocsection-2"><a href="#Sudėtis"><span class="tocnumber">2</span> <span class="toctext">Sudėtis</span></a></li><li class="toclevel-1 tocsection-3"><a href="#Svarba"><span class="tocnumber">3</span> <span class="toctext">Svarba</span></a></li><li class="toclevel-1 tocsection-4"><a href="#Atmosferos_sandara"><span class="tocnumber">4</span> <span class="toctext">Atmosferos sandara</span></a></li><li class="toclevel-1 tocsection-5"><a href="#Taip_pat_skaitykite"><span class="tocnumber">5</span> <span class="toctext">Taip pat skaitykite</span></a></li></ul></div><p></p><h2><span id="Sl.C4.97gis"></span><span class="mw-headline" id="Slėgis">Slėgis</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Redaguoti skyrelį: Slėgis">redaguoti</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera&amp;action=edit&amp;section=1" title="Redaguoti skyrelį: Slėgis">redaguoti vikitekstą</a><span class="mw-editsection-bracket">]</span></span></h2><p>Atmosferos <a href="/wiki/Sl%C4%97gis" title="Slėgis">slėgis</a>&#160;– tai dujų paviršiui statmena jėga, veikianti ploto vienetą. Slėgį lemia planetos <a href="/wiki/Gravitacija" title="Gravitacija">gravitacinė jėga</a> bei dujų stulpo masė virš tam tikro ploto. Kylant slėgis mažėja, nes mažėja aukščiau esančių dujų masė.</p><p>Vidutinis Žemės atmosferos slėgis <a href="/wiki/J%C5%ABros_lygis" title="Jūros lygis">jūros lygyje</a>&#160;– 101,325 <a href="/wiki/Pa" class="mw-redirect mw-disambig" title="Pa">Pa</a>.</p><h2><span id="Sud.C4.97tis"></span><span class="mw-headline" id="Sudėtis">Sudėtis</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Redaguoti skyrelį: Sudėtis">redaguoti</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera&amp;action=edit&amp;section=2" title="Redaguoti skyrelį: Sudėtis">redaguoti vikitekstą</a><span class="mw-editsection-bracket">]</span></span></h2><div class="thumb tright"><div class="thumbinner" style="width:252px;"><a href="/wiki/Vaizdas:Top_of_Atmosphere.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/be/Top_of_Atmosphere.jpg/250px-Top_of_Atmosphere.jpg" width="250" height="166" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/be/Top_of_Atmosphere.jpg/375px-Top_of_Atmosphere.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/be/Top_of_Atmosphere.jpg/500px-Top_of_Atmosphere.jpg 2x" data-file-width="3027" data-file-height="2010" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Vaizdas:Top_of_Atmosphere.jpg" class="internal" title="Padidinti"></a></div>Žemės atmosferos dujos <a href="/wiki/M%C4%97lyna" title="Mėlyna">mėlynos</a> spalvos spindulius išsklaido labiausiai, todėl žiūrint iš kosmoso Žemę juosia mėlynas ratilas</div></div></div><p>Pradinė atmosferos sudėtis susijusi su temperatūra ir chemine sudėtimi planetos formavimosi zonoje <a href="/wiki/Saul%C4%97s_sistemos_susidarymas" class="mw-redirect" title="Saulės sistemos susidarymas">Saulės sistemos susidarymo</a> metu. Tačiau pradinė sudėtis bėgant laikui kinta.</p><p><a href="/wiki/Marsas_(planeta)" title="Marsas (planeta)">Marso</a> ir <a href="/wiki/Venera_(planeta)" title="Venera (planeta)">Veneros</a> atmosferose vyrauja <a href="/wiki/Anglies_dioksidas" title="Anglies dioksidas">anglies dioksidas</a>, randama <a href="/wiki/Azotas" title="Azotas">azoto</a>, <a href="/wiki/Argonas" title="Argonas">argono</a>, <a href="/wiki/Deguonis" title="Deguonis">deguonies</a> ir šiek tiek kitų dujų. Pirmykštėje Žemės atmosferoje taip pat buvo daug anglies dioksido ir mažai deguonies. Dabartinė <a href="/wiki/%C5%BDem%C4%97" title="Žemė">Žemės</a> atmosferos sudėtis&#160;– biocheminių procesų rezultatas. Vystantis gyvybei ir augant bendrai organizmų biomasei, anglies dioksidas buvo sunaudotas <a href="/wiki/Fotosintez%C4%97" title="Fotosintezė">fotosintetinant</a> organinius junginius, o į atmosferą išmestas deguonis kaip šalutinis produktas. Žemės atmosferą sudaro (pagal tūrį) 78,08&#160;% azoto, 20,95&#160;% deguonies, įvairus kiekis vandens garų (vidutiniškai 0,247&#160;%), 0,93&#160;% argono, 0,038&#160;% anglies dioksido, <a href="/wiki/Helis" title="Helis">helio</a> ir kitų <a href="/wiki/Inertin%C4%97s_dujos" title="Inertinės dujos">inertinių dujų</a>, <a href="/wiki/Vandenilis" title="Vandenilis">vandenilio</a>, lakių teršalų pėdsakai.</p><p>Šaltos ir didelės gravitacijos planetos (<a href="/wiki/Jupiteris_(planeta)" title="Jupiteris (planeta)">Jupiteris</a>, <a href="/wiki/Saturnas" title="Saturnas">Saturnas</a>, <a href="/wiki/Uranas_(planeta)" title="Uranas (planeta)">Uranas</a> ir <a href="/wiki/Nept%C5%ABnas_(planeta)" title="Neptūnas (planeta)">Neptūnas</a>) gali išlaikyti atmosferas iš mažos molekulinės masės dujų&#160;– helio, vandenilio.</p><p>Saturno palydovas <a href="/wiki/Titanas_(palydovas)" title="Titanas (palydovas)">Titanas</a> ir Neptūno palydovas <a href="/wiki/Tritonas_(palydovas)" title="Tritonas (palydovas)">Tritonas</a> turi daugiausia iš azoto sudarytas atmosferas. Panaši ir <a href="/wiki/Plutonas_(nyk%C5%A1tukin%C4%97_planeta)" title="Plutonas (nykštukinė planeta)">Plutono</a> atmosfera, tik ji sušąla Plutonui nutolus nuo Saulės.</p><p>Kiti Saulės sistemos objektai dėl mažos masės ar aukštos temperatūros turi itin plonas, nepusiausvyras atmosferas. Tai <a href="/wiki/M%C4%97nulis" title="Mėnulis">Mėnulis</a> (<a href="/wiki/Natris" title="Natris">natrio</a> garai), <a href="/wiki/Merkurijus_(planeta)" title="Merkurijus (planeta)">Merkurijus</a> (natrio garai), <a href="/wiki/Europa_(palydovas)" title="Europa (palydovas)">Europa (palydovas)</a> (deguonis), <a href="/wiki/Ijo_(palydovas)" title="Ijo (palydovas)">Ijo</a> (siera), <a href="/wiki/Enceladas_(palydovas)" title="Enceladas (palydovas)">Enceladas</a> (vandens garai).</p><h2><span class="mw-headline" id="Svarba">Svarba</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Redaguoti skyrelį: Svarba">redaguoti</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera&amp;action=edit&amp;section=3" title="Redaguoti skyrelį: Svarba">redaguoti vikitekstą</a><span class="mw-editsection-bracket">]</span></span></h2><p>Geologiniu požiūriu, atmosfera formuoja planetos <a href="/wiki/Reljefas" title="Reljefas">reljefą</a>&#160;– <a href="/wiki/V%C4%97jas" title="Vėjas">vėjai</a> perneša dulkes, eroduoja paviršių, supusto sąnašas. Reljefą formuoja ir <a href="/wiki/%C5%A0erk%C5%A1nas" title="Šerkšnas">šerkšnas</a> bei <a href="/wiki/Krituliai" title="Krituliai">krituliai</a>, priklausantys nuo atmosferos sudėties. Atmosfera lemia planetos klimatą. Atmosferos sudėtis yra svarbus faktorius, lemiantis galimybę susiformuoti <a href="/wiki/Gyvyb%C4%97" title="Gyvybė">gyvybei</a>.</p><p>Atmosfera saugo Žemės paviršių nuo <a href="/wiki/Meteoras" title="Meteoras">meteorų</a>&#160;– didžioji jų dalis sudega dėl oro trinties, paviršių paprastai pasiekia tik labai menkos dalelės (gramai, gerokai rečiau&#160;– kilogramai ar tonos meteoritinės medžiagos). Atmosfera apsaugo Žemės paviršių nuo pražūtingų gyvybei <a href="/wiki/Ultravioletiniai_spinduliai" title="Ultravioletiniai spinduliai">ultravioletinių</a> ir <a href="/wiki/Rentgeno_spinduliai" title="Rentgeno spinduliai">rentgeno</a> spindulių. Dėl atmosferos poveikio vyksta vandens apytaka (<a href="/wiki/Hidrologinis_ciklas" title="Hidrologinis ciklas">hidrologinis ciklas</a>). Atmosfera švelnina dienos ir nakties temperatūros skirtumą.</p><h2><span class="mw-headline" id="Atmosferos_sandara">Atmosferos sandara</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Redaguoti skyrelį: Atmosferos sandara">redaguoti</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera&amp;action=edit&amp;section=4" title="Redaguoti skyrelį: Atmosferos sandara">redaguoti vikitekstą</a><span class="mw-editsection-bracket">]</span></span></h2><div class="thumb tright"><div class="thumbinner" style="width:242px;"><a href="/wiki/Vaizdas:Atmosferos_temp_schema.png" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/lt/thumb/b/b2/Atmosferos_temp_schema.png/240px-Atmosferos_temp_schema.png" width="240" height="299" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/lt/thumb/b/b2/Atmosferos_temp_schema.png/360px-Atmosferos_temp_schema.png 1.5x, //upload.wikimedia.org/wikipedia/lt/thumb/b/b2/Atmosferos_temp_schema.png/480px-Atmosferos_temp_schema.png 2x" data-file-width="610" data-file-height="759" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Vaizdas:Atmosferos_temp_schema.png" class="internal" title="Padidinti"></a></div>Žemės atmosferos temperatūrinė schema</div></div></div><p>Atmosferą sudaro penki pagrindiniai sluoksniai.</p><ul><li><a href="/wiki/Troposfera" title="Troposfera">Troposfera</a>&#160;– apatinis tankiausias sluoksnis, sudarantis 90&#160;% visos atmosferos masės. Jos storis virš <a href="/wiki/Pusiaujas" title="Pusiaujas">pusiaujo</a> siekia 16–18&#160;km, o virš <a href="/wiki/A%C5%A1igalis" title="Ašigalis">ašigalių</a>&#160;– tik 8–10&#160;km.</li></ul><p>Troposferoje: susitelkę beveik visi atmosferos vandens garai, formuojasi <a href="/wiki/Debesys" class="mw-redirect" title="Debesys">debesys</a> ir <a href="/wiki/Krituliai" title="Krituliai">krituliai</a>; vyksta vertikalusis ir horizontalusis oro judėjimas; didėjant aukščiui, temperatūra kinta; didėjant aukščiui, <a href="/wiki/Atmosferos_sl%C4%97gis" title="Atmosferos slėgis">atmosferos slėgis</a> ir deguonies kiekis mažėja.</p><ul><li><a href="/wiki/Stratosfera" title="Stratosfera">Stratosfera</a>&#160;– ganėtinai tankus sluoksnis, esantis nuo 10–16 iki 45–55&#160;km aukščio. Oras čia labai sausas, temperatūra aukštėjant kyla (kitaip nei troposferoje). Ją sudaro jonizuoti deguonies ir azoto oksido atomai (dėl to dar vadinama jonosfera). Dujų temperatūra termosferoje išauga iki 1500–2000&#160;°C. Čia atsiranda <a href="/wiki/Poliarin%C4%97_pa%C5%A1vaist%C4%97" title="Poliarinė pašvaistė">poliarinė pašvaistė</a> ir vyksta staigūs magnetinio lauko svyravimai.</li><li><a href="/wiki/Egzosfera" title="Egzosfera">Egzosfera</a>&#160;– sluoksnis, laipsniškai pereinantis į <a href="/wiki/Tarpplanetin%C4%97_erdv%C4%97" title="Tarpplanetinė erdvė">tarpplanetinę erdvę</a>.
