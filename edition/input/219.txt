Story : Introduction,<a href="new.baby.html">The Baby</a>,<a href="MM1.html">Manchester Mark 1</a>,<a href="FM1.html">Ferranti Mark 1</a></font></em><hr><h2 align="center">Introduction to the Mark 1</h2><p>The Small-Scale Experimental Machine, known as SSEM, or<a href="new.baby.html">the "Baby"</a>, was designed and built at the Universityof Manchester, and made its first successful run of a program on<b>June 21st 1948</b>.It was the first machine that had all the components now classically regardedas characteristic of the basic computer. Most importantly it was the firstcomputer that could store not only data but <em>any (short!) user program</em>in electronic memory and process it at electronic speed. (Also, the electronicmemory was a true Random Access Memory (RAM), in contrast with the othercomputers being proposed which were based on a<a href="notes.html#acousticdelay">much less flexible storage mechanism</a>).<p>Although the components of the Baby were small and simple, it so clearlydemonstrated the potential of the Stored-Program Electronic Computer that itwas immediately decided to design and build a more powerful and useable machine,the <a href="MM1.html">Manchester Mark 1</a>. Also by <b>October 1948</b> theGovernment had requested Ferranti Ltd. to build a commercial machine to thisnew design.<p>The Manchester Mark 1 not only expanded the size and power of the basic Babycomponents but added some important innovations, in particular a magnetic drumfor auxiliary storage, the original ancestor of the disc.<p>The evolution from the Baby to the Mark 1 took place in a number of stages, andby <b>April 1949</b> an Intermediary Version was available with most of theincreased power (though manual intervention was required to make transfersbetween drum and RAM). In this form it was used for scientific research in theUniversity in 1949.  The full facilities of the Manchester Mark 1 wereoperational by around <b>October 1949</b>.<p>The <a href="FM1.html">Ferranti Mark 1</a> had the same basic architectureas the Manchester Mark 1, but it was better engineered and included a number of enhancements that made it a significantly faster and more powerfulmachine. It was the world's first commercially available computer, with thefirst machine delivered in <b>February 1951</b>. The second machinesold publicly went to the University of Toronto in 1952, and was used in thedesign of the St. Lawrence Seaway.<p>Ferranti then produced a revised version, the<a href="/web/20081026080604/http://www.computer50.org/kgill/mark1/FM1star.html">Ferranti Mark 1*</a>, which hadessentially the same architecture, but with a reduced but improvedorder code.
