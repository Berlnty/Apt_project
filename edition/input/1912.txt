For jordens atmosfære se <a href="/wiki/Jordens_atmosf%C3%A6re" title="Jordens atmosfære">Jordens atmosfære</a>.</i></dd></dl><p><b>Atmosfære</b> (fra <a href="/wiki/Gammelgresk" title="Gammelgresk">gammelgresk</a>: ἀτμός [atmós], «damp»<sup id="cite_ref-PDL1_1-0" class="reference"><a href="#cite_note-PDL1-1">&#91;1&#93;</a></sup> + σφαῖρα [spʰaíra], «kule»,<sup id="cite_ref-PDL2_2-0" class="reference"><a href="#cite_note-PDL2-2">&#91;2&#93;</a></sup> det vil si ‘dampkule’) er det generelle navnet på et lag med <a href="/wiki/Gass" title="Gass">gass</a> som kan ligge rundt et legeme med stor nok <a href="/wiki/Masse" title="Masse">masse</a>.<sup id="cite_ref-OSC_3-0" class="reference"><a href="#cite_note-OSC-3">&#91;3&#93;</a></sup> Gassene er tiltrukket av <a href="/wiki/Tyngdekraft" title="Tyngdekraft">tyngdekraften</a> til legemet, og holdt fast hvis tyngdekraften er stor nok og  atmosfæretemperaturen er lav. Noen planeter består hovedsakelig av diverse gasser, og har dermed svært dype atmosfærer (se <a href="/wiki/Gasskjempe" title="Gasskjempe">gasskjempe</a>).</p><p><a href="/wiki/Jorda" class="mw-redirect" title="Jorda">Jorda</a>, <a href="/wiki/Venus" title="Venus">Venus</a>, <a href="/wiki/Mars_(planet)" title="Mars (planet)">Mars</a>, <a href="/wiki/134340_Pluto" class="mw-redirect" title="134340 Pluto">Pluto</a> og to av <a href="/wiki/Naturlig_satellitt" title="Naturlig satellitt">månene</a> til de ytre planetene (<a href="/wiki/Titan_(m%C3%A5ne)" title="Titan (måne)">saturnmånen Titan</a> og <a href="/wiki/Triton_(m%C3%A5ne)" title="Triton (måne)">neptunmånen Triton</a>) har atmosfærer. I tillegg er kjempeplanetene i det ytre solsystemet (<a href="/wiki/Jupiter" title="Jupiter">Jupiter</a>, <a href="/wiki/Saturn" title="Saturn">Saturn</a>, <a href="/wiki/Uranus" title="Uranus">Uranus</a> og <a href="/wiki/Neptun_(planet)" title="Neptun (planet)">Neptun</a>) hovedsakelig sammensatt av gasser. Andre legemer i <a href="/wiki/Solsystemet" title="Solsystemet">solsystemet</a> har ekstremt tynne atmosfærer: <a href="/wiki/M%C3%A5nen" title="Månen">Månen</a> (<a href="/wiki/Natrium" title="Natrium">natriumgass</a>), <a href="/wiki/Merkur" title="Merkur">Merkur</a> (<a href="/wiki/Natrium" title="Natrium">natriumgass</a>), <a href="/wiki/Europa_(m%C3%A5ne)" title="Europa (måne)">jupitermånen Europa</a> (<a href="/wiki/Oksygen" title="Oksygen">oksygen</a>) og <a href="/wiki/Io_(m%C3%A5ne)" title="Io (måne)">jupitermånen Io</a> (<a href="/wiki/Svoveldioksid" title="Svoveldioksid">Svoveldioksid</a>).</p><p><a href="/wiki/Jordens_atmosf%C3%A6re" title="Jordens atmosfære">Jordens atmosfære</a> er en særskilt atmosfære fordi den inneholder gasser skapt av <a href="/wiki/Liv" title="Liv">livet</a> på jorda; samtidig gjør den det mulig for liv å eksistere.</p><h2><span class="mw-headline" id="Referanser">Referanser</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosf%C3%A6re&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Rediger avsnitt: Referanser">rediger</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosf%C3%A6re&amp;action=edit&amp;section=1" title="Rediger avsnitt: Referanser">rediger kilde</a><span class="mw-editsection-bracket">]</span></span></h2><div class="mw-references-wrap"><ol class="references"><li id="cite_note-PDL1-1"><b><a href="#cite_ref-PDL1_1-0">^</a></b> <span class="reference-text"><cite class="citation web">Henry George Liddell, Robert Scott. <a rel="nofollow" class="external text" href="http://www.perseus.tufts.edu/hopper/text?doc=Perseus%3Atext%3A1999.04.0057%3Aentry%3Da%29tmo%2Fs">«ἀτμός»</a>. <i><span></span></i>A Greek-English Lexicon<i><span></span></i>.
