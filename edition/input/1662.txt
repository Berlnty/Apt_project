href="/wiki/%E0%AE%95%E0%AE%BE%E0%AE%9F%E0%AF%8D%E0%AE%9A%E0%AE%BF%E0%AE%95%E0%AF%8D_%E0%AE%95%E0%AE%B2%E0%AF%88" title="காட்சிக் கலை">காட்சிக்கலை</a>களுள் ஒன்றாகக் கொள்ளப்படுகிறது. ஆனாலும் காட்சி <a href="/wiki/%E0%AE%B5%E0%AE%BF%E0%AE%B3%E0%AE%AE%E0%AF%8D%E0%AE%AA%E0%AE%B0%E0%AE%AE%E0%AF%8D" title="விளம்பரம்">விளம்பரமும்</a><sup id="cite_ref-advertising_4-0" class="reference"><a href="#cite_note-advertising-4">&#91;4&#93;</a></sup>, <a href="/wiki/%E0%AE%85%E0%AE%B2%E0%AE%99%E0%AF%8D%E0%AE%95%E0%AE%BE%E0%AE%B0%E0%AE%AE%E0%AF%8D" title="அலங்காரம்">அலங்கார</a> வனப்பும் கலைகளின் ஈர்ப்பு மையங்களாகும். <a href="/wiki/%E0%AE%87%E0%AE%9A%E0%AF%88" title="இசை">இசை</a>, <a href="/wiki/%E0%AE%85%E0%AE%B0%E0%AE%99%E0%AF%8D%E0%AE%95%E0%AF%81" title="அரங்கு">அரங்கு</a>, <a href="/wiki/%E0%AE%A4%E0%AE%BF%E0%AE%B0%E0%AF%88%E0%AE%AA%E0%AF%8D%E0%AE%AA%E0%AE%9F%E0%AE%AE%E0%AF%8D" title="திரைப்படம்">திரைப்படம்</a>, <a href="/wiki/%E0%AE%A8%E0%AE%9F%E0%AE%A9%E0%AE%AE%E0%AF%8D" title="நடனம்">நடனம்</a>, <a href="/wiki/%E0%AE%A8%E0%AE%BE%E0%AE%9F%E0%AE%95%E0%AE%AE%E0%AF%8D" title="நாடகம்">நாடகம்</a>, உள்ளிட்ட ஏனைய <a href="/wiki/%E0%AE%85%E0%AE%B0%E0%AE%99%E0%AF%8D%E0%AE%95%E0%AF%87%E0%AE%B1%E0%AF%8D%E0%AE%B1%E0%AE%AE%E0%AF%8D" title="அரங்கேற்றம்">அரங்கேற்றல்</a> கலைகள், <a href="/wiki/%E0%AE%87%E0%AE%B2%E0%AE%95%E0%AF%8D%E0%AE%95%E0%AE%BF%E0%AE%AF%E0%AE%AE%E0%AF%8D" title="இலக்கியம்">இலக்கியம்</a>, <a href="/wiki/%E0%AE%8A%E0%AE%9F%E0%AE%95%E0%AE%B5%E0%AE%BF%E0%AE%AF%E0%AE%B2%E0%AF%8D" title="ஊடகவியல்">ஊடகங்கள்</a>, போன்றவையும் கலையின் அகன்ற வரையறையுள் அடங்கும்.<sup id="cite_ref-OD_2-1" class="reference"><a href="#cite_note-OD-2">&#91;2&#93;</a></sup><sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup></p><p>17ஆம் நூற்றாண்டு வரையிலும் கலை, <a href="/wiki/%E0%AE%A4%E0%AE%BF%E0%AE%B1%E0%AE%AE%E0%AF%88" title="திறமை">திறமை</a>க்கும், <a href="/wiki/%E0%AE%86%E0%AE%B3%E0%AF%81%E0%AE%AE%E0%AF%88" title="ஆளுமை">ஆளுமை</a>க்கும் ஒப்பான அறிவியல் நுட்பத்தின் பகுதியாகக் கொள்ளப்பட்டது. அதன் பின்னர் <a href="/wiki/%E0%AE%85%E0%AE%B4%E0%AE%95%E0%AE%BF%E0%AE%AF%E0%AE%B2%E0%AF%8D" title="அழகியல்">அழகியல்</a> முதன்மை பெற்று <a href="/w/index.php?title=%E0%AE%95%E0%AE%B1%E0%AF%8D%E0%AE%AA%E0%AE%A9%E0%AF%88&amp;action=edit&amp;redlink=1" class="new" title="கற்பனை (இன்னமும் எழுதப்படவில்லை)">கற்பனை வளம்</a>, <a href="/wiki/%E0%AE%A4%E0%AE%BF%E0%AE%B1%E0%AE%A9%E0%AF%8D" title="திறன்">திறன்</a> சார்ந்து பயனுறு கலைகளாகவும், <a href="/wiki/%E0%AE%A8%E0%AF%81%E0%AE%A3%E0%AF%8D%E0%AE%95%E0%AE%B2%E0%AF%88" title="நுண்கலை">நுண்கலை</a>களாகவும் பகுத்தாயப்பட்டன. கலையானது நிலை, நிகழ்வின் நகலாக்கம், <a href="/wiki/%E0%AE%95%E0%AE%A4%E0%AF%88" class="mw-redirect" title="கதை">கதைப்புனைவு,</a> நிகழ்வின் வெளிப்பாடு, உணர்ச்சிகளின் தொடர்பு மற்றும் இதர தரவுகளைக் கொண்டிருக்கலாம். உரோமாயர்களின் கலை வரலாறானது மனிதனின் <a href="/wiki/%E0%AE%9A%E0%AE%AE%E0%AE%AF%E0%AE%AE%E0%AF%8D" title="சமயம்">மதம்</a> class="reference"><a href="#cite_note-6">&#91;6&#93;</a></sup> </p><p>எக்கலையானாலும் அதற்குரிய கலைப் பின்புலம் (கலை பற்றிய அறிவு, அழகியல் பார்வை), செய்திறன், சமூகப் பயன்பாடு என்பன இருக்கும்.<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">&#91;7&#93;</a></sup><sup id="cite_ref-8" class="reference"><a href="#cite_note-8">&#91;8&#93;</a></sup><sup id="cite_ref-9" class="reference"><a href="#cite_note-9">&#91;9&#93;</a></sup> எனினும் இது தான் கலை என்றும், இது கலை அல்ல என்றும் எச்செயற்பாட்டையும் அறுதியிட்டுச் சொல்ல முடியாது. மேலும் கலை சமூகம் சார்ந்ததாகவும் இருக்கின்றது. அதாவது, பொருள் படைத்தவர்கள் கலை, பொதுமக்கள் கலை, அல்லது இன அடிப்படையிலான கலை என பல சமூகத் தாக்கங்களும் கலைக்கு உண்டு. எனவே கலை பல நிலைகளிலும், பல்வேறு தளங்களிலும் ஆயப்படவேண்டிய ஒன்று. class="mw-headline" id="கட்புலக்_கலைகள்">கட்புலக் கலைகள்</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a பரிணாம முறையில் நகலாகவோ, கற்பனையாகவோ காட்சிப்படுத்தப்படும் கலைகள் ஆகும்.</p><h5><span id=".E0.AE.93.E0.AE.B5.E0.AE.BF.E0.AE.AF.E0.AE.AE.E0.AF.8D"></span><span class="mw-headline" id="ஓவியம்">ஓவியம்</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%E0%AE%95%E0%AE%B2%E0%AF%88&amp;action=edit&amp;section=4" title="ஓவியம் பகுதியைத் தொகு">தொகு</a><span class="mw-editsection-bracket">]</span></span></h5><div role="note" class="hatnote relarticle mainarticle">முதன்மை கட்டுரை: <a href="/wiki/%E0%AE%95%E0%AE%A3%E0%AF%8D%E0%AE%A3%E0%AE%BE%E0%AE%9F%E0%AE%BF" title="கண்ணாடி">கண்ணாடி</a>, <a href="/wiki/%E0%AE%A4%E0%AF%81%E0%AE%A3%E0%AE%BF" title="துணி">துணி</a>, <a title="வண்ணப்பூச்சு">வண்ணப்பூச்சு</a>களைப் பயன்படுத்தி வரையப்படும் அழகியல் சார்ந்த செயற்பாடு ஓவியக்கலை ஆகும். வரைபவரின் கருத்தியல், நோக்கம் போன்றவற்றை வெளிப்படுத்தும் வண்ணங்கள், அச்சுகளால் வெளிப்படுத்தும் கலை ஆகும். உடல் ஓவியம், கேலிச் சித்திரம், காபி ஓவியம், துணி ஓவியம், கண்ணாடி ஓவியம், குகை ஓவியம், போன்றவை ஓவியக்கலையின் பல்வேறு வடிவங்களாகும்.</p><h5><span id=".E0.AE.9A.E0.AE.BF.E0.AE.B1.E0.AF.8D.E0.AE.AA.E0.AE.AE.E0.AF.8D"></span><span class="mw-headline" id="சிற்பம்">சிற்பம்</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%E0%AE%95%E0%AE%B2%E0%AF%88&amp;action=edit&amp;section=5" title="சிற்பம் பகுதியைத் தொகு">தொகு</a><span class="mw-editsection-bracket">]</span></span></h5><div role="note" class="hatnote relarticle mainarticle">முதன்மை கட்டுரை: <a class="new" title="கல் (இன்னமும் எழுதப்படவில்லை)">கல்</a>, <a href="/wiki/%E0%AE%95%E0%AE%A3%E0%AF%8D%E0%AE%A3%E0%AE%BE%E0%AE%9F%E0%AE%BF" title="கண்ணாடி">கண்ணாடி</a>, <a href="/wiki/%E0%AE%89%E0%AE%B2%E0%AF%8B%E0%AE%95%E0%AE%AE%E0%AF%8D" title="உலோகம்">உலோகம்</a> முதலியவற்றால் செதுக்கியோ, வார்த்தோ செய்யப்படும் முப்பரிமாண கலைப்பொருள் <a href="/wiki/%E0%AE%9A%E0%AE%BF%E0%AE%B1%E0%AF%8D%E0%AE%AA%E0%AE%AE%E0%AF%8D" title="சிற்பம்">சிற்பம்</a> ஆகும். தனிச் சிற்பம், புடைப்புச் சிற்பம், செதுக்குச் சிற்பம், இயங்கியல் சிற்பம், அடுக்கற்கலைச் சிற்பம் போன்றவை சிற்பங்களின் வகைகளாகும். உரோமானிய, கிரேக்க, எகிப்திய, இந்தியக் கலைச்சிற்பங்கள் உலகப்பிரசித்தி பெற்றன. பெரும்பாலான சிற்பங்கள் கடவுள், மதம், இனத்தலைவர்கள், மற்றும் பண்பாடு சார்ந்த கதைமாந்தர்களின் உருவங்களை உருவகிக்கின்றன.</p><h5><span id=".E0.AE.92.E0.AE.B3.E0.AE.BF.E0.AE.AA.E0.AF.8D.E0.AE.AA.E0.AE.9F.E0.AE.AE.E0.AF.8D"></span><span class="mw-headline" id="ஒளிப்படம்">ஒளிப்படம்</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%E0%AE%95%E0%AE%B2%E0%AF%88&amp;action=edit&amp;section=6" href="/wiki/%E0%AE%92%E0%AE%B3%E0%AE%BF%E0%AE%AA%E0%AF%8D%E0%AE%AA%E0%AE%9F%E0%AE%AE%E0%AF%8D" title="ஒளிப்படம்">ஒளிப்படம்</a></div><p>ஒளிப்படம், நிழற்படம் (அ) புகைப்படமானது, ஒளி எதிரொளிப்பின் மூலம் பொருட்கள் தாம் வெளிவிடுகின்ற (அ) அதன் மீது தெளித்து வெளிவருகின்ற ஒளியினால் ஓர் ஒளியுணர் மேற்பரப்பில் அதன் பிம்பத்தைப் பதிவிட்ட படத்தைக் குறிக்கும். அழகுற எடுக்கப்படும் அச்சுப்பிரதி (அ) ஒளிப்படம் பல்வேறு கலை வேலைப்பாடுகளுடன் நிகழ்வை வரலாற்றில் பதிவிக்கின்றன.இது பொதுவாக ஒளிப்பட முறை, எண்மிய ஒளிப்பட முறை என இருவகைப்படும். இவற்றின் நிறம் மற்றும் தரத்தின் அடிப்படையிலும் ஒளிப்படங்கள் வேறுபடும்.</p><h4><span id=".E0.AE.85.E0.AE.B0.E0.AE.99.E0.AF.8D.E0.AE.95.E0.AE.BE.E0.AE.9F.E0.AE.B2.E0.AF.8D_.E0.AE.95.E0.AE.B2.E0.AF.88.E0.AE.95.E0.AE.B3.E0.AF.8D"></span><span class="mw-headline" href="/w/index.php?title=%E0%AE%95%E0%AE%B2%E0%AF%88&amp;action=edit&amp;section=7" title="அரங்காடல் கலைகள் பகுதியைத் தொகு">தொகு</a><span class="mw-editsection-bracket">]</span></span></h4><p>அரங்குகளின் மூலம் அரங்கேற்றப்படும் கலைகள் அரங்காடல் கலைகள் ஆகும். கலைகளின் சிறப்புமையைக் கொண்டு <a href="/wiki/%E0%AE%85%E0%AE%B0%E0%AE%99%E0%AF%8D%E0%AE%95%E0%AF%81" title="அரங்கு">அரங்குகள்</a> மாறுபடும்.</p><h5><span id=".E0.AE.87.E0.AE.9A.E0.AF.88"></span><span class="mw-headline" id="இசை">இசை</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%E0%AE%95%E0%AE%B2%E0%AF%88&amp;action=edit&amp;section=8" title="இசை பகுதியைத் தொகு">தொகு</a><span class="mw-editsection-bracket">]</span></span></h5><div role="note" class="hatnote relarticle mainarticle">முதன்மை கட்டுரை: <a href="/wiki/%E0%AE%87%E0%AE%9A%E0%AF%88" title="இசை">இசை</a></div><p>ஒழுங்கு செய்யப்பட்ட கட்டுப்படுத்தப்பட்ட ஒலிகளால் ஒருங்கிணைக்கப்படும் கூறு இசை ஆகும். ஒலி அளவு, இசைக் கருவிகளின் ஒலிப்பினம், அதிர்வுகள் போன்றவை இசையை மென்மையாக்கும் காரணிகளாகும். மேற்கத்திய, இந்திய, சீன இசைகள் தங்களின் பண்பாடடுடன் ஒன்றியமைந்தாகும்.</p><h5><span id=".E0.AE.A8.E0.AE.9F.E0.AE.A9.E0.AE.AE.E0.AF.8D"></span><span class="mw-headline" பகுதியைத் தொகு">தொகு</a><span class="mw-editsection-bracket">]</span></span></h5><div role="note" class="hatnote relarticle mainarticle">முதன்மை கட்டுரை: <a href="/wiki/%E0%AE%A8%E0%AE%9F%E0%AE%A9%E0%AE%AE%E0%AF%8D" title="நடனம்">நடனம்</a></div><p><a href="/wiki/%E0%AE%A4%E0%AE%BE%E0%AE%B3%E0%AE%AE%E0%AF%8D_(%E0%AE%87%E0%AE%9A%E0%AF%88)" title="தாளம் (இசை)">தாளத்துக்கும்</a>, இசைக்கும் ஒத்தசைவாக உடலை நகர்த்தும் கலை வடிவம் நடனம். சமூகம், பண்பாடு, சமயம் சார்ந்தோ, சார்பற்ற மகிழ்ச்சிக்காகவோ நிகழ்த்திக்காட்டும் நிகழ்கலையாகவோ கொள்ளப்படும். சில விலங்கினங்களில் இனப்பெருக்கத்திற்காக துணையைக் கவரும் விதமாக அவைகளால் நிகழ்த்தப்படுகிறது.</p><p>இந்திய(<a href="/wiki/%E0%AE%95%E0%AE%A4%E0%AE%95%E0%AE%B3%E0%AE%BF" title="கதகளி">கதகளி</a>, <a புனையப்படுவது கதை. திரைப்படங்களில் அதன் காட்சிப்படுத்தலுக்கேற்ப திரைக்கதைகளாக வடிவம் பெறுகின்றன.</p><p>கதைகள் பெரும்பாலும் இயல்பான நிகழ்வை மிகைப்படுத்தி கற்பனைத்திறனை மிகுவித்து உரைப்பதாகும். கதை,
