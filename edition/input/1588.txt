title="Gedhèkaké"></a></div>Kagunan</div></div></div><p><b>Kagunan</b> iku pérangan <a href="/wiki/Kabudayan" class="mw-redirect" title="Kabudayan">kabudayan</a> kang ana gayutané karo wujud rasa apik utawa éndah. Rasa apik iku anané ing <a href="/w/index.php?title=Batin&amp;action=edit&amp;redlink=1" class="new" title="Batin (kaca ora ana)">batin</a>, ing pikiran <a href="/wiki/Manungsa" title="Manungsa">manungsa</a>. Kagunan iku fungsiné kanggo nyaluraké rasa éndah ing pikiran, tegesé, kagunan iku prosès nyiptaaké wujud-wujud kang ndadèkaké <a href="/wiki/Ati" title="Ati">ati</a> ngrasa seneng, puas, bungah kanggo memapki sisih batin/spiritual kauripan manungsa.</p><p>Sapérangan <a href="/wiki/Wong" class="mw-disambig" title="Wong">wong</a> duwé anggepan yèn Kagunan iku mung pelengkap, Sajatiné kagunan iku wis ana kawit manungsa lair, dadi ing saben individu, jiwa kagunan iku tuwuh bareng karo èlmu sawenèh individu mau.</p><h2><span class="mw-headline" id="Fungsi">Fungsi</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Kagunan&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Besut pérangan: Fungsi">besut</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Kagunan&amp;action=edit&amp;section=1" title="Besut pérangan: Fungsi">besut sumber</a><span class="mw-editsection-bracket">]</span></span></h2><p>Fungsi kagunan iku kanggo ngalusaké perasaan, pikiran uga watek/kepribadian, lewih lanjut kanggo ngalusaké tingkah polah utawa tingkah laku manungsa. Kasil pribadi kang wis dibekeli rasa alus iku mengko dadi duwé sipat <a href="/w/index.php?title=Santun&amp;action=edit&amp;redlink=1" class="new" title="Santun (kaca ora ana)">santun</a>, <a href="/w/index.php?title=Toleran&amp;action=edit&amp;redlink=1" class="new" title="Toleran (kaca ora ana)">toleran</a> lan <a href="/w/index.php?title=Wicaksana&amp;action=edit&amp;redlink=1" class="new" title="Wicaksana (kaca ora ana)">wicaksana</a>. Mulane kawit cilik manungsa iku wis moali dibekeli momotan kagunan upamané liwat <a href="/wiki/Tembang_bocah" title="Tembang bocah">tembang bocah</a> saking <a href="/w/index.php?title=Biyung&amp;action=edit&amp;redlink=1" class="new" title="Biyung (kaca ora ana)">biyunge</a>, <a href="/wiki/Tembang" title="Tembang">tembang</a> <a href="/wiki/Tak_Lelo_Lelo_Lelo_Ledung" title="Tak Lelo Lelo Lelo Ledung">Tak Lelo Lelo Lelo Ledung</a> contoné, utawa saking <a href="/wiki/Dong%C3%A8ng" title="Dongèng">dongèng</a> kang dicritaaken <a href="/wiki/Rama" title="Rama">ramane</a>. Fungsi liyané kanggo numbuhke sipat <a href="/w/index.php?title=Kreatif&amp;action=edit&amp;redlink=1" class="new" title="Kreatif (kaca ora ana)">kreatif</a> mligi ing bocah-bocah.</p><p><a href="/wiki/Dolanan" title="Dolanan">Dolanan</a> iku donya kagunan <a href="/wiki/Bocah" title="Bocah">bocah</a> kang terus dilakoni nganti bocah iku melebu <a href="/wiki/Sekolah" class="mw-disambig" title="Sekolah">sekolah</a>. Pelajaran <a href="/w/index.php?title=Nggambar&amp;action=edit&amp;redlink=1" class="new" title="Nggambar (kaca ora ana)">nggambar</a> ing sekolah iku cara kanggo nuwuhaké rasa kagunan ing jiwa bocah.</p><h2><span class="mw-headline" id="Werna-Werna_Kagunan">Werna-Werna Kagunan</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Kagunan&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Besut pérangan: Werna-Werna Kagunan">besut</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Kagunan&amp;action=edit&amp;section=2" title="Besut pérangan: Werna-Werna Kagunan">besut sumber</a><span class="mw-editsection-bracket">]</span></span></h2><ul><li><a href="/wiki/Sastra" title="Sastra">Sastra</a>, ya iku <a href="/wiki/Tembung" title="Tembung">tembung</a> silihan saka Basa <a href="/wiki/Sansekerta" class="mw-redirect" title="Sansekerta">Sansekerta</a>; "<i>śāstra</i>", kang tegesé "tulisan" utawa "buku", saka tembung dhasar "<i>śās-</i>" kang tegesé "weruh". Ing <a href="/wiki/Basa_Jawa" title="Basa Jawa">basa Jawa</a>, tembung iki racaké dianggo ngarani "kasusastran" utawa sajinising tulisan kang duwé arti utawa kaendahan tartamtu.</li><li><a href="/w/index.php?title=Kagunan_rupa&amp;action=edit&amp;redlink=1" class="new" title="Kagunan rupa (kaca ora ana)">Kagunan rupa</a>,ya iku cabang kagunan kang wujud <a href="/w/index.php?title=Karya_kagunan&amp;action=edit&amp;redlink=1" class="new" title="Karya kagunan (kaca ora ana)">karya kagunanne</a> bisa dipirsani/disawang lan bisa diraba. Kagunan rupa iku ngolah konsèp <a href="/wiki/Garis" title="Garis">garis</a>, <a href="/w/index.php?title=Babagan&amp;action=edit&amp;redlink=1" class="new" title="Babagan (kaca ora ana)">babagan</a>, wujud, <a href="/wiki/Volume" title="Volume">volume</a>, <a href="/wiki/Kelir" title="Kelir">kelir</a>, <a href="/wiki/Tekstur" class="mw-redirect" title="Tekstur">tekstur</a> lan pencahayaan kang kabèh kudu manut aturan-aturan <a href="/wiki/Estetika" class="mw-redirect" title="Estetika">estetika</a>.</li><li><a href="/w/index.php?title=Kagunan_pagelaran&amp;action=edit&amp;redlink=1" class="new" title="Kagunan pagelaran (kaca ora ana)">Kagunan pagelaran</a>, ya iku jinising karya kagunan kang merlokaké <a href="/w/index.php?title=Panggung&amp;action=edit&amp;redlink=1" class="new" title="Panggung (kaca ora ana)">panggung</a> utawa <a href="/wiki/Pelataran" class="mw-redirect" title="Pelataran">pelataran</a> kanggo nampilké kasil <a href="/w/index.php?title=Karya_kagunan&amp;action=edit&amp;redlink=1" class="new" title="Karya kagunan (kaca ora ana)">karya kagunanne</a>.</li><li><a href="/wiki/Dolanan_Bocah" title="Dolanan Bocah">Dolanan Bocah</a> ya iku arupa kagunan dolanan bocah-bocah cilik. Ing jaman saiki, dolanan mau suwéning suwé wis ora dikenal manèh, amarga kadhesek karo dolanan kang luwih modhèren.</li></ul><h2><span class="mw-headline" id="Kagunan_miturut_Wewengkon">Kagunan miturut Wewengkon</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Kagunan&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Besut pérangan: Kagunan miturut Wewengkon">besut</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Kagunan&amp;action=edit&amp;section=3" title="Besut pérangan: Kagunan miturut Wewengkon">besut sumber</a><span class="mw-editsection-bracket">]</span></span></h2><p>Wujud kagunan iku lengkèt karo enggon utawa papan asal jinisé kagunan iku.
