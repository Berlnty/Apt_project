class="mw-indicators mw-body-content"></div><h1 id="firstHeading" class="firstHeading" lang="sl">Atmosfera nebesnega telesa</h1>			<div id="bodyContent" class="mw-body-content">				<div id="siteSub" class="noprint">Iz Wikipedije, proste enciklopedije</div>				<div id="contentSub"></div>								<div id="jump-to-nav" class="mw-jump">					Skoči na:					<a href="#mw-head">navigacija</a>, 					<a href="#p-search">iskanje</a>				</div>				<div id="mw-content-text" lang="sl" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><div class="dablink">Za druge pomene glej <a href="/wiki/Atmosfera" class="mw-disambig" title="Atmosfera">Atmosfera</a>.</div><div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Slika:Mars_atmosphere.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/7d/Mars_atmosphere.jpg/220px-Mars_atmosphere.jpg" width="220" height="246" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/7d/Mars_atmosphere.jpg/330px-Mars_atmosphere.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/7d/Mars_atmosphere.jpg/440px-Mars_atmosphere.jpg 2x" data-file-width="1205" data-file-height="1349" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Slika:Mars_atmosphere.jpg" class="internal" title="Povečaj"></a></div>Marsova redka atmosfera</div></div></div><p><b>Atmosfera</b> (po sodobni <a href="/wiki/Latin%C5%A1%C4%8Dina" title="Latinščina">latinski</a> skovanki <i>atmosphaera</i> iz 17. stoletja, <a href="/wiki/Gr%C5%A1%C4%8Dina" title="Grščina">grško</a> ἀτμός [<i>atmos</i>] –&#160;»para«&#160;in σφαῖρα [<i>sphaira</i>] –&#160;»krogla«)<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> je plast <a href="/wiki/Plin" title="Plin">plinov</a>, ki obdaja <a href="/wiki/Planet" title="Planet">planet</a> ali drugo <a href="/wiki/Nebesno_telo" title="Nebesno telo">nebesno telo</a> z dovolj veliko <a href="/wiki/Masa" title="Masa">maso</a>, da jo zadrži njegova <a href="/wiki/Gravitacija" class="mw-redirect" title="Gravitacija">gravitacija</a>. Drug pomemben dejavnik je <a href="/wiki/Temperatura" title="Temperatura">temperatura</a>, ki vpliva na hitrost gibanja molekul plinov. Nebesno telo lahko zadrži atmosfero skozi <a href="/wiki/Geolo%C5%A1ka_%C4%8Dasovna_lestvica" title="Geološka časovna lestvica">geološka obdobja</a>, če je povprečna hitrost molekul manjša od šestine <a href="/wiki/Ube%C5%BEna_hitrost" title="Ubežna hitrost">ubežne hitrosti</a>.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup></p><p><a href="/wiki/Zemlja" title="Zemlja">Zemljina</a> atmosfera se imenuje <a href="/wiki/Ozra%C4%8Dje" title="Ozračje">ozračje</a>, sestavljena je iz 78&#160;% <a href="/wiki/Du%C5%A1ik" title="Dušik">dušika</a>, 21&#160;% <a href="/wiki/Kisik" title="Kisik">kisika</a>, približno 0,01&#160;% <a href="/wiki/Voda" title="Voda">vodne</a> <a href="/wiki/Para" title="Para">pare</a> in drugih plinov v sledeh, predvsem <a href="/wiki/Argon" title="Argon">argona</a> in <a href="/wiki/Ogljikov_dioksid" title="Ogljikov dioksid">ogljikovega dioksida</a>. Tej zmesi pravimo <a href="/wiki/Zrak" title="Zrak">zrak</a>. Plini, predvsem para in ogljikov dioksid, ujamejo del <a href="/wiki/Infrarde%C4%8De_valovanje" title="Infrardeče valovanje">infrardečega valovanja</a> z Zemlje in ga oddajo nazaj proti površju (t. i. <a href="/wiki/U%C4%8Dinek_tople_grede" title="Učinek tople grede">učinek tople grede</a>), zato je temperatura ozračja višja kot bi se jo pričakovalo samo glede na ravnotežje med prihajajočo sončno energijo ter oddano toploto. Poleg tega <a href="/wiki/Ozon" title="Ozon">ozon</a> v vrhnjih plasteh ozračja odbija visokoenergijsko <a href="/wiki/Ultravijoli%C4%8Dno_valovanje" title="Ultravijolično valovanje">ultravijolično valovanje</a>. Sestava ozračja je edistvena med vsemi poznanimi planeti, zaradi dolgotrajnega delovanja živih <a href="/wiki/Organizem" title="Organizem">organizmov</a>, ki sooblikujejo ozračje s procesi <a href="/wiki/Fotosinteza" title="Fotosinteza">fotosinteze</a> in <a href="/wiki/Celi%C4%8Dno_dihanje" title="Celično dihanje">celičnega dihanja</a>.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup><sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup></p><p>Podobna sestava atmosfere na <a href="/wiki/Zunajoson%C4%8Dni_planet" title="Zunajosončni planet">eksoplanetu</a> bi nakazovala, da <a href="/wiki/Naseljivi_planet" title="Naseljivi planet">gosti</a> <a href="/wiki/Zunajzemeljsko_%C5%BEivljenje" class="mw-redirect" title="Zunajzemeljsko življenje">življenje</a>, podobno temu na Zemlji, zato se astronomi osredotočajo na to značilnost pri iskanju ustreznih planetov.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup></p><p>Drugi planeti <a href="/wiki/Oson%C4%8Dje" title="Osončje">Osončja</a> imajo bistveno različne atmosfere, odvisno od njihove mase in oddaljenosti od <a href="/wiki/Sonce" title="Sonce">Sonca</a>. V <a href="/wiki/Venera" title="Venera">Venerini</a> in <a href="/wiki/Mars" title="Mars">Marsovi</a> atmosferi prevladuje ogljikov dioksid, vendar je Venerina atmosfera gosta, z oblaki iz <a href="/wiki/%C5%BDveplova_kislina" title="Žveplova kislina">žveplove kisline</a>, Marsova pa izredno redka. <a href="/wiki/Plinski_velikan" class="mw-redirect" title="Plinski velikan">Plinski velikani</a> <a href="/wiki/Jupiter" title="Jupiter">Jupiter</a>, <a href="/wiki/Saturn" title="Saturn">Saturn</a>, <a href="/wiki/Uran_(planet)" title="Uran (planet)">Uran</a> in <a href="/wiki/Neptun" title="Neptun">Neptun</a> imajo zelo goste atmosfere iz <a href="/wiki/Vodik" title="Vodik">vodika</a>, <a href="/wiki/Helij" title="Helij">helija</a>, <a href="/wiki/Metan" title="Metan">metana</a> in <a href="/wiki/Amonijak" title="Amonijak">amonijaka</a>, poleg njih pa ima omembe vredno atmosfero še Saturnov satelit <a href="/wiki/Titan_(luna)" title="Titan (luna)">Titan</a>, prav tako iz metana in amonijaka.<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup></p><h2><span class="mw-headline" id="Sklici_in_opombe">Sklici in opombe</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera_nebesnega_telesa&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Spremeni razdelek: Sklici in opombe">uredi</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera_nebesnega_telesa&amp;action=edit&amp;section=1" title="Spremeni razdelek: Sklici in opombe">uredi kodo</a><span class="mw-editsection-bracket">]</span></span></h2><div class="reflist references-column-count references-column-count-2" style="-moz-column-count: 2;
