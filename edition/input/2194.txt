<i>Homo sapiens</i> − <i>umni čovjek</i>) je <a href="/wiki/%C5%BDivo_bi%C4%87e" class="mw-redirect" title="Živo biće">živo biće</a> koje ima visoko razvijen <a href="/wiki/Mozak" Današnji čovjek razvio se prije oko 200.000 godina na prostoru <a href="/wiki/Afrika" title="Afrika">Afrike</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> Ljudska populacija je u 2011. godini premašila brojku od 7 milijardi stanovnika.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup> Čovjek je <a href="/wiki/Misao" title="Misao">misaono</a> biće (može zaključivati, crtati, razmišljati), društveno biće (živi u <a href="/wiki/Dru%C5%A1tvo" title="Društvo">društvu</a>), duhovno biće (razlikuje <a href="/wiki/Dobro" title="Dobro">dobro</a> od <a href="/wiki/Zlo" title="Zlo">zla</a>) i prirodno biće (dio je žive <a href="/wiki/Priroda" title="Priroda">prirode</a>).</p><p></p><div id="toc" class="toc"><div VE</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%C4%8Covjek&amp;action=edit&amp;section=2" title="Uredi odlomak: Prenatalni razvoj">uredi</a><span data-file-width="128" data-file-height="128" /></a><i>Podrobniji članak o temi: <a href="/wiki/Prenatalni_razvoj_%C4%8Dovjeka" title="Prenatalni razvoj čovjeka">Prenatalni razvoj čovjeka</a></i><br /></p><p>Da bi nastalo novo ljudsko biće potreban je <a href="/wiki/Spolni_odnos" class="mw-redirect" title="Spolni odnos">spolni odnos</a> između muškarca i žene, a u novije vrijeme moguć je nastanak i putem <a href="/wiki/Umjetna_oplodnja" title="Umjetna oplodnja">umjetne oplodnje</a>. Do <a href="/wiki/Za%C4%8De%C4%87e" class="mw-redirect" title="Začeće">začeća</a> dolazi spajanjem <a href="/wiki/Spermij" title="Spermij">spermija</a> i <a href="/wiki/Jajna_stanica" title="Jajna stanica">jajne stanice</a> u tijelu <a href="/wiki/Majka" title="Majka">majke</a>. Nastaje oplođeno jajašce koje ima 46 <a href="/wiki/Kromosom" class="mw-redirect" title="Kromosom">kromosoma</a> (23 od oca i 23 od majke). Prva ljudska stanica sadrži složen genetski nacrt svake pojedinosti poput <a href="/wiki/Spol" title="Spol">spola</a>, visine, boje kože itd. Na <a href="/wiki/DNK" class="mw-redirect" title="DNK">DNK</a>-a zapisane su sve potrebne <a href="/wiki/Geni" class="mw-redirect" title="Geni">genetske</a> informacije za daljni razvoj. Njih ima, toliko koliko bi stalo podataka u pet setova <a href="/wiki/Encyclop%C3%A6dia_Britannica" title="Encyclopædia Britannica">Enciklopædije Britannice</a>. Jako mnogo podataka zapisano je u jako minijaturnom obliku. Kada bi se na jedno mjesto sakupili svi genetski podaci s DNK-a 5 milijardi ljudi, bili bi veličine samo dvije tablete.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup> Postepenim razvojem od prve stanice nastaje <a href="/wiki/Embrij" class="mw-redirect" title="Embrij">embrij</a>, a nakon tri mjeseca prerasta u ljudski plod ili <a href="/wiki/Fetus" class="mw-redirect" title="Fetus">fetus</a>.</p><h3><span id="Od_ro.C4.91enja_do_starosti"></span><span class="mw-headline" id="Od_rođenja_do_starosti">Od rođenja do starosti</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%C4%8Covjek&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Uredi odlomak: Od rođenja do starosti">uredi VE</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%C4%8Covjek&amp;action=edit&amp;section=3" title="Uredi odlomak: Od rođenja do starosti">uredi</a><span class="mw-editsection-bracket">]</span></span></h3><p><a href="/wiki/%C5%BDena" title="Žena">Žena</a> rađa <a href="/wiki/Dijete" class="mw-redirect" title="Dijete">dijete</a> nakon obično devet mjeseci <a href="/wiki/Trudno%C4%87a" title="Trudnoća">trudnoće</a>, no moguće je i prijevremeno rađanje u sedmom ili osmom mjesecu, međutim to je razdoblje rizično za djetetov život jer mu nisu još potpuno razvijeni respiratorni organi. Djetetova pluća se razvijaju u 8. mjesecu trudnoće, pa ako se dijete tad rodi, obično dođe do komplikacija, dok ako se rodi u 7. mjesecu, dijete ide u inkubator i pluća se razvijaju bez šoka i prekida.</p><p>Zbog spontanog ili namjernog <a href="/wiki/Poba%C4%8Daj" title="Pobačaj">pobačaja</a>, trudnoća se može prekinuti. <a href="/wiki/Ra%C4%91anje" class="mw-redirect" title="Rađanje">Rađanje</a> je proces ovisan o veličini djeteta i njegovoj sposobnosti prilagođavanja na porodni kanal za vrijeme spuštanja te o snazi i pravilnosti trudova te o otporu <a href="/wiki/Zdjelica" title="Zdjelica">zdjelice</a> i mekih tkiva dna zdjelice. <a href="/wiki/Novoro%C4%91en%C4%8De" title="Novorođenče">Novorođenče</a> je obično teško 3-4 <a href="/wiki/Kilogram" title="Kilogram">kilograma</a> i visoko 50-60 <a href="/wiki/Centimetar" class="mw-redirect" title="Centimetar">centimetara</a>.</p><p>Nakon prve godine života, novorođenče prelazi u razdoblje djetinjstva. Rano djetinjstvo traje do 6. godine života, srednje djetinjstvo do 9. godine života, a kasno djetinjstvo do 12. godine života. Djetinjstvo je vrijeme početka školovanja.</p><p>Za vrijeme <a href="/wiki/Pubertet" title="Pubertet">puberteta</a> dolazi do tjelesnog, spolnog, psihičkog i drugih oblika sazrijevanja. Kod djevojčica događa se prva <a href="/wiki/Menstruacija" class="mw-redirect" title="Menstruacija">menstruacija</a> i <a href="/wiki/Ovulacija" title="Ovulacija">ovulacija</a>. Razvijaju se <a href="/wiki/Spolni_organi" title="Spolni organi">spolni organi</a> kod oba <a href="/wiki/Spol" title="Spol">spola</a>. <a href="/wiki/Adolescencija" title="Adolescencija">Adolescencija</a> je razdoblje nakon puberteta, tijekom kojeg dolazi do daljnjeg razvoja <a href="/wiki/Osobnost" class="mw-redirect" title="Osobnost">osobnosti</a>, osamostaljivanja i sazrijevanja. Rast <a href="/wiki/Tijelo_(biologija)" title="Tijelo (biologija)">tijela</a> uglavnom traje do 30. godine, a spolna zrelost doseže se od 12. do 15. godine života (<a href="/wiki/Pubertet" title="Pubertet">pubertet</a>).</p><p>U zrelom razdoblju života, čovjek se ostvaruje na poslovnom i <a href="/wiki/Obitelj" title="Obitelj">obiteljskom</a> planu. Zdrav i dobro uhranjen čovjek u prosjeku doživi 70-80 <a href="/wiki/Godina" title="Godina">godina</a>. Dokazano je da maksimalni životni vijek iznosi 120 godina.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup> <a href="/wiki/Smrt" title="Smrt">Smrt</a> je prestanak <a href="/wiki/Biologija" title="Biologija">bioloških</a> funkcija. <a href="/wiki/Ljudska_visina" title="Ljudska visina">Visina</a> većine odraslih ljudi iznosi između 150 i 200 centimetara, a <a href="/wiki/Te%C5%BEina" title="Težina">težina</a> između 50 i 90 kilograma. Jedna od vrlo bitnih osobina čovjeka je uspravan hod.</p><h2><span class="mw-headline" id="Anatomija">Anatomija</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%C4%8Covjek&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Uredi odlomak: Anatomija">uredi VE</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%C4%8Covjek&amp;action=edit&amp;section=4" title="Uredi odlomak: Anatomija">uredi</a><span class="mw-editsection-bracket">]</span></span></h2><p><a href="/wiki/Datoteka:Vista-xmag.png" class="image"><img alt="Vista-xmag.png" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e5/Vista-xmag.png/18px-Vista-xmag.png" width="18" height="18" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e5/Vista-xmag.png/27px-Vista-xmag.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e5/Vista-xmag.png/36px-Vista-xmag.png 2x" data-file-width="128" data-file-height="128" /></a><i>Podrobniji članak o temi: <a href="/wiki/Anatomija_%C4%8Dovjeka" title="Anatomija čovjeka">Anatomija čovjeka</a></i><br /></p><p>Ljudsko tijelo sastoji se od <a href="/wiki/Glava" title="Glava">glave</a>, <a href="/wiki/Ud" class="mw-redirect" title="Ud">udova</a> i <a href="/wiki/Trup" class="mw-disambig" title="Trup">trupa</a>. Oni se sastoje od <a href="/wiki/Organ" title="Organ">organa</a> i različitih <a href="/wiki/Tkivo" title="Tkivo">tkiva</a>. Dopunjuju se čineći ljudski organizam. <a href="/wiki/Kosti" class="mw-redirect" title="Kosti">Kosti</a> i <a href="/wiki/Mi%C5%A1i%C4%87i" class="mw-redirect" title="Mišići">mišići</a> (koji pomoću antagonističkog stezanja i opuštanja pokreću kosti) čovjeku omogućuju <a href="/wiki/Kretanje" class="mw-redirect" title="Kretanje">kretanje</a>. Glavni dio čovjekova kostura čini <a href="/wiki/Kralje%C5%BEnica" title="Kralježnica">kralježnica</a>, koja povezuje kosti udova, glave i trupa.</p><p>Ljudi se međusobno razlikuju po fizičkom izgledu i karakternim osobinama.</p><h2><span class="mw-headline" id="Fiziologija">Fiziologija</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a data-file-width="128" data-file-height="128" /></a><i>Podrobniji članak o temi: <a href="/wiki/Fiziologija_%C4%8Dovjeka" title="Fiziologija čovjeka">Fiziologija čovjeka</a></i><br /></p><p>Fiziologija je <a href="/wiki/Znanost" title="Znanost">znanost</a> o mehaničkim, fizičkim i biokemijskim funkcijama ljudi. Fiziologija se fokusira na <a href="/wiki/Organ" title="Organ">organe</a> i <a href="/wiki/Tkiva" class="mw-redirect" title="Tkiva">tkiva</a>. Promatra tijelo kao skupinu sustava s međusobnom interakcijom, svaki s vlastitom kombinacijom funkcija i svrhom. Fiziologija čovjeka bliska je <a href="/w/index.php?title=Fiziologija_%C5%BEivotinja&amp;action=edit&amp;redlink=1" class="new" title="Fiziologija životinja (stranica ne postoji)">fiziologiji životinja</a> pa je mnogo spoznaja dobiveno <a href="/wiki/Eksperiment" class="mw-redirect" title="Eksperiment">eksperimentima</a> na <a href="/wiki/%C5%BDivotinja" class="mw-redirect" title="Životinja">životinjama</a>. Anatomija i fiziologija usko su povezana područja: anatomija, istražuje <a href="/wiki/Oblik" title="Oblik">oblike</a>, a fiziologija proučava funkcije, uče se zajedno kao dio <a href="/wiki/Medicina" title="Medicina">medicinskog</a> kurikuluma. Mnoge fiziološke varijable (kao razina <a href="/wiki/Glukoza" title="Glukoza">glukoze</a> u krvi, tjelesna temperatura, pH krvi itd.) moraju se održavati u uskim granicama kako bi se održalo <a href="/wiki/Zdravlje" title="Zdravlje">zdravlje</a>. Prevladavajuća tema fiziologije je o <a href="/wiki/Homeostaza" title="Homeostaza">homeostazi</a>, održavanju stabilnog unutrašnjeg okruženja unatoč vanjskim flaktuacijama. Primarna funkcija mnogih organskih sustava jest održavanje homeostaze. Npr., mokraćni sustav pomaže kontroliranju količine <a href="/wiki/Voda" title="Voda">vode</a> u organizmu kao i održavanje pH vrijednosti krvi i različitih otpadnih produkata, a krvožilni sustav osigurava tkivima stalnu opskrbu kisikom i hranjivim tvarima i uklanja štetne produkte.</p><p>Proučavanje promjene fizioloških procesa u bolesti jest <a href="/wiki/Patofiziologija" title="Patofiziologija">patofiziologija</a>.</p><h2><span id="Evolucijski_razvoj_.C4.8Dovjeka"></span><span class="mw-headline" id="Evolucijski_razvoj_čovjeka">Evolucijski razvoj čovjeka</span><span
