kolorea, eboluzioaren margoa</div></div></div><p><b>Gizakia</b> edo <i><b>Homo sapiens</b></i> (<a href="/wiki/Latin" title="Latin">latinetik</a> <i>ezagutzen duen gizakia</i>) <a href="/wiki/Primate" title="Primate">primate</a> <a href="/w/index.php?title=Bipedismo&amp;action=edit&amp;redlink=1" class="new" title="Bipedismo (sortu gabe)">bipedo</a> bat da, <a href="/wiki/Hominidae" title="Hominidae">Hominidae</a> familiaren barnean (<a href="/wiki/Gorila" title="Gorila">gorila</a>, <a href="/wiki/Txinpantze" title="Txinpantze">txinpantze</a> eta <a href="/wiki/Pongo" title="Pongo">orangutanak</a> barnebiltzen dituena). Beste <a href="/wiki/Ugaztun" title="Ugaztun">ugaztunekin</a> alderatuta eta era berean beste <a href="/wiki/Ornodun" title="Ornodun">ornodun</a> guztiekin, <a href="/wiki/Garun" title="Garun">burmuin</a> tamaina erlatibo oso handia du, arrazonatzea, <a href="/wiki/Hizkuntza" title="Hizkuntza">hizkuntza</a> eta barne bizitza edukitzea baimentzen diona. Bizkarra <a href="/w/index.php?title=Ortogrado&amp;action=edit&amp;redlink=1" class="new" title="Ortogrado (sortu gabe)">ortogradoa</a> denez eta beraz besoak libre dituenez <a href="/wiki/Tresna" title="Tresna">tresnak</a> asmatzeko aukera du, burmuinarekin ere egin ahal duena. Tresnak asmatzen dituen izaki bakarra ez bada ere hauen kopurua eta diseinua askoz handiagoa da. <a href="/wiki/Azido_desoxirribonukleiko" title="Azido desoxirribonukleiko">DNAren</a> ikerketetako datuetan oinarriturik, ondorioztatu da gizaki modernoa <a href="/wiki/Afrika" title="Afrika">Afrikan</a> sortu zela orain dela gutxi gorabehera 200.000 urte.<sup id="cite_ref-Smithsonian_1-0" class="reference"><a href="#cite_note-Smithsonian-1">[1]</a></sup> Gaur egun <a href="/wiki/Kontinente" title="Kontinente">kontinente</a> guztietan aurki daitezke, baita <a href="/wiki/Antartika" title="Antartika">Antartikan</a> ere (nahiz eta oso kopuru txikian), eta 6.600 milioi baino gehiago ziren <a href="/wiki/2007" title="2007">2007an</a>.<sup id="cite_ref-popclock_2-0" class="reference"><a href="#cite_note-popclock-2">[2]</a></sup></p><p>Primate gehienek bezala gizakia <a href="/w/index.php?title=Animalia_sozial&amp;action=edit&amp;redlink=1" class="new" title="Animalia sozial (sortu gabe)">animalia soziala</a> da naturalki eta beste batzuen beharra izan ohi du garapenerako. Hala ere gizakiak <a href="/wiki/Komunikazio" title="Komunikazio">komunikazio</a> gaitasun handiak ditu eta horrekin antolaketaren inguruko ideiak garatu ahal ditu. Gizakiek <a href="/wiki/Gizarte_egitura" title="Gizarte egitura">egitura sozial</a> konplexuak sortzen dituzte, elkarren artean kooperatzen eta lehiatzen duten taldeekin, <a href="/wiki/Familia" title="Familia">familiatik</a> hasita talde politiko, zientifiko eta ekonomikoetaraino. Elkarrekintza honek ohitura, erritual, herri, balore, arau eta legeak sortu ditu, gizakiaren elkarbizitzaren oinarri direnak. Gizakiak, era berean, <a href="/wiki/Estetika" title="Estetika">estetikaren</a> balorazioa du, <a href="/wiki/Kultura" title="Kultura">kultura</a> berrikuntzen, <a href="/wiki/Arte" title="Arte">artearen</a>, <a href="/wiki/Literatura" title="Literatura">literaturaren</a> eta <a href="/wiki/Musika" title="Musika">musikaren</a> oinarrietako bat dena.</p><p>Gizakiek ere interesa dute euren inguruan dagoena ulertzeko eta hauek <a href="/wiki/Zientzia" title="Zientzia">zientzia</a>, <a href="/wiki/Filosofia" title="Filosofia">filosofia</a>, <a href="/wiki/Mitologia" title="Mitologia">mitologia</a> eta <a href="/wiki/Erlijio" title="Erlijio">erlijioaren</a> bitartez azaltzeko. Gaitasun honek hainbat natura gai ezagutzera eraman ditu eta gai dira <a href="/wiki/Su" title="Su">sua</a> erabiltzeko, <a href="/wiki/Sukaldaritza" title="Sukaldaritza">sukaldatze</a>, <a bat.</div></div></div><p>Giza gorputza motak oso aldakorrak dira. Nahiz eta gorputzaren tamaina oso determinaturik dagoen <a href="/wiki/Gene" title="Gene">geneengatik</a> <a href="/wiki/Dieta" title="Dieta">dieta</a> eta <a href="/wiki/Ariketa_fisiko" title="Ariketa fisiko">ariketa fisikoak</a> ere asko eragiten dute honen garapenean. Gizakiaren batez besteko altuera 1,5-1,8 metro artekoa da baina oso aldakorra da lekurik leku.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup><sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup> Gizakiak guztiz gai dira bi hanken gainean ibiltzeko eta modu horretan <a href="/wiki/Esku" title="Esku">eskuak</a> libre uzten dira, objektuak manipulatzea baimenduz. Beste primateek bezala aurrez-aurre jar daitekeen erpurua dute.</p><p>Giza gorputza, berrehun <a href="/wiki/Hezur" title="Hezur">hezurretik</a> gora, ehun <a href="/wiki/Artikulazio" title="Artikulazio">artikulazio</a> eta seiehun eta berrogeita hamar <a href="/wiki/Gihar" title="Gihar">gihar</a>, elkarrekin koordinatuak eta batera funtzionatzen duten egitura da. Gorputzak haren jarrera eutsi, higitu eta beste ekintza asko egin ahal ditu atal guztien elkarlanarekin. Hau guztia aztertzen duen <a href="/wiki/Zientzia" title="Zientzia">zientzia</a> da giza anatomia.</p><p><br />Gizakiak beste primate batzuekin alderatua nahiko ile gutxikoa da, nahiz eta <a href="/wiki/Buru" title="Buru">buru</a>, <a href="/wiki/Besape" title="Besape">besape</a> eta <a href="/w/index.php?title=Hankatarte&amp;action=edit&amp;redlink=1" class="new" title="Hankatarte (sortu gabe)">hankatartean</a> asko hazten diren. Hala ere txinpanze batek haina ile folikulo ditu gorputzean. Gizakien ileak motzagoak, finagoak eta pigmentu gutxiagokoak dira, eta ikusten zailagoak.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup></p><div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Fitxategi:Inuit_women_1907.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/68/Inuit_women_1907.jpg/220px-Inuit_women_1907.jpg" width="220" height="267" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/68/Inuit_women_1907.jpg/330px-Inuit_women_1907.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/68/Inuit_women_1907.jpg/440px-Inuit_women_1907.jpg 2x" data-file-width="2018" data-file-height="2453" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Fitxategi:Inuit_women_1907.jpg" class="internal" title="Handitu"></a></div>Emakume <a href="/wiki/Inuit" title="Inuit">Inuit</a> bat 1907 urtearen inguruan</div></div></div><p>Gizakiaren azalaren eta ilearen kolorea <a href="/wiki/Melanina" title="Melanina">melanina</a> izeneko <a href="/wiki/Pigmentu" title="Pigmentu">pigmentuaren</a> araberakoa da. Kolorea marroi oso ilunetik larrosa oso argira joan daiteke eta ilea <a href="/w/index.php?title=Horaila&amp;action=edit&amp;redlink=1" class="new" title="Horaila (sortu gabe)">horaila</a>, <a href="/wiki/Gorri" title="Gorri">gorria</a>, <a href="/wiki/Arre" class="mw-disambig" title="Arre">arrea</a> eta, batez ere, <a href="/wiki/Beltz" title="Beltz">beltza</a> izan daiteke.<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup> Zientzialari gehiene ustez aldaketa hauek <a href="/wiki/Eguzki-erradiazio" title="Eguzki-erradiazio">eguzki erradiaziok</a> igortzen dituen izpi <a href="/wiki/Ultramore" title="Ultramore">ultramoreetatik</a> babesteko garatu zen, melanina eguzkiaren aurrean blokeatzaile ona baita.<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">[7]</a></sup> Larruaren pigmentazioa gaur egun geografikoki estratifikatua dago eta orokorrean izpi ultramoreen mailarekin du harremana. Gizakiaren larruak eguzkiaren indarrarekin batera iluntzeko gaitasuna ere badu, beltzarantzea.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">[8]</a></sup><sup id="cite_ref-9" class="reference"><a href="#cite_note-9">[9]</a></sup></p><p>Egunero behar den <a href="/wiki/Lo" title="Lo">lo</a> kopurua 7-8 ordu artekoa da pertsona nagusi batean eta 9-10 artean ume batean. Zaharrak direnek 6-7 ordu inguruan egiten dituzte egunean. Jendarte modernoetan oso ohikoa da hau baino gutxiago lo egitea eta honek egoera txarrak sortu ahal ditu, nekea, agresibitatea eta bestelako gaixotasunak kasu.</p><p>Gizakiak espezie <a href="/wiki/Eukarioto" title="Eukarioto">eukariota</a> bat dira. <a href="/wiki/Zelula" title="Zelula">Zelula</a> <a href="/wiki/Diploide" title="Diploide">diploide</a> bakoitzak 23 <a href="/wiki/Kromosoma" title="Kromosoma">kromosoma</a> pare ditu, horietako bakoitza gurasoen batetik jasota. 22 pare <a href="/wiki/Autosoma" title="Autosoma">autosomak</a> dira eta beste pare bat <a href="/wiki/Sexu_ugalketa" title="Sexu ugalketa">sexuaren</a> determinaziorako. Uste da 20.000 eta 25.000 gene artean dituela. Beste ugaztun batzuetan gertatzen den bezala gizakiek <a href="/w/index.php?title=XY_sexu-determinazio_sistema&amp;action=edit&amp;redlink=1" class="new" title="XY sexu-determinazio sistema (sortu gabe)">XY sexu-determinazio sistema</a> dute, non <a href="/wiki/Emakume" title="Emakume">emakumeek</a> XX kromosoma parea duten eta <a href="/wiki/Gizon" title="Gizon">gizonek</a> XY. X kromosoma handiagoa da eta informazio gehiago du Yk baino. Honegatik X kromosoman agertzen diren gaixotasun genetiko batzuek eragin handiagoa dute gizonengan emakumeengan baino, <a href="/wiki/Hemofilia" title="Hemofilia">hemofilia</a> kasu.</p><h3><span class="mw-headline" id="Hortzeri_formula">Hortzeri formula</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gizaki&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Hortzeri formula»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Gizaki&amp;action=edit&amp;section=3" title="Aldatu atal hau: «Hortzeri formula»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h3><p>Gizakiek 2 <a href="/wiki/Ebakortz" title="Ebakortz">ebakortz</a>, <a href="/wiki/Letagin" title="Letagin">letagin</a> 1, 2 <a href="/w/index.php?title=Aurreko_hagin&amp;action=edit&amp;redlink=1" class="new" title="Aurreko hagin (sortu gabe)">aurreko hagin</a> eta 2 <a href="/wiki/Hagin_(argipena)" class="mw-disambig" title="Hagin (argipena)">hagin</a> dituzte barail-erdi bakoitzean. Bere formula, beraz honakoa da:</p><table style="margin: 0 0 0.5em 1em; border-collapse:collapse; text-align:center;" border="1" cellpadding="0"><tr><td 5 asterekin.</div></div></div><p>Gizakiaren <a href="/w/index.php?title=Bizitza_ziklo&amp;action=edit&amp;redlink=1" class="new" title="Bizitza ziklo (sortu gabe)">bizitza zikloa</a> beste <a href="/w/index.php?title=Karenadun&amp;action=edit&amp;redlink=1" class="new" title="Karenadun (sortu gabe)">karenadunen</a> antzekoa da. Gizaki berriak modu <a href="/wiki/Bibiparo" title="Bibiparo">bibiparoan</a> sortzen dira <a href="/w/index.php?title=Fertilizazio&amp;action=edit&amp;redlink=1" class="new" title="Fertilizazio (sortu gabe)">fertilizazioan</a>. <a href="/wiki/Espermatozoide" title="Espermatozoide">Espermatozoide</a> batek <a href="/wiki/Obulu" title="Obulu">obulu</a> bat ernaltzen du emearen barruan <a href="/wiki/Sarpen_(sexualitatea)" title="Sarpen (sexualitatea)">sarketaren</a> ondoren. Hala ere <a href="/wiki/In_vitro_ernalketa" title="In vitro ernalketa">in vitro ernalkuntza</a> dela eta hau kanpoan ere egin daiteke. Obulo fertilizatua, <a href="/wiki/Zigoto" title="Zigoto">zigoto</a> izenekoa, <a href="/wiki/Umetoki" title="Umetoki">umetokiaren</a> barruan garatzen doa <a href="/wiki/Enbrioi" title="Enbrioi">enbrioi</a> batean bilakatu arte. 38 asteren buruan (9 hilabete) <a href="/wiki/Umeki" title="Umeki">umeki</a> bilakatu eta ondoren jaiotzen da. Guztiz hazi denean emakumearen gorputzetik atera eta arnasa bere kasa hartzeko gai da, <a href="/wiki/Ume" class="mw-disambig" title="Ume">ume</a> bat bilakatuz. Momentu honetan kultura moderno gehienek <a href="/wiki/Pertsona_(argipena)" class="mw-disambig" title="Pertsona (argipena)">pertsona</a> gisa onartzen dute eta beraz legearen babes osoa izango du, nahiz eta jurisdikzio batzuetan pertsona izatea uteroaren barrura ere hedatzen den.</p><p>Beste espezie batzuekin alderatua gizakien jaiotza arrikutsua da. 24 ordu baino gehiago irauten duten <a href="/wiki/Erditze" title="Erditze">erditzeak</a> ez dira arraroak eta umearen edo amaren heriotza eragin dezakete. Umekiaren buruaren tamaina dela eta (burmuin handia izateko beharrezkoa) eta emakumean duen <a href="/wiki/Aldaka" title="Aldaka">aldaka</a> estua dela eta (bi hanken gainean ibiltzeko beharrezkoa dena) ozta-ozta sartzen da bata bestean zehar.<sup id="cite_ref-10" class="reference"><a href="#cite_note-10">[10]</a></sup><sup id="cite_ref-11" class="reference"><a href="#cite_note-11">[11]</a></sup>. <a href="/wiki/XX._mendea" title="XX. mendea">XX. mendean</a> zehar <a href="/wiki/Medikuntza" title="Medikuntza">medikuntzan</a> egon diren berrikuntzak direla eta erditzean hiltzeko aukerak asko murriztu dira. Hala ere garapen bidean dauden herrialde askotan oraindik ere arriskuak badirau, non erditzea dela-eta dagoen heriotza tasa 100 aldiz handiagoa baita.<sup id="cite_ref-12" class="reference"><a href="#cite_note-12">[12]</a></sup></p><p>Herrialde garatuetan <a href="/wiki/Haur_jaioberri" title="Haur jaioberri">jaioberriek</a> 3-4 kg. inguruko pisua dute eta 50 - 60 cm. arteko tamaina dute.<sup id="cite_ref-13" class="reference"><a href="#cite_note-13">[13]</a></sup>, Hala ere jaiotzan pisu gutxi izatea ohikoa da herrialde garatuetan eta umeen heriotza tasan eragin handia du.<sup id="cite_ref-14" class="reference"><a href="#cite_note-14">[14]</a></sup> Umea jaio berria denean oso ahula da eta urte nahiko behar ditu hazten jarraitzeko. Heldutasun sexuala 12-15 urte inguruan heltzen da. <a href="/wiki/Neska" title="Neska">Neskek</a> 18 urte arte hazten jarraitzen dute eta <a href="/wiki/Mutil" title="Mutil">Mutilek</a> 21 urte arte. Modu honetan gizakien bizitzaren zikloa haurtzaro, <a href="/wiki/Nerabezaro" title="Nerabezaro">nerabezaro</a>, <a href="/wiki/Gaztaro" title="Gaztaro">gaztaro</a>, <a href="/wiki/Heldu" title="Heldu">heldutasun</a> eta zahartaroan bana daiteke. Hala ere azkeneko atal hauek ez dira oso finkoak eta iraupen ezberdina dute.</p><p><a href="/wiki/Bizi_itxaropen" title="Bizi itxaropen">Bizi itxaropena</a> ezberdina da herrialde ezberdinetan. Lehen Munduko populazioa pixkanaka-pixkanaka zahartzen ari da, batez besteko adina 40 urtekoa da gaur egun (<a href="/wiki/Monako" title="Monako">Monako</a> da altuena, 45,1 urterekin). Hirugarren munduan, ordea, batez besteko adina 15-20 urte artean dago; <a href="/wiki/Uganda" title="Uganda">Uganda</a> da gazteena, 14,8 urterekin. <a href="/wiki/Hong_Kong" title="Hong Kong">Hong Kongen</a> bizi itxaropena 84,8 urtekoa da jaiotzean emakumeengan eta 78,9 gizonengan, <a href="/wiki/Swazilandia" title="Swazilandia">Swazilandian</a>, ordea (eta batez ere <a href="/wiki/Hartutako_Immuno_Eskasiaren_Sindromea" title="Hartutako Immuno Eskasiaren Sindromea">HIESa</a> dela eta) 31,3 urtekoa da bi sexuentzat.<sup id="cite_ref-15" class="reference"><a href="#cite_note-15">[15]</a></sup> Bost europarretik batek 60 urte baino gehiago ditu baina bakarrik 20tik 1 heltzen da adin horretara <a href="/wiki/Afrika" title="Afrika">Afrikan</a>.<sup id="cite_ref-16" class="reference"><a href="#cite_note-16">[16]</a></sup></p><p>100 urte baino gehiago dituzten biztanleen kopurua 210.000 inguru zen <a href="/wiki/2002" title="2002">2002an</a> <a href="/wiki/Nazio_Batuen_Erakundea" title="Nazio Batuen Erakundea">Nazio Batuen Erakundearen</a> arabera.<sup id="cite_ref-17" class="reference"><a href="#cite_note-17">[17]</a></sup> <a href="/wiki/Jeanne_Calment" title="Jeanne Calment">Jeanne Calment</a> da ezagutzen dugun pertsonarik zaharrena, 122 urterekin. Munduan dauden 60 urteko 100 emakumeko 81 gizon daude.</p><h3><span class="mw-headline" id="Elikadura">Elikadura</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gizaki&amp;veaction=edit&amp;section=5" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Elikadura»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Gizaki&amp;action=edit&amp;section=5" title="Aldatu atal hau: «Elikadura»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h3><p>Hasierako <i>Homo sapiens</i>ek bilketa-ehiza metodo bat erabiltzen zuten elikagaiak lortzeko. Euren dietan urtaroetako landareak (fruituak, haziak, tuberkuloak, zizak...) eta ehiza tartekatzen zuten. Uste denez gizakiek gutxienez <i><a href="/wiki/Homo_erectus" title="Homo erectus">Homo erectus</a></i>etik aurrera <a href="/wiki/Su" title="Su">sua</a> erabili dute <a href="/wiki/Sukaldaritza" title="Sukaldaritza">sukaldaritzarako</a>.</p><p>Gizakiak <a href="/wiki/Orojale" title="Orojale">orojaleak</a> dira, bai landareak bai animaliak jan ditzakete. Bakarrik bietako bat jaten duten gizaki taldeek edo gizabanakoek beharrezkoak diren sustantzia batzuen eskasia izan ohi dute. Animaliak bakarrik jaten dituztenek gaixotasun batzuk dituzte eta bakarrik landareak jaten dituztenek <a href="/wiki/B12_bitamina" title="B12 bitamina">B12 bitaminaren</a> eskasia izan ohi dute.</p><p>Gizakien dieta oso ondo aurkeztua dago bere kulturan eta elikagaien zientzia bat egotea ekarri du. Orokorrean gizakien 2tik 8 astera iraun dezakete janaririk gabe, duten <a href="/wiki/Gantz" title="Gantz">gantzaren</a> arabera. Urarik gabe, ordea, 3-4 egun baino ezin da iraun. Hala ere janaririk ez edukitzeak arazo handiak sortzen ditu eta urtero 300.000 bat gizaki hiltzen dira elikagai eskasiagatik.<sup id="cite_ref-18" class="reference"><a href="#cite_note-18">[18]</a></sup> Umeen elikadura txarrak ere gaixotasun ugari sortzen ditu.<sup id="cite_ref-19" class="reference"><a href="#cite_note-19">[19]</a></sup> Hala ere munduan dagoen elikagaien banaketa ez da ona eta <a href="/wiki/Obesitate" title="Obesitate">loditasuna</a> agertzen da munduko leku askotan, baita <a href="/wiki/Izurri" title="Izurri">epidemia</a> gisa ere. Herrialde garatuetan eta garatzen bidean dauden batzuetan heriotza tasa altua dago gehiegizko loditasuna eta horrek sortzen dituen gaixotasunak direla eta. Loditasuna xahutzen direnak baino <a href="/wiki/Kaloria" title="Kaloria">kaloria</a> gehiago jateagatik sortzen da eta gizakietan beste animalietan baino ohikoagoa izaten da.</p><p>Azkeneko 10.000 urtetan <a href="/wiki/Nekazaritza" title="Nekazaritza">nekazaritzaren</a> sorrera eta garapena eman da.<sup id="cite_ref-20" class="reference"><a href="#cite_note-20">[20]</a></sup><sup id="cite_ref-21" class="reference"><a href="#cite_note-21">[21]</a></sup><sup id="cite_ref-22" class="reference"><a href="#cite_note-22">[22]</a></sup> Gertakarik honek asko aldatu du gizakiek jaten dituzten elikagaiak. Hiriak eta populazioaren gehikuntza ekarri du, dentsitatea igotzea eta gaixotasunen garapena eta kutsadura azkarragoa izatea. Denbora eta munduko gune bakoitzean jaten diren eta elikagaiak eta hauek prestatzen erak asko aldatu eta aldatzen dira.</p><h2><span class="mw-headline" id="Historia">Historia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gizaki&amp;veaction=edit&amp;section=6" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Historia»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Gizaki&amp;action=edit&amp;section=6" title="Aldatu atal hau: «Historia»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><h3><span class="mw-headline" id="Eboluzioa">Eboluzioa</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gizaki&amp;veaction=edit&amp;section=7" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Eboluzioa»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Gizaki&amp;action=edit&amp;section=7" title="Aldatu atal hau: «Eboluzioa»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h3><div style="font-style: italic; Bi espezie hauek arbaso bera dute eta duten antolaketa dela eta bereizten dira nagusiki: (beste ezaugarri anatomiko batzuk ere badituzte, hala ere) <a href="/wiki/Bonobo" title="Bonobo">Bonoboa</a> <a href="/wiki/Matriarkatu" title="Matriarkatu">matriarkala</a> da eta txinpantzea <a href="/wiki/Patriarkatu" title="Patriarkatu">patriarkala</a> da. <a href="/wiki/Genoma" title="Genoma">Genoma</a> osoaren sekuentziazioa eginda R. Lehenengoak, <a href="/w/index.php?title=Afrikatik_kanpora&amp;action=edit&amp;redlink=1" class="new" title="Afrikatik kanpora (sortu gabe)">Afrikatik kanpora</a> hipotesia, gizakia <a href="/wiki/Afrika" title="Afrika">Afrikan</a> sortu zela dio eta bertatik kanpora joan zela zeuden beste Homo guztiak ordezkatuz. <a href="/w/index.php?title=Hipotesi_multiregionala&amp;action=edit&amp;redlink=1" class="new" title="Hipotesi multiregionala (sortu gabe)">Hipotesi multiregionalak</a> dioenez gizakiaren eboluzioa leku batean baino gehiagotan eman zen aldi berean hominido populazio ezberdinetatik abiatuz.<sup id="cite_ref-29" class="reference"><a href="#cite_note-29">[29]</a></sup></p><p><a href="/w/index.php?title=Lynn_Jorde&amp;action=edit&amp;redlink=1" class="new" title="Lynn Jorde (sortu gabe)">Lynn Jorde</a> eta <a href="/w/index.php?title=Henry_Harpending&amp;action=edit&amp;redlink=1" class="new" title="Henry Harpending (sortu gabe)">Henry Harpending</a> <a href="/w/index.php?title=Genetista&amp;action=edit&amp;redlink=1" class="new" title="Genetista (sortu gabe)">genetistek</a> proposatu dutenez gizakien barnean dauden <a href="/wiki/Azido_desoxirribonukleiko" title="Azido desoxirribonukleiko">DNA</a> aldaketak oso txikiak dira beste hurbileko espezie batzuekin alderatua. Hau azaltzeko <a href="/wiki/Pleistozeno" title="Pleistozeno">Pleistozenoa</a> bitarte gizakien populazioa oso txikia izan behar zuela ondorioztatu da, agian 10.000 baino gutxiago eta aukerak daude 1.000 inguru izateko. Horrela oso bariabilitate genetiko txikia egongo litzateke. Hau azaltzeko <a href="/w/index.php?title=Toba_katastrofearen_teoria&amp;action=edit&amp;redlink=1" class="new" title="Toba katastrofearen teoria (sortu gabe)">Toba katastrofearen teoria</a> erabili da.</p><p>Gizakiaren eboluzioan aldaketa <a href="/wiki/Morfologia_(argipena)" class="mw-disambig" title="Morfologia (argipena)">morfologiko</a> eta <a href="/wiki/Fisiologia" title="Fisiologia">fisiologiko</a> garrantzitsuak egon dira <i>Homo</i> eta <i>Pan</i> bereiztu zirenetik. Lehenengo aldaketa (bai denboran eta bai definizioan erabiltzen delako) <a href="/wiki/Hominina" title="Hominina">Hominina</a> <a href="/w/index.php?title=Azpileinu&amp;action=edit&amp;redlink=1" class="new" title="Azpileinu (sortu gabe)">azpileinua</a> definitzeko erabiltzen dena da. Aldaketa hau bi hanken gainean ibiltzera moldatzea litzateke, zuhaitzetako (denbora guztian edo zati batean) bizitza batetik. Horretarako hainbat aldaketa eman ziren <a href="/wiki/Belaun" title="Belaun">belaunen</a> <a href="/w/index.php?title=Valgus&amp;action=edit&amp;redlink=1" class="new" title="Valgus (sortu gabe)">valgusean</a>, gorputzaren goikaldearen indarrean eta besoen eta hanken arteko luzeran (hanka luzeak besoekin alderatua). Honen ondoren burezurraren tamaina eta horrekin batera <a href="/wiki/Garun" title="Garun">burmuinarena</a> handitu ziren. Gaur egun tamaina hau 1.400 cm³ inguruko bolumena da, <a href="/wiki/Gorila" title="Gorila">gorila</a> eta txinpanzeetan baino askoz handiagoa. Gainera eskuen erabileran hobekuntza nabarmenak egon ziren hortzeriaren tamaina murrizten zen bitartean. <a href="/wiki/Letagin" title="Letagin">Letaginaren</a> txikitzea eman da. <a href="/wiki/Laringe" title="Laringe">Laringearen</a> eta <a href="/w/index.php?title=Hezur_hioide&amp;action=edit&amp;redlink=1" class="new" title="Hezur hioide (sortu gabe)">hezur hioidearen</a> jaitsiera ere eman zen, hitz egiteko gaitasuna ematen duena. Garapenari loturik burmuinaren hazte tamainan <a href="/w/index.php?title=Heterokronia&amp;action=edit&amp;redlink=1" class="new" title="Heterokronia (sortu gabe)">heterokronia</a> bat ematen da, ikasteko prozesua luzatuz (haurtzaroa luzatzen baita). Burmuinaren tamainaz gain honen berrantolakuntza bat ere eman zen, adibidez hitz egiteko beharrezkoak diren <a href="/w/index.php?title=Brokaren_area&amp;action=edit&amp;redlink=1" class="new" title="Brokaren area (sortu gabe)">Brokaren area</a> eta <a href="/w/index.php?title=Bernickeren_area&amp;action=edit&amp;redlink=1" class="new" title="Bernickeren area (sortu gabe)">Bernickeren arearen</a> tamaina eta egoera. Gizaki emeetan (<a href="/wiki/Emakume" title="Emakume">emakumea</a>) aldaketak daude <a href="/wiki/Obulazio" title="Obulazio">obulazioan</a> eta honen bukaeran <a href="/wiki/Estrogeno" title="Estrogeno">estrogeno</a> kopurua 1/3 baino ez da jaisten, beste primate batzuetan ia guztiz desagertzen den bitartean. Gizakiak gainera <a href="/w/index.php?title=Kultura_material&amp;action=edit&amp;redlink=1" class="new" title="Kultura material (sortu gabe)">kultura materiala</a> du eta hauek edukitzeak statusa ere adierazten du. Aldaketa guztien ordenak eztabaida sutsuak pizten ditu gaur egun.<sup id="cite_ref-30" class="reference"><a href="#cite_note-30">[30]</a></sup><sup id="cite_ref-31" class="reference"><a Normalean talde <a href="/wiki/Nomada" title="Nomada">nomada</a> txikietan bizi ziren. <a href="/wiki/Neolito" title="Neolito">Neolitikoan</a> <a href="/wiki/Nekazaritza" title="Nekazaritza">nekazaritza</a> sortu zen eta horrekin batera gehiegizko janariaren produkzioa. Gizakiak egonkortu ziren eta lehenego herriguneak sortu. Horrek animalien <a href="/wiki/Etxekotze" title="Etxekotze">etxekotzea</a> eta <a href="/wiki/Metalgintza" class="mw-redirect" title="Metalgintza">metalgintza</a> ekarri zuen. Nekazaritzak komertzioa eta kooperazioa handiagotu zituen gizarte konplexuagoak sortzen. Herriak <a href="/wiki/Zibilizazio" title="Zibilizazio">zibilizazio</a> bilakatu ziren <a href="/wiki/Ekialde_Hurbila" title="Ekialde Hurbila">Ekialde Hurbilean</a>, <a href="/wiki/Mesopotamia" title="Mesopotamia">Mesopotamia</a> inguruan.</p><p>Orain dela 6.000 urte inguru lehenengo proto<a href="/wiki/Estatu" title="Estatu">estatuak</a> sortu ziren <a href="/wiki/Mesopotamia" title="Mesopotamia">Mesopotamian</a>, <a href="/wiki/Antzinako_Egipto" title="Antzinako Egipto">Antzinako Egipton</a> eta <a href="/wiki/Indus_haraneko_Zibilizazioa" class="mw-redirect" title="Indus haraneko Zibilizazioa">Indus haranean</a>. Hauek defendatzeko <a href="/wiki/Gudari" title="Gudari">militarrak</a> sortu ziren eta horrekin batera <a href="/wiki/Gobernu" title="Gobernu">gobernu</a> <a href="/wiki/Burokrazia" title="Burokrazia">burokratikoak</a> eta administrazioa. Estatuen artean kooperazioa eta konpetentzia sortu zen <a href="/wiki/Gerra" title="Gerra">gerrak</a> sortuz. Orain dela 2000 eta 3000 urte artean <a href="/wiki/Persia" class="mw-disambig" title="Persia">Persiarrek</a>, <a href="/wiki/Txinako_Herri_Errepublika" title="Txinako Herri Errepublika">Txinatarrek</a> eta <a href="/wiki/Antzinako_Erroma" title="Antzinako Erroma">Erromatarrek</a> konkistaren bitartez inperio erraldoiak sortu zituzten, lehenengo <a href="/wiki/Inperio" title="Inperio">inperioak</a> sortuz. <a href="/wiki/Erlijio" title="Erlijio">Erlijio</a> garrantzitsuak sortu ziren <a href="/wiki/Ekialde_Hurbila" title="Ekialde Hurbila">Ekialde Hurbilean</a>, <a href="/wiki/Judaismo" title="Judaismo">Judaismoa</a> kasu eta <a href="/wiki/Hinduismo" title="Hinduismo">Hinduismoa</a> Hego <a href="/wiki/Asia" title="Asia">Asian</a>.</p><p><a href="/wiki/Erdi_Aroa" title="Erdi Aroa">Erdi Aroaren</a> erdialdean ideia berri eta iraultzaileak sortu ziren. <a href="/wiki/Txinako_Herri_Errepublika" title="Txinako Herri Errepublika">Txinan</a> <a href="/wiki/Inprimatzeko_makina" title="Inprimatzeko makina">inprenta</a> eta <a href="/wiki/Iparrorratz" title="Iparrorratz">iparrorratza</a> asmatu zituzten eta <a href="/w/index.php?title=Islamaren_Urrezko_Aroa&amp;action=edit&amp;redlink=1" class="new" title="Islamaren Urrezko Aroa (sortu gabe)">Islamaren Urrezko Aroa</a> izan zen zientzialari ugarirekin. Hurrengo 500 urtetan <a href="/wiki/Inperialismo" title="Inperialismo">inperialismoaren</a> garapenak <a href="/wiki/Amerika" title="Amerika">Amerika</a>, <a href="/wiki/Asia" title="Asia">Asia</a> eta <a href="/wiki/Afrika" title="Afrika">Afrikako</a> leku ugari Europarren kontrolpean utzi zituzten, <a href="/wiki/Independentzia" title="Independentzia">independentziaren</a> aldeko mugimenduak sortzen. <a href="/wiki/Zientzia_Iraultza" class="mw-redirect" title="Zientzia Iraultza">Zientzia Iraultza</a> <a href="/wiki/XVII._mendea" title="XVII. mendea">XVII. mendean</a> eta <a href="/wiki/Industria_Iraultza" title="Industria Iraultza">Industria Iraultza</a> <a href="/wiki/XVIII._mendea" title="XVIII. mendea">XVIII.</a> eta <a href="/wiki/XIX._mendea" title="XIX. mendea">XIX. mendean</a> <a href="/wiki/Garraio" title="Garraio">garraiobide</a> ugari sortu zituen, <a href="/wiki/Trenbide" title="Trenbide">trenbidea</a> eta <a href="/wiki/Automobil" title="Automobil">autoa</a> kasu. <a href="/wiki/Energia" title="Energia">Energia</a> lortzeko bideak ugaritu ziren, <a href="/wiki/Argindar" title="Argindar">argindarra</a> eta <a href="/wiki/Harrikatz" title="Harrikatz">ikatza</a> gisa. Honek populazio dentsitate eta poluzio arazo handiak sortu ditu.</div></div></div><p>Hasierako giza populazio finkoek <a href="/wiki/Ur" title="Ur">ura</a> behar zuten gertu bizi ahal izateko eta bizi ereduaren arabera beste baliabide natural batzuk, lur emankorra gisa (nekazaritza eta abeltzantza egin ahal izateko). Beste batzuek urtean zehar ehiza egon ahal izatea ere behar zuten. Hala ere gizakiek gaitasun handia dute euren ingurunea aldatzeko metodo ezberdinen bitartez: <a href="/wiki/Ureztatze" title="Ureztatze">ureztatzea</a>, [hirigintza]], <a href="/wiki/Eraikuntza" title="Eraikuntza">eraikuntza</a>, <a href="/wiki/Garraio" title="Garraio">garraioa</a> eta baliabideak sortuz. Merkatu eta garraio egituraren bitartez baliabideetatik gertu egotea ez da beharrezkoa jada eta horregatik herri edo hiri baten hazte gaitasuna ez dago zuzenean lortuta bertan dituen baliabideekin.</p><p>Teknologiak gizakia kontinente guztietan eta <a href="/wiki/Klima" title="Klima">klima</a> anitzetan bizitzeko gaitasuna eman dio. Azkeneko hamarkadetan gizakien <a href="/wiki/Antartika" title="Antartika">Antartika</a> esploratu dute, <a href="/wiki/Ozeano" title="Ozeano">ozeanoen</a> hondoak eta <a href="/wiki/Espazio" title="Espazio">espazioa</a>, nahiz eta bertan bizirautea denbora luzez zaila egiten den. 6.600 milioi biztanle baino gehiagorekin gizakiak dira <a href="/wiki/Ugaztun" title="Ugaztun">ugaztunen</a> artean talderik ugarienetarikoa. Gizakien gehiengoa&#160;%61a <a href="/wiki/Asia" title="Asia">Asian</a> bizi da.&#160;%14 Ameriketan,&#160;%13a <a href="/wiki/Afrika" title="Afrika">Afrikan</a> eta&#160;%12a <a href="/wiki/Europa" title="Europa">Europan</a>. Gainontzeko&#160;%0,5a <a href="/wiki/Ozeania" title="Ozeania">Ozeanian</a> bizi da.</p><p>Gizakia <a href="/w/index.php?title=Ingurune_ekologiko_itxi&amp;action=edit&amp;redlink=1" class="new" title="Ingurune ekologiko itxi (sortu gabe)">ingurune ekologiko itxi</a> batean bizitzeko gaitasuna badu ere hau oso garestia da. Antartika eta espazioan bizitzea epe laburretan egin izan du eta gehienez jota lurretik kanpo 30 bat biztanle eduki ditu aldi berean. <a href="/wiki/1969" title="1969">1969tik</a> <a href="/wiki/1972" title="1972">1972era</a> biztanleria hori gehitu zen, batez ere <a href="/wiki/Ilargia" title="Ilargia">Ilargiaren</a> esplorazioan, 200 bat izatera heldu baitziren. <a href="/wiki/2007" title="2007">2007an</a> oraindik ez da beste planetarik bisitatu nahiz eta <a href="/wiki/Nazioarteko_Espazio_Estazioa" title="Nazioarteko Espazio Estazioa">Nazioarteko Estazio Espaziala</a> <a href="/wiki/2000" title="2000">2000ko</a> <a href="/wiki/Urriaren_31" title="Urriaren 31">urriaren 31n</a> jaurti zenetik populazio egonkorra dagoen espazioan. Hainbat <a href="/wiki/Robot" title="Robot">robot</a> ere egin ditu beste planeta batzuk ikertzeko.</p><p><a href="/wiki/1800" title="1800">1800etik</a> <a href="/wiki/2000" title="2000">2000ra</a> giza populazioa 6koiztu egin zen (1.000 milioitik 6.000 milioira). <a href="/wiki/2004" title="2004">2004an</a> populazioaren&#160;%39,7a inguru (2.500 milioi) <a href="/wiki/Hiri" title="Hiri">hirietan</a> bizi ziren eta espero da <a href="/wiki/XXI._mendea" title="XXI. mendea">XXI. mendean</a> hau igoko dela. Gizakia hirietan biltzeak <a href="/wiki/Kutsadura" title="Kutsadura">poluzio</a> eta <a href="/w/index.php?title=Krime&amp;action=edit&amp;redlink=1" class="new" title="Krime (sortu gabe)">krime</a> mota batzuk sortu ditu.<sup id="cite_ref-33" class="reference"><a href="#cite_note-33">[33]</a></sup> batez ere hirien zentruan eta periferiako hainbat auzotan. Hala ere <a href="/wiki/Kultura" title="Kultura">kultura</a> handiagoa eta gosetei erresitentzia handiagoa sortu du.</p><p>Gizakien bere ingurunean oso efektu dramatikoa dute. Uste denez iraganean gizakien ehizagatik hainbat <a href="/wiki/Espezie" title="Espezie">espezie</a> galbidera eraman ziren eta horregatik gizakia <a href="/w/index.php?title=Superpredatzaile&amp;action=edit&amp;redlink=1" class="new" title="Superpredatzaile (sortu gabe)">superpredatzaile</a> gisa ezagutu ohi da.<sup id="cite_ref-34" class="reference"><a href="#cite_note-34">[34]</a></sup> Gaur egun poluzioa dela eta <a href="/wiki/Klima_aldaketa" title="Klima aldaketa">klima aldaketari</a> laguntza ematea posibleena litzateke.<sup id="cite_ref-35" class="reference"><a href="#cite_note-35">[35]</a></sup>. Klima aldaketa hauek <a href="/w/index.php?title=Holozenoko_estintzio&amp;action=edit&amp;redlink=1" class="new" title="Holozenoko estintzio (sortu gabe)">Holozenoko estintzioaren</a> arrazoietako bat litzateke; horrela jarraituz gero hurrengo mendean espezieen erdia ere desagertu ahal da.<sup id="cite_ref-36" class="reference"><a href="#cite_note-36">[36]</a></sup><sup id="cite_ref-37" class="reference"><a href="#cite_note-37">[37]</a></sup></p><h2><span class="mw-headline" id="Psikologia">Psikologia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gizaki&amp;veaction=edit&amp;section=10" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Psikologia»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Gizaki&amp;action=edit&amp;section=10" title="Aldatu atal hau: «Psikologia»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><div style="font-style: italic; padding-left: 2em; margin-bottom: 0.5em;">Sakontzeko, irakurri: «<a href="/wiki/Psikologia" title="Psikologia">Psikologia</a>» eta «<a href="/wiki/Burmuin" class="mw-redirect" title="Burmuin">Burmuin</a>»</div><p>Giza <a href="/wiki/Garun" title="Garun">burmuina</a> nerbio-sistema zentralaren erdigunea da eta zentzu guztien koordinazio lekua. Burmuinak autonomoak diren hainbat ekintza kontrolatzen ditu, hala nola arnasa-hartzea eta liseriketa. Burmuinak kontzienteak diren beste batzuk ere kontrolataen ditu, nahita eginiko ekintzak, hala nola, pentsatzea, arrazoitzea eta abstrakzioa. Prozesu hauek <a href="/wiki/Psikologia" title="Psikologia">psikologiak</a> aztertzen dituenak dira.</p><p>Giza burmuina horrelako ekintzetan oso garatua dagoela esaten da eta, beraz, adimentsuagoa dela beste espezie batzuk baino. Beste animalia batzuk tresna sinpleak sortzeko gai dira baina gizakien <a href="/wiki/Teknologia" title="Teknologia">teknologia</a> askoz garatuagoa da, aldaketa etengabeekin. Antzinako gizakiek zerabiltzaten tresnak ere edozien animaliak egiten dituenak baino askoz landuagoak ziren.</p><p><a href="/wiki/Antropologia" title="Antropologia">Antropologia</a> modernoak <a href="/wiki/Charles_Darwin" title="Charles Darwin">Charles Darwinek</a> esandakoa hartzen du oinarritzat:</p><table width="33%" style="float:left;background-color:inherit;border-collapse:collapse;border-style:none;margin: .5em .75em;"><tr><td width="20" valign="top" style="color:silver;font-size:40px;font-family:'Times New Roman',serif;font-weight:bold;text-align:left;padding:2 2px;">«</td><td align="left" style="padding:0 10px">...gizakiaren eta beste goi-animalien arteko garunen arteko ezbertsinasuna, hain da handia ezen ez da klase bat baizik eta gradu bat.</td><td width="20" valign="bottom" style="color:silver;font-size:40px;font-family:'Times New Roman',serif;font-weight:bold;text-align:right;padding:2 2px;">»</td></tr><tr><td colspan="3" style="padding-top: 10px"><p style="font-size:smaller;line-height:1em;text-align: right"><b><cite style="font-style:normal;">—Charles Darwin</cite></b></p></td></tr></table><h3><span class="mw-headline" id="Sexualitatea_eta_maitasuna">Sexualitatea eta maitasuna</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gizaki&amp;veaction=edit&amp;section=11" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Sexualitatea eta maitasuna»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Gizaki&amp;action=edit&amp;section=11" title="Aldatu atal hau: «Sexualitatea eta maitasuna»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h3><div style="font-style: italic; padding-left: 2em; margin-bottom: 0.5em;">Sakontzeko, irakurri: «<a href="/wiki/Giza_sexualitatea" class="mw-redirect" title="Giza sexualitatea">Giza sexualitatea</a>» eta «<a href="/wiki/Maitasuna" class="mw-redirect" title="Maitasuna">Maitasuna</a>»</div><p><a href="/wiki/Giza_sexualitatea" class="mw-redirect" title="Giza sexualitatea">Giza sexualitateak</a> funtzio soziala betetzen du, birsortze biologikoaz gain: intimitate fisikoa sortzen du, loturak eta gizabanakoen arteko jerarkiak. Sexuak traszendentzia izpirituala du ohitura batzuetan. Sexua, gainera, <a href="/w/index.php?title=Hedonistika&amp;action=edit&amp;redlink=1" class="new" title="Hedonistika (sortu gabe)">hedonistikaren</a> bitartez sariak lortzeko ere bidera daiteke. Gogo sexuala, edo <a href="/wiki/Libido" title="Libido">libidoa</a>, gizakiek ahalik eta azkarren sexua edukitzeko gogoa da eta maitasuna, estasia eta <a href="/wiki/Jeloskortasun" title="Jeloskortasun">jeloskortasuna</a> ekar dezake.</p><p>Sexualitateak garrantzi handia du gizakien artean eta hau <a href="/wiki/Obulazio" title="Obulazio">obulazio</a> izkutuan, <a href="/wiki/Sexu_dimorfismo" title="Sexu dimorfismo">dimorfismo sexual</a> handian (<a href="/wiki/Txinpantze" title="Txinpantze">txinpanzeekin</a> alderatua) eta emeek obulaziotik kanpo sexu-erlazioak izateko duten gaitasunean ikusten da.</p><p>Horretaz gain gizakiek sexu jarrera ugariak eta oso anitzak dituzte: <a href="/wiki/Heterosexualitate" title="Heterosexualitate">Heterosexualitateaz</a> gain <a href="/wiki/Homosexualitate" title="Homosexualitate">homosexualitatea</a> ere badago. Sexuak funtzio birsorgailuaz gain, bestelakoak ere izan ditzake eta horregatik gizonak (arrak) emakumearen (emearen) aluan egindako <a href="/wiki/Sarpen_(sexualitatea)" title="Sarpen (sexualitatea)">sarpenaz</a> gain, bi sexuen <a href="/wiki/Uzki" title="Uzki">ipurtzuloan</a> egin dezake eta <a href="/wiki/Felazio" title="Felazio">felazioa</a> zein <i><a href="/wiki/Cunnilingus" title="Cunnilingus">cunnilingus</a></i>a ere praktikatu, besteak beste. Gainera gizakiek familia-planifikazioa egiten dute eta hainbat neurri har ditzakete birsortzea lortu edo ekiditeko.</p><p>Gizakien artean eraso sexualak eta gaixotasun psikosexualak ugariak dira.</p><h2><span class="mw-headline" id="Kultura">Kultura</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gizaki&amp;veaction=edit&amp;section=12" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Kultura»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Gizaki&amp;action=edit&amp;section=12" title="Aldatu atal hau: «Kultura»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><div style="font-style: italic; padding-left: 2em; margin-bottom: 0.5em;">Sakontzeko, irakurri: «<a href="/wiki/Kultura" title="Kultura">Kultura</a>»</div><p><a href="/wiki/Kultura" title="Kultura">Kulturaren</a> definizio bat ematea zaila bada ere talde batek dituen ezberdintasun material, intelektuak, emozional eta izpiritualen multzo gisa uler daiteke, artea, literatura, bizimodua, balore sistema, ohiturak, erritualak eta siniskerak barne hartuta. Giza biologia eta giza portaera eta kulturaren arteko lotura oso estua da eta horregatik zaila egiten da bi gai horiek ezberdintzea. Horregatik gai batzuk arlo batean edo bestean jartzea konbentzioz erabaki ohi da.</p><p>Kulturaren barruan baloreak, <a href="/wiki/Arau_sozial" title="Arau sozial">arau sozialak</a> eta asmakuntzak sartzen dira. Kultura baten baloreek <a href="/wiki/Etika" title="Etika">etika</a> zer den zehazten dute. Oso loturik agertzen dira <a href="/w/index.php?title=Arau&amp;action=edit&amp;redlink=1" class="new" title="Arau (sortu gabe)">arauak</a> eta <a href="/wiki/Ohitura_(argipena)" class="mw-disambig" title="Ohitura (argipena)">ohiturak</a>. Asmakizunek hiru kontzeptu horiek biltzen dituzte.</p><p><a href="/wiki/Antropologia" title="Antropologia">Antropologiak</a> duen <a href="/wiki/Kultura" title="Kultura">kulturaren</a> ulermenak oso zaila egiten du jende askoren artean <a href="/wiki/Animalia" title="Animalia">animalia</a> bat direla gogoraraztea.</p><h3><span class="mw-headline" id="Hizkuntza">Hizkuntza</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gizaki&amp;veaction=edit&amp;section=13" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Hizkuntza»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Gizaki&amp;action=edit&amp;section=13" title="Aldatu atal hau: «Hizkuntza»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h3><div style="font-style: italic; padding-left: 2em; margin-bottom: 0.5em;">Sakontzeko, irakurri: «<a href="/wiki/Hizkuntza" title="Hizkuntza">Hizkuntza</a>»</div><div class="thumb tright"><div class="thumbinner" style="width:252px;"><a href="/wiki/Fitxategi:Surfacegyri.JPG" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/86/Surfacegyri.JPG/250px-Surfacegyri.JPG" width="250" height="225" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/86/Surfacegyri.JPG/375px-Surfacegyri.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/86/Surfacegyri.JPG/500px-Surfacegyri.JPG 2x" data-file-width="1664" data-file-height="1496" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Fitxategi:Surfacegyri.JPG" class="internal" title="Handitu"></a></div>Hizkuntzaren oinarria burmuinean dago. Hizkuntza sortzeko hainbat zonalde aktibatu behar dira.</div></div></div><p>Gizakiek kontzeptuak, ideiak eta argibideak batak-besteari hitzez eta idazten igaroarazteko duten gaitasuna ez da ikusten beste espezie guztietan. Hitz egiteko gaitasuna gizakiaren oinarrizko definizioak sartzen da. <a href="/wiki/Hizkuntza" title="Hizkuntza">Hizkuntza</a> gizakien arteko komunikazioren zentruan kokatzen da eta gainera <a href="/wiki/Nazio" title="Nazio">nazioak</a>, <a href="/wiki/Kultura" title="Kultura">kulturak</a> eta <a href="/wiki/Etnia" title="Etnia">talde etnikoak</a> batzen dituen ezaugarria da.</p><p><a href="/wiki/Idazkera_sistema" title="Idazkera sistema">Idazkeraren</a> asmakuntza <a href="/wiki/K._a._4._milurtekoa" title="K. a. 4. milurtekoa">K. a. 4. milurtekoan</a> hainbat hizkuntza objektu materialen gainean fosilizatzea ekarri zuen eta eboluzio kulturalean garrantzi handia izan zuen. Hizkuntza oso loturik dago <a href="/wiki/Ohitura_(argipena)" class="mw-disambig" title="Ohitura (argipena)">ohitura</a> eta <a href="/wiki/Erlijio" title="Erlijio">erlijioari</a>.</p><p><a href="/wiki/Hizkuntzalaritza" title="Hizkuntzalaritza">Hizkuntzalaritzak</a> hizkuntzen egitura eta euren artean dagoen harremana deskribatzen du. Gaur egun 6.000 bat hizkuntza ezberdin daude mudnuan, zeinu hizkuntzak barne, eta beste hainbat <a href="/wiki/Hizkuntza_hil" title="Hizkuntza hil">desagertutako hizkuntzak</a> dira.</p><h3><span id="Artea.2C_literatura_eta_musika"></span><span class="mw-headline" id="Artea,_literatura_eta_musika">Artea, literatura eta musika</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gizaki&amp;veaction=edit&amp;section=14" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Artea, literatura eta musika»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Gizaki&amp;action=edit&amp;section=14" title="Aldatu atal hau: «Artea, literatura eta musika»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h3><div style="font-style: italic; padding-left: 2em; margin-bottom: 0.5em;">Sakontzeko, irakurri: «<a href="/wiki/Artea" class="mw-redirect mw-disambig" title="Artea">Artea</a>», «<a href="/wiki/Literatura" title="Literatura">Literatura</a>» eta «<a href="/wiki/Musika" title="Musika">Musika</a>»</div><div class="thumb tleft"><div class="thumbinner" style="width:202px;"><a href="/wiki/Fitxategi:Junkanoo2.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a5/Junkanoo2.jpg/200px-Junkanoo2.jpg" width="200" height="150" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a5/Junkanoo2.jpg/300px-Junkanoo2.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a5/Junkanoo2.jpg/400px-Junkanoo2.jpg 2x" data-file-width="800" data-file-height="600" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Fitxategi:Junkanoo2.jpg" class="internal" title="Handitu"></a></div>Musika mundu osoan ematen den fenomenoa da eta oso erabilia da ospakizunetan</div></div></div><p>Artea gizakia gizaki denetik egon dela esan daiteke, historiaurreko artetik gaur egungo arte garaikiderarte. Artea gizakiek duten ezaugarri berezietako bat da eta beste espezieetatik bereizten du.</p><p>Gaur egun artea ulertzko modua gizakiek materialak inpultso kreatibo baten bitartez egiteko gaitasun gisa definitzen da. Artea beste lanetatik desberdintzen da ez dagoelako beharra edo behar biologiko bat asetzen, baizik eta errekreazioa.</p><p><a href="/wiki/Musika" title="Musika">Musika</a> fenomeno intuitibo natura bat da, <a href="/wiki/Erritmo" title="Erritmo">erritmoan</a>, <a href="/wiki/Harmonia" class="mw-disambig" title="Harmonia">harmonian</a> eta <a href="/wiki/Melodia" class="mw-redirect" title="Melodia">melodian</a> oinarritua. Musika entzutea baliteke unibertsiala izatea eta entretenimendu moduan ulertu ahal izatea orokorra izatea. Musika ikasi eta ulertzea, bestalde, nahiko ohikoak diren diziplinak dira. <a href="/wiki/Etnia" title="Etnia">Etnia</a> bakoitzak bere musika motak ditu eta aldakortasun handia dago.</p><p><a href="/wiki/Literatura" title="Literatura">Literatura</a>, idatzi eta, batzuetan, ahoz transmititzen den lanak, <a href="/wiki/Prosa" title="Prosa">prosa</a>, <a href="/wiki/Olerkigintza" title="Olerkigintza">olerkiak</a>, <a href="/wiki/Drama" title="Drama">drama</a> eta fikzio zein ez-fikzioa biltzen ditu.
