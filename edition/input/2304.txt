Tailandan pohjoižes.</div></div></div><p><b>Ristit</b> (<a href="/wiki/Latinan_kel%27" title="Latinan kel&#39;">latin</a>.: <i>Homo sapiens</i>) om <a href="/w/index.php?title=Ristitud_(sugu)&amp;action=edit&amp;redlink=1" class="new" title="Ristitud (sugu) (mugošt lehtpol&#39;t ei ole)">Ristitud-sugun</a> erik (Homo) <a href="/w/index.php?title=Gominidad&amp;action=edit&amp;redlink=1" class="new" title="Gominidad (mugošt lehtpol&#39;t ei ole)">Gominidad</a>-sugukundaspäi <a href="/w/index.php?title=Primatad&amp;action=edit&amp;redlink=1" class="new" title="Primatad (mugošt lehtpol&#39;t ei ole)">Primatad</a>-heimkundas, üks'jäine eläb nügüdläižen aigan. Hän om kundaline olii <a href="/w/index.php?title=Mel%27&amp;action=edit&amp;redlink=1" class="new" title="Mel&#39; (mugošt lehtpol&#39;t ei ole)">melenke</a> da <a href="/w/index.php?title=Tunde&amp;action=edit&amp;redlink=1" class="new" title="Tunde (mugošt lehtpol&#39;t ei ole)">tuntkenke</a>, istorižen töhišton da <a href="/wiki/Kul%27tur" title="Kul&#39;tur">kul'turan</a> subjekt. Ristit sündui Mal <a href="/w/index.php?title=Evol%C3%BCcii&amp;action=edit&amp;redlink=1" class="new" title="Evolücii (mugošt lehtpol&#39;t ei ole)">evolücijan processan</a> rezul'tataks — <a href="/w/index.php?title=Antropogenez&amp;action=edit&amp;redlink=1" class="new" title="Antropogenez (mugošt lehtpol&#39;t ei ole)">antropogenezas</a>, sen tedoiduz jätkustub.</p><p><b><a href="/w/index.php?title=Ristitkund&amp;action=edit&amp;redlink=1" class="new" title="Ristitkund (mugošt lehtpol&#39;t ei ole)">Ristitkund</a></b> om ristitližiden individoiden ühtmuz.</p><h2><span class="mw-headline" id="Etimologii">Etimologii</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ristit&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Redaktiruida jaguz Etimologii">redaktiruida</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ristit&amp;action=edit&amp;section=1" title="Redaktiruida jaguz Etimologii">redaktiruida purde</a><span class="mw-editsection-bracket">]</span></span></h2><p><a href="/wiki/Veps%C3%A4n_kel%27" title="Vepsän kel&#39;">Vepsän</a> sana "ristit" tuleb "<a href="/w/index.php?title=Rist&amp;action=edit&amp;redlink=1" class="new" title="Rist (mugošt lehtpol&#39;t ei ole)">rist</a>" i "risttä" vaihišpäi, sikš ende ristituikš sai kucta vaiše <a href="/w/index.php?title=Valatuz&amp;action=edit&amp;redlink=1" class="new" title="Valatuz (mugošt lehtpol&#39;t ei ole)">valatadud</a> mehid. Sid' nece nimetuz leviganzi kaikile mehile.</p><h2><span class="mw-headline" id="Erinijad_pirdad">Erinijad pirdad</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ristit&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Redaktiruida jaguz Erinijad pirdad">redaktiruida</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ristit&amp;action=edit&amp;section=2" title="Redaktiruida jaguz Erinijad pirdad">redaktiruida purde</a><span class="mw-editsection-bracket">]</span></span></h2><p>Ristitun erigoičijad päpirdad oma püštoiged kävund, šingotadud päaivod, meletandan olend i <a href="/wiki/Kel%27" title="Kel&#39;">kel'</a>. Hän voib meletada, sekoida, tehta joudjališt <a href="/w/index.php?title=Vali%C4%8Dend&amp;action=edit&amp;redlink=1" class="new" title="Valičend (mugošt lehtpol&#39;t ei ole)">valičendad</a>, otta vastusenpidandad tegos, pidada moralid meles. Ristit om kuvazumatoi londusen olendale, hänen organad da palad ei voigoi eläda kaičendata avoinudes ümbrištos živatoiden erineden. Vastkarin, mehed mahttas säta i kävutada <a href="/wiki/L%C3%A4moi" title="Lämoi">lämoid</a>, azegid da tökaluid.</p><p><a href="/wiki/Ristitun_hibj" title="Ristitun hibj">Ristitun hibj</a> erineb oblezjanaspäi:</p><ul><li>jaugad pit'kemb käzid</li><li>sel'grodan S-vuitte form</li><li>lapak rindstai</li><li>bembel'vuitte laba surenke barbhanke, toižed barbhad penemb, kaik barbhad oma lühüdad</li><li>peiglon olend käzivardel</li><li>väll' karvkate</li><li>järedad päaivod, 3..4 kerdad suremb</li><li>sor'miden korväččed pölused</li></ul><table class="metadata plainlinks mbox-small" style="border:1px solid #aaa;
