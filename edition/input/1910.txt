id="firstHeading" class="firstHeading" lang="nn">Atmosfære</h1>			<div id="bodyContent" class="mw-body-content">				<div id="siteSub" class="noprint">Frå Wikipedia – det frie oppslagsverket</div>				<div id="contentSub"></div>								<div id="jump-to-nav" class="mw-jump">					Gå til:					<a href="#mw-head">navigering</a>, 					<a href="#p-search">søk</a>				</div>				<div id="mw-content-text" lang="nn" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><dl><dd><div><i>Denne artikkelen handlar om gasslaget kring ein planet. For måleininga, sjå <a href="/wiki/Standardatmosf%C3%A6re" title="Standardatmosfære">standardatmosfære</a>.</i></div></dd></dl><div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Fil:Himmel_ueber_dem_Bielersee03.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/05/Himmel_ueber_dem_Bielersee03.jpg/220px-Himmel_ueber_dem_Bielersee03.jpg" width="220" height="165" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/05/Himmel_ueber_dem_Bielersee03.jpg/330px-Himmel_ueber_dem_Bielersee03.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/05/Himmel_ueber_dem_Bielersee03.jpg/440px-Himmel_ueber_dem_Bielersee03.jpg 2x" data-file-width="2272" data-file-height="1704" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Fil:Himmel_ueber_dem_Bielersee03.jpg" class="internal" title="Forstørr"></a></div><a href="/wiki/Jordatmosf%C3%A6ren" title="Jordatmosfæren">Jordatmosfæren</a> gjer at <a href="/wiki/Himmel" title="Himmel">himmelen</a> ser <a href="/wiki/Bl%C3%A5" title="Blå">blå</a> ut om dagen og <a href="/wiki/Raud" title="Raud">raud</a> når sola står i horisonten. <i>(Foto: Bernhard Witz)</i></div></div></div><p><b>Atmosfære</b> er det generelle namnet på eit lag med <a href="/wiki/Gass" title="Gass">gass</a> som kan ligge rundt ein lekam med stor nok <a href="/wiki/Masse" title="Masse">masse</a>. Gassane er tiltrekt av <a href="/wiki/Tyngdekrafta" class="mw-redirect" title="Tyngdekrafta">tyngdekrafta</a> til lekamen, og halde fast viss tyngdekrafta er stor nok og atmosfæretemperaturen er låg. Nokre planetar består hovudsakleg av diverse gassar, og har dermed svært djupe atmosfærar (sjå <a href="/w/index.php?title=Gasskjempe&amp;action=edit&amp;redlink=1" class="new" title="Gasskjempe (sida finst ikkje)">gasskjempe</a>).</p><p><a href="/wiki/Jorda" title="Jorda">Jorda</a>, <a href="/wiki/Planeten_Venus" title="Planeten Venus">Venus</a>, <a href="/wiki/Planeten_Mars" title="Planeten Mars">Mars</a>, <a href="/wiki/Planeten_Pluto" class="mw-redirect" title="Planeten Pluto">Pluto</a> og to av <a href="/wiki/Naturleg_satellitt" title="Naturleg satellitt">månane</a> til dei ytre planetane (<a href="/wiki/Saturnm%C3%A5nen_Titan" title="Saturnmånen Titan">saturnmånen Titan</a> og <a href="/wiki/Neptunm%C3%A5nen_Triton" title="Neptunmånen Triton">neptunmånen Triton</a>) har atmosfærar. I tillegg er kjempeplanetane i det ytre solsystemet (<a href="/wiki/Planeten_Jupiter" title="Planeten Jupiter">Jupiter</a>, <a href="/wiki/Planeten_Saturn" title="Planeten Saturn">Saturn</a>, <a href="/wiki/Planeten_Uranus" title="Planeten Uranus">Uranus</a> og <a href="/wiki/Planeten_Neptun" title="Planeten Neptun">Neptun</a>) hovudsakleg samansett av gassar. Andre lekamar i <a href="/wiki/Solsystemet" title="Solsystemet">solsystemet</a> har ekstremt tynne atmosfærar: <a href="/wiki/M%C3%A5nen" title="Månen">Månen</a> (<a href="/wiki/Natrium" title="Natrium">natriumgass</a>), <a href="/wiki/Planeten_Merkur" title="Planeten Merkur">Merkur</a> (<a href="/wiki/Natrium" title="Natrium">natriumgass</a>), <a href="/wiki/Jupiterm%C3%A5nen_Europa" title="Jupitermånen Europa">jupitermånen Europa</a> (<a href="/wiki/Oksygen" title="Oksygen">oksygen</a>) og <a href="/wiki/Jupiterm%C3%A5nen_Io" title="Jupitermånen Io">jupitermånen Io</a> (<a href="/wiki/Svovel" title="Svovel">svovel</a>).</p><p><a href="/wiki/Jordatmosf%C3%A6ren" title="Jordatmosfæren">Jordatmosfæren</a> er ein særskild atmosfære fordi han inneheld gassar skapt av <a href="/wiki/Liv" class="mw-disambig" title="Liv">livet</a> på jorda; samstundes gjer han det mogleg for liv å vera til her.</p><div class="spire" style="clear: both;"><a href="/wiki/Fil:P_space.png" class="image" title="Spire"><img alt="Spire" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8b/P_space.png/20px-P_space.png" width="20" height="18" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8b/P_space.png/30px-P_space.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8b/P_space.png/40px-P_space.png 2x" data-file-width="77" data-file-height="68" /></a> Denne <a href="/wiki/Kategori:Astronomispirer" title="Kategori:Astronomispirer">astronomiartikkelen</a> er ei <b><a href="/wiki/Wikipedia:Spire" title="Wikipedia:Spire">spire</a></b>.
