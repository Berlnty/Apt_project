OK</a></span></td></tr></tbody></table><p><b>Ihminen</b> (<i>Homo sapiens</i>) on ainoa nykyisin elossa oleva <a href="/wiki/Ihmiset" title="Ihmiset">ihmisten</a> (<i>Homo</i>) sukuun kuuluva <a href="/wiki/Nis%C3%A4kk%C3%A4%C3%A4t" title="Nisäkkäät">nisäkäs</a>. Ihminen kuuluu <a href="/wiki/K%C3%A4delliset" title="Kädelliset">kädellisten</a> (<i>Primates</i>) lahkon <a href="/wiki/Isot_ihmisapinat" title="Isot ihmisapinat">isojen ihmisapinoiden</a> (<i>Hominidae</i>) heimoon ja on nykyisin elävistä kädellisistä laajimmalle levinnyt, <a href="/wiki/%C3%84lykkyys" title="Älykkyys">älykkäin</a> ja runsaslukuisin laji. Ihmisiä on noin 7,5 miljardia. Ihmisellä on kehittyneet <a href="/wiki/Ihmisaivot" title="Ihmisaivot">aivot</a>, joiden ansiosta ihminen kykenee <a href="/wiki/Abstrakti" class="mw-disambig" title="Abstrakti">abstraktiin</a> <a href="/wiki/Ajattelu" title="Ajattelu">ajatteluun</a>, <a href="/wiki/Kieli" title="Kieli">kieleen</a> ja itsetarkkailuun. Tämä on yhdessä pystyasennon ja <a href="/wiki/Esine" title="Esine">esineiden</a> käsittelyyn kykenevien <a href="/wiki/Yl%C3%A4raaja" title="Yläraaja">yläraajojen</a> kanssa antanut ihmiselle kyvyn käyttää <a href="/wiki/Ty%C3%B6kalu" title="Työkalu">työkaluja</a> monipuolisemmin kuin mikään toinen eläinlaji.</p><p>Ihmiset ovat muiden kädellisten tapaan luonnostaan <a href="/wiki/Yhteis%C3%B6" title="Yhteisö">yhteisöllisiä</a>, mutta ihmiset ovat erityisen kyvykkäitä käyttämään <a href="/wiki/Viestint%C3%A4" title="Viestintä">viestintää</a> ja kieltä <a href="/wiki/Idea" title="Idea">ideoiden</a> välittämiseen sekä menneisyyden ja tulevaisuuden pohtimiseen. Ihmiset muodostavat monimutkaisia yhteistyötä tekeviä ja <a href="/wiki/Kilpailu" title="Kilpailu">kilpailevia</a> sosiaalisia ryhmiä, <a href="/wiki/Perhe" title="Perhe">perheistä</a> aina <a href="/wiki/Kansakunta" title="Kansakunta">kansakuntiin</a> asti. Ihmisyhteisön perustan muodostavat kielen kautta välittyvät <a href="/wiki/Rituaali" title="Rituaali">rituaalit</a>, <a href="/wiki/Traditio" class="mw-redirect" title="Traditio">traditiot</a>, <a href="/wiki/Arvo" title="Arvo">arvot</a> ja <a href="/wiki/Laki" title="Laki">lait</a>. Ihmiset tekevät ainoana lajina <a href="/wiki/Varhaisihmisen_tulenk%C3%A4ytt%C3%B6" title="Varhaisihmisen tulenkäyttö">tulen</a>, <a href="/wiki/Ruoanlaitto" title="Ruoanlaitto">kypsentävät ruoan</a>, <a href="/wiki/Pukeutuminen" title="Pukeutuminen">pukeutuvat</a> ja käyttävät lukuisia <a href="/wiki/Tekniikka" title="Tekniikka">tekniikoita</a>. Ihmisellä on myös <a href="/wiki/Taide" title="Taide">taiteen</a> tapaisia <a href="/wiki/Kulttuuri" title="Kulttuuri">kulttuuri</a>-<a href="/wiki/Innovaatio" title="Innovaatio">innovaatioita</a>, kuten <a href="/wiki/Maalaustaide" title="Maalaustaide">maalaustaide</a>, <a href="/wiki/Arkkitehtuuri" title="Arkkitehtuuri">arkkitehtuuri</a>, <a href="/wiki/Kirjallisuus" title="Kirjallisuus">kirjallisuus</a> ja <a href="/wiki/Musiikki" title="Musiikki">musiikki</a>.</p><div id="toc" class="toc"><div class="toctitle" lang="fi" class="toctext">Aiheesta muualla</span></a></li></ul></div><h2><span class="mw-headline" id="Kehitys">Kehitys</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Muokkaa osiota Kehitys">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=1" title="Muokkaa osiota Kehitys">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h2><h3><span class="mw-headline" id="Suku">Suku</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Muokkaa osiota Suku">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=2" title="Muokkaa osiota Suku">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Ihmisen_evoluutio" title="Ihmisen evoluutio">Ihmisen evoluutio</a></i></div></dd></dl><p>Ihmislaji kuuluu <a href="/wiki/Ihmiset" title="Ihmiset">ihmisten</a> (<i>Homo</i>) sukuun, joka on kehittynyt <a href="/wiki/Ihmisapinat" title="Ihmisapinat">ihmisapinoista</a> (<i>Hominidae</i>) ja <a href="/wiki/K%C3%A4delliset" title="Kädelliset">kädellisistä</a> (<i>Primates</i>). Lähimmät ihmislajin elossa olevat sukulaiset ovat <a href="/wiki/Simpanssi" title="Simpanssi">simpanssi</a> ja <a href="/wiki/Bonobo" title="Bonobo">bonobo</a>. Simpanssien kehityslinja erkani ihmisistä noin 5–6 miljoonaa vuotta sitten ja <a href="/wiki/Gorilla" title="Gorilla">gorillojen</a> noin 7–8 miljoonaa vuotta sitten.</p><h3><span class="mw-headline" id="Nykyihminen">Nykyihminen</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Muokkaa osiota Nykyihminen">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=3" title="Muokkaa osiota Nykyihminen">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><div class="thumb tright"><div class="thumbinner" style="width:252px;"><a href="/wiki/Tiedosto:Map-of-human-migrations.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/37/Map-of-human-migrations.jpg/250px-Map-of-human-migrations.jpg" width="250" height="179" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/37/Map-of-human-migrations.jpg/375px-Map-of-human-migrations.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/37/Map-of-human-migrations.jpg/500px-Map-of-human-migrations.jpg 2x" data-file-width="889" data-file-height="635" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Tiedosto:Map-of-human-migrations.jpg" class="internal" title="Suurenna"></a></div>Kartta varhaisista ihmisen migraatioista <a href="/wiki/Mitokondrio" title="Mitokondrio">mitokondrioiden</a> perusteella  (tuhatta vuotta ennen nykyhetkeä).</div></div></div><p><a href="/wiki/Taksonomia" title="Taksonomia">Taksonomisen luokittelun</a> mukaan nykyisin elävät ihmiset kuuluvat <i>Homo sapiens</i> -lajin alalajiin <i>Homo sapiens sapiens</i>. Laji ja alalaji on ainoa ihmisten suvun elossaoleva, muiden suvun jäsenten kuoltua sukupuuttoon.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup></p><p>Nykytiedon (v. 2017) mukaan nykyihminen kehittyi Afrikassa noin 300&#160;000 vuotta sitten <a href="/wiki/Pleistoseeni" title="Pleistoseeni">myöhäispleistoseenisella aikakaudella</a>. Varhaisimmat löydöt ovat <a href="/wiki/Marokko" title="Marokko">Marokosta</a> <a href="/wiki/Jebel_Irhoud" class="mw-redirect" title="Jebel Irhoud">Jebel Irhoudista</a> löydetyt noin 300&#160;000 vuoden ikäisiksi ajoitetut fossiilit, jotka edustavat nykyihmisen varhaisinta kehitysvaihetta. Niiden aivokoppa tosin vaikuttaa hiukan nykyisin elävien ihmisten aivokoppaa alkeellisemmalta.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup> Ennen vuotta 2017 vanhimpina nykyihmisen jäännöksinä pidettiin Etiopiasta vuonna 1967 löydettyjä kalloja <i>Omo I</i> ja <i>Omo II</i>, jotka on ajoitettu noin 195&#160;000 vuotta vanhoiksi.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup></p><p>Yleisesti vanhentuneeksi katsotun <a href="/wiki/Monialuehypoteesi" title="Monialuehypoteesi">monialuehypoteesin</a> mukaan ihmisiä olisi kehittynyt maapallolla samanaikaisesti monella alueella.<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">&#91;6&#93;</a></sup> Ihmisten DNA-perimä verrattuna muihin samaan aikaan eläneisiin lajeihin kuitenkin viittaa myöhäisellä <a href="/wiki/Pleistoseeni" title="Pleistoseeni">pleistoseenikaudella</a> ihmispopulaation olleen erittäin pieni, maksimissaan 10&#160;000 yksilöä, mikä johti ihmisten pieneen <a href="/wiki/Geenipooli" title="Geenipooli">geenipooliin</a>.<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">&#91;7&#93;</a></sup> Ihmisten evoluutiossa tärkeässä asemassa oli kasvanut aivotilavuus, joka on tyypillisesti aikuisilla nykyihmisillä noin 1&#160;400&#160;cm³. Kasvanut älykkyys ja kehittyneet kommunikointitaidot johtivat ihmiskunnan nopeaan sosiaaliseen ja kulttuurilliseen kehitykseen.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">&#91;8&#93;</a></sup><sup id="cite_ref-9" class="reference"><a href="#cite_note-9">&#91;9&#93;</a></sup></p><h4><span id="Levitt.C3.A4ytyminen_koko_maapallolle"></span><span class="mw-headline" id="Levittäytyminen_koko_maapallolle">Levittäytyminen koko maapallolle</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Muokkaa osiota Levittäytyminen koko maapallolle">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=4" title="Muokkaa osiota Levittäytyminen koko maapallolle">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h4><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Nykyihmisen_levitt%C3%A4ytyminen_maapallolla" title="Nykyihmisen levittäytyminen maapallolla">Nykyihmisen levittäytyminen maapallolla</a></i></div></dd></dl><p>Lähi-idästä on löytöjä nykyihmisen varhaisesta asutuksesta 125&#160;000 vuoden takaa.<sup id="cite_ref-10" class="reference"><a href="#cite_note-10">&#91;10&#93;</a></sup> Aikaisemmin ajateltiinkin, että ihmiset siirtyivät tuohon aikaan ensi kerran pois Afrikan mantereelta.<sup id="cite_ref-:0_11-0" class="reference"><a href="#cite_note-:0-11">&#91;11&#93;</a></sup> Kuitenkin vuonna 2018 julkaistiin tutkimus, jonka mukaan ihmiset levittäytyivät Afrikan ulkopuolelle jo 174 000–188 000 vuotta sitten.<sup id="cite_ref-:0_11-1" class="reference"><a href="#cite_note-:0-11">&#91;11&#93;</a></sup> Nykyihmiset palasivat alueelle useita kertoja, ja läpimurto sieltä ulos tapahtui noin 60&#160;000 vuotta sitten. Nykyihmiset levittäytyivät Etelä-Aasiaan noin 50&#160;000 vuotta sitten ja saapuivat Australiaan viimeistään 40&#160;000 vuotta sitten.<sup id="cite_ref-12" class="reference"><a href="#cite_note-12">&#91;12&#93;</a></sup> Euroopasta on löytynyt nykyihmisen jäänteitä 41 000–45 000 vuoden takaa.<sup id="cite_ref-13" class="reference"><a href="#cite_note-13">&#91;13&#93;</a></sup> Viimeisimpänä ihminen saapui <a href="/wiki/Alaska" title="Alaska">Alaskan</a> kautta Aasiasta Amerikkaan 10&#160;000–30&#160;000 vuotta sitten.</p><p>Varhaisimmat ihmiset olivat keräilijä-metsästäjiä, joiden elämäntyyli oli sopeutunut savannien elämään. Myöhemmät ihmiset omaksuivat maanviljelyn ja vakituiset asuinpaikat. Ihmiset voivat asua useimmissa paikoissa maailmaa. Asutusta on nykyisin kaikilla mantereilla, joskaan Etelämantereella se ei ole vakituista<sup id="cite_ref-IUCN_1-1" class="reference"><a href="#cite_note-IUCN-1">&#91;1&#93;</a></sup>. Ihminen on vieraillut <a href="/wiki/Avaruus" title="Avaruus">avaruudessa</a>, ja <a href="/wiki/Kansainv%C3%A4linen_avaruusasema" title="Kansainvälinen avaruusasema">kansainvälisellä avaruusasemalla</a> on ihmisiä jatkuvasti<sup id="cite_ref-IUCN_1-2" class="reference"><a href="#cite_note-IUCN-1">&#91;1&#93;</a></sup>. Suurin osa ihmisistä, 61 prosenttia, elää <a href="/wiki/Aasia" title="Aasia">Aasiassa</a>. Loput ovat jakautuneet maapallolle seuraavasti: ihmisistä 14&#160;% on <a href="/wiki/Amerikan_manner" title="Amerikan manner">Amerikassa</a>, 13&#160;% <a href="/wiki/Afrikka" title="Afrikka">Afrikassa</a>, 12&#160;% <a href="/wiki/Eurooppa" title="Eurooppa">Euroopassa</a> ja 0,5&#160;% <a href="/wiki/Oseania" title="Oseania">Oseaniassa</a>.</p><p>Ihmispopulaatio on viimeisenä kahtena vuosisatana kasvanut miljardista seitsemään miljardiin henkeen. 2,5 miljardia ihmistä (39,7&#160;%) elää <a href="/wiki/Kaupunki" title="Kaupunki">kaupungeissa</a> tai niiden välittömässä läheisyydessä.<sup><span style="background: #FAD6A5;"><i><a href="/wiki/Wikipedia:Merkitse_l%C3%A4hteet" title="Wikipedia:Merkitse lähteet">lähde?</a></i></span></sup></p><h2><span class="mw-headline" id="Ihmisen_biologia">Ihmisen biologia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=5" class="mw-editsection-visualeditor" title="Muokkaa osiota Ihmisen biologia">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=5" title="Muokkaa osiota Ihmisen biologia">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h2><p>Ihminen on monisoluinen <a href="/wiki/Aitotumaiset" title="Aitotumaiset">aitotumallinen</a> eliö, joka koostuu noin 100 biljoonasta <a href="/wiki/Solu" title="Solu">solusta</a> ja noin 70-prosenttisesti vedestä. Ihmisyksilön kehitys alkaa, kun <a href="/wiki/Siitti%C3%B6" title="Siittiö">siittiö</a> <a href="/wiki/Hedelm%C3%B6itys" title="Hedelmöitys">hedelmöittää</a> munasolun. Sen jakautuessa muodostuu <a href="/wiki/Alkio" title="Alkio">alkio</a>, joka muuttuu <a href="/wiki/Siki%C3%B6" title="Sikiö">sikiöksi</a> kahdeksannella viikolla hedelmöityksestä. Täysikasvuisuuden ihminen saavuttaa <a href="/wiki/Murrosik%C3%A4" title="Murrosikä">murrosiän</a> päättyessä. Merkittäviä orgaanisia järjestelmiä ihmisessä ovat <a href="/wiki/Ihmisen_luuranko" title="Ihmisen luuranko">luuranko</a>, <a href="/wiki/Hermot" class="mw-redirect" title="Hermot">hermoverkosto</a>, <a href="/wiki/Verenkierto" title="Verenkierto">verenkierto</a>, <a href="/wiki/Ihmisen_lihaksisto" title="Ihmisen lihaksisto">lihaksisto</a> ja <a href="/wiki/Hengitys" title="Hengitys">hengitysjärjestelmä</a>. Sukupuolia ihmisellä on kaksi, <a href="/wiki/Mies" title="Mies">mies</a> ja <a href="/wiki/Nainen" title="Nainen">nainen</a>.</p><h3><span class="mw-headline" id="Anatomia_ja_fysiologia">Anatomia ja fysiologia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=6" class="mw-editsection-visualeditor" title="Muokkaa osiota Anatomia ja fysiologia">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=6" title="Muokkaa osiota Anatomia ja fysiologia">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Ihmisen_anatomia" title="Ihmisen anatomia">Ihmisen anatomia</a></i></div></dd></dl><div class="thumb tright"><div class="thumbinner" style="width:252px;"><a href="/wiki/Tiedosto:Skeleton_diagram.svg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/42/Skeleton_diagram.svg/250px-Skeleton_diagram.svg.png" width="250" height="353" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/42/Skeleton_diagram.svg/375px-Skeleton_diagram.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/42/Skeleton_diagram.svg/500px-Skeleton_diagram.svg.png 2x" data-file-width="744" data-file-height="1052" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Tiedosto:Skeleton_diagram.svg" class="internal" title="Suurenna"></a></div>Ihmisen luuranko.</div></div></div><p>Ihmisvartalojen tyyppejä on useita, ja niiden koko ja ominaisuudet määräytyvät geeniperimän ja ympäristötekijöiden, kuten <a href="/wiki/Ruokavalio" title="Ruokavalio">ruokavalion</a> ja liikunnan, mukaan. Aikuisen ihmisen pituus vaihtelee keskimäärin 150&#160;cm:n ja 200&#160;cm:n välillä. Naiset ovat keskimäärin miehiä pienempiä. Keskipituuksissa on myös alueellista eroavaisuutta. Suomalaisten keskipituus on miehillä 178,5 cm ja naisilla 168,0 cm vuonna 2014 tehdyn tutkimuksen mukaan. Suomalaiset ovat maailmantilastossa sijoilla 16. miehissä ja 13. naisissa.<sup id="cite_ref-14" class="reference"><a href="#cite_note-14">&#91;14&#93;</a></sup> Ihmisen <a href="/wiki/Tiheys" title="Tiheys">tiheys</a> on suurin piirtein sama kuin veden; luu- ja lihaskudos ovat vettä tiheämpiä ja rasvakudos vettä harvempaa. Rinta- ja vatsaontelo pienentävät keskitiheyttä.<sup id="cite_ref-15" class="reference"><a href="#cite_note-15">&#91;15&#93;</a></sup></p><p>Ihmiset ovat omaksuneet pystyasentoisen liikkumistavan ja pystyvät näin täysin kaksiraajaiseen liikkumiseen. Siten yläraajat vapautuvat käsittelemään muita esineitä. Koska fysiologia ei ole vielä täysin kehittynyt pystyasentoiseen liikkumiseen, ihmisten liikkumismuoto tuottaa heille vaikeuksia erityisesti myöhemmässä iässä, ja vammojen sattuessa kriittisiin kohtiin voi liikkumiskyky kadota hetkellisesti täysin.</p><p>Ihmisellä ruumiin karvoitus on vähentynyt huomattavasti. Osaltaan joillain yksilöillä karvoituksen vähentyminen on enemmän näennäistä kuin todellista. Heillä karvoja on yhtä tiheässä kuin muillakin ihmisapinoilla, mutta ne ovat lyhyempiä ja huomaamattomampia, koska ne ovat vähäpigmenttisempiä. Joidenkin alueiden ihmisillä karvoitus on vähentynyt huomattavasti voimakkaammin toisten alueiden ihmisiin verrattuna. Karvoituksen puutetta on pyritty selittämään erilaisilla teorioilla. Karvattomuutta on pidetty etenkin metsästyksen aikana tarvittavana jäähdytysjärjestelmänä ja myös ulkoloisten määrää vähentävänä. Myös seksuaalivalinnalla ja muilla sosiaalisilla suhteilla lienee osuutensa karvattomuuteen.<sup id="cite_ref-16" class="reference"><a href="#cite_note-16">&#91;16&#93;</a></sup></p><p>Toisaalta muun ihon vähäisen karvapeitteen vastapainoksi useimmilla ihmisillä on päälaellaan karvat&#160;– <a href="/wiki/Hiukset" title="Hiukset">hiukset</a>&#160;– jotka kasvavat peräti pidemmiksi kuin minkään muun eläinlajin karvat. Tosin joillakin tietyn perimän omaavilla ihmisillä hiukset jäävät lyhyeksi ja hyvin kiharaiseksi, lisäksi monet miehet vanhemmiten <a href="/wiki/Kalju" title="Kalju">kaljuuntuvat</a>. Ihmisen päälaen karvoituksen huikea pituus on ollut arvoitus tieteilijöille. On arveltu, että pitkät hiukset olisivat olleet varhaisina aikoina tärkeitä seksuaalisen valinnan kannalta, vaikka myöhemmin monien alueiden kulttuuri on kehittynyt siihen suuntaan, ettei hiusten vapaata kasvua pidetä jommallakummalla tai molemmilla sukupuolilla toivottavana.</p><p>Ihmisten ihon ja hiusten väri johtuu kudosten väriaineesta eli <a href="/wiki/Pigmentti" title="Pigmentti">pigmentistä</a>, johon ihmisillä ja muilla eläimillä vaikuttaa <a href="/wiki/Melaniini" title="Melaniini">melaniini</a>. <a href="/wiki/Ihmisen_ihonv%C3%A4ri" title="Ihmisen ihonväri">Ihonväri</a> vaihtelee tummanruskeasta vaaleanpunaiseen. Ihonväri on kehittynyt suojaamaan Auringon haitalliselta <a href="/wiki/Ultraviolettis%C3%A4teily" title="Ultraviolettisäteily">UV-säteilyltä</a> ja <a href="/wiki/Foolihappo" title="Foolihappo">foolihapon</a> hajoamiselta, mutta sallimaan <a href="/wiki/D-vitamiini" title="D-vitamiini">D-vitamiinin</a> muodostumisen iholla auringonvalon vaikutuksesta. Tyypillistä on, että pohjoisten seutujen ihmisten iho on vaalea, ja päiväntasaajaa kohti mentäessä populaation keskimääräinen ihonväri tummuu. <a href="/wiki/Tukka" class="mw-redirect" title="Tukka">Tukan</a> väri voi olla vaalea, tumma, ruskea, punainen tai niiden sekoitus. Musta ja ruskea lienevät tavallisimmat, luonnostaan punaiset harvinaisimmat, ja lähinnä eurooppalaisten keskuudessa esiintyvä ominaisuus. Ihmisten iholla on myös ominaisuus <a href="/wiki/Rusketus" title="Rusketus">ruskettua</a> auringonpaisteessa suojautuakseen ultraviolettisäteilyltä. Ruskettuminen on sitä voimakkaampaa mitä vaaleampi iho ihmisellä on luonnostaan.</p><p>Aikuisen ihmisen <a href="/wiki/Nukkuminen" title="Nukkuminen">unentarve</a> on keskimäärin 7–8 tuntia ja 3–18-vuotiaan lapsen 9–12 tuntia vuorokaudessa, riippuen iästä. Alle kolmevuotiaiden unentarve on suurin. Vanhemmat ihmiset <a href="/wiki/Nukkuminen" title="Nukkuminen">nukkuvat</a> vain 6–7 tuntia päivässä.<sup><a href="/wiki/Ohje:Kerro_kuka_tekee" title="Ohje:Kerro kuka tekee"><i>kenen mukaan?</i></a></sup> Univaje heikentää ihmisen suorituskykyä.</p><p>Normaalisti ihmisellä on 46 <a href="/wiki/Kromosomi" title="Kromosomi">kromosomia</a> (23 paria) jokaisessa solussa, joissa on valmistuneiden kartoitusten mukaan noin 3,9 miljardia ”emäspaikkaa”. Mahdollisia <a href="/wiki/Geeni" title="Geeni">geenejä</a> ihmisellä on noin 20&#160;000. Human Genome -projekti on viime aikoina tehnyt merkittävää työtä kartoituksessa. Ihmisen <a href="/wiki/DNA" title="DNA">DNA</a>-perimä on noin 96-prosenttisesti<sup id="cite_ref-17" class="reference"><a href="#cite_note-17">&#91;17&#93;</a></sup> sama kuin <a href="/wiki/Simpanssi" title="Simpanssi">simpanssin</a>, joka on ihmisen lähimpiä elossa olevia sukulaisia. Erotuksena kaikista muista ”tavallisista” kromosomeista eli autosomeista, sukupuolikromosomi määrää ihmisen sukupuolen, kuten muillakin nisäkkäillä.</p><h3><span id="Syntym.C3.A4.2C_kasvu.2C_kehitys.2C_vanheneminen.2C_kuolema"></span><span class="mw-headline" id="Syntymä,_kasvu,_kehitys,_vanheneminen,_kuolema">Syntymä, kasvu, kehitys, vanheneminen, kuolema</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=7" class="mw-editsection-visualeditor" title="Muokkaa osiota Syntymä, kasvu, kehitys, vanheneminen, kuolema">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=7" title="Muokkaa osiota Syntymä, kasvu, kehitys, vanheneminen, kuolema">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Ihmisen samoin kuin muidenkin organismien elämään kuuluu <a href="/wiki/Synnytys" title="Synnytys">syntymä</a>, <a href="/w/index.php?title=Kasvu&amp;action=edit&amp;redlink=1" class="new" title="Kasvu (sivua ei ole)">kasvu</a>, <a href="/wiki/Vanheneminen" title="Vanheneminen">vanheneminen</a> ja lopulta väistämätön <a href="/wiki/Kuolema" title="Kuolema">kuolema</a>.</p><h4><span id="Syntym.C3.A4"></span><span class="mw-headline" id="Syntymä">Syntymä</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=8" class="mw-editsection-visualeditor" title="Muokkaa osiota Syntymä">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=8" title="Muokkaa osiota Syntymä">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h4><div class="thumb tright"><div class="thumbinner" style="width:252px;"><a href="/wiki/Tiedosto:HumanNewborn.JPG" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/38/HumanNewborn.JPG/250px-HumanNewborn.JPG" width="250" height="170" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/38/HumanNewborn.JPG/375px-HumanNewborn.JPG 1.5x, href="/wiki/Tiedosto:HumanNewborn.JPG" class="internal" title="Suurenna"></a></div>Vastasyntynyt lapsi.</div></div></div><p>Naisen sisällä oleva <a href="/wiki/Munasolu" title="Munasolu">munasolu</a> hedelmöittyy yleensä miehen ja naisen välisessä <a href="/wiki/Yhdynt%C3%A4" title="Yhdyntä">yhdynnässä</a> miehen <a href="/wiki/Sperma" class="mw-redirect" title="Sperma">spermalla</a>, jolloin nainen tulee <a href="/wiki/Raskaus" title="Raskaus">raskaaksi</a>. Tosin nykyisin <a href="/wiki/L%C3%A4%C3%A4ketiede" title="Lääketiede">lääketiede</a> on kehittynyt huimasti, ja ihmiset hallitsevat useita tekniikoita, kuten <a href="/wiki/Keinohedelm%C3%B6itys" class="mw-redirect" title="Keinohedelmöitys">keinohedelmöityksen</a> ja muita perinteisistä tavoista poikkeavia tapoja saattaa nainen raskaaksi. Hedelmöittynyt munasolu eli tsygootti jakautuu naisen kohdussa muodostaen alkion, josta myöhemmin kehittyy sikiö. Lapsi tulee tavallisesti ulos naisen <a href="/wiki/Kohtu" title="Kohtu">kohdusta</a> <a href="/wiki/Em%C3%A4tin" title="Emätin">emättimen</a> kautta, ja tällöin terve vastasyntynyt osaa hengittää itsenäisesti. Joissakin tapauksissa lapsi syntyy <a href="/wiki/Keisarileikkaus" title="Keisarileikkaus">keisarileikkauksella</a>. Moderneissa kulttuureissa syntynyttä lasta pidetään henkilönä, ja sitä suojaavat kaikki samat lait kuin täysikasvuisiakin ihmisiä. Joidenkin maiden lait pitävät jo sikiötä henkilönä sen vielä ollessa naisen kohdun sisässä. Kaikissa maissa <a href="/wiki/Abortti" title="Abortti">abortti</a> eli raskaudenkeskeytys ei ole laillinen.</p><p>Muihin nisäkäslajeihin verrattuna ihmisen syntymä on kohtalaisen monimutkainen tapahtuma, mikä johtuu sikiön suuresta päästä. Synnytys voi kestää jopa päivän verran ja saattaa olla hyvin kivulias äidille. Joskus äiti, lapsi tai molemmat kuolevat synnytyksessä tai sen yhteydessä. Lapsikuolleisuus on vähentynyt merkittävästi varakkaissa maissa lääketieteen kehityttyä 1900-luvulla. Luonnollinen synnytys ilman puhtaita olosuhteita ja ammattitaitoista avustusta on vielä yleistä kehitysmaissa.</p><p>Ihmislapsi syntyy noin yhdeksän kuukautta kestäneen raskausajan jälkeen. Varakkaissa maissa täysiaikaisen vastasyntyneen paino on noin 3–4&#160;kg ja pituus 45–60&#160;cm.</p><h4><span id="Lapsuus_ja_kasvuik.C3.A4"></span><span class="mw-headline" id="Lapsuus_ja_kasvuikä">Lapsuus ja kasvuikä</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=9" class="mw-editsection-visualeditor" title="Muokkaa osiota Lapsuus ja kasvuikä">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=9" title="Muokkaa osiota Lapsuus ja kasvuikä">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h4><div class="thumb tright"><div class="thumbinner" style="width:252px;"><a href="/wiki/Tiedosto:Turku_main_library,_new_part,_children%27s_section.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/49/Turku_main_library%2C_new_part%2C_children%27s_section.jpg/250px-Turku_main_library%2C_new_part%2C_children%27s_section.jpg" width="250" height="306" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/49/Turku_main_library%2C_new_part%2C_children%27s_section.jpg/375px-Turku_main_library%2C_new_part%2C_children%27s_section.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/49/Turku_main_library%2C_new_part%2C_children%27s_section.jpg/500px-Turku_main_library%2C_new_part%2C_children%27s_section.jpg 2x" data-file-width="1045" data-file-height="1280" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Tiedosto:Turku_main_library,_new_part,_children%27s_section.jpg" class="internal" title="Suurenna"></a></div>Lapsia Turun pääkirjastossa.</div></div></div><p>Lapsi kasvaa syntymää seuraavina kolmena elinvuotenaan noin puoli metriä, jonka jälkeen pituuskasvu tasoittuu noin 4–7&#160;cm vuosikasvuksi noin 7–12 seuraavan ikävuoden ajaksi, jonka jälkeen kasvu murrosiässä tilapäisesti kiihtyy sukupuolihormonien vauhdittamana, ja lopulta hidastuu ja ajan myötä päättyy. Suomalaiset tytöt kasvavat noin 15–16 ikävuoteen saakka, pojat kasvavat vielä jopa 17–20 vuoden ikäisinä.</p><p>Alle 10-vuotiailla tytöillä ja pojilla on vähän sukupuolieroja. Sukupuolten eriytyvä kehitys käynnistyy murrosiässä. Suomalaisilla tytöillä murrosiän fyysinen kehitys käynnistyy tavallisimmin 10–12-vuotiaana, joillakin yksilöillä myös aikaisemmin ja osalla myöhemminkin. <a href="/wiki/Kuukautiskierto" title="Kuukautiskierto">Kuukautiset</a> alkavat yleensä kaksi vuotta myöhemmin, eli hyvin usein 12–14-vuotiaana. Tytöillä pituuskasvupyrähdys on poikia lievempi ja ajoittuu pojista poiketen murrosiän alkuun. Tytöillä pituuskasvu päättyy poikia varhemmin ja pojat kasvavat pidemmiksi sillä heidän luunsa kasvavat pituutta pidempään. Pojilla murrosikä alkaa tavallisesti 11–13-vuotiaana, aluksi hitaasti <a href="/wiki/Genitaali" class="mw-redirect" title="Genitaali">genitaalien</a> kasvamisena. Selviä merkkejä, kuten nopeaa pituuskasvua tai <a href="/wiki/%C3%84%C3%A4nenmurros" title="Äänenmurros">äänenmurrosta</a> ilmenee vasta 13–15-vuotiailla pojilla. Pojat saavuttavat sukukypsyyden tavallisesti hiukan ennen pituuskasvun kiihtymistä ja <a href="/wiki/%C3%84%C3%A4nenmurros" title="Äänenmurros">äänenmurrosta</a> eli tavallisesti noin 13–14-vuotiaana.</p><p>Murrosiässä tytöt jäävät lyhyemmiksi ja heille kehittyvät rinnat ja leveä lantio. Tyttöjen rasvaprosentti kohoaa ja ääni säilyttää lapsen korkean äänen piirteitä. Tyttöjen henkinen kehitys on murrosiässä nopeampaa ja ovat valmiimpia aikuisuuteen ja lasten saamiseen. Murrosiässä pojat kasvavat pidemmiksi ja tulevat lihaksikkaammiksi sekä ääni madaltuu eikä äänessä säily lapsen korkean äänen piirteitä. Poikien liikunnallinen suorituskyky ylittää tyttöjen liikunnallisen suorituskyvyn. Poikien henkisessä kypsymisessä kestää pidempään.</p><h4><span id="Aikuisik.C3.A4_ja_vanheneminen"></span><span class="mw-headline" id="Aikuisikä_ja_vanheneminen">Aikuisikä ja vanheneminen</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=10" class="mw-editsection-visualeditor" title="Muokkaa osiota Aikuisikä ja vanheneminen">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=10" title="Muokkaa osiota Aikuisikä ja vanheneminen">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h4><p>Murrosiän mentyä on aika jolloin ihminen on fyysisesti huippukunnossa. Ihmisen fyysinen kehitys saavuttaa huippunsa noin 21–22-vuotiaana. Tässä iässä myös naisen hedelmällisyys on yleensä kaikkein korkeimmillaan. Miehen testosteronitasot saavuttavat huippunsa vasta noin 24–25-vuotiaana. Noin 25 vuoden iästä alkaen ihon kollageenituotanto alkaa hiipua, elimistön toipuminen rasitustiloista hidastuu ja naisen hedelmällisyys varsinaisesti alkaa laskea. Tämä prosessi johtaa siihen, että kolmenkymmenenvuoden iän jälkeen kasvojen iholle alkaa ilmaantua pieniä juonteita ja ryppyjä hitaalla tahdilla, ja naisen raskaaksi tulemisen mahdollisuus pienenee. Myöhemmin, neljänkymmenen vuoden iästä eteenpäin ihmisen aineenvaihdunnan hidastuessa keski-ikäinen lihoaa helpommin, kuin alle 30-vuotias.Naisilla merkittävä vanhenemiseen liittyvä ilmiö on <a href="/wiki/Vaihdevuodet" title="Vaihdevuodet">vaihdevuodet</a>.</p><p>Ihmisen psyykkinen kehitys ja kypsyminen sen sijaan jatkuvat pitkälle aikuisikään. Havaintokyky alkaa heiketä <a href="/wiki/Keski-ik%C3%A4" title="Keski-ikä">keski-iässä</a>, mutta useimmat kognitiiviset eli ajatteluun liittyvät toiminnot säilyvät hyvinä noin 60–70 ikävuoteen asti. Kielelliset toiminnot alkavat heiketä yleensä vasta 80 ikävuoden jälkeen. Usein vanhuuden psyykkisten toimintojen huomattava huonontuminen liittyy johonkin dementoivaan sairauteen, kun taas terveiden vanhusten kognitiiviset toiminnot heikkenevät vain vähän. <sup id="cite_ref-18" class="reference"><a href="#cite_note-18">&#91;18&#93;</a></sup></p><p>Kaikki ihmiset vanhenevat, mutta terveellisillä elämäntavoilla, runsaalla liikunnalla ja positiivisella elämänasenteella sitä voidaan osittain hidastaa. Keskimääräinen ihmisten elinikä vaihtelee suuresti maittain. Naiset elävät keskimäärin useita vuosia miehiä kauemmin.</p><h4><span class="mw-headline" id="Kuolema">Kuolema</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=11" class="mw-editsection-visualeditor" title="Muokkaa osiota Kuolema">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=11" title="Muokkaa osiota Kuolema">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h4><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Kuolema" title="Kuolema">Kuolema</a></i></div></dd></dl><p>Kuolema on ihmisen elämän päätepiste ja voi aiheutua äkillisesti onnettomuudesta tai rikoksen uhriksi joutumisesta, jossa jokin tai jotkin elimet ovat vaurioituneet tai tuhoutuneet. Ihminen voi kuolla myös eri sairauksiin, kuten syöpiin, sydän- ja hengityselinsairauksiin tai aivosairauksiin. Vanhuuteen voi myös kuolla, jolloin vaikkapa sydämen vajaatoiminta tulee niin suureksi, että sydän ei enää voi pumpata. </p><p>Epäselvissä tapauksissa kuolinsyy ja -aika voidaan todeta ruumiinavauksessa, jonka suorittaa patologi. Kuolinhetki voidaan määritellä verenkierron ja hengityksen loppumisena (sydänkuolema, kliininen kuolema) tai aivojen toiminnan loppumisena (aivokuolema) ja se määritellään kuolemanmerkkien kuten lautumien ja kuolonkankeuden (rigor mortis) sekä ruumiin jäähtymisen ja hajoamisen avulla. </p><p>Yleensä ihmisen kuoltua hänen läheisensä <a href="/wiki/Hautaus" title="Hautaus">hautaavat</a> vainajan kuolleen ruumiin ja pystyttävät muistomerkin. Useimmille ihmisyhteisöille on ominaista myös halu uskoa kuolemanjälkeiseen elämään tai <a href="/wiki/Sielu" title="Sielu">sielun</a> <a href="/wiki/Kuolemattomuus" title="Kuolemattomuus">kuolemattomuuteen</a>.</p><h3><span class="mw-headline" id="Rotu_ja_etnisyys">Rotu ja etnisyys</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=12" class="mw-editsection-visualeditor" title="Muokkaa osiota Rotu ja etnisyys">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=12" title="Muokkaa osiota Rotu ja etnisyys">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Ihmisrotu" title="Ihmisrotu">Ihmisrotu</a></i></div></dd></dl><table style="" class="metadata plainlinks ambox ambox-content"><tbody><tr><td class="ambox-image"><div style="width:52px;text-align:center"> <a href="/wiki/Tiedosto:Question_book-4.svg" class="image"><img alt="Question book-4.svg" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/64/Question_book-4.svg/50px-Question_book-4.svg.png" width="50" height="39" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/64/Question_book-4.svg/75px-Question_book-4.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/64/Question_book-4.svg/100px-Question_book-4.svg.png 2x" data-file-width="262" data-file-height="204" /></a></div></td><td class="ambox-text"><b>Tähän artikkeliin tai osioon ei ole merkitty lähteitä, joten tiedot kannattaa tarkistaa muista tietolähteistä.</b><br /><small>Voit lisätä artikkeliin <a href="/wiki/Wikipedia:Tarkistettavuus" title="Wikipedia:Tarkistettavuus">tarkistettavissa olevia</a> lähteitä ja merkitä ne <a href="/wiki/Wikipedia:Merkitse_l%C3%A4hteet" title="Wikipedia:Merkitse lähteet">ohjeen</a> mukaan.</small><br /></td></tr></tbody></table><p>Eri maantieteellisillä alueilla asuu ihmispopulaatioita, joiden ulkonäkö, kieli ja kulttuuri eroavat toisistaan. Eri populaatiot ovat erilaisia pääasiassa siksi, koska ne ovat historiansa aikana kokeneet erilaisia vaiheita. Ryhmien on täytynyt selviytyä eri puolilla maapalloa hyvinkin erilaisissa olosuhteissa. Kulttuurillinen, kielellinen ja geneettinen kehitys on kulkenut eri polkuja ja kehittänyt vaihtelua ja paikallisia erikoisuuksia. </p><p>Yleensä ihmiset määrittelevät itsensä johonkin ryhmään <a href="/wiki/Rotu" title="Rotu">rodun</a> ja <a href="/wiki/Etnisyys" class="mw-redirect" title="Etnisyys">etnisyyden</a> perusteella, tosin tieteen kannalta rodun käsitteestä ei olla lainkaan yksimielisiä. Luokittelu yleensä perustuu esivanhempiin ja näkyviin ominaisuuksiin, kuten ihonväriin ja kasvonmuotoihin. Etnisesti ihmiset luokittelevat itsensä kielen, kulttuurin, perinteiden, uskonnon tai syntymäpaikan mukaan johonkin ryhmään, mikä lisää yhteenkuuluvuutta. Etnisyyttä voidaan kutsua myös <a href="/wiki/Kansallisuus" class="mw-redirect" title="Kansallisuus">kansallisuudeksi</a>, mutta termit eivät kaikissa tapauksissa ole yhteneväiset. Etnisyys tai kansallisuus on monille tärkeä samaistumisen kohde, ikään kuin laajennettu perhe; sen kautta monet kokevat kuuluvansa perhettä ja sukua laajempaan ryhmään, jolla on merkittäviä yhdistäviä ja muista ryhmistä erottavia tekijöitä. Etnisyyteen tai kansallisuuteen kohdistuva aatteellinen itsetietoisuus ja tiedostava samaistuminen on 1700- ja 1800-lukujen taitteessa kehittynyt <a href="/wiki/Nationalismi" title="Nationalismi">nationalismi</a>. Nationalismin eli kansallisuusaatteen vaikutuksesta monet tahtovat etnisen ryhmänsä saavan hallita tiettyä aluetta <a href="/wiki/Itsehallinto" class="mw-disambig" title="Itsehallinto">itsehallinnollisella</a> tai <a href="/wiki/Itsen%C3%A4isyys" title="Itsenäisyys">itsenäisenä</a>. Etnisten ryhmien ristiriitaiset aluevaatimukset ovat johtaneet moniin sotiin.</p><p>Lähes kaikki ihmiset elävät jonkin valtion tai yhteiskunnan alueella ja useimmiten ovat jonkun valtion tai yhteiskunnan virallisia jäseniä eli <a href="/wiki/Kansalaisuus" title="Kansalaisuus">kansalaisia</a>. Kansalaisuus on eri asia kuin kansallisuus tai etnisyys, sillä saman valtion kansalaisina voi olla useita etnisyyksiä. Myös valtio tai yhteiskunta voi olla voimakkaan samaistumisen kohde. Tällaista samaistumista kutsutaan <a href="/wiki/Patriotismi" class="mw-redirect" title="Patriotismi">patriotismiksi</a>.</p><h3><span class="mw-headline" id="Ravinto">Ravinto</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=13" class="mw-editsection-visualeditor" title="Muokkaa osiota Ravinto">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=13" title="Muokkaa osiota Ravinto">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><div class="thumb tright"><div class="thumbinner" style="width:252px;"><a href="/wiki/Tiedosto:La_Boqueria.JPG" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/be/La_Boqueria.JPG/250px-La_Boqueria.JPG" width="250" height="188" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/be/La_Boqueria.JPG/375px-La_Boqueria.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/be/La_Boqueria.JPG/500px-La_Boqueria.JPG 2x" data-file-width="1280" data-file-height="960" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Tiedosto:La_Boqueria.JPG" class="internal" title="Suurenna"></a></div>Erilaisia <a href="/wiki/Hedelm%C3%A4_(ruoka)" title="Hedelmä (ruoka)">hedelmiä</a>. Ihmiset yhä harvemmin keräävät itse ravintonsa luonnosta.</div></div></div><p>Ihmiset tarvitsevat elääkseen säännöllisesti <a href="/wiki/Ruoka" title="Ruoka">ruokaa</a> ja <a href="/wiki/Juoma" title="Juoma">juomaa</a>. Ruoan puute ihmisellä johtaa jo lyhyessä ajassa nälkäkuolemaan, mutta vedenpuute on sitäkin vakavampi: se johtaa <a href="/wiki/Dehydraatio" title="Dehydraatio">dehydraatioon</a> ja kuolemaan jo muutamassa päivässä. Vettä tai muita nesteitä ihminen juo keskimäärin 1,5 litraa päivässä. Ihmiset voivat syödä elääkseen sekä <a href="/wiki/Kasvit" title="Kasvit">kasvikunnan</a> että <a href="/wiki/El%C3%A4inkunta" title="Eläinkunta">eläinkunnan</a> tuotteita. Ihminen on siis lajina kaikkiruokainen. Ihmiset ovat käyttäneet sekä liha- että kasvisravintoa jo metsästäjä-keräilijä-kaudella. Ihminen voi kuitenkin tulla toimeen myös pelkällä kasvisravinnolla, jota osa ihmisistä käyttää nykyään eettisistä tai terveydellisistä syistä. 10&#160;000 vuotta sitten ihmiset omaksuivat <a href="/wiki/Maanviljely" title="Maanviljely">maanviljelyskulttuurin</a>, joka johti populaation nopeaan kasvuun ruoan riittäessä yhä useammalle.</p><p>Nykyisin tavaraliikenne on parantunut, ja yhä useammilla ihmisillä on mahdollisuus saada erilaisia ruoka-aineita eri puolilta maailmaa, eikä valikoima rajoitu enää vain paikalliseen tarjontaan.</p><p>Nälkä ja aliravitsemus on myös yksi suurimpia ihmiskunnan ongelmia. Aliravittuja ja nälkäisiä ihmisiä elää maapallolla lähes miljardi, lähinnä kehitysmaissa Afrikassa ja Etelä-Aasiassa. Vuosittain lähes 10 miljoonaa ihmistä kuolee nälkään ja aliravitsemukseen.<sup id="cite_ref-19" class="reference"><a href="#cite_note-19">&#91;19&#93;</a></sup> Monet kehittyneet maat auttavat köyhiä kehitysmaita elintarvikeavun kautta torjumaan nälkää.</p><h3><span id="Aivot.2C_mieli_ja_tietoisuus"></span><span class="mw-headline" id="Aivot,_mieli_ja_tietoisuus">Aivot, mieli ja tietoisuus</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=14" class="mw-editsection-visualeditor" title="Muokkaa osiota Aivot, mieli ja tietoisuus">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=14" title="Muokkaa osiota Aivot, mieli ja tietoisuus">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p><a href="/wiki/Ihmisaivot" title="Ihmisaivot">Ihmisen aivot</a> ovat <a href="/wiki/Keskushermosto" title="Keskushermosto">keskushermoston</a> tärkein osa. Ihmisen aivojen arvellaan olevan tehokkaammat kuin minkään muun nykyisen tai edesmenneen lajin. Ihmiset ovat tiettävästi myös ainoa laji, jota voidaan kutsua tieteellisesti todella <a href="/wiki/%C3%84lykkyys" title="Älykkyys">älykkääksi</a>. Aivojen paino on vain noin 2&#160;% koko ruumiinpainosta, mutta ne käyttävät 25&#160;% ihmisen hengitysilman hapesta. Ihmisaivojen oikeaa ja vasenta puoliskoa yhdistävä aivokurkiainen sisältää noin 100 miljoonaa hermosyytä.Aikuisen miehen aivot painavat keskimäärin 1 424&#160;grammaa, mutta paino alenee iän myötä 1 395 grammaan. Naisen aivot painavat noin 1 265&#160;grammaa. </p><p>Ihminen on yksi seitsemästä lajista, joka läpäisee <a href="/wiki/Peilitesti" title="Peilitesti">peilitestin</a>, tosin yleensä alle kaksivuotiaat ihmislapset eivät läpäise sitä. Testattavan pitää tunnistaa oma itsensä <a href="/wiki/Peili" title="Peili">peilistä</a>. Muut lajit, jotka läpäisevät testin, ovat <a href="/wiki/Simpanssi" title="Simpanssi">simpanssi</a>, <a href="/wiki/Norsu" class="mw-redirect" title="Norsu">norsu</a>, <a href="/wiki/Bonobo" title="Bonobo">bonobo</a>, <a href="/wiki/Oranki" class="mw-redirect" title="Oranki">oranki</a>, <a href="/wiki/Delfiinit" title="Delfiinit">delfiini</a> ja <a href="/wiki/Kyyhkynen" class="mw-redirect" title="Kyyhkynen">kyyhkynen</a>. Yleensä muiden eläinlajien edustajat reagoivat omaan peilikuvaansa samalla tavalla kuin lajikumppaniinsa.<sup id="cite_ref-20" class="reference"><a href="#cite_note-20">&#91;20&#93;</a></sup></p><h2><span class="mw-headline" id="Yhteiskunta">Yhteiskunta</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a tiiviissä vuorovaikutuksessa tietyllä maantieteellisellä alueella. Kaikki yhteiskunnat yhdessä muodostavat koko ihmiskunnan. <a href="/wiki/Kulttuuri" title="Kulttuuri">Kulttuuri</a> tarkoittaa tietylle ihmisryhmälle ominaista käytöstä, tapoja ja historiaa. Ihmisillä on monia arvoja, normeja ja perinteitä, ja ne yhdistävät ryhmiä. </p><p>Vastaavaa sosiaalista kanssakäymistä ja yhteiskuntaa on myös esimerkiksi <a href="/wiki/Muurahaiset" title="Muurahaiset">muurahaisilla</a> ja linnuilla. Ihmiset eroavat muista lajeista kuitenkin monimuotoisen sosiaalisen kanssakäymisen ja kulttuurin sekä voimakkaan <a href="/wiki/Kulttuurievoluutio" title="Kulttuurievoluutio">kulttuurievoluutio</a>n ansiosta.</p><p>Evoluutiossa ihmisen keinoja selvitä muita lajeja paremmin on ollut keskeisesti älykkyys, ihmisen aivot. Muita vastaavia keinoja, osin aivoihin perustuen, ovat olleet esimerkiksi</p><ul><li>kommunikaatiotekniikka, kieli</li><li>muu teknologia</li><li><a href="/wiki/Tieto" title="Tieto">tieto</a> ja ymmärrys</li><li><a href="/wiki/Organisaatio" title="Organisaatio">organisaatiotaito</a>, joukon voima ja sen mahdollistama <a href="/wiki/Erikoistuminen" title="Erikoistuminen">erikoistuminen</a>.</li><li>pitkälle kehittyneet <a href="/wiki/K%C3%A4det" class="mw-redirect" title="Kädet">kädet</a>, jotka mahdollistavat työkalujen käytön ja hienomotoriikan</li></ul><p>Ihmiset pitävät itseään hyvinä tilanteiden arvioinnissa, yleiskäsityksen muodostamisessa, asiayhteyksien tajuamisessa, mallien tunnistamisessa ja vaistojen käytössä. Toisaalta ihmiselle tuottavat vaikeuksia käsitellä asioita rinnakkain, muistaa yksityiskohtia, suorittaa rutiinitehtäviä, luopua vääristä päätelmistä ja hallita stressiä. Kuitenkin kaikkien näiden ominaisuuksien suhteen yksilöllinen vaihtelu on varsin suurta: joillekin ihmisille esimerkiksi suorittava rutiininomainen työ sopii hyvin ja monien ihmisten paineensietokyky on kehittynyt erittäin vahvaksi.</p><h3><span class="mw-headline" id="Kieli">Kieli</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=16" class="mw-editsection-visualeditor" title="Muokkaa osiota Kieli">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=16" title="Muokkaa osiota Kieli">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Kieli" title="Kieli">Kieli</a></i></div></dd></dl><p>Ihmiset eroavat muista lajeista kielellisillä kyvyillään, joiden ansiosta he voivat jakaa keskenään ideoita, suunnitella tulevaisuutta ja siirtää hankittua tietoa jälkipolville. Kieli on järjestelmä, jossa ihminen ilmaisee ajatuksensa kielellisillä merkeillä, abstraktioilla. Näitä merkkejä nimitetään kielellisiksi ilmauksiksi. Puhutuissa kielissä kuuluvia merkkejä tuotetaan puhe-elimillä. Kirjoitetussa kielessä kielelliset ilmaukset on tehty silmin nähtäviksi eli luettaviksi. Viitotuissa kielissä kielelliset ilmaukset ovat visuaalisia, eivät kuuluvia. Kieliä tutkii <a href="/wiki/Kielitiede" title="Kielitiede">kielitiede</a>. Ihmisillä on tällä hetkellä käytössä eri kieliä noin 6&#160;000, ja lukemattomat kielet ovat hävinneet aikojen saatossa. Ihmiskunnan kielellinen monimuotoisuus on kuitenkin vähentymässä ja <a href="/wiki/Kuollut_kieli" title="Kuollut kieli">kieliä kuolee</a> yhä kasvavalla nopeudella, johtuen suurelta osin <a href="/wiki/Globalisaatio" title="Globalisaatio">globalisaatiosta</a>. Valtaosan kielistä pelätään kuolevan tämän vuosisadan aikana. </p><p><a href="/wiki/Kirjoitustaito" class="mw-redirect" title="Kirjoitustaito">Kirjoitustaidon</a> ihmiset kehittivät noin 5&#160;000 vuotta sitten. Se oli tehokas tapa säilöä tietoa muistiin jälkipolville, ja sillä oli huomattava merkitys ihmisten <a href="/wiki/Kulttuurievoluutio" title="Kulttuurievoluutio">kulttuurievoluutiossa</a>.</p><h3><span class="mw-headline" id="Taiteet">Taiteet</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=17" class="mw-editsection-visualeditor" title="Muokkaa osiota Taiteet">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=17" title="Muokkaa osiota Taiteet">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><div class="thumb tright"><div class="thumbinner" style="width:252px;"><a href="/wiki/Tiedosto:Lorenzo_Lippi_001.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/37/Lorenzo_Lippi_001.jpg/250px-Lorenzo_Lippi_001.jpg" width="250" height="296" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/37/Lorenzo_Lippi_001.jpg/375px-Lorenzo_Lippi_001.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/37/Lorenzo_Lippi_001.jpg/500px-Lorenzo_Lippi_001.jpg 2x" data-file-width="2024" data-file-height="2397" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Tiedosto:Lorenzo_Lippi_001.jpg" class="internal" title="Suurenna"></a></div><i>Musiikin allegoria</i> (noin 1594), <a href="/w/index.php?title=Lorenzo_Lippi&amp;action=edit&amp;redlink=1" class="new" title="Lorenzo Lippi (sivua ei ole)">Lorenzo Lippin</a> maalaus kuvaa naista, joka kirjoittaa <a href="/wiki/Partituuri" title="Partituuri">partituuria</a></div></div></div><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Taide" title="Taide">Taide</a></i></div></dd></dl><p><a href="/wiki/Kuvataide" title="Kuvataide">Kuvataidetta</a> on esiintynyt ihmiskunnassa aina esihistoriallisesta ajasta lähtien. Se on erikoinen ilmaisun, viestinnän, kannanoton ja mielihyvän tuottamisen muoto, eikä vastaavaa muilla lajeilla esiinny.</p><p><a href="/wiki/Musiikki" title="Musiikki">Musiikki</a> on ihmisen kulttuurille ominainen ääneen perustuva taiteen ja viestinnän muoto. Sen synnyn arvellaan liittyneen puheen kehittymiseen, ja siihen ovat saattaneet vaikuttaa luonnon äänien – kuten lintujen – matkiminen, työliikkeiden rytmi ja ihmisen leikkisä ja seurallinen perusolemus. Nykyisin musiikki on yksi levinneimmistä viihteen muodoista ihmiskunnan keskuudessa.</p><p><a href="/wiki/Kirjallisuus" title="Kirjallisuus">Kirjallista</a> perinnettä edeltää ihmiskunnan suullisen kerronnan, tarinoiden, kaskujen ja tarujen pitkä perinne. Kirjallista taidetta edustavat <a href="/wiki/Runous" title="Runous">runot</a>, <a href="/wiki/Draama" title="Draama">draama</a> ja <a href="/wiki/Proosa" title="Proosa">proosa</a>.</p><h3><span class="mw-headline" id="Motivaatio_ja_tunteet">Motivaatio ja tunteet</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=18" class="mw-editsection-visualeditor" title="Muokkaa osiota Motivaatio ja tunteet">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=18" title="Muokkaa osiota Motivaatio ja tunteet">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p><a href="/wiki/Motivaatio" title="Motivaatio">Motivaation käsitettä</a> on käytetty kuvaamaan sitä 1) mikä aktivoi organismin (<i>aktivaatio</i>) 2) mikä saa organismin valitsemaan juuri tietyn vasteen tai käyttäytymisen (<i>valinta/selektio ja ohjaaminen</i>) ja 3) miksi organismi vastaa samaan ärsykkeeseen välillä eri tavoilla (<i>valmius vasteeseen</i>). Yksinkertaisesti motivaatioteorian tulisi vastata kysymykseen ”miksi käyttäydymme siten kuin käyttäydymme”. Motivaatioteorioiden jaoksi on ehdotettu niin sanottua push-pull-dikotomiaa: toisissa teorioissa oletetaan jokin ”työntävä”, organismin sisäinen motivaatiovoima (perinteisesti tension/sisäisen jännitteen vähentäminen, tension reduction), toisissa taas jokin ”vetävä”, organismin ulkopuolella oleva motivaatiovoima (perinteisesti palkkiot/rangaistukset). <a href="/wiki/Hedonismi" title="Hedonismi">Hedonistisia</a> motivaatioteorioita yhdistää korostus organismin pyrkimisestä nautinnon tavoitteluun ja ahdistuksen/kivun välttämiseen. Tärkeitä motivaatiokäsitteitä ovat muun muassa vietti, tarve, tavoite, attribuutio ja flow.<sup id="cite_ref-22" class="reference"><a href="#cite_note-22">&#91;22&#93;</a></sup></p><p><a href="/wiki/Tunne" title="Tunne">Tunne</a> on kokonaisuus <a href="/wiki/Kompleksisuus" title="Kompleksisuus">kompleksisen</a> järjestelmän, kuten ihmisen, koordinoimiseksi tiettyyn perustilanteeseen. Esimerkiksi <a href="/wiki/Pelko" title="Pelko">pelon</a> tunne herkistää <a href="/wiki/Aisti" title="Aisti">aisteja</a>, keskittää huomiota, pienentää pintaverenkiertoa (pienemmät vauriot taistelutilanteessa), nostaa sydämen lyöntitiheyttä, erittää vereen adrenaliinia ja niin edelleen. Vaikka ihmiset tietävätkin tunteista nykyään paljon, ovat ne edelleenkin tieteellinen mysteeri. Näitä asioita tutkii erityisesti <a href="/wiki/Psykologia" title="Psykologia">psykologia</a>.</p><h3><span class="mw-headline" id="Rakkaus_ja_seksuaalisuus">Rakkaus ja seksuaalisuus</span><span Botticelli">Botticellin</a> maalaus <i><a href="/wiki/Venuksen_syntym%C3%A4_(Botticelli)" title="Venuksen syntymä (Botticelli)">Venuksen syntymä</a></i></div></div></div><p><a href="/wiki/Rakkaus" title="Rakkaus">Rakkaus</a> on yksi ihmisen voimakkaimmista tunteista. Se voi kohdistua toiseen ihmiseen, aatteeseen, uskontoon, taiteeseen, luontoon tai johonkin muuhun asiaan. Aikuisten välisissä suhteissa keskeistä on seksuaalisuus, joka on tärkeä keino rakkauden ilmaisemiseksi. Kaikki rakkaussuhteet eivät kuitenkaan sisällä lainkaan seksuaalista ulottuvuutta, eivätkä kaikki seksuaaliset suhteet ole rakkaussuhteita. Vanhempien jälkeläisiään (usein myös muiden lapsia) kohtaan tuntemassa rakkaudessa on keskeisenä vaikuttimena biologinen huoltamisvaisto eli hoivavietti.</p><p>Seksuaalisuus ei ole keskeistä rakkaudessa, vaikka se onkin yksi niistä merkittävistä tunnealueista (kuten hoivavietti), jotka limittyvät osittain rakkauden kanssa. Seksuaalisuus on keskeistä vain niissä rakkaussuhteissa, jotka ovat seksisuhteita. Lisäksi seksuaalisuudella on merkittävä - joskaan ei yleensä keskeinen - osa myös joissakin muissa ihmissuhteissa.</p><p><a href="/wiki/Ihmisen_seksuaalisuus" title="Ihmisen seksuaalisuus">Seksuaalisuus</a> on ihmisellä esiintyvä ilmiö. Eläimillä sukupuolista käyttäytymistä esiintyy lähinnä vain silloin kun hedelmöityminen on mahdollista, ja eläinten kosiskelu- ja parittelutoiminnot ovat vaistonvaraisia ja kiinteästi lajisidonnaisia. Ihmisten seksuaalisuus on monimutkaisinta kaikista eläimistä, eikä ihmisten välinen seksuaalinen kanssakäyminen useimmiten liity suoraan lisääntymiseen. Ihmiset voidaan jakaa seksuaalisten halujensa kohteen perusteella esimerkiksi <a href="/wiki/Aseksuaalisuus" title="Aseksuaalisuus">a</a>-, <a href="/wiki/Heteroseksuaalisuus" title="Heteroseksuaalisuus">hetero</a>-, <a href="/wiki/Homoseksuaalisuus" title="Homoseksuaalisuus">homo</a>- ja <a href="/wiki/Biseksuaalisuus" title="Biseksuaalisuus">biseksuaaleihin</a>, joista heteroseksuaalisuutta pidetään yleisimpänä seksuaalisena suuntautumisena. Suuntautumisten välinen rajanveto on kuitenkin usein vaikeaa ellei mahdotonta, ja eräät seksologit uskovatkin ihmisten olevan monien muiden eläinlajien tavoin luonnostaan biseksuaaleja.<sup><span style="background: #FAD6A5;"><i><a href="/wiki/Wikipedia:Merkitse_l%C3%A4hteet" title="Wikipedia:Merkitse lähteet">lähde?</a></i></span></sup></p><h3><span class="mw-headline" id="Uskonto">Uskonto</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=20" class="mw-editsection-visualeditor" title="Muokkaa osiota Uskonto">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=20" title="Muokkaa osiota Uskonto">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Uskonto" title="Uskonto">Uskonto</a></i></div></dd></dl><p>Uskonto tarkoittaa ihmisen <a href="/wiki/Usko" title="Usko">uskoa</a> <a href="/wiki/Jumala" title="Jumala">jumaliin</a> tai muuhun yli-inhimilliseen mahtiin, hänen turvautumistaan siihen sekä siihen liittyvää <a href="/wiki/Pyh%C3%A4" title="Pyhä">pyhyyden</a> kokemusta. Nämä ovat uskonnossa usein muotoutuneet oppijärjestelmäksi ja ilmenevät erilaisina yhteisöllisinä tapoina ja <a href="/wiki/Palvonta" title="Palvonta">palvontamenoina</a>.<sup id="cite_ref-23" class="reference"><a href="#cite_note-23">&#91;23&#93;</a></sup> Yleensä uskonnot sisältävät myös moraalisia arvoja ja/tai sääntöjä, joiden mukaan ihmisten oletetaan elävän. Uskonnon määritelmiä on lukuisia eikä niistä vallitse yksimielisyyttä. On ajateltu muun muassa, että ihminen on kehittänyt erilaisia uskonnonmuotoja tietämättömyyden pelossa.<sup id="cite_ref-24" class="reference"><a href="#cite_note-24">&#91;24&#93;</a></sup>Yhden näkemyksen mukaan uskonnon keskeinen merkitys ihmiselle on vastata olemassaoloon liittyviin, eksistentiaalisiin kysymyksiin, esimerkiksi ”mitä tapahtuu kuoleman jälkeen”, ”mitä merkitystä elämällä on” ja niin edelleen.<sup id="cite_ref-25" class="reference"><a href="#cite_note-25">&#91;25&#93;</a></sup></p><h3><span class="mw-headline" id="Filosofia">Filosofia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=21" class="mw-editsection-visualeditor" title="Muokkaa osiota Filosofia">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=21" title="Muokkaa osiota Filosofia">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Filosofia" title="Filosofia">Filosofia</a></i></div></dd></dl><p>Filosofia on noin 600–400&#160;eaa. välisenä aikana antiikin Kreikassa syntynyt ajatteluperinne ja oppiala, joka pyrkii tutkimaan todellisuuden perimmäistä olemusta, tiedon yleisiä ehtoja, kauneuden ja arvojen olemassaoloa, hyvän yhteiskunnan ehtoja, ihmisenä olemisen luonnetta sekä näihin kytkeytyviä yleisiä teemoja.</p><h3><span class="mw-headline" id="Tiede_ja_tekniikka">Tiede ja tekniikka</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=22" class="mw-editsection-visualeditor" title="Muokkaa osiota Tiede ja tekniikka">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=22" title="Muokkaa osiota Tiede ja tekniikka">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Tiede" title="Tiede">Tiede</a>, <a href="/wiki/Tekniikka" title="Tekniikka">Tekniikka</a></i></div></dd></dl><p>Tiede viittaa systemaattiseen tiedonhankintajärjestelmään, jolla ihmiset pyrkivät ymmärtämään todellisuutta yhä tarkemmin. Tiede perustuu teorioihin, joiden perusteella luodaan ennustuksia ja joita sitten testataan kokeilla. Moderni tiede syntyi Euroopassa uudella ajalla. Tieteen tekemät ennustukset usein hyödyttävät yhteiskuntaa tai yksilöitä, jotka käyttävät niitä. Tiede ja erityisesti teknologian kehitys ovat olleet yksi keskeisimmistä ihmisten kulttuuria ohjaavista tekijöistä.</p><p>Varhaisimmat tekniikan alkumerkit ovat hyvin vanhoja, jopa 3,4 miljoonaa vuotta<sup id="cite_ref-26" class="reference"><a href="#cite_note-26">&#91;26&#93;</a></sup>. Tekniikan ollessa näin vanhaa on ilmeistä, että tekniikka ei ole vaikuttanut vain ihmisen kulttuuriin vaan myös ihmisen evoluutioon, minkä seurauksena ihminen ilman tekniikkaa on luonnossa varsin avuton eläin. Samanaikaisesti nyky-ihmisen ilmaantuessa 200&#160;000 vuotta sitten on varhaisesta kivikaudesta löytöjen perusteella siirrytty keskiseen kivikauteen, jolle on tyypillistä hienostuneemmin tehdyt kivityökalut ja nuolen kärjet.<sup id="cite_ref-27" class="reference"><a href="#cite_note-27">&#91;27&#93;</a></sup> Myös vaatetuksen käyttö on noihin aikoihin ollut niin laajaa, että ihmisen loisesta, <a href="/wiki/T%C3%A4i" class="mw-redirect" title="Täi">täistä</a>, kehittyi vain ihmisen vaatteissa elävä alalaji, <a href="/wiki/Vaatet%C3%A4i" title="Vaatetäi">vaatetäi</a>.</p><h3><span class="mw-headline" id="Valtio_ja_politiikka">Valtio ja politiikka</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=23" class="mw-editsection-visualeditor" title="Muokkaa osiota Valtio ja politiikka">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=23" title="Muokkaa osiota Valtio ja politiikka">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Valtio" title="Valtio">Valtio</a>, <a href="/wiki/Politiikka" title="Politiikka">politiikka</a></i></div></dd></dl><p>Valtio on nykyisin keskeinen yhteiskunnallisen organisoitumisen muoto. Sitä edelsi esimerkiksi klaani, heimo tai <a href="/wiki/P%C3%A4%C3%A4llikk%C3%B6kunta" title="Päällikkökunta">päällikkökunta</a>.</p><p>Sovinnaisen käsityksen mukaan politiikka tarkoittaa ihmisyhteisön päätöksentekoa ja muuta siihen liittyvää toimintaa. Laajimman määrittelyn mukaan politiikka on toimimista jonkin asian puolesta jotain toista asiaa vastaan.</p><h4><span id="V.C3.A4kivalta"></span><span class="mw-headline" id="Väkivalta">Väkivalta</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=24" class="mw-editsection-visualeditor" title="Muokkaa osiota Väkivalta">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=24" title="Muokkaa osiota Väkivalta">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h4><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/V%C3%A4kivalta" title="Väkivalta">Väkivalta</a></i></div></dd></dl><div class="thumb tright"><div class="thumbinner" style="width:252px;"><a href="/wiki/Tiedosto:Rwandan_Genocide_Murambi_skulls.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/db/Rwandan_Genocide_Murambi_skulls.jpg/250px-Rwandan_Genocide_Murambi_skulls.jpg" width="250" height="149" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/db/Rwandan_Genocide_Murambi_skulls.jpg/375px-Rwandan_Genocide_Murambi_skulls.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/d/db/Rwandan_Genocide_Murambi_skulls.jpg 2x" data-file-width="500" data-file-height="298" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Tiedosto:Rwandan_Genocide_Murambi_skulls.jpg" class="internal" title="Suurenna"></a></div>Sodat ja vainot ovat näyttävimpiä esimerkkejä väkivallasta, mutta suurin osa väkivallasta tapahtuu pienemmässä mittakaavassa, ja jää siksi vähemmälle huomiolle.</div></div></div><p>Ihmisille kuten muillekin kädellisille ja nisäkkäille on luonteenomaista ryhmien väliset aggressiot, jotka useimmiten rajoittuvat uhkailuun. Väkivaltaa esiintyy runsaasti niin ihmisyksilöiden kuin yhteisöjenkin välillä. Ihmisryhmään kohdistuvaa väkivaltaa kutsutaan esimerkiksi vainoksi. <a href="/wiki/Sota" title="Sota">Sota</a> on kahden ihmisyhteisön väkivaltainen yhteenotto. Vaikka yhteisö ei olisi osallisena missään sodassa tai vainossa, sen yksilöt voivat kuitenkin kohdata väkivaltaa toistensa taholta. Yhteiskunnat pyrkivät rajoittamaan väkivaltaa tai monopolisoimaan sen vain tietyille tahoille. Monissa kulttuureissa väkivaltaa sekä ihannoidaan että paheksutaan. Väkivalta voi olla myös henkistä.</p><p>Väkivalta on ihmisille useimmiten vallankäyttöä, sillä pyritään alistamaan toinen tai saavuttamaan jokin tavoite.</p><h3><span class="mw-headline" id="Sota">Sota</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=25" class="mw-editsection-visualeditor" title="Muokkaa osiota Sota">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=25" title="Muokkaa osiota Sota">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Sota" title="Sota">Sota</a></i></div></dd></dl><p>Arkeologisten löytöjen ja kulttuuriantropologisen vertailun perusteella on päätelty, että ihmiset ovat käyneet heimojen tai klaanien välisiä sotia noin 10&#160;000–12&#160;000 vuoden ajan. Tämä tekee sodasta varsin nuoren ilmiön ihmisen biologisessa ja kulttuurisessa evoluutiossa.<sup id="cite_ref-28" class="reference"><a href="#cite_note-28">&#91;28&#93;</a></sup> Viime vuosituhansien aikana sotiminen on kuitenkin ollut yleistä monien kulttuurien keskuudessa. Ihmiskunnan historian tuhoisin sota on ollut <a href="/wiki/Toinen_maailmansota" title="Toinen maailmansota">toinen maailmansota</a>, jonka seurauksena kuoli yli 50 miljoonaa ihmistä.</p><div class="thumb tleft"><div class="thumbinner" style="width:252px;"><a href="/wiki/Tiedosto:Market-Chichicastenango.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/62/Market-Chichicastenango.jpg/250px-Market-Chichicastenango.jpg" width="250" height="150" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/62/Market-Chichicastenango.jpg/375px-Market-Chichicastenango.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/62/Market-Chichicastenango.jpg/500px-Market-Chichicastenango.jpg 2x" data-file-width="1200" data-file-height="720" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Tiedosto:Market-Chichicastenango.jpg" class="internal" title="Suurenna"></a></div>Kaupankäyntiä kirkon portailla <a href="/wiki/Guatemala" title="Guatemala">Guatemalassa</a>.</div></div></div><h3><span id="Kaupank.C3.A4ynti"></span><span class="mw-headline" id="Kaupankäynti">Kaupankäynti</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=26" class="mw-editsection-visualeditor" title="Muokkaa osiota Kaupankäynti">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=26" title="Muokkaa osiota Kaupankäynti">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="disambig"><i>Pääartikkeli&#58; <a href="/wiki/Kaupank%C3%A4ynti" title="Kaupankäynti">Kaupankäynti</a></i></div></dd></dl><p>Ihmisillä kaupankäynti tarkoittaa tavaran tai palvelun vaihdantaa, jossa vaihdannan tapahtumaan ei liity tiiviin henkilökohtaisen suhteen solmimista. Alun perin ihmiset vaihtoivat tavaroita toisiin tavaroihin, mutta kyse oli usein pikemminkin sosiaalisia suhteita vahvistavien lahjojen vaihdosta kuin kaupankäynnistä. <a href="/wiki/Raha" title="Raha">Rahan</a> keksiminen tehosti kaupankäyntiä. Kun vaihdannassa käytetään rahaa, voidaan erottaa ostaminen ja myyminen erillisinä tapahtumina. Toisten eläinten ei tiedetä käyvän kovinkaan merkittävää kauppaa.</p><h3><span id="Suhde_ymp.C3.A4rist.C3.B6.C3.B6n"></span><span class="mw-headline" id="Suhde_ympäristöön">Suhde ympäristöön</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ihminen&amp;veaction=edit&amp;section=27" class="mw-editsection-visualeditor" title="Muokkaa osiota Suhde ympäristöön">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ihminen&amp;action=edit&amp;section=27" title="Muokkaa osiota Suhde ympäristöön">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><div class="thumb tright"><div class="thumbinner" style="width:192px;"><a href="/wiki/Tiedosto:Pollution_de_l%27air.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b3/Pollution_de_l%27air.jpg/190px-Pollution_de_l%27air.jpg" width="190" height="223" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/b/b3/Pollution_de_l%27air.jpg 1.5x" data-file-width="231" data-file-height="271" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Tiedosto:Pollution_de_l%27air.jpg" class="internal" title="Suurenna"></a></div>Fossiilisten polttoaineiden käyttö ja niiden korvaaminen on keskeisessä asemassa sekä hyvinvoinnin kehittymisessä että kestävän kehittymisen turvaamisessa.</div></div></div><p>Monet <a href="/wiki/Mets%C3%A4st%C3%A4j%C3%A4-ker%C3%A4ilij%C3%A4t" title="Metsästäjä-keräilijät">metsästäjä-keräilijäkansat</a> ja varhaiset maanviljelijätkin elivät melko lailla ympäristönsä kantokyvyn sisällä. Poikkeuksiakin on. Ihmistoiminnalle erityisen herkissä ekosysteemeissä esimerkiksi monet suuret riistaeläimet tapettiin sukupuuttoon. Paikallisesti melko varhaiskantaisetkin yhteiskunnat hävittivät jopa kokonaisia ekosysteemejä, kuten <a href="/wiki/P%C3%A4%C3%A4si%C3%A4issaari" title="Pääsiäissaari">Pääsiäissaarilla</a>.<sup id="cite_ref-29" class="reference"><a href="#cite_note-29">&#91;29&#93;</a></sup> </p><p>Erityisesti <a href="/wiki/Teollinen_vallankumous" title="Teollinen vallankumous">teollisen vallankumouksen</a> myötä tilanne muuttui ratkaisevasti. Ihminen on viime vuosisatoina levittäytynyt yhä laajemmalle ja ottanut käyttöönsä yhä enemmän <a href="/wiki/Luonnonvarat" title="Luonnonvarat">luonnonvaroja</a>, sekä uusiutuvia että uusiutumattomia. Yhä enemmän maa-aluetta on otettu ihmisten aktiiviseen käyttöön ja viety se näin pois muulta luonnolta. Maata on sekä rakennettu että muutettu viljelykäyttöön. Monia uusiutuvia luonnonvaroja on alettu hyödyntää yli niiden uusiutumiskapasiteetin, minkä myötä niiden uusiutumisvauhti ja -kyky on heikentynyt. Jotkut uusiutuvat luonnonvarat, kuten monet eliölajit, on käytetty kokonaan loppuun siten, että ne ovat tuhoutuneet kokonaan. Teollisen vallankumouksen keskeinen osa on ollut uusiutumattomien ei-kierrättettävien luonnonvarojen käyttöönotto. Näitä ovat etupäässä fossiiliset polttoaineet ja fissiokelpoinen uraani eli toistaiseksi keskeisimmät energialähteet. Myös kierrätyskelpoisten uusiutumattomien luonnovarojen käyttö on lisääntynyt huomattavasti. Neitseellisen varojen ehtyessä ja niiden hyödyntämisen vaikeutuessa on alettu <a href="/wiki/Kierr%C3%A4tys" title="Kierrätys">kierrättää</a>.</p><p>Mm. sairaanhoidossa ja ravitsemuksessa saavutetun edistyksen myötä ihmisen <a href="/wiki/V%C3%A4est%C3%B6nkasvu" title="Väestönkasvu">väestönkasvu</a> kiihtyi selvästi 1900-luvulla, jolloin sitä myös alettiin pitää vakavana ongelmana. Kuolleisuus pieneni mutta syntyvyyden lasku seurasi vasta viiveellä perässä. Väestönkasvu on jo hidastunut ensimmäisenä teollistuneissa maissa ja näin oletetaan käyvän myös vasta teollistumassa olevissa maissa. 2000-luvun alussa valtaosa ihmisistä elää esiteollisessa kurjuudessa ja köyhyydessä, sekä useat myös nälässä tai kunnollisten elintarvikkeiden ja veden puutteessa. Eri tieteenalat tutkivat ihmistä ja ihmisen toimintaa eri tavalla ja osin eri lähtökohdista. Niitä ovat esimerkiksi:</p><ul><li><a href="/wiki/Antropologia" title="Antropologia">Antropologia</a> on yleisnimitys ihmisten tutkimiselle. Monen ihmistä tutkivan tieteen voidaan katsoa kuuluvan antropologiaan.</li><li><a href="/wiki/V%C3%A4est%C3%B6tiede" title="Väestötiede">Väestötiede</a> eli demografia</li><li><a title="Kielitiede">Kielitiede</a></li><li><a href="/wiki/Sosiologia" title="Sosiologia">Sosiologia</a></li><li><a href="/wiki/Historia" title="Historia">Historia</a></li><li><a href="/wiki/Arkeologia" title="Arkeologia">Arkeologia</a></li></ul><h2><span id="Katso_my.C3.B6s"></span><span class="mw-headline" id="Katso_myös">Katso myös</span><span class="mw-editsection"><span
