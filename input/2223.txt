дан</div>				<div id="contentSub"></div>								<div id="jump-to-nav" class="mw-jump">					Өтүү:					<a href="#mw-head">багыттоо</a>, 					<a href="#p-search">издөө</a>				</div>				<div id="mw-content-text" lang="ky" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Bhutan_(8026014211).jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/24/Bhutan_%288026014211%29.jpg/220px-Bhutan_%288026014211%29.jpg" width="220" height="330" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/24/Bhutan_%288026014211%29.jpg/330px-Bhutan_%288026014211%29.jpg 1.5x, class="magnify"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Bhutan_(8026014211).jpg" class="internal" title="Чоңойтуу"></a></div>Адам</div></div></div><p><b>Адам</b> (киши) - <a href="/wiki/%D0%96%D0%B5%D1%80" title="Жер">Жердеги</a> тирүү <a href="/wiki/%D0%9E%D1%80%D0%B3%D0%B0%D0%BD%D0%B8%D0%B7%D0%BC" title="Организм">организмдердин</a> ичинен өзүнүн өрчүп-өсүшүндө эң жогорку баскычка жеткен, коомдук-тарыхый ишмердүүлүктүн жана маданияттын жаратуучусу жана ээси.</p><p>Адам <a href="/wiki/%D0%96%D0%B0%D0%BD%D1%8B%D0%B1%D0%B0%D1%80%D0%BB%D0%B0%D1%80" title="Жаныбарлар">жаныбарлардан</a> айырмаланып, <a href="/wiki/%D0%90%D0%BA%D1%8B%D0%BB-%D1%8D%D1%81" title="Акыл-эс">акыл-эс</a>, ойлоо жөндөмдүүлүгүнө ээ, эмгек куралын жасап, аны менен <a href="/wiki/%D0%90%D0%B9%D0%BB%D0%B0%D0%BD%D0%B0-%D1%87%D3%A9%D0%B9%D1%80%D3%A9" title="Айлана-чөйрө">айлана-чөйрөгө</a> таасир этип, өзгөртө алат; мээ түзүлүшү татаал, сүйлөөгө жөндөмдүү. Адам гендик жактан жаныбарлар менен тектеш, ал аң-сезимдүү коомдук жандыкка айлангыча узакка созулган биологиялык, социалдык, тарыхый жолду басып өткөн. Адам сымал маймылдын жашоо үчүн <a href="/wiki/%D0%A2%D0%B0%D0%B1%D0%B8%D1%8F%D1%82" title="Табият">табияттагы</a> даяр нерселерди курал катары колдонушу кийин эмгек куралын жасашына түрткү берген. Адамдын эң негизги жөндөмдүүлүгү эмгек куралын жасап, аны табиятты өзгөртүү үчүн пайдаланышы. Адам сымал маймылдын адамга айланышына шарт түзгөн эмгек болгон. Эмгек анын аң-сезиминин, сезүү органдарынын жана мээсинин иштешин жакшыртып, дене түзүлүшүн өзгөртүп, <a href="/wiki/%D0%90%D0%BD%D1%82%D1%80%D0%BE%D0%BF%D0%BE%D0%B3%D0%B5%D0%BD%D0%B5%D0%B7" title="Антропогенез">антропогенездин</a> улам жаңы баскычына көтөрүп, миллиондогон жылдан кийин жаңы биологиялык түрдү пайда кылган. Эмгек аң-сезимдин туундусу катары сүйлөө жөндөмдүүлүгүн жараткан. Анатомиялык-морфологиялык бөтөнчөлүктөрү жагынан сүт эмүүчүлөрдүн <a href="/wiki/%D0%9F%D1%80%D0%B8%D0%BC%D0%B0%D1%82%D1%82%D0%B0%D1%80" title="Приматтар">приматтар</a> түркүмүндөгү «гоминид» (латынча homo – адам) тукумуна кирет. Гоминиддердин түпкү теги катары саналган, мындан 1 млн 600 жыл мурда Чыгыш Африкада жашаган хомохабилис (Homo habilis ишке жөндөмдүү А.) келип чыккан. Археологиялык казындыларда хомохабилистин сөөктөрү менен кошо колго жасалган жөнөкөй эмгек куралдары да табылган. Эмгек куралын жасоо питекантроптордун убагында эле эзелки Адамдын аң-сезимине таасир этип, өз ара пикир алышуу зарылдыгын туудурган. Ушуга байланыштуу алардын мээ түзүлүшү да өзгөрө баштаган. Мындай эволюциялык өзгөрүүлөр Адам коомунун алгачкы доорунда эле башталган. Буга Европа, Африка, Азияда жашаган неандерталдардын көп калдыктары далил.</p><p>Төртүнчүлүк доордун аяк ченинде неандерталдарда коомдук мамилелер өнүгө баштаган. Кийинчерээк алардын ордун кроманьондор баскан. Алар илимде Homo sapiens (акылы бар А.) деген атка конгон, неандерталдарга караганда татаалыраак курал жасоого жөндөмдүү болушкан. Алардын дене түзүлүшү менен түр түспөлүндө да туруктуу морфологиялык жана расалык белгилер пайда болгон. Ошентип, <a href="/wiki/%D0%AD%D0%B2%D0%BE%D0%BB%D1%8E%D1%86%D0%B8%D1%8F" title="Эволюция">эволюциялык</a> жана тарыхый өнүгүүнүн натыйжасында калыптанган Адам Жер жүзүндөгү ар түрдүү <a href="/wiki/%D0%9C%D0%B0%D1%82%D0%B5%D1%80%D0%B8%D0%BA%D1%82%D0%B5%D1%80" class="mw-redirect" title="Материктер">материктерде</a> жашап, ошол жердин ысык-суугуна жана тиричилик шарттарына ыңгайлашып отурган. Натыйжада Адамдын сырткы келбети, түспөлү өзгөрүп, ар кайсы жерде түрдүү расалар калыптанган.</p><p>Адамдын органдары жана системалары ткандардан куралат. <a href="/wiki/%D0%A1%D0%BA%D0%B5%D0%BB%D0%B5%D1%82" class="mw-redirect" title="Скелет">Скелет</a>, <a href="/wiki/%D0%9C%D1%83%D1%83%D0%BD_(%D0%B0%D0%BD%D0%B0%D1%82%D0%BE%D0%BC%D0%B8%D1%8F)" title="Муун (анатомия)">муундар</a> жана <a href="/wiki/%D0%91%D1%83%D0%BB%D1%87%D1%83%D2%A3_%D1%82%D0%BA%D0%B0%D0%BD%D1%8B" title="Булчуң тканы">булчуңдар</a> <a href="/wiki/%D0%9A%D1%8B%D0%B9%D0%BC%D1%8B%D0%BB" title="Кыймыл">кыймыл</a> функциясын аткарат. Адамдын тамактанышы тамак сиңирүү органдары аркылуу жүрөт, <a href="/wiki/%D0%94%D0%B5%D0%BC_%D0%B0%D0%BB%D1%83%D1%83" title="Дем алуу">дем алуу</a> жолу жана өпкө аркылуу дем алат. Зат алмашууда пайда болгон зыяндуу заттар <a href="/wiki/%D0%91%D3%A9%D0%B9%D1%80%D3%A9%D0%BA" title="Бөйрөк">бөйрөк</a>, <a href="/wiki/%D0%A2%D0%B5%D1%80%D0%B8" title="Тери">тери</a> аркылуу денеден сыртка чыгарылып, <a href="/wiki/%D0%9A%D0%B0%D0%BD_(%D0%B0%D0%BD%D0%B0%D1%82%D0%BE%D0%BC%D0%B8%D1%8F)" title="Кан (анатомия)">кан курамын</a> туруктуу сактап турат. Ткандарды жана <a href="/wiki/%D0%9A%D0%BB%D0%B5%D1%82%D0%BA%D0%B0" title="Клетка">клеткаларды</a> азык, <a href="/wiki/%D0%9A%D1%8B%D1%87%D0%BA%D1%8B%D0%BB%D1%82%D0%B5%D0%BA" title="Кычкылтек">кычкылтек</a>, гормон менен камсыз кылуу, зыяндуу заттарды бөлүп чыгаруу милдетин <a href="/wiki/%D0%96%D2%AF%D1%80%D3%A9%D0%BA" title="Жүрөк">жүрөк</a>, кантамыр жана <a href="/wiki/%D0%9B%D0%B8%D0%BC%D1%84%D0%B0" title="Лимфа">лимфа</a> системалары аткарат. Жалпы денеде ткань, орган системаларынын бири-бири менен байланышын <a href="/wiki/%D0%91%D0%BE%D1%80%D0%B1%D0%BE%D1%80%D0%B4%D1%83%D0%BA_%D0%BD%D0%B5%D1%80%D0%B2_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0%D1%81%D1%8B" title="Борбордук нерв системасы">борбордук нерв системасы</a> башкарат.</p><p>Адамдын бою <a href="/wiki/%D0%90%D0%BD%D1%82%D1%80%D0%BE%D0%BF%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Антропология">антропологияда</a> жана <a href="/wiki/%D0%9C%D0%B5%D0%B4%D0%B8%D1%86%D0%B8%D0%BD%D0%B0" title="Медицина">медицинада</a> дененин морфологиялык өсүшүн, мүчөлөрдүн бири бирине шайкеш келишин мүнөздөйт. Жаңы төрөлгөн эркек ымыркайдын боюнун орточо узундугу 54 см, кыздыкы 53 см болот. Аялдын боюнун өсүшү 17-18, эркектики 20-23 жашка чыкканда токтолот. Бул процесс акцелерацияга байланыштуу. Адам бою 45 жаштан кийин ар бир он жылда 1-1,5 смге кыскара баштайт. Бул негизинен омуртка кемирчектеринин, тамандын жукарышынан ж. б. себептерден болот. Адамдын конституциясы дененин <a href="/wiki/%D0%A2%D1%83%D0%BA%D1%83%D0%BC_%D0%BA%D1%83%D1%83%D0%B3%D1%83%D1%87%D1%82%D1%83%D0%BA" title="Тукум куугучтук">тукум куума</a> жана көнүмүш касиеттеринин негизинде калыптанган функциялык жана морфологиялык өзгөчөлүктөрү болуп эсептелет.
