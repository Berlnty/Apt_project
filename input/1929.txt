title="Elektrana">Elektrana</a> u <a href="/wiki/Novi_Meksiko" title="Novi Meksiko">Novom Meksiku</a></div></div></div><p><b>Atmosfera</b> (<a href="/wiki/Starogr%C4%8Dki_jezik" title="Starogrčki jezik">grč</a>. <i>ἀτμός</i>: dah, para + <a href="/wiki/Sfera" title="Sfera">sfera</a>) je <a href="/wiki/Plin" class="mw-redirect" title="Plin">plinoviti</a> omotač oko <a href="/wiki/Zemlja_(planet)" class="mw-redirect" title="Zemlja (planet)">Zemlje</a> ili nekog drugog <a href="/wiki/Nebesko_tijelo" title="Nebesko tijelo">nebeskog tijela</a>. Većina planeta ima neki oblik atmosfere, a rjeđe plinoviti omotač mogu imati i neki mjeseci poput <a href="/wiki/Titan_(mjesec)" title="Titan (mjesec)">Titana</a> koji kruži oko <a href="/wiki/Saturn_(planet)" class="mw-redirect" title="Saturn (planet)">Saturna</a>. Atmosfera nije statički sustav oko nebeskih tijela, već se mijenja u vremenu (promatrano na skali <a href="/wiki/Geolo%C5%A1ko_razdoblje" class="mw-redirect" title="Geološko razdoblje">geoloških razdoblja</a> poput <a href="/wiki/Eon_(geologija)" title="Eon (geologija)">eona</a> i <a href="/wiki/Era" class="mw-disambig" title="Era">era</a>). Na primjer, <a href="/wiki/Zemljina_atmosfera" title="Zemljina atmosfera">atmosfera Zemlje</a> u njezinih prvih 500 milijuna godina značajno se razlikovala od današnje, kada su prevladavali otrovnih <a href="/wiki/Sumpor" title="Sumpor">sumporni</a> i <a href="/wiki/Du%C5%A1ik" title="Dušik">dušikovi</a> spojevi. Također, smatra se da je <a href="/wiki/Mars_(planet)" class="mw-redirect" title="Mars (planet)">Mars</a> u prvih nekoliko stotina milijuna godina svoje geološke povijesti imao znatno gušću atmosferu nego li je to danas slučaj, no zbog male mase <a href="/wiki/Planet" class="mw-redirect" title="Planet">planeta</a>, djelovanja <a href="/wiki/Sun%C4%8Dev_vjetar" class="mw-redirect" title="Sunčev vjetar">Sunčeva vjetra</a> i hladnoće veći dio je odnešen u međuplanetarni prostor ili je zamrznut. Atmosfera <a href="/wiki/Sunce" title="Sunce">Sunca</a> i ostalih <a href="/wiki/Zvijezda" title="Zvijezda">zvijezda</a> sastoji se od plinovite mase iznad <a href="/wiki/Fotosfera" title="Fotosfera">fotosfere</a>. Atmosferu imaju i svi planeti <a href="/wiki/Sun%C4%8Dev_sustav" class="mw-redirect" title="Sunčev sustav">Sunčeva sustava</a>, osim <a href="/wiki/Merkur" title="Merkur">Merkura</a>, na kojem plinovi postoje samo u neznatnim količinama. <sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup></p><p></p><div id="toc" class="toc"><div class="toctitle" lang="sh" dir="ltr" title="Uredi sekciju: Zemljina atmosfera">uredi izvor</a><span class="mw-editsection-bracket">]</span></span></h2><dl><dd><div class="noprint relarticle mainarticle"><i>Glavni članak: <a href="/wiki/Zemljina_atmosfera" title="Zemljina atmosfera">Zemljina atmosfera</a></i></div></dd></dl><p>Pod Zemljinoj atmosferom se razumijeva <a href="/wiki/Zrak" class="mw-redirect" title="Zrak">zrak</a> (smjesa plinova koja okružuje Zemlju i sudjeluje u <a href="/wiki/Zemljina_rotacija" title="Zemljina rotacija">njezinoj vrtnji</a>). Osnovni su sastojci zraka: <a href="/wiki/Du%C5%A1ik" title="Dušik">dušik</a> (78,08%), <a href="/wiki/Kisik" class="mw-redirect" title="Kisik">kisik</a> (20,95%), <a href="/wiki/Argon" title="Argon">argon</a> (0,93%), te u promjenjivim količinama <a href="/wiki/Vodena_para" title="Vodena para">vodena para</a> (0 do 4%) i <a href="/wiki/Ugljikov_dioksid" class="mw-redirect" title="Ugljikov dioksid">ugljikov dioksid</a> (0,03%), a u neznatnim količinama <a href="/wiki/Vodik" title="Vodik">vodik</a>, <a href="/wiki/Helij" title="Helij">helij</a>, <a href="/wiki/Ozon" title="Ozon">ozon</a>, <a href="/wiki/Metan" title="Metan">metan</a>, <a href="/wiki/Amonijak" title="Amonijak">amonijak</a>, <a href="/wiki/Ugljikov_monoksid" class="mw-redirect" title="Ugljikov monoksid">ugljikov monoksid</a>, <a href="/wiki/Kripton" title="Kripton">kripton</a> i <a href="/wiki/Ksenon" title="Ksenon">ksenon</a>. Količina vodene pare u atmosferi promjenjiva je, ovisno o temperaturi, a udio se smanjuje i s visinom. U najnižim slojevima atmosfere ima sitnih čestica <a href="/wiki/Anorganska_kemija" class="mw-redirect" title="Anorganska kemija">anorganskoga</a> (<a href="/wiki/Soli" class="mw-redirect" title="Soli">sol</a>, prašina, <a href="/wiki/Pepeo" class="mw-redirect" title="Pepeo">pepeo</a>) i <a href="/wiki/Organska_kemija" class="mw-redirect" title="Organska kemija">organskog</a> podrijetla (<a href="/wiki/Pelud" title="Pelud">pelud</a> i <a href="/wiki/Mikrobiologija" title="Mikrobiologija">mikrobi</a>), te <a href="/wiki/Olovo_(element)" class="mw-redirect" title="Olovo (element)">olovnih</a>, <a href="/wiki/Du%C5%A1ik" title="Dušik">dušikovih</a> i <a href="/wiki/Sumpor" title="Sumpor">sumpornih</a> spojeva i čađe, što je posljedica emisije <a href="/wiki/Industrija" title="Industrija">industrije</a> i <a href="/wiki/Promet" class="mw-redirect" title="Promet">prometa</a>. Pojedini <a href="/wiki/Oksidi" class="mw-redirect" title="Oksidi">oksidi</a> dušika i sumpora veliki su zagađivači, osobito zbog djelovanja <a href="/wiki/Kisele_ki%C5%A1e" class="mw-redirect" title="Kisele kiše">kiselih kiša</a> i utječu na čovjekovo <a href="/wiki/Zdravlje" title="Zdravlje">zdravlje</a>. Posebnu važnost ima ugljikov dioksid, zbog uloge u zagrijavanju atmosfere (<a href="/wiki/Efekt_staklenika" title="Efekt staklenika">efekt staklenika</a>). Količina ugljikovog dioksida posljednjih stotinu godina neprekidno se povećava zbog izgaranja <a href="/wiki/Fosilna_goriva" title="Fosilna goriva">fosilnih goriva</a>. U posljednjih desetak godina porasla je za više od 10%.</p><h3><span id="Razli.C4.8Dita_podru.C4.8Dja_u_atmosferi"></span><span class="mw-headline" id="Različita_područja_u_atmosferi">Različita područja u atmosferi</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Uredi sekciju: Različita područja u atmosferi">uredi - уреди</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;action=edit&amp;section=2" title="Uredi sekciju: Različita područja u atmosferi">uredi izvor</a><span class="mw-editsection-bracket">]</span></span></h3><p>Sastav zraka u bitnome se ne mijenja do visine od oko 90 <a href="/wiki/Metar" title="Metar">km</a>. Taj se sloj naziva homosfera, a iznad te visine dolazi heterosfera. Zbog slabijeg utjecaja Zemljine <a href="/wiki/Gravitacija" title="Gravitacija">gravitacije</a> u njoj prevladavaju lakši plinovi, kao vodik i helij. U najdonjim slojevima atmosfere, u vrlo malim koncentracijama, prisutan je ozon. Na visinama većima od 10 km, a posebno u sloju između 20 i 50 km (<a href="/wiki/Ozonski_omota%C4%8D" title="Ozonski omotač">ozonosfera</a>), <a href="/wiki/Ozon" title="Ozon">ozon</a> nastaje fotodisocijacijom <a href="/wiki/Molekula" class="mw-redirect" title="Molekula">molekularnoga</a> <a href="/wiki/Kisik" class="mw-redirect" title="Kisik">kisika</a> djelovanjem <a href="/wiki/Sun%C4%8Deva_svjetlost" title="Sunčeva svjetlost">Sunčeva zračenja</a>. Najviše koncentracije ozona nalaze se između 20 i 30 km visine i mogu dosegnuti vrijednosti čak od 10 dijelova na milijun (ppm; 10 000 ppm = 1%). Zbog unošenja antropogenih plinova (<a href="/w/index.php?title=Klorofluorougljici&amp;action=edit&amp;redlink=1" class="new" title="Klorofluorougljici (još nenapisan)">klorofluorovodika</a> i drugih), ozonski se sloj povremeno može znatno smanjiti i time znatno povećati <a href="/wiki/Ultraljubi%C4%8Dasto_zra%C4%8Denje" title="Ultraljubičasto zračenje">ultraljubičasto zračenje</a> na površini Zemlje (<a href="/wiki/Ozonske_rupe" title="Ozonske rupe">ozonske rupe</a>). U najdonjih 10 km atmosfere ozon nastaje kao nusprodukt fotokemijskih procesa vezanih uz <a href="/wiki/One%C4%8Di%C5%A1%C4%87avanje_zraka" class="mw-redirect" title="Onečišćavanje zraka">onečišćenje zraka</a> i važan je sastojak fotokemijskoga <a href="/w/index.php?title=Smog&amp;action=edit&amp;redlink=1" class="new" title="Smog (još nenapisan)">smoga</a>, koji štetno djeluje na zdravlje ljudi, životinja i bilja te na materijalna dobra.</p><h4><span class="mw-headline" id="Troposfera">Troposfera</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Uredi sekciju: Troposfera">uredi - уреди</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;action=edit&amp;section=3" title="Uredi sekciju: Troposfera">uredi izvor</a><span class="mw-editsection-bracket">]</span></span></h4><dl><dd><div class="noprint relarticle mainarticle"><i>Glavni članak: <a href="/wiki/Troposfera" title="Troposfera">Troposfera</a></i></div></dd></dl><p>Ovisno o promjeni <a href="/wiki/Temperatura" title="Temperatura">temperature</a> zraka, atmosfera se može podijeliti na više slojeva. U najnižem sloju, troposferi, temperatura zraka opada u prosjeku 6,5 <a href="/wiki/Celzij" class="mw-redirect" title="Celzij">°C</a> na 1 <a href="/wiki/Metar" title="Metar">km</a>. Pojavljuju se <a href="/wiki/Oblak" title="Oblak">oblaci</a>, padaju <a href="/wiki/Oborine" class="mw-redirect" title="Oborine">oborine</a>, pušu <a href="/wiki/Vjetar" class="mw-redirect" title="Vjetar">vjetrovi</a> i zbivaju se sve promjene koje čine <a href="/wiki/Vrijeme_(klima)" title="Vrijeme (klima)">vrijeme</a>. Pri tlu prosječna temperatura zraka na Zemlji iznosi 15 °C, a kada ne bi bilo atmosfere, temperatura bi bila niža za 33 °C. Troposfera u <a href="/wiki/Tropi" title="Tropi">tropskim predjelima</a> doseže u prosjeku visinu od kojih 16 km i na vrhu ima temperaturu od oko –80 °C, u umjerenim i polarnim predjelima ima visinu između 7 i 11 km te na gornjoj granici temperaturu od oko –40 do –60 °C.</p><h4><span class="mw-headline" id="Stratosfera">Stratosfera</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Uredi sekciju: Stratosfera">uredi - уреди</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;action=edit&amp;section=4" title="Uredi sekciju: Stratosfera">uredi izvor</a><span class="mw-editsection-bracket">]</span></span></h4><dl><dd><div class="noprint relarticle mainarticle"><i>Glavni članak: <a href="/wiki/Stratosfera" title="Stratosfera">Stratosfera</a></i></div></dd></dl><p>Iznad troposfere nalazi se 1 do 2 km debeo prijelazni sloj, <a href="/wiki/Tropopauza" title="Tropopauza">tropopauza</a>, gdje se temperatura zraka gotovo ne mijenja, a zatim na visinama sve do 50 km stratosfera, u kojoj temperatura u prosjeku najprije lagano, a potom naglije raste te na svojem vrhu doseže vrijednosti kao pri tlu. U najvišim dijelovima troposfere te u donjim dijelovima stratosfere vjetrovi mogu dosegnuti brzine i do 400 <a href="/wiki/Kilometar_na_sat" title="Kilometar na sat">km/h</a> (<a href="/w/index.php?title=Mlazna_struja&amp;action=edit&amp;redlink=1" class="new" title="Mlazna struja (još nenapisan)">mlazna struja</a>). Na visinama između 20 i 30 km katkad se pojavljuju <a href="/wiki/Polarni_stratosferski_oblaci" title="Polarni stratosferski oblaci">sedefasti oblaci</a>, sastavljeni od čestica <a href="/wiki/Led" title="Led">leda</a>.</p><h4><span class="mw-headline" id="Mezosfera">Mezosfera</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;veaction=edit&amp;section=5" class="mw-editsection-visualeditor" title="Uredi sekciju: Mezosfera">uredi - уреди</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;action=edit&amp;section=5" title="Uredi sekciju: Mezosfera">uredi izvor</a><span class="mw-editsection-bracket">]</span></span></h4><dl><dd><div class="noprint relarticle mainarticle"><i>Glavni članak: <a href="/wiki/Mezosfera" class="mw-disambig" title="Mezosfera">Mezosfera</a></i></div></dd></dl><p>Atmosferski sloj između 50 i 80 km naziva se mezosfera. U njemu temperatura zraka opada s visinom da bi na vrhu sloja dosegnula vrijednosti od –85 do –90 °C. Na visinama između 75 i 90 km mogu se u sjevernim <a href="/wiki/Zemljopisna_%C5%A1irina" class="mw-redirect" title="Zemljopisna širina">zemljopisnim širinama</a> rijetko vidjeti i svjetleći noćni oblaci (srebrnasti oblaci) sastavljeni vjerojatno od <a href="/wiki/Led" title="Led">leda</a> istaložena na <a href="/wiki/Meteor" title="Meteor">meteornoj</a> prašini.</p><h4><span class="mw-headline" id="Termosfera">Termosfera</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;veaction=edit&amp;section=6" class="mw-editsection-visualeditor" title="Uredi sekciju: Termosfera">uredi - уреди</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;action=edit&amp;section=6" title="Uredi sekciju: Termosfera">uredi izvor</a><span class="mw-editsection-bracket">]</span></span></h4><dl><dd><div class="noprint relarticle mainarticle"><i>Glavni članak: <a href="/wiki/Termosfera" title="Termosfera">Termosfera</a></i></div></dd></dl><p>Iznad 80 km pa sve do 500 km proteže se termosfera, u kojoj temperatura ponovno raste s visinom. Sloj između 60 i 400 km naziva se i <a href="/wiki/Ionosfera" title="Ionosfera">ionosfera</a>, zato što postoji više <a href="/wiki/Ion" class="mw-redirect" title="Ion">ioniziranih</a> slojeva zraka nastalih djelovanjem <a href="/wiki/Ultraljubi%C4%8Dasto_zra%C4%8Denje" title="Ultraljubičasto zračenje">ultraljubičastog zračenja</a>, <a href="/wiki/Rendgenske_zrake" class="mw-redirect" title="Rendgenske zrake">rendgenskih zraka</a>, te naglih struja <a href="/wiki/Elektron" title="Elektron">elektrona</a> sa Sunca (<a href="/wiki/Sun%C4%8Dev_vjetar" class="mw-redirect" title="Sunčev vjetar">Sunčev vjetar</a>). U sloju između 60 i 90 km postoji tzv. D-sloj, između 90 i 150 km E-sloj (sloj Heaviside-Kennely) te na visinama između 150 i 400 km F-sloj (Appletonov sloj). Ti slojevi provode <a href="/wiki/Elektri%C4%8Dna_struja" title="Električna struja">električnu struju</a> i reflektiraju <a href="/wiki/Radio_valovi" class="mw-redirect" title="Radio valovi">radio valove</a> sa Zemlje. Prije pojave umjetnih telekomunikacijskih <a href="/wiki/Satelit" class="mw-disambig" title="Satelit">satelita</a>, to je bio jedini mogući način bežičnog prijenosa radio valova na velike udaljenosti. Svjetlo <a href="/wiki/Meteor" title="Meteor">meteora</a> počinje se vidjeti već na visinama od 170 km, dok se <a href="/wiki/Polarna_svjetlost" title="Polarna svjetlost">polarna svjetlost</a> pojavljuje na visinama između 70 i 1000 km.</p><h4><span class="mw-headline" id="Egzosfera">Egzosfera</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;veaction=edit&amp;section=7" class="mw-editsection-visualeditor" title="Uredi sekciju: Egzosfera">uredi - уреди</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;action=edit&amp;section=7" title="Uredi sekciju: Egzosfera">uredi izvor</a><span class="mw-editsection-bracket">]</span></span></h4><dl><dd><div class="noprint relarticle mainarticle"><i>Glavni članak: <a href="/wiki/Egzosfera" title="Egzosfera">Egzosfera</a></i></div></dd></dl><p>Iznad ionosfere, sve do udaljenosti od nekih 9 600 km, proteže se egzosfera; unutar nje na visinama iznad 500 km započinje <a href="/wiki/Magnetosfera" title="Magnetosfera">magnetosfera</a>, u kojoj struje naelektrizirane čestice zraka pod djelovanjem <a href="/wiki/Zemljino_magnetsko_polje" class="mw-redirect" title="Zemljino magnetsko polje">Zemljina magnetskoga polja</a>, a <a href="/wiki/Ionizacija" title="Ionizacija">ionizacija</a> je toliko jaka da čestice čine <a href="/wiki/Plazma" class="mw-disambig" title="Plazma">plazmu</a> (smjesa nabijenih <a href="/wiki/Ion" class="mw-redirect" title="Ion">iona</a> i negativnih <a href="/wiki/Elektron" title="Elektron">elektrona</a>). Posebno je visoka koncentracija nabijenih čestica u sloju između 3000 i 15 000 km (<a href="/w/index.php?title=Van_Allenovi_pojasi_zra%C4%8Denja&amp;action=edit&amp;redlink=1" class="new" title="Van Allenovi pojasi zračenja (još nenapisan)">Van Allenovi pojasi zračenja</a>). Granica magnetosfere različito je udaljena od Zemlje, ovisno o položaju prema Suncu: drži se da se ta granica (magnetopauza) nalazi približno 60 000 km prema Suncu, a daleko više na suprotnoj strani, što pak ovisi o aktivnosti Sunca. Izvan toga prostora jakost magnetskog polja jednaka je jakosti magnetskog polja u interplanetarnom prostoru pa se drži da tu prestaje utjecaj Zemlje i počinje međuplanetarni prostor.</p><h3><span class="mw-headline" id="Fizikalna_svojstva">Fizikalna svojstva</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;veaction=edit&amp;section=8" class="mw-editsection-visualeditor" title="Uredi sekciju: Fizikalna svojstva">uredi - уреди</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;action=edit&amp;section=8" title="Uredi sekciju: Fizikalna svojstva">uredi izvor</a><span class="mw-editsection-bracket">]</span></span></h3><p><a href="/wiki/Gusto%C4%87a_zraka" title="Gustoća zraka">Gustoća zraka</a> brzo opada s visinom: prosječna je vrijednost pri tlu 1.295 <a href="/wiki/Kilogram_po_metru_kubnom" title="Kilogram po metru kubnom">kg/m³</a>, na 1 km visine 1.112 kg/m³, a na 5 km visine gustoća zraka pada već na 0.736 kg/m³. S tim je u svezi i promjena <a href="/wiki/Tlak" title="Tlak">tlaka</a> zraka ovisno o visini: u prosjeku je pri tlu tlak zraka 101.3 k<a href="/w/index.php?title=Pa&amp;action=edit&amp;redlink=1" class="new" title="Pa (još nenapisan)">Pa</a>, na 1 km visine oko 90 kPa, na 5.5 km visine 50 kPa. Na oko 16 km visine tlak zraka iznosi 1/100 tlaka pri tlu, a na visini nešto ispod 50 km 1/1000 tlaka zraka pri tlu, dok je gustoća zraka samo 0.001 kg/m³. Na visinama od nekoliko stotina km zrak je toliko rijedak da pojedine molekule prijeđu put i od više stotina metara, pa i kilometara, prije negoli se sudare s drugom česticom, pa se zato, s obzirom na gustoću zraka, već i na tim visinama može govoriti o zrakopraznom prostoru; zbog toga se gornja granica atmosfere ne može precizno odrediti.</p><p>Na gornjoj granici atmosfere ozračenje (iradijancija) je 1365 do 1373 W/m² (<a href="/wiki/Sun%C4%8Deva_konstanta" title="Sunčeva konstanta">Sunčeva konstanta</a>). Od ukupne Sunčeve energije u godišnjem se prosjeku 30% raspršuje i reflektira natrag u svemir (od toga 20% od oblaka, 4% od Zemljine površine i 6% od zračnih slojeva), dok se 19% energije apsorbira u oblacima i atmosferi. Ostatak od 51% apsorbira Zemljina površina. Od te količine velik se dio (23%) troši za isparivanje vodenih površina, oko 7% gubi se <a href="/w/index.php?title=Prijenos_topline&amp;action=edit&amp;redlink=1" class="new" title="Prijenos topline (još nenapisan)">vođenjem topline</a> i <a href="/wiki/Konvekcija" title="Konvekcija">konvektivnim</a> strujama zraka, a ostatak od 21% izračuje se, kao <a href="/wiki/Toplinsko_zra%C4%8Denje" class="mw-redirect" title="Toplinsko zračenje">toplinsko zračenje</a>, prema atmosferi. Većinu te energije apsorbiraju <a href="/wiki/Vodena_para" title="Vodena para">vodena para</a> u atmosferi, oblaci i <a href="/wiki/Stakleni%C4%8Dki_plinovi" title="Staklenički plinovi">staklenički plinovi</a>, ali se i od te količine najveći dio protuzračenjem vraća prema Zemlji, pa se na taj način u godišnjem prosjeku održava srednja temperatura zraka, uzimajući pritom Zemlju i atmosferu u cjelini gotovo stalnom, odnosno tako se održava toplinska ravnoteža cijeloga sustava Zemlja – atmosfera.</p><p>Mjerenja stanja atmosfere pri tlu provode se u <a href="/wiki/Meteorolo%C5%A1ka_postaja" title="Meteorološka postaja">meteorološkim postajama</a>, na visinama do 40 km radiosondama, <a href="/wiki/Zrakoplov" title="Zrakoplov">zrakoplovima</a> i <a href="/wiki/Balon" title="Balon">balonima</a>, dok se <a href="/wiki/Raketa" title="Raketa">rakete</a>, koje nose meteorološke instrumente i preko <a href="/wiki/Radio" title="Radio">radija</a> javljaju podatke na Zemlju, upotrebljavaju za istraživanja atmosfere do visine od oko 800 km. U novije doba mjerenja stanja atmosfere u cijelome sloju od tla do kakvih 900 km provode se meteorološkim <a href="/wiki/Satelit" class="mw-disambig" title="Satelit">satelitima</a> i satelitima za praćenje stanja čovjekova <a href="/w/index.php?title=Okoli%C5%A1&amp;action=edit&amp;redlink=1" class="new" title="Okoliš (još nenapisan)">okoliša</a>.</p><h3><span id="One.C4.8Di.C5.A1.C4.87avanje_zraka"></span><span class="mw-headline" id="Onečišćavanje_zraka">Onečišćavanje zraka</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;veaction=edit&amp;section=9" class="mw-editsection-visualeditor" title="Uredi sekciju: Onečišćavanje zraka">uredi - уреди</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosfera_(plinoviti_omota%C4%8D)&amp;action=edit&amp;section=9" title="Uredi sekciju: Onečišćavanje zraka">uredi izvor</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="noprint relarticle mainarticle"><i>Glavni članak: <a href="/wiki/One%C4%8Di%C5%A1%C4%87avanje_zraka" class="mw-redirect" title="Onečišćavanje zraka">Onečišćavanje zraka</a></i></div></dd></dl><p>Svaki otklon sastava atmosfere od prirodnih uvjeta zbog prisutnosti drugih plinova, para ili lebdećih čestica u koncentracijama koje ne uzrokuju izravnu štetu za ljude, faunu ili floru zovemo onečišćenjem atmosfere. Ako se ljudskom djelatnošću dostigne ili prekorači granica tolerancije, tako da se prisutnošću stranih tvari u atmosferi ugrožava <a href="/wiki/Biosfera" title="Biosfera">biosfera</a> ili ljudsko <a href="/wiki/Zdravlje" title="Zdravlje">zdravlje</a>, govori se o zagađivanju (zagađenju) atmosfere. Ljudskom djelatnošću u atmosferu dospijevaju različite tvari, među kojima su najvažnije <a href="/wiki/Sumpor" title="Sumpor">sumporni</a> i <a href="/wiki/Du%C5%A1ik" title="Dušik">dušični</a> spojevi, <a href="/wiki/Organski_spojevi" class="mw-redirect" title="Organski spojevi">organski spojevi</a>, halogenidi, <a href="/wiki/Ugljikov_monoksid" class="mw-redirect" title="Ugljikov monoksid">ugljikov monoksid</a>, čestične i <a href="/wiki/Radioaktivnost" title="Radioaktivnost">radioaktivne tvari</a>. Dio tih tvari ne stiže u atmosferu primarnom emisijom, nego nastaje u zagađenoj atmosferi kao posljedica kemijskih ili fotokemijskih reakcija (npr. <a href="/wiki/Ozon" title="Ozon">ozon</a> u nižim slojevima troposfere). Emisija ili nastajanje takvih tvari u atmosferi uzrokuje oštećenja na biljkama i materijalima, štetno djeluje na ljudsko zdravlje i mijenja prirodna svojstva atmosfere. Atmosfera apsorbira glavninu zračenja iz svemira i tako zaštićuje organizme na Zemlji od njegova štetna djelovanja. Također apsorbira glavninu Sunčeva elektromagnetskoga zračenja, propuštajući samo zračenje u području valnih duljina od 300 do 2500 <a href="/wiki/Metar" title="Metar">nm</a> (blisko <a href="/wiki/Ultraljubi%C4%8Dasto_zra%C4%8Denje" title="Ultraljubičasto zračenje">ultraljubičasto</a>, vidljivo i blisko <a href="/wiki/Infracrveno_zra%C4%8Denje" title="Infracrveno zračenje">infracrveno zračenje</a>). Apsorpcijom <a href="/wiki/Elektromagnetsko_zra%C4%8Denje" title="Elektromagnetsko zračenje">elektromagnetskoga zračenja</a> valnih duljina manjih od 300 nm onemogućuje se štetno djelovanje ultraljubičastog zračenja na žive organizme.</p><h2><span class="mw-headline" id="Reference">Reference</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a
