Είναι πιθανότερο η ατμόσφαιρα να συγκρατηθεί από έναν <a href="/wiki/%CE%9F%CF%85%CF%81%CE%AC%CE%BD%CE%B9%CE%BF_%CF%83%CF%8E%CE%BC%CE%B1" title="Ουράνιο σώμα">ουράνιο σώμα</a> όταν η βαρύτητα είναι ισχυρή και η <a href="/wiki/%CE%98%CE%B5%CF%81%CE%BC%CE%BF%CE%BA%CF%81%CE%B1%CF%83%CE%AF%CE%B1" title="Θερμοκρασία">θερμοκρασία</a> της ατμόσφαιρας χαμηλή.</p><p>Η <a href="/wiki/%CE%91%CF%84%CE%BC%CF%8C%CF%83%CF%86%CE%B1%CE%B9%CF%81%CE%B1_%CF%84%CE%B7%CF%82_%CE%93%CE%B7%CF%82" title="Ατμόσφαιρα της Γης">ατμόσφαιρα της Γης</a> αποτελείται κυρίως από <a href="/wiki/%CE%86%CE%B6%CF%89%CF%84%CE%BF" title="Άζωτο">άζωτο</a> και σε μικρότερο βαθμό από <a href="/wiki/%CE%9F%CE%BE%CF%85%CE%B3%CF%8C%CE%BD%CE%BF" title="Οξυγόνο">οξυγόνο</a>, το οποίο χρησιμοποιείται από τους οργανισμούς για την <a href="/wiki/%CE%91%CE%BD%CE%B1%CF%80%CE%BD%CE%BF%CE%AE" title="Αναπνοή">αναπνοή</a>, και <a href="/wiki/%CE%94%CE%B9%CE%BF%CE%BE%CE%B5%CE%AF%CE%B4%CE%B9%CE%BF_%CF%84%CE%BF%CF%85_%CE%AC%CE%BD%CE%B8%CF%81%CE%B1%CE%BA%CE%B1" title="Διοξείδιο του άνθρακα">διοξείδιο του άνθρακα</a>, το οποίο χρησιμοποιείται από τα <a href="/wiki/%CE%A6%CF%85%CF%84%CE%AC" class="mw-redirect" title="Φυτά">φυτά</a>, τα <a href="/wiki/%CE%A6%CF%8D%CE%BA%CE%B7" title="Φύκη">φύκη</a> και τα <a href="/wiki/%CE%9A%CF%85%CE%B1%CE%BD%CE%BF%CE%B2%CE%B1%CE%BA%CF%84%CE%AE%CF%81%CE%B9%CE%B1" title="Κυανοβακτήρια">κυανοβακτήρια</a> για τη <a href="/wiki/%CE%A6%CF%89%CF%84%CE%BF%CF%83%CF%8D%CE%BD%CE%B8%CE%B5%CF%83%CE%B7" title="Φωτοσύνθεση">φωτοσύνθεση</a>. Η ατμόσφαιρα βοηθάει στην προστασία των ζωντανών οργανισμών από γενετικές βλάβες από την ηλιακή <a href="/wiki/%CE%A5%CF%80%CE%B5%CF%81%CE%B9%CF%8E%CE%B4%CE%B7%CF%82_%CE%B1%CE%BA%CF%84%CE%B9%CE%BD%CE%BF%CE%B2%CE%BF%CE%BB%CE%AF%CE%B1" title="Υπεριώδης ακτινοβολία">υπεριώδη ακτινοβολία</a>, τον <a href="/wiki/%CE%97%CE%BB%CE%B9%CE%B1%CE%BA%CF%8C%CF%82_%CE%AC%CE%BD%CE%B5%CE%BC%CE%BF%CF%82" title="Ηλιακός άνεμος">ηλιακό άνεμο</a> και τις <a href="/wiki/%CE%9A%CE%BF%CF%83%CE%BC%CE%B9%CE%BA%CE%AD%CF%82_%CE%B1%CE%BA%CF%84%CE%AF%CE%BD%CE%B5%CF%82" title="Κοσμικές ακτίνες">κοσμικές ακτίνες</a>. Η σημερινή της σύνθεση είναι αποτέλεσμα δισεκατομμυρίων χρόνων βιοχημικών μεταβολών στην παλαιοατμόσφαιρα από τους ζωντανούς οργανισμούς.</p><p>Ο όρος αστρική ατμόσφαιρα περιγράφει την εξωτερική περιοχή ενός <a href="/wiki/%CE%86%CF%83%CF%84%CF%81%CE%BF" class="mw-redirect" title="Άστρο">άστρου</a>, και περιλαμβάνει τυπικά το τμήμα από την αδιαφανή <a href="/wiki/%CE%A6%CF%89%CF%84%CF%8C%CF%83%CF%86%CE%B1%CE%B9%CF%81%CE%B1" class="mw-redirect" title="Φωτόσφαιρα">φωτόσφαιρα</a> και προς τα έξω.
