Erityisesti sana taide viittaa <a href="/wiki/Kuvataide" title="Kuvataide">kuvataiteisiin</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup> </p><p>Taide on yksi <a href="/wiki/Kulttuuri" title="Kulttuuri">kulttuurin</a> peruskäsitteistä. Taidetta voi olla myös sellainen, joka aiheuttaa kokijassaan muitakin kuin <a href="/wiki/Esteettinen" class="mw-redirect" title="Esteettinen">esteettisyyden</a> tunteita. </p><div id="toc" class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Taide&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Muokkaa osiota Taiteen määritelmiä">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Taide&amp;action=edit&amp;section=1" title="Muokkaa osiota Taiteen määritelmiä">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h2><p><a href="/wiki/Immanuel_Kant" title="Immanuel Kant">Immanuel Kantin</a> mukaan ihminen ei voi saada tietoa ympäristön todellisesta olemuksesta, vaan ympäröivä todellisuus ilmentyy <a href="/wiki/Havainto" title="Havainto">havainnoissamme</a> sellaisena, kuin <a href="/wiki/Aisti" title="Aisti">aistimme</a> ne hahmottavat. <a href="/wiki/Estetiikka" title="Estetiikka">Estetiikka</a> ei siis myöskään kerro mitään tarkasteltavasta esineestä itsestään, vaan tarkastelijan tunnepohjaisesta suhtautumisesta esineen <a href="/wiki/Mielikuva" title="Mielikuva">mielikuvaan</a>. Kantin toisen määritelmän mukaan <a href="/wiki/Taideteos" title="Taideteos">taideteoksia</a> voidaan verrata luonnon tuotoksiin: luonnossa kaikki on välttämätöntä seurausta <a href="/wiki/Luonnonlaki" class="mw-redirect" title="Luonnonlaki">luonnonlaeista</a>, ja vaikka taideteokset syntyvät ikään kuin vapaudessa, noudattaa taiteilija niissäkin luonnon määräämiä sääntöjä, ja taide jäljittelee siten luontoa.</p><h3><span class="mw-headline" id="Mimesis">Mimesis</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Taide&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Muokkaa osiota Mimesis">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Taide&amp;action=edit&amp;section=2" title="Muokkaa osiota Mimesis">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p><a href="/w/index.php?title=Taiteen_teoria&amp;action=edit&amp;redlink=1" class="new" title="Taiteen teoria (sivua ei ole)">Taiteen teorian</a> keskeinen käsite on <a href="/wiki/Mimesis" title="Mimesis">mimesis</a>, joka tarkoittaa luonnon tai todellisuuden jäljittelyä ja esittämistä taiteen keinoin. Taideteos voi äärimmillään olla tarkka <a href="/w/index.php?title=J%C3%A4ljenn%C3%B6s&amp;action=edit&amp;redlink=1" class="new" title="Jäljennös (sivua ei ole)">jäljennös</a> jo olemassa olevasta kohteesta tai jotain sellaista, mitä emme kykene edes hahmottamaan. Teoksen ei tarvitse olla kuitenkaan esine, vaan se voi olla vaikkapa <a href="/wiki/Teatteri" title="Teatteri">teatteri-</a>, <a href="/wiki/Musiikki" title="Musiikki">musiikki-</a> tai <a href="/wiki/Performanssi" title="Performanssi">performanssiesitys</a> ja sellaisenaan mahdoton toistaa.</p><p><a href="/wiki/Platon" title="Platon">Platon</a> ajatteli, että taide vain jäljittelee jo olemassa olevia esineitä. Esimerkiksi tehdessään <a href="/wiki/Tuoli" title="Tuoli">tuolia</a> puuseppä jäljittelee <a href="/wiki/Tuoli" title="Tuoli">tuolin</a> <a href="/wiki/Idea" title="Idea">ideaa</a>, kun taas tuolia maalaava <a href="/wiki/Taidemaalari" title="Taidemaalari">taidemaalari</a> jäljittelee jäljitelmää, missä Platonin mukaan ei ole järkeä.</p><h3><span class="mw-headline" id="Mediaominaisuudet">Mediaominaisuudet</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Taide&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Muokkaa osiota Mediaominaisuudet">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Taide&amp;action=edit&amp;section=3" title="Muokkaa osiota Mediaominaisuudet">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Taidetta voi lähestyä tarkastelemalla taideteosten välityskeinoja eli sen <a href="/wiki/Media" title="Media">mediaominaisuuksia</a>. <a href="/wiki/Kuvataide" title="Kuvataide">Kuvataiteen</a> <a href="/wiki/Kuva" title="Kuva">kuva</a>, <a href="/wiki/Musiikki" title="Musiikki">musiikin</a> <a href="/wiki/%C3%84%C3%A4ni" title="Ääni">ääni</a>, <a href="/wiki/Tanssi" title="Tanssi">tanssin</a> <a href="/wiki/Liike_(fysiikka)" title="Liike (fysiikka)">liike</a>, <a href="/wiki/Kirjallisuus" title="Kirjallisuus">kirjallisuuden</a> <a href="/wiki/Sana" title="Sana">sana</a> tai <a href="/wiki/Teatteri" title="Teatteri">teatterin</a> <a href="/w/index.php?title=N%C3%A4ytt%C3%A4m%C3%B6tulkinta&amp;action=edit&amp;redlink=1" class="new" title="Näyttämötulkinta (sivua ei ole)">näyttämötulkinta</a> saattavat kuitenkin rajoittaa tarkastelua, sillä varsinkin <a href="/wiki/Nykytaide" title="Nykytaide">nykytaiteessa</a> taide välittyy useiden keinojen avulla.</p><h3><span class="mw-headline" id="Historia">Historia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Taide&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Muokkaa osiota Historia">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Taide&amp;action=edit&amp;section=4" title="Muokkaa osiota Historia">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Kaikilla taiteilla on <a href="/wiki/Historia" title="Historia">historiansa</a>, joissa tarkastellaan niiden kehitystä. Taiteen tekemiseen on vaikuttanut aina myös <a href="/w/index.php?title=Taiteen_tutkimus&amp;action=edit&amp;redlink=1" class="new" title="Taiteen tutkimus (sivua ei ole)">taiteen tutkimus</a>. <a href="/wiki/Taidesuuntaus" title="Taidesuuntaus">Taidesuuntaukset</a> ovat historian eri aikoina vallinneita <a href="/wiki/Tyyli" title="Tyyli">tyylejä</a>. Taiteen vastaanottajan on hyvä tuntea niitä, sillä tunnetun mietelmän mukaan taide on velkaa toisille taideteoksille usein enemmän kuin todellisuudelle.</p><h3><span id="Teht.C3.A4v.C3.A4"></span><span class="mw-headline" id="Tehtävä">Tehtävä</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Taide&amp;veaction=edit&amp;section=5" class="mw-editsection-visualeditor" title="Muokkaa osiota Tehtävä">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Taide&amp;action=edit&amp;section=5" title="Muokkaa osiota Tehtävä">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Taideteoksella ei tarvitse olla käytännön tehtävää. Silti myös käytännölliset asiat voivat olla taidetta. Teos voi olla esimerkiksi samanaikaisesti sekä esteettisen mielihyvän, että <a href="/wiki/Taidekauppa" title="Taidekauppa">kaupankäynnin</a> kohde. <a href="/wiki/Typografia" title="Typografia">Typografialla</a> on tarkasti määritelty tehtävä, edistää luettavuutta.</p><h3><span id="Esitt.C3.A4v.C3.A4t_ja_ei-esitt.C3.A4v.C3.A4t"></span><span class="mw-headline" id="Esittävät_ja_ei-esittävät">Esittävät ja ei-esittävät</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Taide&amp;veaction=edit&amp;section=6" class="mw-editsection-visualeditor" title="Muokkaa osiota Esittävät ja ei-esittävät">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Taide&amp;action=edit&amp;section=6" title="Muokkaa osiota Esittävät ja ei-esittävät">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Taiteet voidaan jakaa esittäviin ja ei-esittäviin. <a href="/wiki/Esitt%C3%A4v%C3%A4_taide" title="Esittävä taide">Esittävät taiteet</a>, kuten <a href="/wiki/Teatteri" title="Teatteri">teatteri</a>, <a href="/wiki/Elokuva" title="Elokuva">elokuva</a>, <a href="/wiki/Tanssi" title="Tanssi">tanssi</a> tai <a href="/wiki/Musiikki" title="Musiikki">musiikki</a> esitetään ajallisessa jatkumossa ja esitys suunnitellaan siten, että yleisön oletetaan seuraavan esitystä alusta loppuun. Esittävät taiteet perustuvat esityksiin, joilla on tavallisesti selkeä alku, keskikohta ja loppu.</p><p><a href="/wiki/Arkkitehtuuri" title="Arkkitehtuuri">Arkkitehtuuri</a>, <a href="/wiki/Maalaustaide" title="Maalaustaide">maalaustaide</a>, <a href="/wiki/Grafiikka" class="mw-redirect" title="Grafiikka">grafiikka</a>, <a href="/wiki/Kuvanveisto" title="Kuvanveisto">kuvanveisto</a>, <a href="/wiki/Muotoilu" title="Muotoilu">muotoilu</a> tai <a href="/wiki/Kirjallisuus" title="Kirjallisuus">kirjallisuuden</a> eri lajit eivät perustu aikaan sidottuihin esityksiin, vaan katsoja tai lukija valitsee ja päättää itse katselutilanteen pituuden tai lukemisajankohdan. Tämän takia niitä kutsutaan ei-esittäviksi taiteiksi.</p><p>Esittäviä ja ei-esittäviä taiteenaloja voidaan myös yhdistää esimerkiksi <a href="/wiki/Performanssi" title="Performanssi">performanssissa</a> tai <a href="/wiki/Videotaide" title="Videotaide">videotaiteessa</a>. </p><div class="thumb tright"><div class="thumbinner" style="width:247px;"><a class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Taide&amp;veaction=edit&amp;section=7" class="mw-editsection-visualeditor" title="Muokkaa osiota Tekijät">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Taide&amp;action=edit&amp;section=7" title="Muokkaa osiota Tekijät">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Taiteen <a href="/wiki/Tekij%C3%A4_(tekij%C3%A4noikeus)" title="Tekijä (tekijänoikeus)">tekijöitä</a> ovat <a href="/wiki/Taiteilija" title="Taiteilija">taiteilijat</a>. Sana on yleisnimitys eri taiteita tekeville. Termiä voidaan käyttää esimerkiksi musiikin, kirjallisuuden, teatterin, kuvataiteen tai muotoilun harjoittajasta. Kaikilla taiteen aloilla on tunnettuja ja tunnustettuja tekijöitä, jotka ovat saavuttaneet vakiintuneita <a href="/wiki/Yleis%C3%B6" title="Yleisö">yleisöjä</a>. Useimmilla taiteenaloilla on oma ammatillinen koulutus. Ensimmäisenä varsinaisena taidekouluna pidetään <a href="/wiki/Giorgio_Vasari" title="Giorgio Vasari">Giorgio Vasarin</a> Firenzeen vuonna 1563 perustamaa <i><a href="/wiki/Accademia_di_belle_arti_di_Firenze" title="Accademia di belle arti di Firenze">Accademia del Disegno</a></i>a. </p><h2><span id="Taiteenlajit.2C_historia_ja_teoriat"></span><span class="mw-headline" id="Taiteenlajit,_historia_ja_teoriat">Taiteenlajit, historia ja teoriat</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Taide&amp;veaction=edit&amp;section=8" class="mw-editsection-visualeditor" title="Muokkaa osiota Taiteenlajit, historia ja teoriat">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Taide&amp;action=edit&amp;section=8" title="Muokkaa osiota Taiteenlajit, historia ja teoriat">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h2><p>Taiteenlajit ovat syntyneet ja vakiintuneet <a href="/wiki/Historia" title="Historia">historian</a> kuluessa eri taiteen muodoista. Esimerkiksi <a href="/wiki/Baletti" title="Baletti">baletti</a> syntyi Italiassa <a href="/wiki/Renessanssi" title="Renessanssi">renessanssin</a> aikaan 1400-luvulla. Alussa balettia esitettiin vain pieninä <a href="/wiki/Episodi" title="Episodi">episodeina</a> <a href="/wiki/Ooppera" title="Ooppera">oopperoissa</a> tai musiikkiesityksissä, mutta se siirtyi myöhemmin Ranskaan, alkaen kukoistaa itsenäisenä ilmiönä kuninkaallisten hoveissa. Itsenäiseksi lajiksi baletti muodostui vasta 1700-luvun toisella puoliskolla.</p><p>Jokaisella taiteenlajilla on historia ja <a href="/wiki/Estetiikka" title="Estetiikka">estetiikka</a>. Tekeminen ja <a href="/wiki/Reseptio" title="Reseptio">vastaanotto</a> tapahtuu tavallisesti vakiintuneiden esteettisten <a href="/wiki/Normi" class="mw-disambig" title="Normi">normien</a> mukaan ja niitä muuttamalla taiteenlajit muuttuvat. Historian joinakin aikoina taiteenlajien väliset rajat ovat jyrkkiä, joskus ne madaltuvat. Esimerkiksi aikaisemmin korostettiin <a href="/wiki/Valokuva" title="Valokuva">valokuvan</a> ja maalaustaiteen erilaisuutta, kun taas nykytaiteessa niiden välinen raja on kaventunut.</p><p>Myös taiteen teoriaa on pidetty tärkeänä. Esimerkiksi <a href="/wiki/Arthur_Danto" title="Arthur Danto">Arthur Danto</a> määritteli <a href="/wiki/Taidemaailma" title="Taidemaailma">taidemaailman</a> käsitteen vuonna 1964, jossa hänen mukaansa: ”Taiteen teorioiden rooli, niin tänään kuin aina ennenkin, on aikaansaada taidemaailma ja taide mahdolliseksi.”</p><h3><span class="mw-headline" id="Kuvataiteet">Kuvataiteet</span><span
