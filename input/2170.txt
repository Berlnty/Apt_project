1758</small></td></tr></table></div><p><b>Bod dynol</b> yw gair <a href="/wiki/Gwyddoniaeth" title="Gwyddoniaeth">y gwyddonydd</a> am <a href="/wiki/Dyn" title="Dyn">ddyn</a>. Person ydyw (neu "greadur deallus") sy'n perthyn i rywogaeth <i><a href="/wiki/Homo_sapiens" title="Homo sapiens">Homo sapiens</a></i>; 'human' yn Saesneg. Mae bod dynol yn greadur <a href="/w/index.php?title=Creaduriaid_deudroed&amp;action=edit&amp;redlink=1" class="new" title="Creaduriaid deudroed (does dim tudalen o'r enw hwn i gael)">deudroed</a> ac wedi'i osod yn y teulu biolegol a elwir yn <a href="/wiki/Hominidae" title="Hominidae">Hominidae</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup><sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup> Mae tystiolaeth genynnol o ran <a href="/wiki/DNA" title="DNA">DNA</a> yn dangos mai cynefin neu darddle bodau dynol yw <a href="/wiki/Affrica" title="Affrica">Affrica</a>, ac iddynt ddod o'r fan honno tua 200,000 o flynydoedd yn ôl.</p><p>Mae gan fodau dynol <a href="/wiki/Ymennydd" title="Ymennydd">ymennydd</a> sydd wedi datblygu llawer pellach na gweddill <a href="/wiki/Anifeiliaid" class="mw-redirect" title="Anifeiliaid">anifeiliaid</a>, yn fiolegol felly. Gall <a href="/wiki/Rheswm" class="mw-disambig" title="Rheswm">resymoli</a> yn haniaethol, gall ymwneud ag <a href="/wiki/Iaith" title="Iaith">iaith</a>, <a href="/w/index.php?title=Mewnsyllu&amp;action=edit&amp;redlink=1" class="new" title="Mewnsyllu (does dim tudalen o'r enw hwn i gael)">mewnsyllu</a> a datrus problemau fel mae nhw'n codi. Mae'r gallu ymenyddol hwn ynghyd â chorff fertigol, gyda breichiau rhydd i symud neu ddal a thrin erfyn yn ei alluogi i ddefnyddio arfau llawer mwy nag unrhyw anifail arall i amddiffyn ei hun, i weithio am fwyd neu i ymosod.</p><p>Mae bodau dynol wedi eu dosbarthu ar hyd a lled y ddaear ar wahân i <a href="/wiki/Antartig" class="mw-redirect" title="Antartig">Antartig</a>. Mae poblogaeth y ddaear bellach yn fwy na 6.7&#160;biliwn, (data Gorffennaf, 2008).<sup id="cite_ref-popclock_3-0" class="reference"><a href="#cite_note-popclock-3">[3]</a></sup> Un isrywogaeth sydd ar gael: <i>Homo sapiens sapiens</i>. Mae'n <a href="/wiki/Mamal" title="Mamal">famal</a> ac felly'n bwydo ei blant gyda <a href="/wiki/Llaeth" title="Llaeth">llaeth</a>.</p><p>Fel <a href="/wiki/Epa" title="Epa">epaod</a>, mae dyn yn gymdeithasol ei natur. Mae wedi mireinio'r grefft o gyfathrebu'n effeithiol er mwyn iddo fynegi ei hun, cyfnewid syniadau a threfnu gweithgareddau. Mae wedi creu strwythurau cymdeithasol cywrain a chymhleth o grwpiau sy'n cydweithio ac yn cystadlu yn erbyn ei gilydd; grwpiau mor wahanol â'r <a href="/wiki/Teulu" title="Teulu">teulu</a> a <a href="/wiki/Cenedl" title="Cenedl">chenhedloedd</a>. Mae'r cydymwneud hwn rhwng dyn a dyn wedi sefydlu dros y milenias diwethaf <a href="/w/index.php?title=Traddodiad&amp;action=edit&amp;redlink=1" class="new" title="Traddodiad (does dim tudalen o'r enw hwn i gael)">draddodiadau</a>, <a href="/wiki/Defod" title="Defod">defodau</a>, <a href="/wiki/Moesau" class="mw-redirect" title="Moesau">moesau</a>, <a href="/w/index.php?title=Gwerthoedd&amp;action=edit&amp;redlink=1" class="new" title="Gwerthoedd (does dim tudalen o'r enw hwn i gael)">gwerthoedd</a> a safon dderbyniol gan gymdeithas drwy gyfundrefn o <a href="/w/index.php?title=Deddf&amp;action=edit&amp;redlink=1" class="new" title="Deddf (does dim tudalen o'r enw hwn i gael)">ddeddfau</a>. Gall dyn werthfawrogi <a href="/w/index.php?title=Harddwch&amp;action=edit&amp;redlink=1" class="new" title="Harddwch (does dim tudalen o'r enw hwn i gael)">harddwch</a> ac <a href="/wiki/Estheteg" title="Estheteg">estheteg</a> sydd ynghyd â'r awydd i fynegi ei hun wedi arwain at <a href="/wiki/Celfyddyd" title="Celfyddyd">gelfyddyd</a>, <a href="/wiki/Llenyddiaeth" title="Llenyddiaeth">llenyddiaeth</a> a <a href="/wiki/Cerddoriaeth" title="Cerddoriaeth">cherddoriaeth</a>.</p><h2><span class="mw-headline" id="Cynhanes_Cymru">Cynhanes Cymru</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bod_dynol&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Golygu&#039;r adran: Cynhanes Cymru">golygu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Bod_dynol&amp;action=edit&amp;section=1" title="Golygu&#039;r adran: Cynhanes Cymru">golygu cod y dudalen</a><span class="mw-editsection-bracket">]</span></span></h2><p>Gellir dweud bod tri math o fod dynol yng Nghymru yn ystod y cyfnod cynhanes (a phrotohanes):</p><ol><li>Dyn Neanderthal cynnar yn <a href="/wiki/Ogof_Bontnewydd" title="Ogof Bontnewydd">Ogof Bontnewydd</a>, ger <a href="/wiki/Llanelwy" title="Llanelwy">Llanelwy</a> - 225,000 <a href="/wiki/CP" title="CP">CP</a></li><li>Dyn Neanderthal clasurol yn <a href="/wiki/Ogof_Coygan" title="Ogof Coygan">Ogof Coygan</a>, <a href="/wiki/Sir_Gaerfyrddin" title="Sir Gaerfyrddin">Sir Gaerfyrddin</a> - 50,000 CP</li><li>Bod dynol modern yn <a href="/wiki/Ogof_Paviland" title="Ogof Paviland">Ogof Paviland</a>, <a href="/wiki/G%C5%B5yr" class="mw-disambig" title="Gŵyr">Gŵyr</a> - 26,000 CP</li></ol><p>Gwyddom hyn gan fod eu hesgyrn wedi'u canfod yno;
