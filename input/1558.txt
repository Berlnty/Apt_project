class="firstHeading" lang="ko">예술</h1>			<div id="bodyContent" class="mw-body-content">				<div id="siteSub" class="noprint">위키백과, 우리 모두의 백과사전.</div>				<div id="contentSub"></div>								<div id="jump-to-nav" class="mw-jump">					이동:					<a href="#mw-head">둘러보기</a>, 					<a href="#p-search">검색</a>				</div>				<div id="mw-content-text" lang="ko" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><p><b>예술</b>(藝術)은 <a href="/wiki/%ED%95%99%EB%AC%B8" title="학문">학문</a>·<a href="/wiki/%EC%A2%85%EA%B5%90" title="종교">종교</a>·<a href="/wiki/%EB%8F%84%EB%8D%95" title="도덕">도덕</a> 등과 같은 <a href="/wiki/%EB%AC%B8%ED%99%94" title="문화">문화</a>의 한 부문으로, 예술 활동(창작, 감상)과 그 성과(예술 작품)의 총칭이다. <a href="/wiki/%EB%AC%B8%ED%95%99" title="문학">문학</a>, <a href="/wiki/%EC%9D%8C%EC%95%85" title="음악">음악</a>, <a href="/wiki/%EB%AF%B8%EC%88%A0" title="미술">미술</a>, <a href="/wiki/%EC%98%81%ED%99%94" title="영화">영화</a>, <a href="/wiki/%EB%AC%B4%EC%9A%A9" class="mw-redirect" title="무용">무용</a> 등의 공연예술이 포함된다. 그러나 이러한 예술작품을 다루는 학문은 <a href="/wiki/%EC%9D%B8%EB%AC%B8%ED%95%99" title="인문학">인문학</a>의 영역이다.</p><p>예술은 사람들을 결합시키고 사람들에게 감정이나 사상을 전달하는 수단이 된다. 과학도 같은 구실을 하기는 하나, 과학은 주로 개념으로 설명하고 예술은 미적 형상(美的形象)으로 설명한다. 예술의 중심 개념은 ‘아름다움’으로서, 만약 미가 결핍되거나 상실되면 예술이라고 말할 수 없다. 그러나 아름다움만으로는 예술이라고 할 수 없으며 어떤 ‘형상’에 의해 표현되어야만 한다. 예술관을 크게 나누면 <a href="/wiki/%EC%95%84%EC%9D%B4%EB%94%94%EC%96%BC%EB%A6%AC%EC%A6%98" class="mw-redirect" title="아이디얼리즘">아이디얼리즘</a> 또는 <a href="/wiki/%EB%A1%9C%EB%A7%A8%ED%8B%B0%EC%8B%9C%EC%A6%98" class="mw-redirect" title="로맨티시즘">로맨티시즘</a>과 <a href="/wiki/%EB%A6%AC%EC%96%BC%EB%A6%AC%EC%A6%98" class="mw-redirect" title="리얼리즘">리얼리즘</a>이 된다. 전자를 대표하는 사람은 헤겔로서 ‘미’는 예술가의 주관적 공상이라고 보는 데 대해, 후자를 주장하는 자는 아리스토텔레스로서 미를 자연의 모방, 혹은 재현이라고 본다. 이 대립은 오늘날까지 계속되고 있다.</p><p>예술학은 예술에 관한 학문으로, 예술 철학과 구별하여 예술을 과학적으로 연구하는 의미로도 쓰인다. 예술 사회학은 예술의 사회학적 연구로서 <a href="/wiki/19%EC%84%B8%EA%B8%B0" title="19세기">19세기</a> 중엽에 일어났던 <a href="/wiki/%EC%8B%A4%EC%A6%9D%EC%A3%BC%EC%9D%98" title="실증주의">실증주의</a>(實證主義)와 함께 발달했다.</p><h2><span id=".EB.B6.84.EB.A5.98"></span><span class="mw-headline" id="분류">분류</span><span
