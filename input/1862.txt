Těleso může být obklopeno atmosférou pouze za předpokladu, že má dostatečnou <a href="/wiki/Hmotnost" title="Hmotnost">hmotnost</a> na to, aby plyn vázalo <a href="/wiki/Gravita%C4%8Dn%C3%AD_s%C3%ADla" class="mw-redirect" title="Gravitační síla">gravitační silou</a>. V&#160;případě některých plynných sloučenin musí být splněna i&#160;další podmínka – dostatečně nízká <a href="/wiki/Teplota" title="Teplota">teplota</a>. Atmosféru Země tvoří z&#160;21&#160;% <a href="/wiki/Kysl%C3%ADk" title="Kyslík">kyslík</a>, ze 78&#160;% <a href="/wiki/Dus%C3%ADk" title="Dusík">dusík</a> a 1&#160;% zabírají <a href="/wiki/Vz%C3%A1cn%C3%A9_plyny" title="Vzácné plyny">vzácné plyny</a> a ostatní prvky.</p><p><a href="/wiki/Hustota" title="Hustota">Hustota</a> plynu je nepřímo úměrná <a href="/wiki/Hmotnost" title="Hmotnost">hmotnosti</a> tělesa. Lehčí plyn (např. <a href="/wiki/Vod%C3%ADk" title="Vodík">vodík</a>) neunikne do <a href="/wiki/Vesm%C3%ADr" title="Vesmír">vesmíru</a>, je-li vázán vyšší gravitační silou. To je případ <a href="/wiki/Plynn%C3%BD_obr" title="Plynný obr">plynných obrů</a> ve <a href="/wiki/Slune%C4%8Dn%C3%AD_soustava" title="Sluneční soustava">sluneční soustavě</a>, např. <a href="/wiki/Jupiter_(planeta)" title="Jupiter (planeta)">Jupitera</a>.</p><p>Atmosféra je důležitou ochranou života před <a href="/wiki/Kosmick%C3%A9_z%C3%A1%C5%99en%C3%AD" title="Kosmické záření">kosmickým zářením</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup></p><p></p><div id="toc" class="toc"><div class="toctitle" lang="cs" dir="ltr" class="mw-editsection-bracket">]</span></span></h3><dl><dd><i>Související informace naleznete také v článku <a href="/wiki/Atmosf%C3%A9ra_Merkuru" title="Atmosféra Merkuru">Atmosféra Merkuru</a>.</i></dd></dl><p>Merkur má velmi tenkou atmosféru, složenou z&#160;atomů vyražených z&#160;jeho povrchu <a href="/wiki/Slune%C4%8Dn%C3%AD_v%C3%ADtr" title="Sluneční vítr">slunečním větrem</a>, což je zapříčiněno slabým gravitačním polem vytvářeným poměrně lehkou planetou.<sup id="cite_ref-stra114_2-0" class="reference"><a href="#cite_note-stra114-2">[2]</a></sup> Protože je povrch Merkura velmi horký, tyto atomy rychle unikají do vesmíru. Takže oproti <a href="/wiki/Zem%C4%9B" title="Země">Zemi</a> nebo <a href="/wiki/Venu%C5%A1e_(planeta)" title="Venuše (planeta)">Venuši</a>, jejichž atmosféry jsou stabilní, Merkurova atmosféra je proměnlivá a musí být neustále doplňována. <a href="/wiki/Tlak" title="Tlak">Tlak</a> atmosféry na povrchu je menší než 5,10<sup>−10</sup> <a href="/wiki/Pascal_(jednotka)" title="Pascal (jednotka)">Pa</a>, tedy v&#160;pozemských měřítkách ultravysoké <a href="/wiki/Vakuum" title="Vakuum">vakuum</a>, daleko vyšší tlak má i&#160;vakuum v&#160;běžné žárovce.<sup id="cite_ref-merkurnasa_3-0" class="reference"><a href="#cite_note-merkurnasa-3">[3]</a></sup><sup id="cite_ref-atmosféra_4-0" class="reference"><a href="#cite_note-atmosféra-4">[4]</a></sup></p><div class="thumb tleft"><div class="thumbinner" style="width:192px;"><a href="/wiki/Soubor:Venus-real_color.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e5/Venus-real_color.jpg/190px-Venus-real_color.jpg" width="190" height="190" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e5/Venus-real_color.jpg/285px-Venus-real_color.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e5/Venus-real_color.jpg/380px-Venus-real_color.jpg 2x" data-file-width="480" data-file-height="480" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Soubor:Venus-real_color.jpg" class="internal" title="Zvětšit"></a></div>Snímek atmosféry Venuše v&#160;pravých barvách. Viditelné jsou pouze svrchní vrcholky mračen obepínající planetu.</div></div></div><h3><span id="Atmosf.C3.A9ra_Venu.C5.A1e"></span><span class="mw-headline" id="Atmosféra_Venuše">Atmosféra Venuše</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosf%C3%A9ra&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Editace sekce: Atmosféra Venuše">editovat</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosf%C3%A9ra&amp;action=edit&amp;section=3" title="Editace sekce: Atmosféra Venuše">editovat zdroj</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><i>Související informace naleznete také v článku <a href="/wiki/Atmosf%C3%A9ra_Venu%C5%A1e" title="Atmosféra Venuše">Atmosféra Venuše</a>.</i></dd></dl><p>Současná představa o&#160;struktuře atmosféry Venuše se zakládá na měřeních uskutečněných sondami typu <a href="/wiki/Program_Ven%C4%9Bra" title="Program Veněra">Veněra</a>, <a href="/wiki/Program_Mariner" title="Program Mariner">Mariner</a>, <a href="/wiki/Program_Pioneer" title="Program Pioneer">Pioneer-Venus</a>, pozemskými pozorováními a teoretickými modely, které umožňují odhadovat chování atmosféry. Venuše je obklopena hustou vrstvou atmosféry, která je tvořená převážně z&#160;<a href="/wiki/Oxid_uhli%C4%8Dit%C3%BD" title="Oxid uhličitý">oxidu uhličitého</a>, dále pak malého množství <a href="/wiki/Dus%C3%ADk" title="Dusík">dusíku</a>, <a href="/wiki/Kysl%C3%ADk" title="Kyslík">kyslíku</a> a <a href="/wiki/Vodn%C3%AD_p%C3%A1ra" title="Vodní pára">vodní páry</a>. Kombinace těchto plynů má za následek vznik silného <a href="/wiki/Sklen%C3%ADkov%C3%BD_efekt" title="Skleníkový efekt">skleníkového efektu</a>, který zvyšuje <a href="/wiki/Teplota" title="Teplota">teplotu</a> <a href="/wiki/Povrch_Venu%C5%A1e" title="Povrch Venuše">povrchu</a> o&#160;více než 400&#160;°C, v&#160;oblastech okolo <a href="/wiki/Zemsk%C3%BD_rovn%C3%ADk" class="mw-redirect" title="Zemský rovník">rovníku</a> dokonce až o&#160;500&#160;°C.</p><h3><span id="Atmosf.C3.A9ra_Zem.C4.9B"></span><span class="mw-headline" id="Atmosféra_Země">Atmosféra Země</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosf%C3%A9ra&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Editace sekce: Atmosféra Země">editovat</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosf%C3%A9ra&amp;action=edit&amp;section=4" title="Editace sekce: Atmosféra Země">editovat zdroj</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><i>Související informace naleznete také v článku <a href="/wiki/Atmosf%C3%A9ra_Zem%C4%9B" title="Atmosféra Země">Atmosféra Země</a>.</i></dd></dl><ul><li>Země má relativně hustou atmosféru složenou ze 78&#160;% <a href="/wiki/Dus%C3%ADk" title="Dusík">dusíku</a>, 21&#160;% <a href="/wiki/Kysl%C3%ADk" title="Kyslík">kyslíku</a>, 1&#160;% <a href="/wiki/Argon" title="Argon">argonu</a> a stopového množství jiných plynů včetně <a href="/wiki/Oxid_uhli%C4%8Dit%C3%BD" title="Oxid uhličitý">oxidu uhličitého</a> a <a href="/wiki/Voda" title="Voda">vodních</a> par. Atmosféra chrání povrch Země před dopadem některých druhů <a href="/wiki/Slune%C4%8Dn%C3%AD_z%C3%A1%C5%99en%C3%AD" class="mw-redirect" title="Sluneční záření">slunečního záření</a>. Její složení je nestabilní a silně ovlivněno biosférou. Jde především o&#160;velké množství volného dvouatomového kyslíku, který vyrábějí pozemské <a href="/wiki/Rostliny" title="Rostliny">rostliny</a> a bez nichž by se kyslík v&#160;atmosféře v&#160;geologicky krátkém čase sloučil s&#160;materiály z&#160;povrchu Země. Volný kyslík v&#160;atmosféře je známkou života. Atmosféru Země dělíme na několik vrstev:</li><li><a href="/wiki/Troposf%C3%A9ra" title="Troposféra">Troposféra</a> – v&#160;troposféře se nachází naše civilizace, a veškerý inteligentní život na Zemi</li><li><a href="/wiki/Stratosf%C3%A9ra" title="Stratosféra">Stratosféra</a> – stratosféra se nachází nad troposférou, od 7–17&#160;km až do 50&#160;km. Teplota stoupá s&#160;nadmořskou výškou.</li><li><a href="/wiki/Mezosf%C3%A9ra" title="Mezosféra">Mezosféra</a> – sahá od 50 km až do 80–85 km.Teplota zde s&#160;nadmořskou výškou naopak klesá.</li><li><a href="/wiki/Termosf%C3%A9ra" title="Termosféra">Termosféra</a> – největší část atmosféry zaujímá termosféra, která sahá od 80–85&#160;km až do 500-1000&#160;km. Teplota zde s&#160;narůstající výškou opět stoupá. Od výšek nad 100&#160;km se utváří např. <a href="/wiki/Pol%C3%A1rn%C3%AD_z%C3%A1%C5%99e" title="Polární záře">polární záře</a>.</li><li><a href="/wiki/Exosf%C3%A9ra" title="Exosféra">Exosféra</a> – nejvyšší vrstva atmosféry sahající do 10–35 tis. km. Plynule navazuje na meziplanetární prostor, kde již gravitace Země nedokáže molekuly plynu trvaleji udržet.</li></ul><h3><span id="Atmosf.C3.A9ra_Marsu"></span><span class="mw-headline" id="Atmosféra_Marsu">Atmosféra Marsu</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosf%C3%A9ra&amp;veaction=edit&amp;section=5" class="mw-editsection-visualeditor" title="Editace sekce: Atmosféra Marsu">editovat</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosf%C3%A9ra&amp;action=edit&amp;section=5" title="Editace sekce: Atmosféra Marsu">editovat zdroj</a><span class="mw-editsection-bracket">]</span></span></h3><div class="thumb tright"><div class="thumbinner" style="width:182px;"><a href="/wiki/Soubor:Mars_atmosphere.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/7d/Mars_atmosphere.jpg/180px-Mars_atmosphere.jpg" width="180" height="202" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/7d/Mars_atmosphere.jpg/270px-Mars_atmosphere.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/7d/Mars_atmosphere.jpg/360px-Mars_atmosphere.jpg 2x" data-file-width="1205" data-file-height="1349" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Soubor:Mars_atmosphere.jpg" class="internal" title="Zvětšit"></a></div>Nad povrchem Marsu je při bočním pohledu viditelná atmosféra (snímek družice Mars Global Surveyor)</div></div></div><dl><dd><i>Související informace naleznete také v článku <a href="/wiki/Atmosf%C3%A9ra_Marsu" title="Atmosféra Marsu">Atmosféra Marsu</a>.</i></dd></dl><p>Mars má dnes velmi řídkou atmosféru, která není schopná zadržovat tepelnou výměnu mezi povrchem a okolním prostorem, což má za následek velké tepelné rozdíly během dne a noci. Tlak na povrchu se pohybuje mezi 600 až 1000&#160;Pa, což je přibližně 100 až 150krát méně než na povrchu Země či jako přibližně ve 30&#160;km nad jejím povrchem. Podobně jako na Zemi ale dochází ke změnám v&#160;atmosféře v&#160;závislosti na sezónních výkyvech, jak se planeta přibližuje a oddaluje od Slunce. V&#160;zimě 25–30&#160;% atmosférického oxidu uhličitého zmrzne na pólech, zatímco v&#160;létě opět <a href="/wiki/Sublimace" title="Sublimace">sublimuje</a> a vrátí se do atmosféry.</p><p>Atmosféra je tvořena převážně z&#160;<a href="/wiki/Oxid_uhli%C4%8Dit%C3%BD" title="Oxid uhličitý">oxidu uhličitého</a> (95,32&#160;%), dále obsahuje: <a href="/wiki/Dus%C3%ADk" title="Dusík">dusík</a> (2,7&#160;%), <a href="/wiki/Argon" title="Argon">argon</a> (1,6&#160;%), <a href="/wiki/Kysl%C3%ADk" title="Kyslík">kyslík</a> (0,13&#160;%), <a href="/wiki/Oxid_uhelnat%C3%BD" title="Oxid uhelnatý">oxid uhelnatý</a> (0,07&#160;%) a <a href="/wiki/Vodn%C3%AD_p%C3%A1ra" title="Vodní pára">vodní páry</a> (0,03&#160;%),<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup> která vzniká sublimací z&#160;<a href="/wiki/Pol%C3%A1rn%C3%AD_%C4%8Depi%C4%8Dky_Marsu" title="Polární čepičky Marsu">polárních čepiček</a>. Mezi ostatní plyny vyskytující se v&#160;atmosféře se pak ještě řadí <a href="/wiki/Neon" title="Neon">neon</a>, <a href="/wiki/Krypton" title="Krypton">krypton</a>, <a href="/wiki/Xenon" title="Xenon">xenon</a>, <a href="/wiki/Ozon" title="Ozon">ozón</a> a <a href="/wiki/Methan" title="Methan">metan</a> (který je možným indikátorem života na Marsu, jelikož podléhá rychlému rozpadu<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup>).</p><h2><span class="mw-headline" id="Odkazy">Odkazy</span><span class="mw-editsection"><span
