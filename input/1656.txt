class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Konst&amp;action=edit&amp;section=1" title="Redigera avsnitt: Bakgrund">redigera wikitext</a><span class="mw-editsection-bracket">]</span></span></h2><h3><span class="mw-headline" id="Historik">Historik</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Konst&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Redigera avsnitt: Historik">redigera</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Konst&amp;action=edit&amp;section=2" title="Redigera avsnitt: Historik">redigera wikitext</a><span class="mw-editsection-bracket">]</span></span></h3><p>Det svenska ordet härstammar ifrån <a href="/wiki/Fornl%C3%A5gtyska" class="mw-redirect" title="Fornlågtyska">fornlågtyskans</a> <i>kunst</i>, med koppling till verbet för <i>kunna</i>.<sup id="cite_ref-ne-ordbok_2-0" class="reference"><a href="#cite_note-ne-ordbok-2"><span class="cite-reference-link-bracket">[</span>2<span class="cite-reference-link-bracket">]</span></a></sup> Det <a href="/wiki/Engelska" title="Engelska">engelska</a> ordet för konst – <i>art</i> – härstammar istället ifrån <a href="/wiki/Latin" title="Latin">latinets</a> <i>ars</i>, vilket ungefär betyder 'arrangemang' eller 'arrangera'.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3"><span class="cite-reference-link-bracket">[</span>3<span class="cite-reference-link-bracket">]</span></a></sup></p><p>I slutet av <a href="/wiki/Medeltiden" title="Medeltiden">medeltiden</a> betecknades alla mer etablerade <a href="/wiki/Hantverk" title="Hantverk">hantverk</a> för <i>konster</i>, och de var större konst ju mer krävande de var. Ett väl utarbetat och egenartat verk kallades <i>konststycke</i> (<a href="/wiki/Tyska" title="Tyska">tyska</a>: <i>Kunststück</i><sup id="cite_ref-4" class="reference"><a href="#cite_note-4"><span class="cite-reference-link-bracket">[</span>a<span class="cite-reference-link-bracket">]</span></a></sup>).<sup id="cite_ref-ne_1-1" class="reference"><a href="#cite_note-ne-1"><span class="cite-reference-link-bracket">[</span>1<span class="cite-reference-link-bracket">]</span></a></sup></p><p>Det dröjde till in på 1700-talet innan konst började definiera summan av ett antal uttrycksformer, där <a href="/wiki/Bildkonst" title="Bildkonst">bildkonst</a>, <a href="/wiki/Arkitektur" title="Arkitektur">arkitektur</a> och <a href="/wiki/Konsthantverk" title="Konsthantverk">konsthantverk</a> – men även <a href="/wiki/Musik" title="Musik">musik</a>, <a href="/wiki/Opera" title="Opera">opera</a>, <a href="/wiki/Teater" title="Teater">teater</a>, <a href="/wiki/Dans" title="Dans">dans</a> och <a href="/wiki/Litteratur" title="Litteratur">litteratur</a> – ingick. Samtidigt förutsatte definitionen en viss sorts nyskapande. Ett <i>konstverk</i> i betydelsen bildkonst är alltid ett viss <a href="/wiki/Fysik" title="Fysik">fysiskt</a> föremål.<sup id="cite_ref-ne_1-2" class="reference"><a href="#cite_note-ne-1"><span class="cite-reference-link-bracket">[</span>1<span class="cite-reference-link-bracket">]</span></a></sup></p><h3><span class="mw-headline" id="Konstbegreppet_idag">Konstbegreppet idag</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Konst&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Redigera avsnitt: Konstbegreppet idag">redigera</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Konst&amp;action=edit&amp;section=3" title="Redigera avsnitt: Konstbegreppet idag">redigera wikitext</a><span class="mw-editsection-bracket">]</span></span></h3><p>Det avgörande värdet i ett konstverk är inte själva föremålet som föremål. Konstvärdet kommer sig av egenskaperna som antingen är kopplade till konstnärens avsikt eller som kan ge en viss upplevelse hos konstkonsumenten.<sup id="cite_ref-ne_1-3" class="reference"><a href="#cite_note-ne-1"><span class="cite-reference-link-bracket">[</span>1<span class="cite-reference-link-bracket">]</span></a></sup></p><p>Idag används oftast begreppet i anknytning till vad som på 1700-talet började benämnas som <a href="/wiki/De_sk%C3%B6na_konsterna" title="De sköna konsterna">de sköna konsterna</a><sup id="cite_ref-ne_1-4" class="reference"><a href="#cite_note-ne-1"><span class="cite-reference-link-bracket">[</span>1<span class="cite-reference-link-bracket">]</span></a></sup> (jämför också <a href="/wiki/De_fria_konsterna" title="De fria konsterna">de fria konsterna</a>). Dessa omfattar bland annat <a href="/wiki/Bildkonst" title="Bildkonst">bildkonst</a>, <a href="/wiki/Arkitektur" title="Arkitektur">arkitektur</a>, <a href="/wiki/Konsthantverk" title="Konsthantverk">konsthantverk</a>, <a href="/wiki/Skulptur" title="Skulptur">skulptur</a> och <a href="/wiki/Musik" title="Musik">musik</a>, det vill säga skapande verksamhet som syftar till emotionella eller expressiva uttryck – eller resultatet av en sådan verksamhet. Konsten hade då tappat sin praktiska funktion<sup id="cite_ref-ne_1-5" class="reference"><a href="#cite_note-ne-1"><span class="cite-reference-link-bracket">[</span>1<span class="cite-reference-link-bracket">]</span></a></sup> utan handlade allt mer om känsla och estetik.</p><p>Denna användning av begreppet är en utveckling ifrån <a href="/wiki/Antiken" title="Antiken">antikens</a> tanke om <a href="/wiki/De_sju_fria_konsterna" class="mw-redirect" title="De sju fria konsterna">de sju fria konsterna</a> (på latin <i>septem artes liberales</i>) vilka var akademiska ämnen som exempelvis <a href="/wiki/Grammatik" title="Grammatik">grammatik</a> och <a href="/wiki/Retorik" title="Retorik">retorik</a> men även musik. Konsten var då starkt förknippat med ett hantverk. Utifrån detta perspektiv omfattade begreppet konst även senare mer handfasta företeelser som <a href="/wiki/Ingenj%C3%B6rskonst" class="mw-redirect" title="Ingenjörskonst">ingenjörskonst</a> eller <a href="/wiki/Stridskonst" title="Stridskonst">stridskonst</a>. I samma anda som den senare användningen brukas begreppet exempelvis fortfarande för <a href="/wiki/Cirkus" title="Cirkus">cirkuskonster</a> och <a href="/wiki/Kampsport" title="Kampsport">kampsport</a> som på engelska kallas <i>martial arts</i>.</p><p>Med konst avses ofta <a href="/wiki/Konstverk" title="Konstverk">konstverken</a> och deras kvaliteter. När man i stället talar om <i>konstarter</i> eller <i>konstens arter</i> är det de kreativa aktiviteter som ligger bakom dessa verk som avhandlas. Denna artikel handlar alltså i första hand om <i>kunnandet</i> – den inomdisciplinära kunskapen – och, som en följd därav, blir det snarare <a href="/wiki/Konstn%C3%A4r" title="Konstnär">konstnärernas</a> perspektiv som dominerar än publikens eller konstkritikernas; det är snarare traditionellt kunnande än historieskrivning och bildförståelse som beskrivs.</p><h2><span class="mw-headline" id="Konstarter">Konstarter</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Konst&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Redigera avsnitt: Konstarter">redigera</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Konst&amp;action=edit&amp;section=4" title="Redigera avsnitt: Konstarter">redigera wikitext</a><span class="mw-editsection-bracket">]</span></span></h2><p>Här redogörs helt kort för de konstarter som brukar kallas <a href="/wiki/De_sk%C3%B6na_konsterna" title="De sköna konsterna">de sköna konsterna</a> (jämför också <a href="/wiki/De_fria_konsterna" title="De fria konsterna">de fria konsterna</a>). class="mw-editsection-bracket">]</span></span></h3><ul><li><a href="/wiki/Konsthantverk" title="Konsthantverk">Konsthantverk</a></li></ul><h2><span class="mw-headline" id="Konstbegreppets_historiska_utveckling">Konstbegreppets historiska utveckling</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Konst&amp;veaction=edit&amp;section=11" class="mw-editsection-visualeditor" title="Redigera avsnitt: Konstbegreppets historiska utveckling">redigera</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Konst&amp;action=edit&amp;section=11" title="Redigera avsnitt: Konstbegreppets historiska utveckling">redigera wikitext</a><span class="mw-editsection-bracket">]</span></span></h2><p>Begreppet "konst" har haft olika betydelser under olika <a href="/wiki/Epok" title="Epok">epoker</a>. Tolkningen och användningen av begreppet skiftar heller inte bara över tid, utan också mellan individer och kulturer.</p><h3><span class="mw-headline" id="Antiken_och_medeltiden">Antiken och medeltiden</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Konst&amp;veaction=edit&amp;section=12" class="mw-editsection-visualeditor" title="Redigera avsnitt: Antiken och medeltiden">redigera</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Konst&amp;action=edit&amp;section=12" title="Redigera avsnitt: Antiken och medeltiden">redigera wikitext</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="noprint"><i>Fördjupning: <a href="/wiki/Konstbegreppet_i_klassisk_filosofi" title="Konstbegreppet i klassisk filosofi">Konstbegreppet i klassisk filosofi</a></i></div></dd></dl><p>I <a href="/wiki/Antiken" title="Antiken">antikens</a> <a href="/wiki/Grekland" title="Grekland">Grekland</a> fanns inget konstbegrepp i dagens mening.<sup id="cite_ref-ne_1-6" class="reference"><a href="#cite_note-ne-1"><span class="cite-reference-link-bracket">[</span>1<span class="cite-reference-link-bracket">]</span></a></sup> Hos dem stod konstbegreppet för praktiskt kunnande, satt i kontrast till teoretiska kunskaper. Termen för detta var <i>techne</i> (τεχνη), det vill säga kunnande, teknik.</p><p>Konst var sådant som man lärde sig genom praktisk utövning eller utövandes produkter, teoretisk kunskap däremot krävde inget utövande – hade man förstått hade man vetandet. Man skilde också på sådana aktiviteter som resulterade i verkliga ting och sådana som producerade bilder av ting. Ända fram till <a href="/wiki/1700-talet" title="1700-talet">1700-talet</a> kallades sådant avbildande för just konst.</p><p>Kopplingen mellan konst och <a href="/wiki/Vetenskap" title="Vetenskap">vetenskap</a> under medeltiden och <a href="/wiki/Ren%C3%A4ssansen" title="Renässansen">renässansen</a> exemplifierades bland annat av <a href="/wiki/Leonardo_da_Vinci" title="Leonardo da Vinci">Leonardo da Vinci</a> och <a href="/wiki/Albrecht_D%C3%BCrer" title="Albrecht Dürer">Albrecht Dürer</a>,<sup id="cite_ref-ne_1-7" class="reference"><a href="#cite_note-ne-1"><span class="cite-reference-link-bracket">[</span>1<span class="cite-reference-link-bracket">]</span></a></sup> människor som blev framgångsrika inom bägge fälten</p><p>I <a href="/wiki/Sverige" title="Sverige">Sverige</a> introducerades begreppet i denna <a href="/wiki/Klassisk" title="Klassisk">klassiska</a> betydelse först under <a href="/wiki/Medeltiden" title="Medeltiden">medeltiden</a> då allt kvalificerat <a href="/wiki/Hantverk" title="Hantverk">hantverk</a> betraktades som "konster", vilket också vanligen innebar att de importerades från <a href="/wiki/Tyskland" title="Tyskland">Tyskland</a>.</p><h3><span class="mw-headline" id="Upplysningstiden">Upplysningstiden</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Konst&amp;veaction=edit&amp;section=13" class="mw-editsection-visualeditor" title="Redigera avsnitt: Upplysningstiden">redigera</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Konst&amp;action=edit&amp;section=13" title="Redigera avsnitt: Upplysningstiden">redigera wikitext</a><span class="mw-editsection-bracket">]</span></span></h3><p>Under 1700-talet, i och med <a href="/wiki/Upplysningen" class="mw-redirect" title="Upplysningen">upplysningen</a>, fick konstbegreppet gradvis en ny innebörd. Innan dess bedömde man konstverk efter deras överensstämmelse med naturen – konstverket lyftes upp från hantverkets domän och kom att allt mer piedestalsättas. Även <a href="/wiki/Konstn%C3%A4r" title="Konstnär">konstnärens</a> roll utvecklades från hantverkarens till "geniets"; från <a href="/wiki/Skr%C3%A5" class="mw-redirect" title="Skrå">skråets</a> kollektiv till den fria individens. Nu omtalades även själva konstobjekten som konst, inte bara kunnandet. <a href="/wiki/Estetik" title="Estetik">Estetik</a> som självständig <a href="/wiki/Filosofi" title="Filosofi">filosofisk</a> disciplin växte fram, och diskussionen om vad som <i>egentligen</i> är konst inleddes.</p><h3><span class="mw-headline" id="Modern_tid">Modern tid</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Konst&amp;veaction=edit&amp;section=14" class="mw-editsection-visualeditor" title="Redigera avsnitt: Modern tid">redigera</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Konst&amp;action=edit&amp;section=14" title="Redigera avsnitt: Modern tid">redigera wikitext</a><span class="mw-editsection-bracket">]</span></span></h3><p><a href="/wiki/1900-talet" title="1900-talet">1900-talets</a> <a href="/wiki/Postmodernism" title="Postmodernism">postmodernism</a> förkastade idén om att konsten skulle ha ett egenvärde eller "inre kärna och mening". Detta har förändrat synen på konstens "äkthet" och frågan om "god" och "dålig" konst. En enda <a href="/wiki/Definition" title="Definition">definition</a> av konstbegreppet anses allmänt inte längre möjlig att åstadkomma, åtminstone inte utifrån estetiska kriterier eller <a href="/wiki/Konstverk" title="Konstverk">konstverken</a> själva. Ett samtida konstbegrepp är det så kallade <a href="/wiki/Institutionell_konstteori" title="Institutionell konstteori">institutionella konstbegreppet</a>, som bygger på <a href="/wiki/Cirkelargument" class="mw-redirect" title="Cirkelargument">cirkelargumentet</a>, att det som konstvärlden behandlar och benämner som konst är konst. I dagligt språkbruk tenderar frågan "är det här konst?" snarare betyda "är detta <i>bra</i><sup id="cite_ref-ne_1-8" class="reference"><a href="#cite_note-ne-1"><span class="cite-reference-link-bracket">[</span>1<span class="cite-reference-link-bracket">]</span></a></sup> konst?".</p><h2><span class="mw-headline" id="Vetenskaper_som_studerar_konst">Vetenskaper som
