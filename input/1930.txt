title="Titan (mesiac)">Atmosféra Titanu</a></div></div></div><p><b>Atmosféra</b> je vo všeobecnosti <a href="/wiki/Plyn" title="Plyn">plynný</a> obal kozmického telesa, napríklad <a href="/wiki/Plan%C3%A9ta" title="Planéta">planéty</a>, <a href="/wiki/Mesiac_(dru%C5%BEica)" title="Mesiac (družica)">mesiaca</a>, <a href="/wiki/Kom%C3%A9ta" title="Kométa">kométy</a> alebo <a href="/wiki/Hviezda" title="Hviezda">hviezdy</a>. Pri telese je pridržiavaná jeho <a href="/wiki/Gravit%C3%A1cia" title="Gravitácia">gravitáciou</a>. Najlepšie preskúmaná je <a href="/wiki/Atmosf%C3%A9ra_Zeme" title="Atmosféra Zeme">atmosféra Zeme</a>.</p><p>Atmosféra <a href="/wiki/Terestri%C3%A1lna_plan%C3%A9ta" title="Terestriálna planéta">terestrických telies</a> (telies s pevným povrchom) je zospodu ohraničená kôrou planéty (<a href="/wiki/Litosf%C3%A9ra" title="Litosféra">litosférou</a>) a zhora sa pozvoľne rozplýva do okolitého <a href="/wiki/Kozmick%C3%BD_priestor" class="mw-disambig" title="Kozmický priestor">kozmického priestoru</a>. Atmosféra <a href="/wiki/Hviezda" title="Hviezda">hviezd</a>, rovnako ako atmosféra <a href="/wiki/Jovi%C3%A1lna_plan%C3%A9ta" title="Joviálna planéta">joviálnych planét</a> je na rozdiel od atmosféry kamenných telies špecifická tým, že nie je možné presne definovať hranicu medzi atmosférou a vlastným telesom. Joviálne planéty sú zložené predovšetkým z plynu, ktorý postupne smerom k jadru telesa hustne, až prechádza do kvapalného <a href="/wiki/Skupenstvo" title="Skupenstvo">skupenstva</a> a neskôr do tuhého. Hviezdy sú tvorené <a href="/wiki/Plazma_(fyzika)" title="Plazma (fyzika)">plazmou</a>, ktorá je vo väčšej hĺbke čoraz hustejšia. Ako hranica medzi atmosférou hviezdy a jej vlastným telesom sa používa sféra, kedy už hustnúca plazma prestane byť priehľadná pre <a href="/wiki/Fot%C3%B3n" title="Fotón">fotóny</a> viditeľného <a href="/wiki/Svetlo" class="mw-disambig" title="Svetlo">svetla</a>. Táto sféra sa nazýva <a href="/wiki/Fotosf%C3%A9ra" title="Fotosféra">fotosféra</a>.</p><p>Ani atmosféra pevných telies nie je izolovaný systém. Vstupuje do interakcie hlavne s povrchom planéty a s jej <a href="/wiki/Magnetosf%C3%A9ra" title="Magnetosféra">magnetosférou</a>. Niektoré vlastnosti, napríklad výšku hornej hranice atmosféry ovplyvňuje <a href="/wiki/Hviezdny_vietor" class="mw-redirect" title="Hviezdny vietor">hviezdny vietor</a>. Atmosféru pevných telies možno z hľadiska <a href="/wiki/Elektrick%C3%BD_n%C3%A1boj" title="Elektrický náboj">elektrického náboja</a> rozdeliť na <i><a href="/w/index.php?title=Neutrosf%C3%A9ra&amp;action=edit&amp;redlink=1" class="new" title="Neutrosféra (stránka neexistuje)">neutrosféru</a></i>, ktorá obsahuje hlavne elektricky neutrálne častice a <i><a href="/wiki/Ionosf%C3%A9ra" title="Ionosféra">ionosféru</a></i>, ktorá sa nachádza nad ňou a obsahuje veľa <a href="/wiki/I%C3%B3n" class="mw-disambig" title="Ión">iónov</a>. Atmosféra hviezd sa obvykle delí na tri základné vrstvy: fotosféru (niekedy ešte nezaraďovanú k atmosfére), <a href="/wiki/Chromosf%C3%A9ra" title="Chromosféra">chromosféru</a> a <a href="/wiki/Kor%C3%B3na" title="Koróna">korónu</a>.</p><p>Nenulový <a href="/wiki/Atmosf%C3%A9rick%C3%BD_tlak" title="Atmosférický tlak">atmosférický tlak</a> je podmienkou pre výskyt <a href="/wiki/Voda" title="Voda">vody</a> v <a href="/wiki/Kvapalina" title="Kvapalina">kvapalnom</a> skupenstve a teda aj nepriamo podmienkou výskytu <a href="/wiki/%C5%BDivot" title="Život">života</a> na kozmickom telese.</p><p></p><div id="toc" class="toc"><div class="toctitle" lang="sk" dir="ltr" xml:lang="sk"><h2>Obsah</h2></div><ul><li class="toclevel-1 tocsection-1"><a class="mw-headline" id="Atmosféry_telies_slnečnej_sústavy">Atmosféry telies slnečnej sústavy</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Upraviť sekciu: Atmosféry telies slnečnej sústavy">upraviť</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;action=edit&amp;section=1" title="Upraviť sekciu: Atmosféry telies slnečnej sústavy">upraviť zdroj</a><span class="mw-editsection-bracket">]</span></span></h2><p>Atmosféru majú všetky <a href="/wiki/Jovi%C3%A1lna_plan%C3%A9ta" title="Joviálna planéta">joviálne planéty</a> slnečnej sústavy, teda <a href="/wiki/Jupiter" title="Jupiter">Jupiter</a>, <a href="/wiki/Saturn" title="Saturn">Saturn</a>, <a href="/wiki/Ur%C3%A1n_(plan%C3%A9ta)" title="Urán (planéta)">Urán</a> a <a href="/wiki/Nept%C3%BAn" title="Neptún">Neptún</a>. Aj všetky <a href="/wiki/Terestri%C3%A1lna_plan%C3%A9ta" title="Terestriálna planéta">terestriálne planéty</a> slnečnej sústavy, čiže <a href="/wiki/Merk%C3%BAr" title="Merkúr">Merkúr</a>, <a href="/wiki/Venu%C5%A1a" title="Venuša">Venuša</a>, <a href="/wiki/Zem" title="Zem">Zem</a> a <a href="/wiki/Mars" title="Mars">Mars</a> majú atmosféru. Atmosféra Merkúru je však taká riedka (jeho atmosférický tlak je len 2.10<sup>-7</sup> Pa), že sa o ňom niekedy hovorí ako o telese bez atmosféry. Najhustejšiu atmosféru z terestriálnych planét má Venuša, ktorej atmosférický tlak na povrchu je až 9321,9 <a href="/wiki/Pascal_(jednotka)" class="mw-redirect" title="Pascal (jednotka)">kPa</a>.</p><p>Atmosféra bola zistená aj na mnohých <a href="/wiki/Mesiac_(dru%C5%BEica)" title="Mesiac (družica)">prirodzených obežniciach</a> veľkých planét, hlavne na <a href="/wiki/Galileove_mesiace" title="Galileove mesiace">Galileiho mesiacoch</a> Jupitera, ako aj na Saturnovom mesiaci <a href="/wiki/Titan_(mesiac)" title="Titan (mesiac)">Titan</a>, ktorý má vôbec najhustejšiu atmosféru zo všetkých mesiacov. Jeho atmosféra je hustejšia ako atmosféra Zeme. Vo všeobecnosti môže mať atmosféru každé teleso, ktorého <a href="/wiki/%C3%9Anikov%C3%A1_r%C3%BDchlos%C5%A5" title="Úniková rýchlosť">úniková rýchlosť</a> na povrchu je menšia ako rýchlosť <a href="/wiki/Teplo" title="Teplo">tepelného</a> pohybu molekúl. Z tohto dôvodu má riedku atmosféru aj <a href="/wiki/Pluto" class="mw-redirect" title="Pluto">Pluto</a>, hoci je menší (a má teda aj menšiu gravitáciu), ako <a href="/wiki/Mesiac" title="Mesiac">Mesiac</a>, ktorý je bez atmosféry. Niektoré telesá majú atmosféru, resp. hustejšiu atmosféru, len dočasne, keď sa na svojej <a href="/wiki/Obe%C5%BEn%C3%A1_dr%C3%A1ha" title="Obežná dráha">dráhe</a> približujú k Slnku. Vtedy sa pod vplyvom zvýšeného slnečného žiarenia časť látok tvoriacich ich povrch <a href="/wiki/Vyparovanie" title="Vyparovanie">vyparuje</a> alebo <a href="/wiki/Sublim%C3%A1cia" class="mw-disambig" title="Sublimácia">sublimuje</a> do atmosféry, atmosféra sa zahusťuje a tým zvyšuje atmosférický tlak. Pri vzďaľovaní sa od Slnka tieto vrstvy atmosféry opäť zamrznú a stanú sa súčasťou povrchu. Príkladom telesa so sezónne sa meniacou atmosférou je Neptúnov mesiac <a href="/wiki/Triton_(mesiac)" class="mw-redirect" title="Triton (mesiac)">Triton</a>, ale aj kométy, ktorých dočasná atmosféra je nazývaná <i><a href="/wiki/Koma_(astron%C3%B3mia)" title="Koma (astronómia)">koma</a></i>.</p><h2><span id="Vlastnosti_atmosf.C3.A9ry"></span><span class="mw-headline" id="Vlastnosti_atmosféry">Vlastnosti atmosféry</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Upraviť sekciu: Vlastnosti atmosféry">upraviť</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;action=edit&amp;section=2" title="Upraviť sekciu: Vlastnosti atmosféry">upraviť zdroj</a><span class="mw-editsection-bracket">]</span></span></h2><p>Bez ohľadu na chemické zloženie majú plynné obaly pevných telies niektoré spoločné vlastnosti. Všeobecne atmosféra pôsobí ako tepelný regulátor, takže čím hustejšiu atmosféru teleso má, tým sú jeho rozdiely medzi <a href="/wiki/De%C5%88" title="Deň">dennou</a> a <a href="/wiki/Noc" title="Noc">nočnou</a> teplotou menšie.</p><p>Atmosféra <a href="/wiki/Absorpcia_(%C5%BEiarenie)" class="mw-disambig" title="Absorpcia (žiarenie)">absorbuje</a> a rozptyľuje časť <a href="/wiki/Elektromagnetick%C3%A9_%C5%BEiarenie" title="Elektromagnetické žiarenie">elektromagnetického žiarenia</a>, ktoré planéta dostáva od Slnka (resp. inej hviezdy). Prepúšťa len určité oblasti <a href="/wiki/Elektromagnetick%C3%A9_spektrum" title="Elektromagnetické spektrum">spektra</a>, ktoré sa nazývajú <i>atmosférické okná</i>. Táto vlastnosť atmosféry je dôležitá pri ochrane <a href="/wiki/%C5%BDiv%C3%A1_bytos%C5%A5" title="Živá bytosť">živých organizmov</a> pred nebezpečnými zložkami žiarenia z kozmu.</p><p>Aj časť žiarenia, ktorá sa celkom nepohltí, je pri prechode atmosférou oslabená. Preto sú nebeské telesá, napríklad hviezdy pri pozorovaní cez atmosféru slabšie, ako pri pozorovaní z <a href="/wiki/Vesm%C3%ADr" title="Vesmír">vesmíru</a>. K najmenšiemu skresleniu jasnosti telies dochádza v <a href="/wiki/Zenit" title="Zenit">zenite</a>, k najväčšiemu v blízkosti <a href="/wiki/Obzor" class="mw-disambig" title="Obzor">obzoru</a>. Svetlo sa v atmosfére tiež láme a odkláňa sa od svojho pôvodného smeru, preto sú hviezdy pri pozorovaní cez atmosféru mierne posunuté oproti svojim skutočným polohám a to smerom k zenitu. Tento jav sa nazýva <a href="/w/index.php?title=Astronomick%C3%A1_refrakcia&amp;action=edit&amp;redlink=1" class="new" title="Astronomická refrakcia (stránka neexistuje)">astronomická refrakcia</a>.</p><p>Atmosféra tiež chráni povrch telesa pred <a href="/wiki/Meteoroid" title="Meteoroid">meteoroidmi</a> a to tým účinnejšie, čím je hustejšia. Telesá vznikajúce do atmosféry veľkými rýchlosťami sa zrážajú s časticami atmosféry, v dôsledku čoho sa začnú prudko zahrievať a vyparovať. Tento proces sa nazýva <i><a href="/wiki/Abl%C3%A1cia_(astron%C3%B3mia)" title="Ablácia (astronómia)">ablácia</a></i> a vytvorí okolo pôvodného telesa žiariaci kužeľ ionizovaných častíc, ktorý nazývame <a href="/wiki/Meteor" title="Meteor">meteor</a>. Ak sa pri tomto deštruktívnom procese nevyparí teleso celé a nejaká jeho časť dopadne na povrch planéty alebo mesiaca, dopadnuté teleso sa nazýva <a href="/wiki/Meteorit" title="Meteorit">meteorit</a>. Veľké meteority vytvárajú <a href="/wiki/Impaktn%C3%BD_kr%C3%A1ter" title="Impaktný kráter">impaktné krátery</a>. Telesá s hustou atmosférou majú všeobecne menej impaktných kráterov na povrchu ako telesá s riedkou atmosférou alebo bez atmosféry, pretože veľa telies pri prelete atmosférou úplne zhorí alebo sa ich veľkosť výrazne zredukuje. Menšie množstvo impaktných kráterov však môžu mať telesá aj vďaka <a href="/wiki/Er%C3%B3zia" title="Erózia">erózii</a> alebo <a href="/wiki/Sope%C4%8Dn%C3%A1_%C4%8Dinnos%C5%A5" title="Sopečná činnosť">sopečnej činnosti</a>.</p><h2><span id="Vznik_a_v.C3.BDvoj_atmosf.C3.A9r"></span><span class="mw-headline" id="Vznik_a_vývoj_atmosfér">Vznik a vývoj atmosfér</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Upraviť sekciu: Vznik a vývoj atmosfér">upraviť</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;action=edit&amp;section=3" title="Upraviť sekciu: Vznik a vývoj atmosfér">upraviť zdroj</a><span class="mw-editsection-bracket">]</span></span></h2><h3><span id="Vznik_atmosf.C3.A9ry"></span><span class="mw-headline" id="Vznik_atmosféry">Vznik atmosféry</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Upraviť sekciu: Vznik atmosféry">upraviť</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;action=edit&amp;section=4" title="Upraviť sekciu: Vznik atmosféry">upraviť zdroj</a><span class="mw-editsection-bracket">]</span></span></h3><p>Pri plynných a plazmových telesách je atmosféra ich prirodzenou súčasťou už od čias ich formovania. V prípade joviálnych planét hovoríme o tzv. <b>prvotnej atmosfére</b>, čo znamená, že vznikla už pri <a href="/wiki/Vznik_a_v%C3%BDvoj_slne%C4%8Dnej_s%C3%BAstavy" title="Vznik a vývoj slnečnej sústavy">formovaní</a> z týchto planét z <a href="/wiki/Protoplanet%C3%A1rny_disk" title="Protoplanetárny disk">protoplanetárneho disku</a>. Chemické zloženie týchto atmosfér sa od čias ich vzniku zmenilo len málo. Pevné telesá nadobudli svoj vzdušný obal uvoľnením plynov zo svojho vnútra, dopadom asteroidov a komét obsahujúcich prchavé látky, alebo gravitačným nabalením materiálu z okolia. O týchto atmosférach sa predpokladá, že podstúpili dlhý vývoj a ich chemické zloženie sa podstatne líši od ich pôvodného zloženia po sformované slnečnej sústavy. Takéto vekom značne zmenené atmosféry sa nazývajú <b>druhotné atmosféry</b>. Druhotné atmosféry sú napríklad značne ochudobnené o <a href="/wiki/Vod%C3%ADk" title="Vodík">vodík</a> a <a href="/wiki/H%C3%A9lium" title="Hélium">hélium</a>, pretože gravitácia telies nepostačovala na udržanie si týchto dvoch najľahších chemických prvkov.</p><h3><span id="Strata_atmosf.C3.A9ry"></span><span class="mw-headline" id="Strata_atmosféry">Strata atmosféry</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;veaction=edit&amp;section=5" class="mw-editsection-visualeditor" title="Upraviť sekciu: Strata atmosféry">upraviť</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;action=edit&amp;section=5" title="Upraviť sekciu: Strata atmosféry">upraviť zdroj</a><span class="mw-editsection-bracket">]</span></span></h3><h4><span id="Term.C3.A1lny_.C3.BAnik"></span><span class="mw-headline" id="Termálny_únik">Termálny únik</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;veaction=edit&amp;section=6" class="mw-editsection-visualeditor" title="Upraviť sekciu: Termálny únik">upraviť</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;action=edit&amp;section=6" title="Upraviť sekciu: Termálny únik">upraviť zdroj</a><span class="mw-editsection-bracket">]</span></span></h4><p>Viaceré telesá v histórii slnečnej sústavy možno mali atmosféru, ale postupne ju čiastočne alebo úplne stratili. Plyn sa z atmosféry stráca, keď <a href="/wiki/R%C3%BDchlos%C5%A5" class="mw-disambig" title="Rýchlosť">rýchlosť</a> jeho častíc presiahne únikovú rýchlosť. To sa stáva najčastejšie pod vplyvom vysokej teploty - vtedy hovoríme o takzvanom termálom úniku. Čím slabšiu gravitáciu má teleso, tým nižšia teplota je dostačujúca na to, aby z nej častice unikali. Významnú úlohu hrá hmotnosť častíc. Najľahšie unikajú voľné atómy najľahšieho plynu - vodíka. Hranica, pri ktorej vodík nezadržateľne opúšťa zemskú atmosféru, je približne 500 km a teplota, pri ktorej k tomu dochádza, nad 1000 <a href="/wiki/Kelvin" title="Kelvin">kelvinov</a>. Hoci priemerná rýchlosť atómov vodíka v tejto výške a pri tejto teplote je stále len polovica z druhej únikovej rýchlosti, častice atmosféry v takejto výške sú už také riedke, že v ich rýchlostiach sú obrovské rozdiely, ktoré sa nestíhajú vyrovnávať. Preto majú niektoré častice už vyššiu rýchlosť, ako je úniková rýchlosť. Zemskú atmosféru opúšťajú každú sekundu tri kilogramy vodíka a 50 gramov hélia (druhý najľahší plyn). Unikajúci vodík možno vidieť na ultrafialových snímkach ako veľké <a href="/wiki/Halo" class="mw-disambig" title="Halo">halo</a> obaľujúce Zem. Tento spôsob postupného "vyparovania sa" atmosféry sa nazýva Jeansov únik. Predpokladá sa, že práve Jeansov únik je vo veľkej miere zodpovedný za to, že o svoju atmosféru prišiel Mesiac.</p><p>Ďalším spôsobom termálneho úniku je strata atmosféry pod vplyvom UV žiarenia hviezdy. Atmosféra planéty absorbuje UV žiarenie a premení ho na teplo. Teplo spôsobí rozpínanie atmosféry, ktoré napokon vyústi do vypudenia celého oblaku molekúl do kozmu. Tento spôsob úniku plynov z atmosféry sa označuje ako hydrodynamický a plyn, ktorý takto opúšťa atmosféru planéty sa nazýva <a href="/w/index.php?title=Planet%C3%A1rny_vietor&amp;action=edit&amp;redlink=1" class="new" title="Planetárny vietor (stránka neexistuje)">planetárny vietor</a>. Hydrodynamický únik sa predpokladá aj v atmosférach <a href="/wiki/Exoplan%C3%A9ta" class="mw-redirect" title="Exoplanéta">exoplnét</a>.</p><h4><span id="Neterm.C3.A1lny_.C3.BAnik"></span><span class="mw-headline" id="Netermálny_únik">Netermálny únik</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;veaction=edit&amp;section=7" class="mw-editsection-visualeditor" title="Upraviť sekciu: Netermálny únik">upraviť</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;action=edit&amp;section=7" title="Upraviť sekciu: Netermálny únik">upraviť zdroj</a><span class="mw-editsection-bracket">]</span></span></h4><p>Netermálny únik zahŕňa nadobudnutie druhej únikovej rýchlosti bez pomoci tepla. Častice môžu nadobudnúť vysoké rýchlosti pri <a href="/wiki/Chemick%C3%A1_reakcia" title="Chemická reakcia">chemických reakciách</a> a vzájomných kolíziách častíc. <a href="/wiki/Elektrick%C3%BD_n%C3%A1boj" title="Elektrický náboj">Elektricky nabité</a> častice - <a href="/wiki/I%C3%B3n" class="mw-disambig" title="Ión">ióny</a> sú veľmi ovplyvňované <a href="/wiki/Magnetick%C3%A9_pole" title="Magnetické pole">magnetickým poľom</a> planéty, ktoré ich väčšinou nepustí. Ak však majú dostatočnú rýchlosť, môžu unikať pozdĺž takzvaných neukotvených <a href="/wiki/Silo%C4%8Diara" title="Siločiara">siločiar</a> magnetického poľa. Iným spôsobom úniku iónu z magnetosféry planéty je jeho kolízia s neutrálnym atómom. Tomu ión vytrhne <a href="/wiki/Elektr%C3%B3n" title="Elektrón">elektrón</a>, čím sa sám neutralizuje a ako neutrálna častica už nespadá pod pôsobenie magnetického poľa, takže môže uniknúť.</p><h4><span class="mw-headline" id="Impakty">Impakty</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;veaction=edit&amp;section=8" class="mw-editsection-visualeditor" title="Upraviť sekciu: Impakty">upraviť</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Atmosf%C3%A9ra_(kozmick%C3%A9ho_telesa)&amp;action=edit&amp;section=8" title="Upraviť sekciu: Impakty">upraviť zdroj</a><span class="mw-editsection-bracket">]</span></span></h4><p>Zrážka s iným telesom spôsobí jednorazové vymrštenie do kozmu oveľa väčšej časti atmosféry, ako za rovnakú časovú jednotku unikne termálnymi či netermálnymi procesmi. Pri náraze sa veľká časť impaktora aj zasiahnutého objektu vyparí. Stúpajúci horúci oblak plynu z obidvoch telies so sebou strháva aj častice atmosféry. Čím je atmosféra redšia, tým väčší fragment z nej pri zrážke unikne, čím sa ešte väčšmi zriedi a nasledujúci impakt bude ešte drvivejší. Takouto reťazovou reakciou môže teleso postupne prísť takmer o celú atmosféru. Najľahšie opäť unikajú chemické prvky s najmenšou hmotnosťou.
