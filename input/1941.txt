class="mbox-text-span">&#160;Iminumungkahi na ang artikulo ay hatiin sa mga artikulo na pinamagatang <i>[[::atmospera|atmospera]]</i> at <i>[[::himpapawid|himpapawid]]</i>, na matutunghayan sa <a (<a href="/wiki/Usapan:Himpapawid" title="Usapan:Himpapawid">Pag-usapan</a>)</span></td></tr></table><p>Ang <b>atmospero</b>, <b>atmospera</b>, o <b>panganorin</b> ay ang "kapisanan ng mga <a href="/wiki/Hangin" title="Hangin">hangin</a> at ng sari-saring mga singaw ng panahon na nakaliligid sa sangdaigdig hanggang sa sanggayong lawak".<sup id="cite_ref-Bansa_1-0" class="reference"><a href="#cite_note-Bansa-1">[1]</a></sup> Tinatawag din itong <b>himpapawid</b>, <b>agay-ay</b>, <b>alangaang</b>, o <b>kahanginan</b> dahil ito ang hangin na bumabalot sa <a href="/wiki/Daigdig" title="Daigdig">daigdig</a>.<sup id="cite_ref-Gaboy_2-0" class="reference"><a href="#cite_note-Gaboy-2">[2]</a></sup> Nagmula ang salitang <i>atmospera</i> sa <a href="/wiki/Wikang_Griyego" title="Wikang Griyego">wikang Griyego</a> na binubuo ng mga salitang ἀτμός - <i>atmos</i>, "singaw"<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup> at σφαῖρα - <i>sphaira</i>, "bilog" o "timbulog"<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup>). Isang sapin ng mga <a href="/wiki/Gas" title="Gas">gas</a> o mga singaw (mga hangin) ang atmospera na maaaring pumaligid sa isang katawang materyal na may sapat na <a href="/wiki/Masa" title="Masa">masa</a><sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup> sa pamamagitan ng <a href="/wiki/Grabidad" title="Grabidad">grabidad</a> ng katawang iyon, at napapanatili sa loob ng matagal na panahon kapag mataas ang grabidad at mababa ang temperatura ng atmospera. May ilang mga <a href="/wiki/Planeta" title="Planeta">planeta</a> ang pangunahing naglalaman lamang ng sari-saring mga gas o singaw, subalit tanging ang kanilang panlabas na sapin o patong (balot) lamang ang tinatawag na atmospera nila (tingnan ang mga <a href="/wiki/Higanteng_gas" class="mw-redirect" title="Higanteng gas">higanteng gas</a>).</p><p>Nilalarawan ng katagang <a href="/w/index.php?title=Atmosperang_istelar&amp;action=edit&amp;redlink=1" class="new" title="Atmosperang istelar (hindi pa naisusulat)">atmosperang istelar</a> o <a href="/w/index.php?title=Atmosperang_pambituin&amp;action=edit&amp;redlink=1" class="new" title="Atmosperang pambituin (hindi pa naisusulat)">atmosperang pambituin</a> ang panlabas na rehiyon ng isang <a href="/wiki/Bituin" title="Bituin">bituin</a>, at karaniwang nagsasama ng bahaging nagsisimula sa malabo o malamlam na <a href="/w/index.php?title=Potospero&amp;action=edit&amp;redlink=1" class="new" title="Potospero (hindi pa naisusulat)">potospero</a> papalabas. Maaaring bumuo ng mga molekulang kumpuwesto ang mga bituing may mababang temperatura sa kanilang panlabas na atmospera. Ang <a href="/w/index.php?title=Atmospera_ng_daigdig&amp;action=edit&amp;redlink=1" class="new" title="Atmospera ng daigdig (hindi pa naisusulat)">atmospera ng daigdig</a> ay naglalaman ng <a href="/wiki/Oksiheno" title="Oksiheno">oksihenong</a> ginagamit ng karamihan sa mga <a href="/wiki/Organismo" title="Organismo">organismo</a> para sa <a href="/wiki/Respirasyon" class="mw-redirect" title="Respirasyon">respirasyon</a> o <a href="/wiki/Paghinga" class="mw-redirect" title="Paghinga">paghinga</a>, at ng <a href="/wiki/Karbong_dioksido" class="mw-redirect" title="Karbong dioksido">karbong dioksidong</a> ginagamit naman ng mga halaman, mga <a href="/wiki/Lumot" title="Lumot">lumot</a>, at mga <a href="/wiki/Cyanobacteria" title="Cyanobacteria">siyanobakterya</a> para sa <a href="/wiki/Potosintesis" title="Potosintesis">potosintesis</a>. Pinupruteksyunan din ng atmospera ng daigdig ang mga organismong may buhay mula sa mga kapinsalaang henetikong dulot ng <a href="/w/index.php?title=Radyasong_ultrabiyoleta&amp;action=edit&amp;redlink=1" class="new" title="Radyasong ultrabiyoleta (hindi pa naisusulat)">radyasong ultrabiyoleta</a> mula sa <a href="/w/index.php?title=Sinag_ng_araw&amp;action=edit&amp;redlink=1" class="new" title="Sinag ng araw (hindi pa naisusulat)">sinag ng araw</a>. Ang pangkasalukuyang komposisyon ng atmospera ng daigdig ay isang produkto ng bilyun-bilyong mga taong pagbabagong biyokimikal ng <a href="/w/index.php?title=Paleoatmospera&amp;action=edit&amp;redlink=1" class="new" title="Paleoatmospera (hindi pa naisusulat)">paleoatmospera</a> ng mga organismong nabubuhay.</p><h2><span class="mw-headline" id="Mga_sanggunian">Mga sanggunian</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Himpapawid&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Baguhin seksiyon: Mga sanggunian">baguhin</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Himpapawid&amp;action=edit&amp;section=1" title="Baguhin seksiyon: Mga sanggunian">baguhin ang batayan</a><span class="mw-editsection-bracket">]</span></span></h2><div class="reflist" style="list-style-type: decimal;"><ol class="references"><li id="cite_note-Bansa-1"><span class="mw-cite-backlink"><a href="#cite_ref-Bansa_1-0">↑</a></span> <span class="reference-text"><span class="citation encyclopaedia">Blake, Matthew (2008).
