Hispania, <a href="/wiki/Saeculum_10" title="Saeculum 10">saeculo decimo</a> exeunte.</div></div></div><p><b>Ars</b> (<a href="https://la.wiktionary.org/wiki/ars" class="extiw" title="wikt:ars">artis, <i>f.</i></a>) est <a href="/wiki/Disciplina" class="mw-redirect" title="Disciplina">disciplina</a> vel operandi modus, partim translaticius partimque novatus, qui <a href="/wiki/Nihil" title="Nihil">nihil</a> est nisi facultas longam seriem actionum specialium exsequendi ut finis quidam attingantur. Sensu stricto, ars contracta est in quadam habilitate, sed, latiore sensu, comprehendit omnes <a href="/wiki/Via" title="Via">vias</a> et modos res bene agendi discendos. </p><p><i>Artifex</i> est qui aliquam artem exerceat. Artifices magna <a href="/wiki/Superbia" title="Superbia">superbia</a> suis operibus gloriantur, saepeque censentur per suum <a href="/wiki/Magnum_opus" title="Magnum opus">magnum opus</a>. <a href="/wiki/Lucretius" title="Lucretius">Lucretius</a> per suum <i><a href="/wiki/De_Rerum_Natura_(Lucretius)" class="mw-redirect" title="De Rerum Natura (Lucretius)">De Rerum Natura</a></i>, <a href="/wiki/Michael_Angelus_Bonarotius" title="Michael Angelus Bonarotius">Michael Angelus Bonarotius</a> per <a href="/wiki/Ars_visualis" class="mw-redirect" title="Ars visualis">Davidis signum</a>, <a href="/wiki/Gulielmus_Shakesperius" title="Gulielmus Shakesperius">Gulielmus Shakesperius</a> per tales <a href="/wiki/Ludus_scaenicus" title="Ludus scaenicus">ludos</a> sicut <i><a href="/wiki/Romaeus_et_Iulia" class="mw-redirect" title="Romaeus et Iulia">Romaeus et Iulia</a></i> et <i><a href="/wiki/Amletus,_Princeps_Daniae" class="mw-redirect" title="Amletus, Princeps Daniae">Amletus, Princeps Daniae</a></i>, <a href="/wiki/Thomas_Edison" title="Thomas Edison">Thomas Edison</a> per <a href="/wiki/Globulus_electricus" title="Globulus electricus">globulum electricum</a>, <a href="/wiki/Gulielmus_Marconi" title="Gulielmus Marconi">Gulielmus Marconi</a> per <a href="/wiki/Radiophonum" title="Radiophonum">radiophonum</a>, etc.</p><p>Olim omnes artes in <a href="/wiki/Artes_liberales" title="Artes liberales">artes liberales</a> et artes illiberales divisae sunt. Artes sic <i>liberales</i> appellabantur quae censebantur hominibus <a href="/wiki/Libertas" title="Libertas">liberis</a> discendi aptae. Per aetates autem, notio de eo quod hominibus liberis aptum est aliquanto mutavit; et, ut exempla infra data monstrant, multae novae artes hominibus liberis dignae tandem creatae sunt, quae inter artes liberales traditionales non censetur. Illae <a href="/wiki/Bellae_artes" title="Bellae artes">bellae artes</a> specialiter ad operis artistici <a href="/wiki/Pulchritudo" title="Pulchritudo">pulchritudinem</a> cultivantur. <a href="/wiki/Ars_ingeniaria" title="Ars ingeniaria">Artes ingeniariae</a>, ad contrarium, sine fine <a href="/wiki/Aesthetica" title="Aesthetica">aesthetico</a>, ortae sunt specialiter ad propositum <a href="/wiki/Oeconomia" class="mw-redirect" title="Oeconomia">oeconomicum</a> <a href="/wiki/Scientia_(ratio)" title="Scientia (ratio)">scientificumque</a>. </p><h2><span class="mw-headline"
