class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Menas&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Redaguoti skyrelį: Architektūra">redaguoti</a><span class="mw-editsection-bracket">]</span></span></h3><dl><dd><div class="noprint relarticle mainarticle"><i>Pagrindinis straipsnis – <a href="/wiki/Architekt%C5%ABra" title="Architektūra">Architektūra</a></i>.</div></dd></dl><p><b>Architektūra</b> (iš <a href="/wiki/Graik%C5%B3_kalba" title="Graikų kalba">gr.</a> <i>αρχη</i> 'vyriausiasis, vadovaujantis' ir <i>τεκτων</i> 'statybininkas, dailidė')&#160;– <a href="/wiki/Pastatas" title="Pastatas">pastatų</a> projektavimo <a class="mw-selflink selflink">menas</a> ir <a href="/wiki/Mokslas" title="Mokslas">mokslas</a>. Pagrindiniai architektūros tikslai yra formuoti aplinkos erdvę, projektuoti <a href="/wiki/Statinys" title="Statinys">statinius</a> bei jų kompleksus. Plačiąja šio žodžio reikšme architektūros objektas yra žmogaus gyvenamosios aplinkos planavimas ir organizavimas, pradedant miestų planavimu, landšafto architektūra ir baigiant interjero dizainu.</p><p>Kaip akademinė disciplina architektūra paprastai skirstoma į dvi pagrindines šakas: <a href="/wiki/Urbanistika" title="Urbanistika">urbanistiką</a> ir <a href="/w/index.php?title=T%C5%ABrin%C4%97_architekt%C5%ABra&amp;action=edit&amp;redlink=1" class="new" title="Tūrinė architektūra (puslapis neegzistuoja)">tūrinę architektūrą</a>. Architektūrinis projektavimas, skirtingai nuo <a href="/wiki/Septyni_laisvieji_menai" title="Septyni laisvieji menai">laisvųjų menų</a>, yra priklausomas nuo daugelio techninių, fizinių, teisinių, socialinių ir klimato faktorių, lemiančių galutinį projektą. Tačiau skirtingai nuo <a href="/wiki/Mokslas" title="Mokslas">taikomųjų mokslų</a> ar <a href="/wiki/In%C5%BEinerija" title="Inžinerija">inžinerijos</a>, iš architektūros tikimasi ne tik techninio sprendimo ir funkcionalumo, bet taip pat ir estetinių principų laikymosi. Architektūros kūrimo procesas apima priemones, naudojamas tradiciniuose menuose&#160;– skulptūroje ir tapyboje&#160;– <a href="/wiki/Erdv%C4%97" title="Erdvė">erdvę</a>, formą, <a href="/wiki/T%C5%ABris" title="Tūris">tūrį</a>, <a href="/wiki/Tekst%C5%ABra" title="Tekstūra">tekstūrą</a>, <a href="/wiki/Spalva" title="Spalva">spalvą</a>, <a href="/wiki/%C5%A0viesa" title="Šviesa">šviesą</a>, <a href="/wiki/%C5%A0e%C5%A1%C4%97lis" title="Šešėlis">šešėlį</a>. Dėl šių aspektų pasireiškimo architektūrinis projektavimas tradiciškai laikomas kūrybiniu procesu, o architektūra bendruoju atveju priskiriama prie meno šakų.</p><h3><span class="mw-headline" Tapybos kūrinio <a href="/wiki/Kompozicija" title="Kompozicija">kompozicijos</a> elementai: <a href="/wiki/Koloritas" title="Koloritas">koloritas</a>, <a href="/w/index.php?title=Pie%C5%A1inys&amp;action=edit&amp;redlink=1" class="new" title="Piešinys (puslapis neegzistuoja)">piešinys</a>, šešėliavimas, potėpis, <a href="/w/index.php?title=Fakt%C5%ABra&amp;action=edit&amp;redlink=1" class="new" title="Faktūra (puslapis neegzistuoja)">faktūra</a>. Tapyba grindžiama lokaliųjų spalvų ir valiorų deriniais arba refleksų ir tonų santykiais. Atsirado <a href="/wiki/JAV" class="mw-redirect" title="JAV">JAV</a> ir Vakarų Europoje kaip reakcija į <a href="/wiki/Minimalizmas" title="Minimalizmas">minimalizmą</a>, pagrįstą teiginiu, kad meno kūrinys iš esmės tėra fizinis daiktas.</p><p>Vadovaujamasi įsitikinimu, kad meno esmė glūdi <a href="/wiki/S%C4%85voka" title="Sąvoka">sąvokose</a>, idėjose arba konceptuose, tiriama, kaip menas suvokiamas <a href="/wiki/Visuomen%C4%97" title="Visuomenė">visuomenėje</a>, kvestionuojami dailės interpretacijos <a href="/w/index.php?title=Stereotipai&amp;action=edit&amp;redlink=1" class="new" title="Stereotipai (puslapis neegzistuoja)">stereotipai</a>, pasitelkus <a href="/wiki/Lingvistika" class="mw-redirect" title="Lingvistika">lingvistiką</a>, <a href="/wiki/Filosofija" title="Filosofija">filosofiją</a>, <a href="/wiki/Filosofija" title="Filosofija">sociologiją</a>, meno kritiką ar kasdieninį patyrimą, analizuojamos meno idėjos.</p><p>Šio meno pagrindas&#160;– pats kūrybinis sumanymas, projektas, idėja. href="/w/index.php?title=Menas&amp;veaction=edit&amp;section=7" class="mw-editsection-visualeditor" title="Redaguoti skyrelį: Literatūra">redaguoti</a><span class="mw-editsection-divider"> | </span><a class="noprint relarticle mainarticle"><i>Pagrindiniai straipsniai – <a href="/wiki/Kalba" title="Kalba">Kalba</a>&#32;ir&#32;<a href="/wiki/Literat%C5%ABra" title="Literatūra">Literatūra</a></i>.</div></dd></dl><p>Terminas <b>literatūra</b> gali būti taikomas tiek pavieniams rašytiniams kūriniams, tiek ir apimant visus rašytinius viso pasaulio, tam tikros kultūros ar tam tikros epochos kūrinius. Terminas kilęs iš <a href="/wiki/Lotyn%C5%B3_kalba" title="Lotynų kalba">lotyniško</a> žodžio <i>littera</i>, reiškiančio raidę arba „individualaus pobūdžio raštas (laiškas)“.</p><p>Literatūra laikoma viena iš meno rūšių, tačiau literatūros kaip meno ribas apibrėžti taip pat yra sunku, nes ne visada aišku kam priskirti, pvz., filosofinius kūrinius, <a href="/wiki/Pjes%C4%97" title="Pjesė">pjeses</a> arba filmų <a href="/wiki/Scenarijus" title="Scenarijus">scenarijus</a>.</p><p><a href="/wiki/Literat%C5%ABra" title="Literatūra">Literatūros</a>, kaip ir kitų menų kūriniai skirstomi į tam tikras grupes&#160;– literatūros rūšis. Jau nuo <a href="/wiki/Antika" title="Antika">antikos</a> laikų samprotaujama apie tris pagrindines literatūros rūšis. Tai <a href="/wiki/Epika" title="Epika">epika</a>, <a href="/wiki/Lyrika" title="Lyrika">lyrika</a> ir <a href="/wiki/Drama" title="Drama">drama</a>. Šios rūšys viena nuo kitos skiriasi kalbinės raiškos, žmogaus vaizdavimo pobūdžiu. Epikoje vyrauja pasakojimas, jos žanrai: <a href="/wiki/Epas" title="Epas">epas</a>, <a href="/wiki/Es%C4%97" title="Esė">esė</a>, <a href="/wiki/Romanas" title="Romanas">romanas</a>, <a href="/wiki/Apysaka" title="Apysaka">apysaka</a>, <a href="/wiki/Apsakymas" title="Apsakymas">apsakymas</a>. Tuo tarpu dramoje svarbiausią vaidmenį užima veiksmas. Jos žanrai&#160;– <a href="/wiki/Tragedija" title="Tragedija">tragedija</a>, <a href="/wiki/Komedija" title="Komedija">komedija</a>, <a href="/wiki/Tragikomedija" title="Tragikomedija">tragikomedija</a>, <a href="/wiki/Drama" title="Drama">drama</a> (siaurąją prasme). Lyrika&#160;– jausmų ir minčių raiškos menas, jos žanrai&#160;– <a href="/wiki/Eil%C4%97ra%C5%A1tis" title="Eilėraštis">eilėraštis</a>, <a href="/wiki/Sonetas" title="Sonetas">sonetas</a>, <a href="/wiki/Trioletas" title="Trioletas">trioletas</a>.</p><h3><span class="mw-headline" id="Vaizduojamasis_menas">Vaizduojamasis menas</span><span class="mw-editsection"><span title="Vaizduojamasis menas">Vaizduojamasis menas</a></i>.</div></dd></dl><p>Vaizduojamasis menas skiriasi nuo plastikos meno, nes naudojamas menininko <a href="/wiki/K%C5%ABnas" class="mw-disambig" title="Kūnas">kūnas</a>, <a href="/wiki/Veidas" title="Veidas">veidas</a>, o plastikos menas naudoja medžiagas, kaip <a href="/wiki/Molis" title="Molis">molis</a>, <a href="/wiki/Metalas" class="mw-redirect" title="Metalas">metalas</a> ar <a href="/wiki/Da%C5%BEai" class="mw-redirect" title="Dažai">dažai</a>, kurie gali būti transformuojami į keletą meno objektų.</p><p>Į vaizduojamąjį meną įtrauktos šios meno formos: <a href="/wiki/Akrobatika" title="Akrobatika">akrobatika</a>, <a href="/wiki/Komedija" title="Komedija">komedijos</a>, <a href="/wiki/%C5%A0okiai" class="mw-redirect" title="Šokiai">šokiai</a>, <a href="/wiki/Magija" title="Magija">magija</a>, <a href="/wiki/Muzika" title="Muzika">muzika</a>, <a href="/wiki/Opera" title="Opera">opera</a>, <a href="/wiki/Operet%C4%97" title="Operetė">operetė</a>, <a href="/w/index.php?title=%C5%BDonglieriai&amp;action=edit&amp;redlink=1" class="new" title="Žonglieriai (puslapis neegzistuoja)">žonglieriai</a>, <a href="/wiki/Kovos_menai" title="Kovos menai">kovos menai</a>.</p><p>Menininkai, kurie dalyvauja šių meno kūrime ir vaizdavime prieš <a href="/wiki/Auditorija" class="mw-redirect" title="Auditorija">auditoriją</a> yra vadinami atlikėjais, dalyviai, <a href="/w/index.php?title=Komikas&amp;action=edit&amp;redlink=1" class="new" title="Komikas (puslapis neegzistuoja)">komikais</a>, šokėjais, <a href="/wiki/Muzikantas" title="Muzikantas">muzikantais</a> ir <a href="/wiki/Dainininkas" title="Dainininkas">dainininkais</a>. Atlikėjai dažnai keičia savo išvaizdą <a href="/wiki/Kostiumas" title="Kostiumas">kostiumų</a> ir <a href="/w/index.php?title=Makia%C5%BEas&amp;action=edit&amp;redlink=1" class="new" title="Makiažas (puslapis neegzistuoja)">makiažo</a> pagalba. Yra ir specializuotų meno formų, kur menininkai atlieka savo kūrinį tiesiogiai auditorijai&#160;– tai vadinama <a href="/wiki/Performansas" title="Performansas">performansu</a>. Dauguma performansų apima tam tikras plastikos formas. Šiuolaikinio šokio era yra kartais priskiriama prie plastikos meno.</p><h3><span class="mw-headline" id="Muzika">Muzika</span><span class="mw-editsection"><span natos.</div></div></div><p><a href="/wiki/Muzika" title="Muzika">Muzika</a>, kaip akademinė disciplina, daugiausia orientuojasi dvejomis kryptimis: muzikos performansas (orkestrai ir koncertų salės) href="/wiki/Pranc%C5%ABz%C5%B3_kalba" title="Prancūzų kalba">pranc.</a> <i>théâtre</i>; <a href="/wiki/Lotyn%C5%B3_kalba" title="Lotynų kalba">lot.</a> <i>theatrum</i>; <a href="/wiki/Graik%C5%B3_kalba" title="Graikų kalba">gr.</a> <i>théatron</i> vieta, skirta parodymui; v. <i>theasthai</i> pasižiūrėti) meninės išraiškos priemonė, išorinio ir vidinio pasaulio įvykių atvaizdavimas <a href="/wiki/Scena" title="Scena">scenoje</a>, dalyvaujant artistams (<a href="/wiki/Aktorius" title="Aktorius">aktoriams</a>) bei <a href="/wiki/Publika" class="mw-redirect" title="Publika">publikai</a>.V amžius laikomas <a href="/wiki/Drama" title="Drama">dramų</a> klestėjimo laikotarpis. Žymiausi <a href="/wiki/Tragedija" title="Tragedija">tragedijų</a> kūrėjai: <a href="/wiki/Sofoklis" title="Sofoklis">Sofoklis</a>, <a href="/wiki/Aischilas" title="Aischilas">Aischilas</a>, <a href="/wiki/Euripidas" title="Euripidas">Euripidas</a>.</p><p>Teatro veiksmo metu pasakojama istorija, naudojant kombinacijos tarp žodžio, gesto, muzikos, šokio, garso. Ir spektaklis&#160;– naudoja bent vieną ar daugiau šių elementų. Be standartinio aprašomojo <a href="/w/index.php?title=Dialogas&amp;action=edit&amp;redlink=1" class="new" title="Dialogas (puslapis neegzistuoja)">dialogo</a> stiliaus, teatras imasi ir kitokių formų: <a href="/wiki/Opera" title="Opera">operos</a>, <a href="/wiki/Baletas" title="Baletas">baleto</a>, pantomimos, <a href="/wiki/Kabuki" title="Kabuki">kabuki</a>, klasikinių Indijos šokių, kinų operos.</p><div class="thumb tright"><div class="thumbinner" style="width:202px;"><a href="/wiki/Vaizdas:Two_dancers.jpg" class="image"><img alt="" class="noprint relarticle mainarticle"><i>Pagrindinis straipsnis – <a href="/wiki/%C5%A0okis" title="Šokis">Šokis</a></i>.</div></dd></dl><p>Šokiai paprastai reiškia žmonių <a href="/wiki/Jud%C4%97jimas" class="mw-disambig" title="Judėjimas">judėjimą</a>. Šokių terminas naudojamas apibūdinti metodus neverbalinio bendravimo tarp žmonių ar <a href="/wiki/Gyv%C5%ABnas" class="mw-redirect" title="Gyvūnas">gyvūnų</a> (<a href="/wiki/Bit%C4%97s" title="Bitės">bičių</a> šokis, <a href="/w/index.php?title=Poravimasis&amp;action=edit&amp;redlink=1" class="new" title="Poravimasis (puslapis neegzistuoja)">poravimosi</a> šokis), judesio ir negyvų objektų, bei tam tikras muzikos formas ir žanras.</p><p><a href="/wiki/Choreografija" title="Choreografija">Choreografija</a> yra šokio mokymasis. Asmuo, kuris mokina šokti yra vadinamas choreografu. Žmonės šoka ir dėl noro atsipalaiduoti. Šokiai skirstomo nuo judėjimo formos ir paskirties:<a href="/wiki/Liaudies_%C5%A1okiai" class="mw-redirect" title="Liaudies šokiai">Liaudies šokiai</a>, <a href="/wiki/Sportas" title="Sportas">sporto</a> <a href="/wiki/Gimnastika" title="Gimnastika">gimnastika</a>, <a href="/wiki/Sinchroninis_plaukimas" title="Sinchroninis plaukimas">sinchroninis plaukimas</a> ir kiti. Kovos menai irgi dažnai priskiriami prie šokių.</p><h2><span class="mw-headline" id="Meno_funkcijos">Meno funkcijos</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a
