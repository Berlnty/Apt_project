1758</small></td></tr></tbody></table><p><b>Čovjek</b> (mn. <b>ljudi</b>) definiše se na <a href="/wiki/Biologija" title="Biologija">biološki</a>, <a href="/wiki/Sociologija" title="Sociologija">sociološki</a> i <a href="/wiki/Duhovni" class="mw-redirect" title="Duhovni">duhovni</a> način.</p><p>Kao riječ i izraz, "čovjek" je <a href="/wiki/Historija" title="Historija">historijski</a>, <a href="/wiki/Religija" title="Religija">vjerski</a> i <a href="/wiki/Kultura" title="Kultura">kulturno</a> uvriježena riječ koja se koristi u smislu označavanja ljudske vrste općenito, tj. pokrivajući sve rodove i polove: muškarce, žene, djecu, <a href="/wiki/Transseksualnost" title="Transseksualnost">transseksualce</a>, itd. U nedostatku boljeg neutralnog izraza, riječ čovjek se koristi kao prevod <a href="/wiki/Latinski_jezik" title="Latinski jezik">latinske</a> riječi <i><a href="/wiki/Homo" class="mw-redirect" title="Homo">homo</a></i> koja označava "osobu." Slično stanje je i u mnogim drugim jezicima, kao naprimjer, u <a href="/wiki/Engleski_jezik" title="Engleski jezik">engleskom</a>, gdje se riječ <i>man</i>, doslovno "<a href="/wiki/Mu%C5%A1karac" title="Muškarac">muškarac</a>," često, mada ne uvijek, koristi u smislu ljudskog bića, pa prema tome uključuje i seksualne polove koji nisu muški, kao naprimjer žene.</p><div id="toc" class="toc"><div aspekt</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%C4%8Covjek&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Uredi odlomak &quot;Biološki aspekt&quot;">uredi</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%C4%8Covjek&amp;action=edit&amp;section=1" title="Uredi odlomak &quot;Biološki aspekt&quot;">uredi izvor</a><span class="mw-editsection-bracket">]</span></span></h2><p>Biološki, čovjek se klasifikuje kao <a href="/wiki/Vrsta_(biologija)" title="Vrsta (biologija)">vrsta</a> <i>homo sapiens</i> (na <a href="/wiki/Latinski_jezik" title="Latinski jezik">latinskom</a>, "mudri čovjek" ili "misleći čovjek"), iz <a href="/wiki/Rod_(biologija)" title="Rod (biologija)">roda</a> <i><a href="/wiki/Homo_(rod)" title="Homo (rod)">homo</a></i> (na latinskom, "čovjek"); <i>homo sapiens</i> je <a href="/w/index.php?title=Dvonogi&amp;action=edit&amp;redlink=1" class="new" title="Dvonogi (stranica ne postoji)">dvonogi</a> <a href="/wiki/Primat_(biologija)" class="mw-redirect" title="Primat (biologija)">primat</a> iz natporodice <a href="/wiki/Hominoidea" class="mw-redirect" title="Hominoidea">hominoidea</a>, u koju spadaju ostali <a href="/wiki/%C4%8Covjekoliki_majmun" class="mw-redirect" title="Čovjekoliki majmun">čovjekoliki majmuni</a>: <a href="/wiki/%C4%8Cimpanza" class="mw-redirect" title="Čimpanza">čimpanze</a>, <a href="/wiki/Gorila" class="mw-redirect" title="Gorila">gorile</a>, <a href="/wiki/Orangutan" class="mw-redirect" title="Orangutan">orangutani</a> i <a href="/w/index.php?title=Gibon&amp;action=edit&amp;redlink=1" class="new" title="Gibon (stranica ne postoji)">giboni</a>.</p><p>Čovjek ima uspravno tijelo, koje mu oslobađa gornje udove za baratanje predmetima, i veoma razvijeni <a href="/wiki/Mozak" title="Mozak">mozak</a>, koji je u stanju da vrši apstraktno <a href="/wiki/Mi%C5%A1ljenje" class="mw-redirect" title="Mišljenje">mišljenje</a>, <a href="/wiki/Govor" title="Govor">govor</a>, <a href="/wiki/Jezik" title="Jezik">jezik</a> i <a href="/w/index.php?title=Introspekcija&amp;action=edit&amp;redlink=1" class="new" title="Introspekcija (stranica ne postoji)">introspekciju</a>. Čini se da se <a href="/wiki/Dvono%C5%A1ci_(biologija)" class="mw-redirect" title="Dvonošci (biologija)">dvonogo</a> kretanje <a href="/wiki/Evolucija" title="Evolucija">evolutivno</a> razvilo prije <a href="/w/index.php?title=Encefalizacija&amp;action=edit&amp;redlink=1" class="new" title="Encefalizacija (stranica ne postoji)">encefalizacije</a>, tj. razvoja <a href="/wiki/Veliki_mozak" class="mw-redirect" title="Veliki mozak">velikog mozga</a>. Istraživanje o porijeklu dvonogog hoda i njegove uloge u evoluciji <a href="/wiki/Mozak" title="Mozak">ljudskog mozga</a> je još uvijek u toku.</p><h2><span id="Sociolo.C5.A1ki_aspekt"></span><span class="mw-headline" id="Sociološki_aspekt">Sociološki aspekt</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%C4%8Covjek&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Uredi odlomak &quot;Sociološki aspekt&quot;">uredi</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%C4%8Covjek&amp;action=edit&amp;section=2" title="Uredi odlomak &quot;Sociološki aspekt&quot;">uredi izvor</a><span class="mw-editsection-bracket">]</span></span></h2><p>Sociološki, čovjek je suštinski <a href="/wiki/Dru%C5%A1tvo" title="Društvo">društveno</a> biće, kao i većina <a href="/wiki/Primat_(biologija)" class="mw-redirect" title="Primat (biologija)">primata</a>. Ljudska bića stvaraju složene društvene strukture koje sačinjavaju saradničke (kooperativne) ili takmičarske (kompetitivne) grupe. U ove spadaju grupe u rasponu od <a href="/wiki/Nacija" title="Nacija">nacije</a> i <a href="/wiki/Dr%C5%BEava" title="Država">države</a> pa sve do <a href="/wiki/Porodica" title="Porodica">porodice</a>.</p><h2><span id="Duhovni.2C_vjerski_i_kulturni_aspekt"></span><span class="mw-headline" id="Duhovni,_vjerski_i_kulturni_aspekt">Duhovni, vjerski i kulturni aspekt</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%C4%8Covjek&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Uredi odlomak &quot;Duhovni, vjerski i kulturni aspekt&quot;">uredi</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%C4%8Covjek&amp;action=edit&amp;section=3" title="Uredi odlomak &quot;Duhovni, vjerski i kulturni aspekt&quot;">uredi izvor</a><span class="mw-editsection-bracket">]</span></span></h2><p>Duhovno, vjerski i kulturno, ljudska bića stvaraju i obogaćuju svoje unutrašnje i vanjske svjetove putem traganja za razumijevanjem, upotrebom i manipulisanjem <a href="/wiki/Nauka" title="Nauka">nauke</a> i <a href="/wiki/Tehnologija"
