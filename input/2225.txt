pagina)">Homines hodiernas corporis conformationes habentes</a> <a href="/wiki/Africa" title="Africa">Africa</a> abhinc <a href="/wiki/Annus" title="Annus">annorum</a> circa 200&#8201; 000 orti sunt, et perfectam <a href="/wiki/Mores" title="Mores">morum</a> hodiernitatem abhinc circa annorum 50&#8201;000 adsecuti sunt.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup></p><p>Quia hominibus est <a href="/wiki/Cerebrum" title="Cerebrum">cerebrum</a> humanum <a href="/w/index.php?title=Encephalizatio&amp;action=edit&amp;redlink=1" class="new" title="Encephalizatio (non est haec pagina)">maxime evolutum</a>, eo est ei <a href="/wiki/Cogitatio" title="Cogitatio">sensus subiectivus abstractionis</a> quo homines <a href="/wiki/Ratio" title="Ratio">ratiocinari</a>, <a href="/wiki/Lingua" title="Lingua">lingua</a> loqui, in <a href="/wiki/Mens" title="Mens">mentem</a> <a href="/w/index.php?title=Introspectio&amp;action=edit&amp;redlink=1" class="new" title="Introspectio (non est haec pagina)">introspicere</a>, <a href="/wiki/Quaestiones" class="mw-redirect" title="Quaestiones">quaestiones</a> censere, et solutiones invenire possunt. Haec mentis facultas, cum erecto <a href="/wiki/Corpus_(physica)" class="mw-redirect" title="Corpus (physica)">corporis</a> habitu et <a href="/wiki/Manus" title="Manus">manibus</a> liberis res tractantibus, hominibus sinit <a href="/wiki/Instrumentum_rusticum" title="Instrumentum rusticum">instrumentis</a> omnium generum uti ad tractandum plus quam quaequam orbis terrarum speciem <a href="/wiki/Vita" title="Vita">vivam</a>. Etenim adhibet <a href="/wiki/Cognitio" title="Cognitio">cognitionis</a> gradum altiorem, sicut <a href="/wiki/Agnitio_sui" class="mw-redirect mw-disambig" title="Agnitio sui">agnitionem sui</a> ("ratiocinor ergo exsisto"), <a href="/wiki/Memoria" title="Memoria">memoriam</a>, <a href="/wiki/Ratio" title="Ratio">rationem</a>, <a href="/w/index.php?title=Ratiocinatio&amp;action=edit&amp;redlink=1" class="new" title="Ratiocinatio (non est haec pagina)">ratiocinationem</a>, et <a href="/wiki/Sapientia" title="Sapientia">sapientiam</a> praebens.<sup id="cite_ref-mc2_4-0" class="reference"><a href="#cite_note-mc2-4">&#91;4&#93;</a></sup><sup id="cite_ref-mcd_5-0" class="reference"><a href="#cite_note-mcd-5">&#91;5&#93;</a></sup><sup id="cite_ref-flanagan_6-0" class="reference"><a href="#cite_note-flanagan-6">&#91;6&#93;</a></sup> Item habent notiones attributionum definientes rerum quae "<a href="/wiki/Persona" title="Persona">personam</a>" conformant.<sup id="cite_ref-mc1_7-0" class="reference"><a href="#cite_note-mc1-7">&#91;7&#93;</a></sup><sup id="cite_ref-mol_8-0" class="reference"><a href="#cite_note-mol-8">&#91;8&#93;</a></sup></p><p>Hominibus est unica peritia ad versanda systemata <a href="/wiki/Communicatio" title="Communicatio">communicationis</a> ut de se dicant, excogitationes inter se permutent, et res communiter <a href="/wiki/Organizatio" title="Organizatio">ordinent</a>. Ei multiplices <a href="/w/index.php?title=Structura_socialis&amp;action=edit&amp;redlink=1" class="new" title="Structura socialis (non est haec pagina)">structuras sociales</a> fingunt, quae in multis gregibus una agentes et contendentes coexsistunt, inter quas <a href="/wiki/Familia" title="Familia">familiae</a>, pleraque <a href="/w/index.php?title=Cognatio&amp;action=edit&amp;redlink=1" class="new" title="Cognatio (non est haec pagina)">cognationis</a> retia, et <a href="/wiki/Civitas" title="Civitas">civitates</a>, a <a href="/wiki/Mos" class="mw-redirect" title="Mos">moribus</a> et <a href="/wiki/Lex" title="Lex">legibus</a> temperatae. <a href="/w/index.php?title=Interactio_socialis&amp;action=edit&amp;redlink=1" class="new" title="Interactio socialis (non est haec pagina)">Interactiones sociales</a> inter homines variissimas statuerunt aestimationes, <a href="/w/index.php?title=Norma_(sociologia)&amp;action=edit&amp;redlink=1" class="new" title="Norma (sociologia) (non est haec pagina)">normas sociales</a>, et <a href="/wiki/Ritus" title="Ritus">ritus</a>, qui una sunt fundamenta societatis humanae. Quia in omni <a href="/wiki/Continens" title="Continens">continente</a>, etiam <a href="/wiki/Antarctica" title="Antarctica">Antarcticam</a>, <a href="/wiki/Habitatio" title="Habitatio">habitant</a>, homines sunt species <a href="/wiki/Distributio_cosmopolita" title="Distributio cosmopolita">cosmopolita</a>.</p><p>Aestimavit <a href="/w/index.php?title=Divisio_Civium_Numeri_Consociationis_Nationum&amp;action=edit&amp;redlink=1" class="new" title="Divisio Civium Numeri Consociationis Nationum (non est haec pagina)">Divisio Civium Numeri Consociationis Nationum</a> anno <a href="/wiki/2011" title="2011">2011</a> <a href="/w/index.php?title=Civium_numerus&amp;action=edit&amp;redlink=1" class="new" title="Civium numerus (non est haec pagina)">civium numerum</a> esse circa 7 <a href="/wiki/Billio" title="Billio">billiones</a>,<sup id="cite_ref-9" class="reference"><a href="#cite_note-9">&#91;9&#93;</a></sup> et aestimavit <a href="/wiki/Ministerium_Census_Civitatum_Foederatarum" title="Ministerium Census Civitatum Foederatarum">Ministerium Census Civitatum Foederatarum</a> eum esse circa 6.97 billiones.<sup id="cite_ref-popclock_10-0" class="reference"><a href="#cite_note-popclock-10">&#91;10&#93;</a></sup> Insigniter desiderant homines eorum <a href="/wiki/Circumiecta_naturalia" title="Circumiecta naturalia">circumiectum</a> intellegere et adfigere, res per <a href="/wiki/Scientia_(ratio)" title="Scientia (ratio)">scientiam</a>, <a href="/wiki/Philosophia" title="Philosophia">philosophiam</a>, <a href="/wiki/Mythologia" title="Mythologia">mythologiam</a>, et <a href="/wiki/Religio" title="Religio">religionem</a> et alias <a href="/wiki/Superstitio" title="Superstitio">superstitiones</a> explicare et tractare petentes. Hoc <a href="/wiki/Natura" title="Natura">naturale</a> noscendi studium evolutionem instrumentorum promotorum et sollertiarum excitavit, quae per <a href="/wiki/Cultura" title="Cultura">culturam</a> traduntur; homines quidem sunt sola species quae <a href="/wiki/Ignis" title="Ignis">ignem</a> sibi facit, <a href="/wiki/Ars_coquinaria" title="Ars coquinaria">cibum sibi coquunt</a>, <a href="/wiki/Vestimenta" title="Vestimenta">induunt sibi vestem</a>, atque multas alias <a href="/wiki/Technologia" title="Technologia">technologias</a> et <a href="/wiki/Ars" title="Ars">artes</a> excogitant et adhibent, praecipue <a href="/wiki/Litterae" title="Litterae">litteras</a>, <a href="/wiki/Musica" title="Musica">musicam</a>, <a href="/wiki/Saltatio" title="Saltatio">saltationem</a>, et alias <a href="/wiki/Ars" title="Ars">artes</a> quae ex aestimatione <a href="/wiki/Pulchritudo" title="Pulchritudo">pulchritudinis</a> dependent. Investigatio hominum est <a href="/wiki/Scientia_(ratio)" title="Scientia (ratio)">scientifica</a> <a href="/wiki/Anthropologia" title="Anthropologia">anthropologiae</a> doctrina. </p><div id="toc" title="Mas">mas</a> <a href="/wiki/Femina" class="mw-disambig" title="Femina">feminaque</a>. Differentiae naturales communes inter eos sunt. Mas longior et fortior quam femina est atque <a href="/wiki/Vox" title="Vox">vocem</a> obscuriorem habet. Femina parem <a href="/wiki/Pectus" class="mw-redirect" title="Pectus">pectoris</a> magni habet qui in muliere <a href="/wiki/Mamma" title="Mamma">mamma</a> appellatur. Quando mulier <a href="/wiki/Infans" class="mw-redirect" title="Infans">infantem</a> nata est, <a href="/wiki/Lac" title="Lac">lac</a> in mamma creatur. Mas plus <a href="/wiki/Pilus" title="Pilus">pilum corporis</a> quam femina habet. De organo coitus et <a href="/wiki/Reproductio" class="mw-redirect" title="Reproductio">reproductioni</a>, mas <a href="/wiki/Penis" title="Penis">penem</a>, femina <a href="/wiki/Vagina" title="Vagina">vaginam</a>, habet.</p><p>Aliquando mutationes naturales sexus in evolutione <a href="/wiki/Fetus" title="Fetus">fetus</a> fieri possunt. Tales homines <a href="/wiki/Intersexualitas" title="Intersexualitas">intersexuales</a> et <a href="/wiki/Transsexualitas" title="Transsexualitas">transsexuales</a> appellantur. Populus quoque est qui eius sexus mutare vult. Saepissime a viro ad mulierem est. Praeter sexus naturales, nonnullus populus talis generis etiam est. Multi tales homines etiam <a href="/wiki/Actrix_pornographica" class="mw-redirect" title="Actrix pornographica">actrices pornographicae</a> sunt. In <a href="/wiki/Pornographia" title="Pornographia">pornographia</a>, nominibus <a href="/wiki/Lingua_Anglica" title="Lingua Anglica">Anglicis</a> tribus noti sunt; <i>transexual</i>, <i>shemale</i> ("eamas") et <i>ladyboy</i> ("dominapuer"). <a href="/wiki/Thailandia" title="Thailandia">Thailandiae</a> saepissime <i>ladyboy</i> appellantur. In Thailandia quoque multus populus tali generi humani est.</p><p>Nonnulli homines transsexuales etiam ut alteri sexus esse se ipsi vident.</p><h3><span class="mw-headline" id="Vita">Vita</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gens_humana&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Recensere partem: Vita">recensere</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Gens_humana&amp;action=edit&amp;section=3" title="Recensere partem: Vita">fontem recensere</a><span class="mw-editsection-bracket">]</span></span></h3><p>Homines sunt species hodierna inter <a href="/wiki/Animalia" title="Animalia">animalia</a> <a href="/wiki/Multicellulare" title="Multicellulare">multicellularia</a>, longaeviorum. Sciuntur igitur homines plus quam centum viginti annos aetate maiores. Talis longaevitas est res <a href="/wiki/Genotypus" title="Genotypus">genotypica</a>, cui <a href="/w/index.php?title=Phaenotypus&amp;action=edit&amp;redlink=1" class="new" title="Phaenotypus (non est haec pagina)">phaenotypus</a> (modus hominum vivendi) favet.</p><h3><span class="mw-headline" id="Evolutio">Evolutio</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gens_humana&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Recensere partem: Evolutio">recensere</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Gens_humana&amp;action=edit&amp;section=4" title="Recensere partem: Evolutio">fontem recensere</a><span class="mw-editsection-bracket">]</span></span></h3><div style="margin-bottom: 0.5em; padding: 2px; text-align: left; font-size: 95%;"><a href="/wiki/Fasciculus:Searchtool.svg" class="image"><img alt="Searchtool.svg" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/61/Searchtool.svg/18px-Searchtool.svg.png" width="18" height="18" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/61/Searchtool.svg/27px-Searchtool.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/61/Searchtool.svg/36px-Searchtool.svg.png 2x" data-file-width="512" data-file-height="512" /></a> <i>Si plus cognoscere vis, vide <a href="/wiki/Evolutio_Hominis" class="mw-redirect" title="Evolutio Hominis">Evolutio Hominis</a></i></div><p><i>Homo sapiens</i> <a href="/wiki/Evolutio_hominis" title="Evolutio hominis">evolutione</a> ex specie <i><a href="/wiki/Homo_erectus" title="Homo erectus">Hominis erecti</a></i> in <a href="/wiki/Africa" title="Africa">Africa</a> evolutus est. Ante circiter centum milia annorum <i>Homo sapiens</i> ex <a href="/wiki/Africa" title="Africa">Africa</a> migrare coepit.</p><p>Omnes homines eiusdem speciei sunt. Varietates conspicuae non sunt scientificae. Analyses <a href="/wiki/Genetica" title="Genetica">geneticae</a> monstrant diversitatem <a href="/wiki/Genum" title="Genum">genorum</a> in hominibus Africanis maximam esse, sed minimam in <a href="/wiki/Europa" title="Europa">Europaeis</a> <a href="/wiki/Asia" title="Asia">Asiaticis</a> <a href="/wiki/America" title="America">Americanisque</a>. Robota primitiva, aut progressiva in sensu hodierno, quae homo non similant pars societatis oblectamento et auxilio iam sunt, sicut <a href="/wiki/Asimo" title="Asimo">Asimo</a> et <a href="/wiki/Nao" title="Nao">Nao</a>.</p><p><a href="/wiki/Androides" title="Androides">Androides</a> viro et gynaecoides mulieri appellantur ea robota quae homines vident, i.e. aspectum et habitum. androides, copia<sup><small><a href="/wiki/Disputatio:Gens_humana" title="Disputatio:Gens humana">?</a></small></sup> a se, etiam habet.</p><p><i><a href="/wiki/Cyborg" title="Cyborg">Cyborg</a></i> est mixtio hominis et partium machinalium, quod aliquando in <a href="/wiki/Cultura_popularis" title="Cultura popularis">cultura populari</a> apparet, sicut in multiserie <i><a href="/wiki/Star_Trek" title="Star Trek">Star Trek</a></i> ubi videmus <i><a href="/wiki/Borg_(Star_Trek)" title="Borg (Star Trek)">Borg</a></i>, malus esse proscriptum, et <i><a href="/wiki/Terminator" class="mw-redirect"
