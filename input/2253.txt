style="text-align: center;" colspan="2"><i><b>Homo sapiens sapiens</b></i><div style="text-align: center; font-size: smaller;"><a href="/wiki/Carl_von_Linn%C3%A9" title="Carl von Linné">Linné</a>,&#160;<a href="/wiki/1758" title="1758">1758</a></div></td></tr></table><p><b>Mennesket</b> (<b>Homo sapiens</b>) er ein art av <a href="/w/index.php?title=Menneskeapane&amp;action=edit&amp;redlink=1" class="new" title="Menneskeapane (sida finst ikkje)">menneskeapane</a> som går oppreist på to bein. Det har ein velutvikla <a href="/wiki/Hjerne" title="Hjerne">hjerne</a> som mogleggjer <a href="/wiki/Spr%C3%A5k" title="Språk">språkbruk</a>, <a href="/wiki/Abstrakt" class="mw-redirect" title="Abstrakt">abstrakt</a> tenking og sjølvforståing. Desse eigenskapane gjer at <a href="/wiki/Art" title="Art">arten</a> bruker <a href="/wiki/Reiskap" title="Reiskap">reiskapar</a> i større utstrekning enn nokon annan. Menneska finst på alle <a href="/wiki/Kontinent" title="Kontinent">kontinenta</a> og har tilpassa seg eit vidt omfang av omgjevnader. Det finst i dag over seks <a href="/wiki/Milliard" title="Milliard">milliardar</a> <a href="/wiki/Folketal" title="Folketal">menneske i verda</a>, fleire enn noko anna <a href="/wiki/Pattedyr" title="Pattedyr">pattedyr</a>.</p><p>Kva som skil menneska frå andre <a href="/wiki/Dyr" title="Dyr">dyr</a> har oppteke forskarar og filosofar i tusenvis av år. Ulike karakteristikkar er føreslått, som at dei bruker <a href="/wiki/Reiskap" title="Reiskap">reiskapar</a>, har <a href="/w/index.php?title=Medkjensle&amp;action=edit&amp;redlink=1" class="new" title="Medkjensle (sida finst ikkje)">medkjensle</a> og høg <a href="/w/index.php?title=Intelligens&amp;action=edit&amp;redlink=1" class="new" title="Intelligens (sida finst ikkje)">intelligens</a>, tek seg av <a href="/wiki/Sjukepleie" title="Sjukepleie">sjuke og funksjonshemma</a>, bruker <a href="/wiki/Spr%C3%A5k" title="Språk">språk</a> og <a href="/wiki/Klede" title="Klede">klede</a>, har utvikla <a href="/wiki/Jordbruk" title="Jordbruk">jordbruk</a>, <a href="/wiki/Vitskap" title="Vitskap">vitskap</a> og <a href="/wiki/Religion" title="Religion">religion</a>, er glade i <a href="/wiki/Kunst" title="Kunst">kunst</a> og <a href="/wiki/Underhaldning" class="mw-redirect" title="Underhaldning">underhaldning</a>. Nokre av desse eigenskapane finst hos andre dyr, men er ikkje så uttalte som hos menneske-arten.<sup title="Det som står før denne taggen bør ha ei kjelde." class="noprint">[<a href="/wiki/Wikipedia:Kjelder" title="Wikipedia:Kjelder"><i>treng&#160;kjelde</i></a>]</sup></p><p>Studiet av mennesket og menneskesamfunnet skjer innanfor alle akademiske retningar, både i <a href="/w/index.php?title=Humanbiologi&amp;action=edit&amp;redlink=1" class="new" title="Humanbiologi (sida finst ikkje)">humanbiologi</a>, <a href="/wiki/Medisin" title="Medisin">medisin</a>, <a href="/wiki/Psykologi" title="Psykologi">psykologi</a>, <a href="/wiki/Antropologi" title="Antropologi">antropologi</a>, <a href="/wiki/%C3%98konomi" title="Økonomi">økonomi</a>, <a href="/wiki/Sosiologi" title="Sosiologi">sosiologi</a> og <a href="/wiki/Lingvistikk" title="Lingvistikk">lingvistikk</a>, for å nemna eit lite utval.</p><p></p><div id="toc" class="toc"><div class="mw-editsection-bracket">]</span></span></h2><div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Fil:Woodstock_redmond_hair.JPG" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/f5/Woodstock_redmond_hair.JPG/220px-Woodstock_redmond_hair.JPG" width="220" height="151" class="thumbimage" class="magnify"><a href="/wiki/Fil:Woodstock_redmond_hair.JPG" class="internal" title="Forstørr"></a></div>Menneske søkjer gjerne saman i grupper.</div></div></div><dl><dd><i>Sjå òg <a href="/wiki/Menneskekroppen" class="mw-redirect" title="Menneskekroppen">Menneskekroppen</a></i></dd></dl><p>Menneska går oppreist på to bakbein og nyttar heile <a href="/w/index.php?title=H%C3%A6l&amp;action=edit&amp;redlink=1" class="new" title="Hæl (sida finst ikkje)">hælen</a>. Høgda varierer, men ligg vanlegvis mellom 145–180 cm for kvinner og 165–195 cm for menn. Vekt kan variera endå meir mellom individ, ettersom menneske lett utviklar <a href="/wiki/Feitt" title="Feitt">feittlager</a> av ekstra føde. «Normalt» vektområde kan seiast å vera 40–100 kg.</p><p>Samanlikna med andre <a href="/w/index.php?title=Primata&amp;action=edit&amp;redlink=1" class="new" title="Primata (sida finst ikkje)">primatar</a> har menneska større <a href="/wiki/Hjerne" title="Hjerne">hjerne</a>. Han er typisk 1&#160;400 <a href="/wiki/Cm%C2%B3" class="mw-redirect" title="Cm³">cm³</a>, dobbelt så stor som hjå <a href="/wiki/Gorilla" title="Gorilla">gorillaer</a> og <a href="/wiki/Sjimpansar" title="Sjimpansar">sjimpansar</a>. Menneska skil seg òg frå slektningane sine ved å ha kortare <a href="/wiki/Arm" title="Arm">armar</a> og ved å ha utvikla ein motståande og roterande <a href="/w/index.php?title=Tommel&amp;action=edit&amp;redlink=1" class="new" title="Tommel (sida finst ikkje)">tommel</a> som opnar for <a href="/w/index.php?title=Finmotorikk&amp;action=edit&amp;redlink=1" class="new" title="Finmotorikk (sida finst ikkje)">finmotorikk</a>. <a href="/wiki/T%C3%A5" title="Tå">Tærne</a> er derimot lite førlege, og blir mest brukte til å halda balansen.</p><p>Menneske har lite <a href="/wiki/H%C3%A5r" title="Hår">hår</a> samanlikna med dei andre primatane. Tett hårvekst finn ein berre visse stader på kroppen, som oppå <a href="/wiki/Hovud" title="Hovud">hovudet</a>, under armane og i <a href="/w/index.php?title=Skrittet&amp;action=edit&amp;redlink=1" class="new" title="Skrittet (sida finst ikkje)">skrittet</a>, men det finst likevel spreidde <a href="/w/index.php?title=H%C3%A5rsekk&amp;action=edit&amp;redlink=1" class="new" title="Hårsekk (sida finst ikkje)">hårsekkar</a> over heile kroppen.</p><p>Medan den vanlegaste hårfargen blant menneske er <a href="/wiki/Svart" title="Svart">svart</a> finst det òg mange andre fargar som går frå nesten <a href="/wiki/Kvit" title="Kvit">kvitt</a> til <a href="/wiki/Brun" title="Brun">brunt</a> eller <a href="/wiki/Raud" title="Raud">raudt</a>/oransje. Den hovudsakleg nakne <a href="/wiki/Hud" title="Hud">huda</a> kan gå frå å vera nesten fargelaus, noko som gjev ein <a href="/wiki/Rosa" title="Rosa">rosa</a> let ettersom <a href="/wiki/Blod" title="Blod">blod</a> viser gjennom, til mørkt <a href="/wiki/Brun" title="Brun">brun</a>. <a href="/wiki/Pigment" title="Pigment">Pigmenta</a> som fargar huda blir kalla <a href="/wiki/Melanin" title="Melanin">melanin</a>, og vernar mot <a href="/wiki/UV-str%C3%A5ling" class="mw-redirect" title="UV-stråling">UV-stråling</a>. Hud som blir utsett for mykje <a href="/wiki/Sollys" class="mw-redirect" title="Sollys">sollys</a> utviklar meir melanin, og blir brunare.</p><h2><span class="mw-headline" id="Livssyklus">Livssyklus</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Menneske&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Endre bolk: Livssyklus">endre</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Menneske&amp;action=edit&amp;section=2" title="Endre bolk: Livssyklus">endre wikiteksten</a><span class="mw-editsection-bracket">]</span></span></h2><p><a href="/wiki/Unge" title="Unge">Ungar</a> blir normalt fødde etter <a href="/wiki/Svangerskap" title="Svangerskap">svangerskap</a> på 38 <a href="/wiki/Veke" title="Veke">veker</a>. Dei er hjelpelause ved <a href="/wiki/F%C3%B8dsel" title="Fødsel">fødselen</a>, og treng omsorg og opplæring i fleire år. I <a href="/w/index.php?title=Alderdom&amp;action=edit&amp;redlink=1" class="new" title="Alderdom (sida finst ikkje)">alderdommen</a>, som også kan vera lang, treng menneske ofte liknande hjelp og omsorg.</p><p>Kjønnsmogninga skjer normalt ved tolv til femten års alder, men særleg gutar kan halde fram med å veksa fram til tjueårsalderen. Menn er teoretisk <a href="/wiki/Fruktbar" class="mw-redirect" title="Fruktbar">fruktbare</a> heile livet, medan kvinner går gjennom ein <a href="/w/index.php?title=Overgangsalder&amp;action=edit&amp;redlink=1" class="new" title="Overgangsalder (sida finst ikkje)">overgangsalder</a> som gjer at dei ikkje kan unnfanga lenger når dei er om lag femti.</p><p>Kor lenge folk lever kjem an på ytre tilhøve. I område der menneske har god tilgang på mat og helsestell er forventa levealder rundt 80, noko lågare for menn og noko høgare for kvinner. I land og tidsrom prega av svolt og sjukdom er levealderen lågare, typisk litt over 30 år. Under gode tilhøve kan menneske bli opptil eitt hundre og tjue år gamle, med <a href="/w/index.php?title=Jeanne_Calment&amp;action=edit&amp;redlink=1" class="new" title="Jeanne Calment (sida finst ikkje)">122 år og 164 dagar</a> som den høgaste målte levealderen til no.</p><h2><span class="mw-headline" id="Levesett">Levesett</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Menneske&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Endre bolk: Levesett">endre</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Menneske&amp;action=edit&amp;section=3" title="Endre bolk: Levesett">endre wikiteksten</a><span class="mw-editsection-bracket">]</span></span></h2><div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Fil:Young_monks_of_Drepung.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8d/Young_monks_of_Drepung.jpg/220px-Young_monks_of_Drepung.jpg" width="220" height="165" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8d/Young_monks_of_Drepung.jpg/330px-Young_monks_of_Drepung.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8d/Young_monks_of_Drepung.jpg/440px-Young_monks_of_Drepung.jpg 2x" data-file-width="800" data-file-height="600" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Fil:Young_monks_of_Drepung.jpg" class="internal" title="Forstørr"></a></div>Unge <a href="/wiki/Tibet" title="Tibet">tibetanske</a> gutar følgjer sin <a href="/wiki/Buddhist" class="mw-redirect" title="Buddhist">buddhistiske</a> kultur.</div></div></div><p>Menneske lever som regel i små grupper, ofte ei form for <a href="/wiki/Familie" title="Familie">familie</a>, som igjen kan vera del av eit større <a href="/wiki/Samfunn" title="Samfunn">samfunn</a>. Gruppene <a href="/wiki/Sosialisering" title="Sosialisering">sosialiserer</a> dei yngre menneska og <a href="/w/index.php?title=Samarbeid&amp;action=edit&amp;redlink=1" class="new" title="Samarbeid (sida finst ikkje)">samarbeider</a> om å klara seg. Det er vanleg å spesialisera seg til ulike oppgåver eller <a href="/wiki/Yrke" title="Yrke">yrke</a> innanfor gruppa eller samfunnet.</p><h3><span class="mw-headline" id="Kultur">Kultur</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Menneske&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Endre bolk: Kultur">endre</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Menneske&amp;action=edit&amp;section=4" title="Endre bolk: Kultur">endre wikiteksten</a><span class="mw-editsection-bracket">]</span></span></h3><p>Ei gruppe vil ha fleire <a href="/wiki/Tradisjon" title="Tradisjon">tradisjonar</a> for korleis ein lever og underheld seg sjølve. <i><a href="/wiki/Kultur" title="Kultur">Kultur</a></i> er eit samleomgrep for desse. Til dømes vil alle grupper ha visse levereglar og <a href="/wiki/Tabu" title="Tabu">tabu</a>. Bryt ein med desse kan ein blir utestengd eller straffa med <a href="/w/index.php?title=Fangenskap&amp;action=edit&amp;redlink=1" class="new" title="Fangenskap (sida finst ikkje)">fangenskap</a>, <a href="/wiki/Vald" title="Vald">vald</a> eller <a href="/wiki/Drap" title="Drap">drap</a>.</p><h3><span class="mw-headline" id="Konflikt">Konflikt</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Menneske&amp;veaction=edit&amp;section=5" class="mw-editsection-visualeditor" title="Endre bolk: Konflikt">endre</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Menneske&amp;action=edit&amp;section=5" title="Endre bolk: Konflikt">endre wikiteksten</a><span class="mw-editsection-bracket">]</span></span></h3><p>Grupper av menneske har ofte ei kjensle av å høyra til som kan gjera at dei går saman mot ei anna gruppe. Konfliktar mellom ulike grupper kan føra til mindre <a href="/w/index.php?title=Sl%C3%A5sting&amp;action=edit&amp;redlink=1" class="new" title="Slåsting (sida finst ikkje)">slåsting</a> eller <a href="/wiki/Krig" title="Krig">krigar</a>.</p><h2><span class="mw-headline" id="Utvikling">Utvikling</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Menneske&amp;veaction=edit&amp;section=6" class="mw-editsection-visualeditor" title="Endre bolk: Utvikling">endre</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Menneske&amp;action=edit&amp;section=6" title="Endre bolk: Utvikling">endre wikiteksten</a><span class="mw-editsection-bracket">]</span></span></h2><p>I følgje genetisk og antropologisk forsking utvikla menneska seg i <a href="/wiki/Afrika" title="Afrika">Afrika</a> <sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> for mellom 100 og 200&#160;000 år sidan. Frå Afrika spreidde dei seg vidare utover i <a href="/wiki/Eurasia" title="Eurasia">Eurasia</a> og deretter <a href="/wiki/Oseania" title="Oseania">Oseania</a> og <a href="/wiki/Amerika" title="Amerika">Amerika</a>.</p><h2><span class="mw-headline" id="Genetikk">Genetikk</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a
