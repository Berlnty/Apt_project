data-file-width="879" data-file-height="882" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Fil:Poi_circles.jpg" class="internal" title="Forstørr"></a></div><a href="/wiki/Fotografi" title="Fotografi">Fotografisk</a> kunst av ein utøvande <a href="/wiki/Poi" title="Poi">poikunstnar</a>. <i>(Foto: Hendrik Kueck)</i></div></div></div><p><b>Kunst</b> (frå <a href="/wiki/Urgermansk" title="Urgermansk">urgermansk</a> <i>kunst</i>, 'kunne') er skapande verksemder som kan ha dekorativt eller kjenslemessig uttrykk, og resultatet av slike verksdemder. Medan 'kunst' opphavleg omfatta all menneskeleg aktivitet, også <a href="/wiki/Handverk" title="Handverk">handverk</a> og <a href="/wiki/Vitskap" title="Vitskap">vitskap</a>, blir det i dag sett på som eit uttrykk for menneskeleg <a href="/w/index.php?title=Skaparevne&amp;action=edit&amp;redlink=1" class="new" title="Skaparevne (sida finst ikkje)">skaparevne</a>. </p><p><a href="/wiki/Menneske" title="Menneske">Menneske</a> i alle kulturar har skapt kunst frå <a href="/wiki/Steinalder" class="mw-redirect" title="Steinalder">steinalderen</a> og fram til våre dagar, både for si eiga skuld og for å skapa <a href="/wiki/Underhalding" title="Underhalding">underhalding</a> eller <a href="/wiki/Venleik" title="Venleik">venleik</a> for andre. Den kunstnariske skapartrongen kan derfor sjå ut til å vera ein viktig del av det å vera menneske.</p><div id="toc" class="toc"><div class="toctitle" lang="nn" <span class="toctext">Kjelder</span></a></li></ul></li></ul></div><h2><span class="mw-headline" id="Kunstomgrepet">Kunstomgrepet</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Kunst&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Endre bolk: Kunstomgrepet">endre</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Kunst&amp;action=edit&amp;section=1" title="Endre bolk: Kunstomgrepet">endre wikiteksten</a><span class="mw-editsection-bracket">]</span></span></h2><div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Fil:Prajnaparamita_Java.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/33/Prajnaparamita_Java.jpg/220px-Prajnaparamita_Java.jpg" width="220" height="295" class="thumbimage" class="magnify"><a href="/wiki/Fil:Prajnaparamita_Java.jpg" class="internal" title="Forstørr"></a></div>Denne <a href="/wiki/Bodhisattva" title="Bodhisattva">Bodhisattvafiguren</a> frå Java blei laga av ein handverkar. Likevel vil dei fleste kalla han eit kunstverk.</div></div></div><p>Omgrepet 'kunst' har hatt ulike tydingar til ulike tider. I mange samfunn, mellom anna i <a href="/wiki/Det_gamle_Hellas" class="mw-redirect" title="Det gamle Hellas">det gamle Hellas</a> stod 'kunst' for praktisk kunne, som ein motsats til <a href="/wiki/Teori" title="Teori">teoretisk</a> kunne. All menneskeleg aktivitet som kravde eit visst handlag var kunst. Dei som skapte kunst var dyktige <a href="/wiki/Handverkar" title="Handverkar">handverkarar</a>. </p><p>I løpet av <a href="/wiki/Renessansen" title="Renessansen">renessansen</a> fekk omgrepet gradvis delvis nytt innhald. I tillegg til den personlege kunna til skaparen snakka ein no òg om det han laga som 'kunst'. I takt med at kunstverka fekk mindre praktisk bruk og blei høgare verdsette for innhaldet, blei òg opphavsmennene viktigare. Desse blei etter kvart sett på som <a href="/wiki/Kunstnar" title="Kunstnar">kunstnarar</a>, ei eiga gruppe menneske fylde av ei <a href="/wiki/Geni" title="Geni">geni</a> al, skapande kraft.</p><p>På slutten av <a href="/wiki/1900-talet" title="1900-talet">1900-talet</a> blei idéen om ein indre kjerne og meining i kunsten forkasta av <a href="/w/index.php?title=Postmodernistisk&amp;action=edit&amp;redlink=1" class="new" title="Postmodernistisk (sida finst ikkje)">postmodernistiske</a> tenkjarar. Kunst blei i staden sett på som eit produkt av prosessar påverka av <a href="/wiki/Samfunn" title="Samfunn">samfunn</a> og <a href="/wiki/Kultur" title="Kultur">kultur</a>. Under denne definisjonen kunne ein ikkje skilja mellom "god" og "dårleg" kunst. Det er i dag vanskeleg å definera kva som er kunst og kva som ikkje er det.</p><p>Mange held fortsatt på den opphavlege tydinga av kunst som noko ein kan skapa om ein har ein viss evne. I <a href="/wiki/Norsk_spr%C3%A5k" class="mw-redirect" title="Norsk språk">norsk</a> finst spor etter denne haldninga i vanlege uttrykk som "det er inga kunst", eller bruken av "kunst" om ting som er skapte ved hjelp av menneskekunne, slik som <a href="/wiki/Kunstfiber" title="Kunstfiber">kunstfiber</a> eller <a href="/wiki/Kunstgj%C3%B8dsel" title="Kunstgjødsel">kunstgjødsel</a>.</p><h2><span class="mw-headline" id="Kunstformer">Kunstformer</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Kunst&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Endre bolk: Kunstformer">endre</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Kunst&amp;action=edit&amp;section=2" title="Endre bolk: Kunstformer">endre wikiteksten</a><span class="mw-editsection-bracket">]</span></span></h2><div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Fil:Puccini_Tosca.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/66/Puccini_Tosca.jpg/220px-Puccini_Tosca.jpg" width="220" height="293" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/66/Puccini_Tosca.jpg/330px-Puccini_Tosca.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/6/66/Puccini_Tosca.jpg 2x" data-file-width="338" data-file-height="450" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Fil:Puccini_Tosca.jpg" class="internal" title="Forstørr"></a></div>Vestleg <a href="/wiki/Opera" title="Opera">opera</a> samanfatter mange ulike kunstformer, som <a href="/wiki/Song" title="Song">song</a>, <a href="/wiki/Dans" title="Dans">dans</a>, <a href="/wiki/Skodespel" title="Skodespel">skodespel</a> og <a href="/wiki/Bildekunst" class="mw-redirect" title="Bildekunst">bildekunst</a>.</div></div></div><p>Det finst ei rekke ulike måtar å uttrykka seg kunstnarisk på. Dei kan delast inn på ulike måtar, til dømes i utøvande kunst, som <a href="/wiki/Musikk" title="Musikk">musikk</a> og <a href="/wiki/Teater" title="Teater">teater</a>, og skapande kunst, som <a href="/wiki/Bildekunst" class="mw-redirect" title="Bildekunst">bildekunst</a> og <a href="/wiki/Arkitektur" title="Arkitektur">arkitektur</a>.
